# \(QQ 1243909887\) 分享：电机控制入门：电流环带宽与时间常数——从物理直觉到PI整定

# 前言

刚开始调试电流环的时候，你大概率会碰到两个概念：**时间常数 τ** 和**带宽 α**。

网上很多资料上来就给公式：Kp = α·L，Ki = α·R。照抄当然能跑起来，但为什么这么算？α 到底是什么？设 1000 还是 3000 有什么区别？

这篇文章试图从物理直觉出发，把\&\#34;时间常数→频域分析→带宽→零极点对消→PI整定\&\#34;这条线串起来。目标读者是刚入门电机控制的新手，所以我会尽量用类比把抽象概念翻译成可感知的东西。

## 电机绕组的自然时间常数 τ = L/R

### 1\.1 一个简单的实验

电机绕组本质上就是一个**电感 L 和电阻 R 的串联**。假设你直接给绕组突加一个直流电压 V₀，电流会怎么变化？

答案是按照指数规律上升：

i\(t\) = V₀/R × \(1 \- e^\(\-t/τ\)\)，其中 τ = L/R

τ 就是**自然时间常数**。它刻画了电流\&\#34;自己往上爬\&\#34;的速度：

|时刻|电流值|含义|
|---|---|---|
|t = 0|0|电感反抗电流突变，电流从零开始|
|t = τ|63\.2% × V₀/R|上升到稳态值的 63%|
|t = 2τ|86\.5% × V₀/R|接近稳态|
|t = 3τ|95\.0% × V₀/R|工程上认为已进入稳态|

### 1\.2 物理直觉：惯性 vs 阻力

τ = L / R

↑     ↑

惯性   阻力

- **L（电感）大 → 惯性大**：电感阻碍电流变化，L 越大，电流越\&\#34;笨重\&\#34;，充电越慢 → τ 越大

- **R（电阻）大 → 充电快**：虽然 R 越大稳态电流越小，但 R 在分母上，意味着能量耗散快，**相对而言**达到稳态的速度更快 → τ 越小

你可以把绕组想象成一个水桶：

- L 是桶的底面积（越大，装到同样水位需要的水越多）

- R 是桶底的漏水孔（越大，水漏得越快，水位更快达到平衡）

### 1\.3 一个真实电机的数据

以一台实际电机为例：L = 0\.10376 mH = 0\.00010376 H，R = 0\.1051 Ω：

τ = L/R = 0\.00010376 / 0\.1051 ≈ 0\.000987 s ≈ 1 ms

**这意味着即使你直接把最大电压怼上去，电流从 0 升到稳态值也需要大约 3 ms。** 这是电机绕组本身的物理极限，跟你的控制器没关系。

> 关键认知：τ 告诉你\&\#34;电机本身有多慢\&\#34;——这是你调 PI 的**出发点**，也是物理上限。

## 为什么要从频域看问题？

讲带宽之前，必须先回答一个前置问题：\*\*为什么放着好好的时间波形不看，非要用频率来思考？

### 2\.1 两种视角

**时域**——你拿示波器看电流，横轴是时间。给一个阶跃指令，看到波形从 0 爬到 1A，花了多少毫秒。直观，但回答不了这些问题：

- 如果指令以 500 Hz 正弦变化，电流跟得上吗？

- 增大 Kp 会不会让系统不稳定？

- 噪声在什么频率？我该降带宽还是加滤波器？

**频域**——不问\&\#34;电流随时间怎么变\&\#34;，而是问\&\#34;系统对不同频率的正弦波各放大多少倍\&\#34;。

为什么可以这样问？**傅里叶定理：任何信号都可以拆成不同频率正弦波的叠加。** 如果你知道系统对每一个频率怎么响应，就能拼出它对任何信号的响应。

### 2\.2 频域分析对控制设计的三个核心价值

**价值一：微分方程变代数方程。** 时域里 V = L·di/dt \+ R·i 是微分方程，每次换输入都要重新解。拉普拉斯变换后变成 I\(s\)/V\(s\) = 1/\(Ls\+R\)，一个跟输入无关的传递函数——拿到它，就知道系统对**任何**输入的响应。

**价值二：稳定性判断不需要解方程。** 时域判断稳定要解出 i\(t\) 看有没有发散项，三阶以上几乎手算不了。频域只需要看开环传递函数在截止频率处的相位滞后有没有超过 180°——算一个点的增益和相位就能判断稳定性。这是频域最大的工程价值。

**价值三：把\&\#34;快慢\&\#34;变成可计算的量。** 时域里你说\&\#34;我要电流响应快一点\&\#34;——多快？对应什么 Kp？说不清。频域里你说\&\#34;我要带宽 α = 3000 rad/s\&\#34;——Kp = α·L = 0\.311，直接出数值。

### 2\.3 一张表对比

||时域|频域|
|---|---|---|
|横轴|时间 t|频率 ω|
|看什么|波形 i\(t\)|Bode 图（幅频 \+ 相频）|
|数学工具|微分方程|传递函数，s = jω|

|电流环例子|Iq\_ref 阶跃 0→1A，看上升时间|看 ω\_c 处增益 = 0 dB，相位裕量多少|

## 带宽到底是什么？

### 3\.1 定义

**带宽 α（Bandwidth）** 是系统能有效跟踪输入信号的最高频率。单位可以是 rad/s 或 Hz（α = 2π·f）。

更精确地说：当输入正弦波的频率达到 α 时，输出的幅值衰减到输入幅值的 0\.707 倍（\-3 dB 点）。

### 3\.2 三个层次理解带宽

#### 层次一：时域——带宽决定响应速度

给定一个阶跃指令（0A → 1A），不同带宽的响应完全不同：

α = 1000 rad/s：电流花 1\.8 ms 慢慢升到 1A    → 像经济型轿车加速

α = 3000 rad/s：电流花 0\.6 ms 快速到位       → 像普通轿车加速

α = 6000 rad/s：电流花 0\.3 ms 瞬间到位       → 像跑车加速

**你不需要记那串绕口的定义，只需要理解：设 α = 3000，就是要求电流环在约 0\.6 ms 内跟上指令。**

#### 层次二：频域——带宽决定\&\#34;跟得上多快的变化\&\#34;

假设给 Iq\_ref 一个正弦指令：Iq\_ref\(t\) = 1A × sin\(ωt\)，改变正弦波的频率 ω：

|ω|f|实际电流幅值|跟踪情况|
|---|---|---|---|
|100 rad/s|16 Hz|1\.00 A|完美跟踪|
|1000 rad/s|159 Hz|0\.95 A|良好|
|**3000 rad/s**|**477 Hz**|**0\.707 A**|**\-3dB 点 = 带宽**|
|6000 rad/s|955 Hz|0\.45 A|严重衰减|
|20000 rad/s|3\.2 kHz|几乎不动|完全跟不上|
|**为什么会衰减？** 回到电压方程 V = L·di/dt \+ i·R。令 i = I₀·sin\(ωt\)，则 L·di/dt = L·I₀·ω·cos\(ωt\)。频率 ω 越高，L·di/dt 项越大（正比于 ω）。但母线电压有上限（比如 24V），当 ω 高到一定程度，你把全部电压都用来克服 L·di/dt 都不够，电流自然跟不上了。||||
|**带宽的物理本质：在电压预算内，能跟上的最高变化频率。**||||

#### 层次三：环路稳定性——带宽是\&\#34;系统不失控的极限\&\#34;

实际系统里有各种延迟：PWM 更新延迟、ADC 采样延迟、计算延迟。这些延迟的总和 T\_d 会带来相位滞后。

延迟的相位滞后 ≈ \-ω × T\_d。当频率高到相位滞后接近 180° 时，**负反馈变成正反馈**——你在该减的时候加，在该加的时候减，系统振荡失控

**所以带宽本质上受到延迟的约束：α\_max ≤ 2π·f\_pwm / 20。**

### 3\.3 一个终极类比

想象传球游戏：

|概念|传球类比|
|---|---|
|电流指令|你喊\&\#34;往左跑！\&\#34;|
|实际电流|接球人往左移动|
|**带宽**|**你喊话的频率上限** — 每秒喊 1 次，接球人能反应过来；每秒喊 100 次，接球人完全跟不上，甚至摔倒|
|T\_d（延迟）|你喊完到对方听到需要的时间 — 如果你喊得太快，对方听到的是过时指令，反而跑错方向|
|L·di/dt|接球人的身体惯性 — 不可能瞬间变向|
|母线电压|你的嗓门大小 — 嗓门越大（电压越高），越能让人加速|

## 零极点对消：PI 控制器如何\&\#34;改造\&\#34;电机

搞清楚时间常数和带宽之后，现在进入正题：**PI 控制器是怎么把电机从一个 τ≈1ms 的\&\#34;慢系统\&\#34;加速到你想要的带宽的？**

### 4\.1 电机的传递函数

电机绕组的传递函数（从电压到电流）：

G\(s\) = I\(s\) / V\(s\) = 1 / \(L·s \+ R\)

这是一个**一阶惯性环节**，在 s = \-R/L 处有一个**极点**。

什么是极点？把传递函数的分母写成零极点形式：L·s \+ R = L\(s \+ R/L\)，当 s = \-R/L 时分母为零，传递函数趋于无穷——虽然在实际频率 s = jω 上你永远不会\&\#34;走到\&\#34;这个点，但它决定了系统随频率衰减的形状。

物理含义：极点 s = \-R/L = \-1/τ 告诉你——**没有控制器时，系统受到扰动后会以 e^\(\-t/τ\) 的速度自己衰减回零**。极点离虚轴越远，衰减越快。

### 4\.2 PI 控制器的传递函数

C\(s\) = Kp \+ Ki/s = \(Kp·s \+ Ki\) / s

这个传函有两个特征：

- **s = 0 处有一个极点**（分母的 s）→ 这就是积分器，DC 增益无穷大，是消除静差的根源

- **s = \-Ki/Kp 处有一个零点**（分子的根）→ 零点在频域里做\&\#34;抬升\&\#34;动作，\+20dB/dec

### 4\.3 对消的逻辑

开环传递函数 = 控制器 × 电机：

L\(s\) = C\(s\)·G\(s\) = \(Kp·s \+ Ki\)/s  ×  1/\(L·s \+ R\)

电机在 s = \-R/L 处有一个极点（它让高频增益衰减）。PI 控制器在 s = \-Ki/Kp 处有一个零点（它让高频增益抬升）。

**零极点对消的核心想法：让 PI 的零点正好落在电机的极点上。**

令 Ki/Kp = R/L    →    PI零点 s = \-Ki/Kp = 电机极点 s = \-R/L

分子里的 \(s\+R/L\) 和分母里的 \(s\+R/L\) 约掉，开环传函变成：

L\(s\) = Kp/L × 1/s

**一个纯粹的积分器。** Bode 图上就是一条干净的 \-20 dB/dec 斜线，没有拐弯。

这为什么好？

- 斜率恒定 → 相位滞后恒为 \-90° → 相位裕量恒为 90° → **理论上永远稳定**

- 开环增益 \|L\(jω\)\| = Kp/\(L·ω\)，令其 = 1（0 dB）求出截止频率：**ω\_c = Kp/L**

### 4\.4 闭环带宽 = Kp/L

对消后，闭环传递函数为：

G\_cl\(s\) = 1 / \(1 \+ \(L/Kp\)·s\)

这又是一个一阶惯性环节，等效时间常数 τ\_eq = L/Kp。一阶系统的 \-3 dB 带宽就是 1/τ\_eq：

α = 1/τ\_eq = Kp/L

**这就是带宽法核心公式的来历：Kp = α·L，Ki = α·R。**

## 带宽法整定 PI 参数

### 5\.1 核心公式

|参数|公式|物理含义|
|---|---|---|
|**Kp**|α × L|克服电感惯性，决定响应速度|
|**Ki**|α × R|补偿电阻压降，消除稳态静差|
|物理含义极其清爽：|||

- 想要响应更快（α 更大）→ Kp 和 Ki 同步增大

- 电机电感大 → 需要更大 Kp 来推动电流变化

- 电机电阻大 → 需要更大 Ki 来补偿稳态压降

### 5\.2 如何选择 α

**理论上限：** α\_max ≤ 2π × f\_pwm / 20

**工程推荐值：**

|应用场景|带宽 α|对应频率|说明|
|---|---|---|---|
|保守调试起点|1000\~2000 rad/s|160\~320 Hz|首次调试，稳字当头|
|**常规运行**|**2000\~4000 rad/s**|**320\~640 Hz**|**大多数情况的首选**|
|高性能伺服|4000\~6000 rad/s|640\~950 Hz|需要低噪声、高 PWM 频率|

### 5\.3 一个实例计算

电机参数：L = 0\.10376 mH，R = 0\.1051 Ω，PWM = 20 kHz

取 α = 3000 rad/s（推荐初值）：

Kp = α × L = 3000 × 0\.00010376 ≈ 0\.311 V/A

Ki = α × R = 3000 × 0\.1051   ≈ 315\.3 V/\(A·s\)

不同带宽的对照：

|带宽 α \(rad/s\)|f \(Hz\)|Kp \(V/A\)|Ki \(V/\(A·s\)\)|上升时间 tr≈1\.8/α|
|---|---|---|---|---|
|1000|159|0\.104|105\.1|1\.8 ms|
|2000|318|0\.208|210\.2|0\.9 ms|
|**3000**|**477**|**0\.311**|**315\.3**|**0\.6 ms**|
|4000|637|0\.415|420\.4|0\.45 ms|
|5000|796|0\.519|525\.5|0\.36 ms|

## 三个概念的关系——一张图总结

三个概念反复出现，这里把它们的关系串起来：

电机物理极限               控制器实现                 最终效果

─────────────        ────────────────        ────────────────

τ = L/R             ω\_c = Kp/L             α ≈ ω\_c

↓                      ↓                      ↓

电流自己爬多快        PI把系统加速到多快       系统实际跟得上多快的指令

↓                      ↓                      ↓

τ ≈ 1 ms           ω\_c = 3000 rad/s       α ≈ 3000 rad/s

↓

tr ≈ 1\.8/α ≈ 0\.6 ms

- **τ = L/R** 告诉你\&\#34;电机本身有多慢\&\#34;——天生极限，≈1 ms

- **Kp 和 α** 告诉你\&\#34;你用控制器把它加速了多少\&\#34;

- **加速比** = α / \(1/τ\) = \(Kp/L\) / \(R/L\) = Kp/R

    - 你设 α = 3000，自然响应 ≈ 1/τ ≈ 1013 rad/s

    - 加速比 ≈ 3 倍——**你把电机的响应速度硬生生提了 3 倍**

一句话：τ 是起点，α 是目标，Kp = α·L 是手段。

## 调试建议

1. **从低带宽起步**：α = 1000 rad/s，观察阶跃响应是否正常

2. **逐步增大**：每次加 500\~1000 rad/s，观察上升时间是否变快、有没有振荡

3. **找到拐点**：持续增大直到电流出现明显振荡或电机发出高频啸叫，记录临界值 α\_critical

4. **留裕量**：最终使用的 α = α\_critical × 0\.6\~0\.7

5. **微调 Ki**：如果启动有过冲，Ki 减小 10\~20%；如果稳态有净差，Ki 增大 10\~20%
