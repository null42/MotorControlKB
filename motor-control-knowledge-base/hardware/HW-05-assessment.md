## 一、参数计算题（★★☆☆☆）

### 题目1：寄生电容提取与计算
**问题：** 从MOSFET数据表中读取：CISS = 3000pF，CRSS = 120pF，COSS = 650pF（VDS=25V时）。请计算：
1. CGS、CGD、CDS的实际值
2. 若VDS从25V升至400V，CGD大约如何变化？（提示：CGD ∝ 1/√VDS）
3. 平均CGD用于损耗计算

**参考答案：**
1. CGD = CRSS = 120pF；CGS = CISS - CGD = 3000 - 120 = 2880pF；CDS = COSS - CGD = 650 - 120 = 530pF
2. CGD与VDS近似反比：CGD(400V) ≈ CGD(25V) × √(25/400) = 120pF × 0.25 = 30pF（高压下CGD显著减小）
3. 平均CGD = 2 × CRSS,spec × √(VDS,spec / VDS,off) = 2 × 120pF × √(25/400) = 2 × 120 × 0.25 = 60pF

### 题目2：栅极驱动功率计算
**问题：** 一个MOSFET的Qg = 80nC，驱动电压VGS = 12V，开关频率fsw = 50kHz。请计算：
1. 栅极驱动功率
2. 等效电容
3. 驱动器温升（驱动器静态电流2mA，封装热阻100°C/W，VCC=12V）

**参考答案：**
1. Pgate = Qg × VGS × fsw = 80nC × 12V × 50kHz = 48mW
2. Ceq = Qg / VGS = 80nC / 12V = 6.67nF
3. Pquiescent = VCC × Iq = 12V × 2mA = 24mW；Ptotal = 48 + 24 = 72mW；ΔT = 0.072W × 100°C/W = 7.2°C → 温升很低，安全

---

### 题目3：开关损耗计算
**问题：** 已知MOSFET参数：VDS = 400V，ID = 10A，开通上升时间tr = 50ns，关断下降时间tf = 30ns，fsw = 100kHz。请计算：
1. 开通损耗
2. 关断损耗
3. 总开关损耗
4. 如果散热能力限制总损耗为8W，是否需要降低开关频率？

**参考答案：**
1. Psw(on) = ½ × VDS × ID × tf × fsw = 0.5 × 400 × 10 × 30ns × 100kHz = 6W
2. Psw(off) = ½ × VDS × ID × tr × fsw = 0.5 × 400 × 10 × 50ns × 100kHz = 10W
3. Psw(total) = 6 + 10 = 16W
4. 16W >> 8W，必须降低频率。fsw_max = Pmax / (½ × VDS × ID × (tr+tf)) = 8 / (0.5 × 400 × 10 × 80×10^-9) ≈ 50kHz。建议fsw降至40kHz以下

## 二、电路设计题（★★★☆☆）

### 题目4：栅极电阻设计
**问题：** 为一个MOSFET（Qg = 71nC，VGS = 10V）设计栅极电阻，驱动器峰值电流2A。请计算：
1. 基于峰值电流的RG
2. 基于目标开关时间50ns的RG
3. 基于振荡抑制的RG（栅极回路电感20nH）
4. 最终选择及理由

**参考答案：**
1. RG = VDRV / Ipeak = 10V / 2A = 5Ω
2. Ceq = Qg / VGS = 71nC / 10V = 7.1nF；RG = tsw / (2.2 × Ceq) = 50ns / (2.2 × 7.1nF) = 3.2Ω
3. RG ≥ √(Lgate / Ceq) = √(20nH / 7.1nF) = √2.82 = 1.68Ω
4. 综合考虑：RG = 3.3Ω（标准值）。理由：满足振荡抑制要求(>1.68Ω)，开关速度较快(接近50ns目标)，峰值电流3A < 驱动器2A限制时需注意——实际Ipeak = 10V/3.3Ω = 3A，超出驱动器2A能力，需增大RG到4.7Ω或选择更大电流驱动器

### 题目5：自举电路设计
**问题：** 为一个48V三相电机控制器设计自举电路，MOSFET的Qg = 50nC，高侧驱动器静态电流50μA，最大导通时间10μs，允许电压跌落0.5V。请计算：
1. 自举电容最小值
2. 选择实际电容值
3. 自举二极管选型要求
4. 最大占空比限制

**参考答案：**
1. CB ≥ (Qgate + Ileak × ton) / ΔVboot = (50nC + 50μA × 10μs) / 0.5V = (50nC + 0.5nC) / 0.5V = 101nF
2. 选择CB = 1μF（留10倍裕度，考虑温度和电压降额）
3. 自举二极管要求：反向耐压VR > VDC = 48V（选100V以上）；正向电流IF > Ipeak（栅极峰值电流）；反向恢复时间trr < 100ns。推荐：US1M（100V/1A/75ns）或UF4007
4. 充电时间 tcharge ≈ 5 × CB × (RG + RD) = 5 × 1μF × (10 + 5) = 75μs；Dmax = 1 - tcharge/Tsw。若Tsw = 50μs(20kHz)：Dmax = 1 - 75/50 < 0 → 无法充电！需增大PWM频率或减小CB。实际：Tsw = 50μs时，1μF太大，改用0.1μF → tcharge = 7.5μs → Dmax = 1 - 7.5/50 = 85%

## 三、故障诊断题（★★★☆☆）

### 题目6：栅极电压振荡诊断
**问题：** 示波器测量MOSFET栅极电压VGS发现明显的振荡，振荡频率约50MHz，幅度约2Vpp。请分析：
1. 振荡的原因
2. 振荡的危害
3. 解决方案

**参考答案：**
1. 原因：栅极回路存在寄生LC谐振。L = PCB走线电感 + 器件引脚电感（约10-50nH），C = CGS + CGD（约1-5nF），谐振频率f = 1/(2π√(LC))。若L=20nH, C=5nF → f = 1/(2π√(100×10^-18)) = 15.9MHz。RG太小无法提供足够阻尼
2. 危害：(a) VGS可能超过最大额定值（±20V）导致栅极氧化层击穿；(b) 误导通/误关断；(c) 增加开关损耗；(d) EMI辐射
3. 解决方案：
   - (a) 增大栅极电阻RG（最简单有效）
   - (b) 缩短栅极走线，减小寄生电感
   - (c) 栅极-源极之间加小电阻或铁氧体磁珠
   - (d) 驱动器紧靠MOSFET放置（<10mm）
   - (e) 栅极-源极之间加齐纳二极管保护（15V）

### 题目7：高侧MOSFET频繁损坏
**问题：** 三相逆变器满载运行时，高侧MOSFET频繁损坏，低侧正常。测量发现VGS波形在高侧导通时有跌落。请分析：
1. 可能的根本原因
2. 诊断流程
3. 解决方案

**参考答案：**
1. 根本原因：自举电容充电不足 → 高侧驱动电压不足 → VGS跌落 → MOSFET未完全导通 → RDS(on)增大 → 导通损耗急剧增加 → 过热烧毁
2. 诊断流程：
   - (a) 测量自举电容两端电压 → 正常应≈VCC-VF，若明显偏低则充电不足
   - (b) 检查低侧导通时间是否足够给自举电容充电
   - (c) 检查自举二极管是否正常（反向漏电流是否过大）
   - (d) 检查自举电容容量是否足够（是否老化失效）
   - (e) 检查高侧最大导通时间是否超过自举电容维持时间
3. 解决方案：
   - (a) 增大自举电容容量
   - (b) 确保低侧在每个PWM周期有足够导通时间
   - (c) 更换低漏电流自举二极管
   - (d) 限制最大占空比（确保充电窗口）
   - (e) 启动时先开通低侧给自举电容预充电

## 四、软硬件关联题（★★★★☆）

### 题目8：dv/dt误导通分析与防护
**问题：** 已知MOSFET参数：CGD = 80pF，dVDS/dt = 8V/ns，RG = 5Ω，RG,I = 3Ω（内部栅极电阻），Vth = 3.5V。请分析：
1. 是否会发生dv/dt误导通？
2. 如果会，给出3种解决方案并计算修正后的VGS
3. 该问题在桥式电路中为什么特别危险？

**参考答案：**
1. VGS = CGD × dVDS/dt × (RG + RG,I) = 80pF × 8V/ns × 8Ω = 80×10^-12 × 8×10^9 × 8 = 5.12V > Vth = 3.5V → **会误导通！**
2. 解决方案：
   - (a) 负压关断VGS(off) = -5V：裕度 = 5 + 3.5 - 5.12 = 3.38V → 安全
   - (b) 降低RG到1Ω：VGS = 80pF × 8V/ns × 4Ω = 2.56V < 3.5V → 安全
   - (c) 选择低CGD器件（20pF）：VGS = 20pF × 8V/ns × 8Ω = 1.28V → 安全
3. 桥式电路特别危险的原因：一个桥臂关断时，对管体二极管反向恢复产生高dVDS/dt → 通过CGD耦合到关断管的栅极 → 如果误导通 → 上下管同时导通 → 直通短路 → 器件烧毁！这是桥式电路最常见的失效模式之一

### 题目9：米勒效应的定量分析
**问题：** 已知MOSFET参数：CGD = 100pF，跨导gfs = 20S，栅极驱动电阻RGD = 10Ω。请计算：
1. 米勒等效电容
2. 米勒平台时间（Qgd = 25nC，驱动电流2A）
3. 米勒效应对开关损耗的影响

**参考答案：**
1. CGD,eqv = CGD × (1 + gfs × RGD) = 100pF × (1 + 20 × 10) = 100pF × 201 = 20.1nF → 米勒效应使CGD"放大"了201倍！
2. tplateau = Qgd / Idrive = 25nC / 2A = 12.5ns
3. 米勒平台期间VDS和ID同时存在 → 这是开关损耗最大的阶段。Psw ∝ (tr + tf) × fsw，而tr+tf中米勒平台占主要部分。如果驱动电流不足（如1A），tplateau = 25ns → 开关损耗翻倍。因此：驱动电流越大 → 米勒平台越短 → 开关损耗越小

### 题目10：IGBT与MOSFET驱动差异及系统影响
**问题：** 对比分析MOSFET（IRF540N）和IGBT（IKW40N60T）在电机驱动应用中的差异：
1. 驱动电压差异及原因
2. 为什么IGBT需要负压关断而MOSFET通常不需要？
3. IGBT短路保护的特殊要求
4. 选择MOSFET还是IGBT的决策框架

**参考答案：**
1. 驱动电压：MOSFET 10-15V，IGBT 15-20V。原因：IGBT的VCE(sat)随VGE增大而降低，VGE=15V时VCE(sat)最低；MOSFET的RDS(on)在VGS=10V时已充分导通
2. IGBT需要负压关断的原因：(a) IGBT的CGD通常比MOSFET大 → dv/dt耦合更严重；(b) IGBT关断时有拖尾电流 → 关断时间更长 → dv/dt更大；(c) IGBT常用于高压（>600V）→ dVDS/dt更大 → 误导通风险更高。负压关断（VGE = -5V ~ -15V）提供额外裕度
3. IGBT短路保护特殊要求：(a) 短路承受时间有限（<10μs），必须快速检测和关断；(b) 使用VCE(sat)检测：开通后延迟3-5μs开始检测，VCE > 7V则判断短路；(c) 软关断：短路时不能硬关断（di/dt太大产生过压），需要缓慢降低VGE
4. 决策框架：
   - 电压<600V + 频率>20kHz → MOSFET（低损耗、快速开关）
   - 电压>600V + 频率<20kHz + 大功率 → IGBT（高耐压、大电流）
   - 电压>1200V → IGBT（MOSFET耐压不够）
   - 高频+高压 → SiC MOSFET（兼顾高频和高压）
