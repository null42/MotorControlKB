# HW-01 电机本体：电驱算法的物理基础

**副标题：从磁路到绕组，理解PMSM的物理本质**

---

## 1. 📌 核心摘要 ★★★☆☆ 🔰📚

**一句话讲清楚**：电机本体是电驱系统的"心脏"，理解其物理特性是开发高质量控制算法的前提——电感决定电流响应速度，磁链决定转矩常数，电阻决定铜损，极对数决定电角速度与机械角速度的关系。

**认知挂钩**：很多人以为电机就是"通电就转"，**这是致命误区！** 实际上，电机是一个**复杂的机电能量转换系统**：电能→磁场→机械能。不理解磁路、绕组、反电动势等物理概念，就无法理解FOC算法中的Park变换、Id/Iq解耦、弱磁控制等核心概念。

**与算法的关联**：
- 🔗 **算法关联**：电感(Ld, Lq) → 决定电流环PI参数设计 → `Kp = L × ωc`，电感不准则PI参数整定失败
- 🔗 **算法关联**：磁链(ψf) → 决定转矩常数 → `Kt = 1.5 × P × ψf`，磁链不准则转矩输出偏差大
- 🔗 **算法关联**：极对数(P) → 决定Park变换角度 → `θe = P × θm`，极对数错则Park变换完全错误
- 电阻(Rs) → 决定铜损 → 影响效率计算
- 齿槽转矩 → 决定转矩脉动 → 影响控制精度

## 3. 💡 直观理解 ★★☆☆☆ 🔰💡

### 类比1：电机就像"旋转的磁铁"

**生活场景**：想象你有两块磁铁，一块固定(定子)，一块可以旋转(转子)。

```
定子磁铁(N极) ────→ 吸引 ────→ 转子磁铁(S极)
                              ↓
                           旋转对齐
```

**关键理解**：
- 定子产生旋转磁场(通过三相绕组)
- 转子跟随旋转磁场(永磁体)
- 控制定子磁场方向 → 控制转子位置

| 特性 | 直流电机 | PMSM |
|------|---------|------|
| 换向方式 | 机械换向器 | 电子换向(FOC) |
| 磁场来源 | 电枢绕组 | 永磁体 |
| 控制复杂度 | 简单 | 复杂(需要Park变换) |
| 效率 | 较低 | 高 |

### 类比2：电感就像"惯性"

**生活场景**：想象你在推一个重物，刚开始很难推动，但一旦动起来，很难停下来。

```
推力(电压) ────→ 重物(电感) ────→ 速度(电流)
   │                │                │
   └──── 惯性 ──────┘

数学关系:
  V = L × di/dt  ↔  F = m × dv/dt (牛顿第二定律)
  
  V(电压) ↔ F(力)
  L(电感) ↔ m(质量)
  i(电流) ↔ v(速度)
```

**关键理解**：电感阻碍电流变化，就像惯性阻碍速度变化。电感越大，电流变化越慢，电流环带宽受电感限制。

### 类比3：极对数就像"齿轮比"

**生活场景**：想象自行车的齿轮，前齿轮转一圈，后齿轮转多圈。

```
机械旋转 ────→ 极对数P ────→ 电角度
   │                            │
   1圈                         P圈

示例(4极电机P=2):
  机械角度: 0° → 180°
  电角度:   0° → 360°
  关系: θe = P × θm = 2 × 180° = 360°
```

**关键理解**：极对数 = 电角度/机械角度的比值，Park变换需要电角度。

### 关键概念速查

**电感(Ld, Lq)**：
- Ld: d轴电感，影响Id电流响应
- Lq: q轴电感，影响Iq电流响应
- 表贴式PMSM: Ld ≈ Lq
- 内置式PMSM: Lq > Ld(磁阻转矩)

**磁链(ψf)**：永磁体产生的磁通量，决定反电动势和转矩常数，单位韦伯(Wb)

**极对数(P)**：转子磁极对数(N+S为一对)，决定电角度与机械角度关系，θe = P × θm

**电阻(Rs)**：定子绕组电阻，决定铜损，影响效率

## 4. 🔬 技术原理 ★★★★☆ 📚🔧

### 4.1 PMSM结构原理

#### 4.1.1 基本结构

```
PMSM（永磁同步电机）结构：

                    定子（Stator）
         ┌─────────────────────────────────┐
         │         ┌───────────┐           │
         │    A    │           │    A'     │  A相绕组
         │   ┌──┐  │    转子    │  ┌──┐     │
         │   │  │  │  (Rotor)  │  │  │     │
         │   └──┘  │   ┌───┐   │  └──┘     │
         │    B    │   │ N │   │    B'     │  B相绕组
         │   ┌──┐  │   │   │   │  ┌──┐     │
         │   │  │  │   │ S │   │  │  │     │
         │   └──┘  │   └───┘   │  └──┘     │
         │    C    │           │    C'     │  C相绕组
         └─────────────────────────────────┘

关键部件：
  1. 定子（Stator）：硅钢片叠压，嵌入三相绕组
  2. 转子（Rotor）：永磁体产生磁场
  3. 绕组（Winding）：三相绕组（A、B、C）
  4. 永磁体（PM）：钕铁硼或铁氧体
```

#### 4.1.2 定子绕组

```
三相绕组空间分布（2极电机）：

                0° (A相轴)
                  │
         240°    │    120°
          (B相)  │   (C相)
            \    │    /
             \   │   /
              \  │  /
               \ │ /
                ─┼─
                 │
               转子

电角度关系：
  A相：θe = 0°
  B相：θe = 120°
  C相：θe = 240°

星形连接（Y接）——大多数PMSM采用：
  相电压 = 线电压/√3，相电流 = 线电流
  优点：中性点可引出，无环流，绝缘要求较低
```

#### 4.1.3 转子结构

```
类型1：表贴式（SPM）
  特点：Ld ≈ Lq，适用于高速应用，但离心力大

类型2：内置式（IPM）
  特点：Ld < Lq，可利用磁阻转矩，结构强度高

类型3：埋入式
  特点：Ld << Lq，磁阻转矩大，适合弱磁控制
```

### 4.2 电机参数详解

#### 4.2.1 电感参数

```
电感的定义：
  L = N × Φ / I

电感对电流的影响：
  V = L × dI/dt
  电流变化率 = 电压 / 电感

  示例：
    L = 1mH, V = 100V
    dI/dt = 100,000 A/s = 100 A/ms
    要使电流增加10A，需要时间：t = 0.1ms = 100μs
```

**Ld和Lq的区别**：

```
d轴（直轴）：与转子磁场方向一致
q轴（交轴）：与转子磁场方向垂直

磁路分析：
  d轴方向：永磁体磁阻大（永磁体磁导率≈空气）
  q轴方向：铁芯磁阻小（铁芯磁导率高）
  因此：Ld < Lq

SPM电机：Ld ≈ Lq（磁路对称）
IPM电机：Ld < Lq（通常Lq = 1.5~3 × Ld）

电感对控制的影响：
┌──────────────────────────────────────────────────────────────┐
│  参数          │ SPM (Ld=Lq)      │ IPM (Ld<Lq)              │
│  ─────────────┼─────────────────┼──────────────────────────│
│  电流响应      │ Id, Iq相同       │ Id响应快(Ld小),Iq慢(Lq大)│
│  磁阻转矩      │ 无               │ 有(与(Lq-Ld)×Id×Iq成正比)│
│  弱磁能力      │ 差               │ 好(利用Ld×Id产生弱磁)    │
│  电流环设计    │ 单一带宽         │ Id, Iq可能需要不同带宽   │
└──────────────────────────────────────────────────────────────┘
```

**电感测量方法**：

```
方法1：静态测量（锁定转子）
  1. 将转子锁定在d轴方向
  2. 施加高频交流电压，测量电流响应
  3. 计算：Ld = V / (2πf × I)
  4. 旋转90°电角度，重复测量得到Lq

方法2：高频注入法
  注入高频电压信号Vd_hf = Vh × sin(ωh × t)
  测量电流响应：
    Ld = Vh / (ωh × Id_hf_amplitude)
    Lq = Vh / (ωh × Iq_hf_amplitude)

方法3：从电机参数表获取
  小功率伺服电机：Ld, Lq = 0.1 ~ 10 mH
  中功率电机：Ld, Lq = 1 ~ 50 mH
  大功率电机：Ld, Lq = 10 ~ 100 mH
```

#### 4.2.2 磁链参数

```
磁链的定义：
  ψ = N × Φ

永磁体磁链（ψf）：
  决定反电动势大小、转矩常数、弱磁电流需求

磁链与反电动势的关系：
  ea = ψf × ωe × sin(θe)

  反电动势常数：Ke = ψf × P

  手推计算示例：
    额定转速 n = 3000 rpm, 额定反电动势 E = 200V, 极对数 P = 4
    ωm = 2π × 3000 / 60 = 314.16 rad/s
    Ke = E / ωm = 0.637 V·s/rad
    ψf = Ke / P = 0.159 Wb = 159 mWb

磁链与转矩的关系：
  SPM电机：Te = 1.5 × P × ψf × Iq
  IPM电机：Te = 1.5 × P × [ψf × Iq + (Ld - Lq) × Id × Iq]
                                       ↑永磁转矩    ↑磁阻转矩

  转矩常数：
    SPM：Kt = 1.5 × P × ψf
    IPM：Kt = 1.5 × P × [ψf + (Ld - Lq) × Id]

  Kt与Ke的关系：Kt = Ke（国际单位制数值上相等）

  手推计算示例：
    P = 4, ψf = 0.15 Wb, Iq = 10A
    Te = 1.5 × 4 × 0.15 × 10 = 9 N·m
    Kt = 1.5 × 4 × 0.15 = 0.9 N·m/A
```

#### 4.2.3 电阻参数

```
电阻的物理意义：
  Rs = ρ × l / A

  对电机的影响：
  1. 铜损：Pcu = 3 × I² × Rs
  2. 电压降：V = I × Rs
  3. 效率：η = Pout / (Pout + Pcu + Pfe + ...)

  电阻随温度变化：
    Rs(T) = Rs(20°C) × [1 + α × (T - 20)]
    α = 铜的温度系数 ≈ 0.00393 /°C

    示例：Rs(20°C) = 1Ω, T = 100°C
    Rs(100°C) = 1 × [1 + 0.00393 × 80] = 1.315Ω
    温升80°C，电阻增加31.5%！

  典型值：
    小功率电机：Rs = 0.1 ~ 10 Ω
    中功率电机：Rs = 0.01 ~ 1 Ω
    大功率电机：Rs = 0.001 ~ 0.1 Ω
```

#### 4.2.4 极对数

```
极对数的定义：P = 极数 / 2

电角度与机械角度：θe = P × θm
电角速度与机械角速度：ωe = P × ωm

同步转速：ns = 60 × f / P

极对数选择原则：
  • 高速应用：少极数（2极、4极）
  • 低速大转矩：多极数（8极、12极、16极）
  • 齿槽转矩考虑：槽数与极数配合
```

### 4.3 电机数学模型

#### 4.3.1 三相静止坐标系（abc）

```
三相电压方程：
  ua = Rs × ia + dψa/dt
  ub = Rs × ib + dψb/dt
  uc = Rs × ic + dψc/dt

问题：方程复杂，有互感项，电感随位置变化，三相耦合难以独立控制
解决方案：坐标变换
```

#### 4.3.2 两相静止坐标系（αβ）——Clarke变换

```
Clarke变换（abc → αβ）：
  iα = ia
  iβ = (ia + 2×ib) / √3

反变换（αβ → abc）：
  ia = iα
  ib = -iα/2 + √3/2 × iβ
  ic = -iα/2 - √3/2 × iβ

优点：减少变量数量（3→2），但电感仍随位置变化
```

#### 4.3.3 两相旋转坐标系（dq）——Park变换

```
Park变换（αβ → dq）：
  id =  iα × cos(θe) + iβ × sin(θe)
  iq = -iα × sin(θe) + iβ × cos(θe)

反变换（dq → αβ）：
  iα = id × cos(θe) - iq × sin(θe)
  iβ = id × sin(θe) + iq × cos(θe)

dq坐标系下的电压方程：
  ud = Rs × id + Ld × d(id)/dt - ωe × Lq × iq
  uq = Rs × iq + Lq × d(iq)/dt + ωe × (Ld × id + ψf)

关键洞察：
  1. d轴与转子磁场方向一致
  2. q轴与转子磁场方向垂直
  3. id控制磁场（弱磁/增磁）
  4. iq控制转矩

dq坐标系下的转矩方程：
  Te = 1.5 × P × [ψf × iq + (Ld - Lq) × id × iq]

优点：电感为常数（不随位置变化），id和iq解耦，便于控制设计
```

### 4.5 电机特性曲线

#### 4.5.1 转矩-转速特性

```
PMSM转矩-转速曲线：

转矩 T
  │ 恒转矩区          弱磁区
  │←─────────────→│←──────────────────→│
  │  ─────────────┐                    │
  │               │＼                   │
  │               │  ＼                 │
  │               │    ＼               │
  └───────────────┼──────────────┼──────→ 转速 n
               基速 nb        最高转速 nmax

恒转矩区：Id = 0(SPM)或最优Id(IPM)，Iq = 额定电流
弱磁区：Id < 0(负电流去磁)，转矩随转速下降，功率近似恒定

基速计算：nb = Vdc / (√3 × Ke)
```

#### 4.5.2 效率曲线

```
损耗构成：Ploss = Pcu + Pfe + Pmech + Pstray
效率：η = Pout / (Pout + Ploss)

典型效率：
  小功率电机：75% ~ 85%
  中功率电机：85% ~ 92%
  大功率电机：92% ~ 96%
```

#### 4.5.3 电流轨迹（Id-Iq平面）

```
工作区域：
  1. 恒转矩区：电流极限圆内
  2. 弱磁区：电压极限圆内
  3. 最大转矩/电流比轨迹（MTPA）

MTPA轨迹（IPM电机）：
  Id = (ψf - √(ψf² + 4×(Lq-Ld)²×Iq²)) / (2×(Lq-Ld))

弱磁轨迹：
  Id = -(ψf/Ld) + √((umax/ωe)² - (Lq × Iq)²) / Ld
```

## 5. 🔗 交叉视角 ★★★☆☆ 💡🔧

> 电机参数不是孤立的硬件指标，它们直接决定了控制算法的行为。本章节揭示硬件参数与算法之间的深层关联。

### 5.1 电感参数(Ld, Lq) → 电流环PI参数设计

**硬件特性**：
```
电感决定电流变化速度：
  di/dt = V / L
  L大 → 电流变化慢 → 带宽低
  L小 → 电流变化快 → 带宽高
```

**🔗 算法关联**：电感直接决定电流环PI参数
```
电流环PI参数计算：
  Kp = L × ωc (ωc = 电流环带宽)
  Ki = R × ωc

  示例：
    Ld = 1mH, Lq = 1.5mH, R = 0.1Ω
    ωc = 1000 rad/s (约160Hz)
    Kp_d = 1mH × 1000 = 1
    Kp_q = 1.5mH × 1000 = 1.5
    Ki = 0.1 × 1000 = 100
```

**参数整定指南**：
```
现象：电流环震荡
诊断：
  1. 检查电感参数是否准确
  2. 检查PI参数是否过大
解决：
  - 降低Kp(电感估计偏大)
  - 增加滤波器
  - 降低电流环带宽
```

### 5.2 磁链参数(ψf) → 转矩常数与Id=0控制

**硬件特性**：
```
转矩常数：Kt = 1.5 × P × ψf
反电动势常数：Ke = P × ψf
关系：Kt = Ke（单位统一后）
```

**🔗 算法关联**：磁链决定转矩常数，直接影响Id=0控制策略
```
Id=0控制策略：
  Te = 1.5 × P × ψf × Iq

  如果ψf不准：
    - 转矩输出偏差
    - 效率计算错误
    - 弱磁控制失效

软件应对：
  1. 离线辨识ψf
  2. 在线自适应
  3. 查表补偿
```

**参数辨识方法**：
```c
// 方法1:空载反电动势法
float psi_f = V_phase / (P * omega_e);

// 方法2:堵转测试法
float psi_f = T_measured / (1.5 * P * Iq);
```

### 5.3 极对数(P) → Park变换角度计算

**硬件特性**：
```
电角度计算：θe = P × θm
示例：4极电机(P=2)，机械角度180° → 电角度360°
```

**🔗 算法关联**：极对数决定Park变换中的电角度
```c
void Park_Transform(float alpha, float beta, float theta_e, float *d, float *q)
{
    float sin_theta = sinf(theta_e);
    float cos_theta = cosf(theta_e);

    *d =  alpha * cos_theta + beta * sin_theta;
    *q = -alpha * sin_theta + beta * cos_theta;
}

// 如果极对数错误：
//   - Park变换角度错误
//   - Id/Iq解耦失败
//   - 电流环震荡
```

**常见错误**：
```
错误1:极对数配置错误
  现象:电机抖动,无法正常运转
  解决:检查电机铭牌,确认极对数

错误2:角度单位混淆
  现象:Park变换结果错误
  解决:统一使用弧度制
```

### 5.4 电阻(Rs) → 效率计算与温度补偿

```
铜损计算：Pcu = 3 × I² × Rs
示例：I = 10A, Rs = 0.1Ω → Pcu = 30W

如果Rs不准：
  - 效率计算错误
  - 温升估计错误
  - 电流环Ki参数偏差
```

### 5.5 磁阻转矩 → MTPA控制策略

```
IPM电机：Lq > Ld
  Te = 1.5 × P × [ψf × Iq + (Ld - Lq) × Id × Iq]
                       ↑永磁转矩         ↑磁阻转矩

MTPA控制(最大转矩电流比)：
  利用磁阻转矩，提高效率
  Id_ref = f(Iq_ref, Ld, Lq, ψf)

控制策略选择：
  SPM(Ld ≈ Lq)：Id=0
  IPM(Lq > Ld)：MTPA、弱磁控制
```

### 5.6 参数测量方法

**离线测量**：
```c

float Ld = V_d_high_freq / (2*pi*f * I_d_high_freq); // 电感：高频注入法

## 6. 🎯 工程案例 ★★★☆☆ 🔧🎯

### 案例1：电流环震荡——电感参数不准

**项目背景**：
```
应用:电动自行车
电机:表贴式PMSM,额定功率500W
控制器:STM32G4,FOC控制
问题:电流环震荡,无法稳定
```

**诊断过程**：
```
步骤1:示波器观察电流波形 → 震荡频率约200Hz
步骤2:检查电流环带宽 → 设计160Hz,接近震荡频率
步骤3:检查电感参数 → 数据表L=0.5mH,实测L=0.8mH(偏差60%!)
步骤4:重新计算PI参数 → 原参数基于错误电感值
```

**根本原因**：电感参数不准！Kp计算偏小 → 带宽不足 → 相位裕度小 → 震荡

**解决方案**：
```
方案A:修正电感参数,重新计算PI
  新Kp = 0.8mH × 1000 = 0.8 (原0.5)
  结果:震荡消除,响应良好 ✅

方案B:降低电流环带宽
  ωc = 800 rad/s (原1000)
  结果:震荡消除,但响应变慢 ⚠️
```

**经验总结**：
1. 电机参数必须实测，不能只看数据表
2. 电感测量方法：高频注入法(在线)、LCR电桥(离线)
3. 电流环带宽应小于电频率的1/10
4. 相位裕度应>45°

### 案例2：弱磁控制失效——d轴电感参数不准

**项目背景**：
```
应用:电动汽车
电机:内置式PMSM,额定功率30kW
控制器:TI C2000,FOC+弱磁控制
问题:高速时电流失控,报过流故障
```

**诊断过程**：
```
步骤1:分析弱磁控制策略 → 采用前馈弱磁控制
步骤2:检查Id电流轨迹 → Id未按预期负值增加
步骤3:检查电压极限圆 → 电压已达极限,但Id未调整
步骤4:检查d轴电感参数 → 数据表Ld=1mH,实测Ld=1.5mH
```

**根本原因**：d轴电感参数不准！弱磁电流计算错误：
```
  Id_weak = -ψf / Ld
  计算值(基于1mH): Id = -100A
  实际需要(基于1.5mH): Id = -67A
  结果:弱磁电流过大 → 饱和 → 失控
```

**解决方案**：
```
方案A:修正电感参数 → 弱磁控制正常 ✅
方案B:采用自适应弱磁控制(在线调整Id,基于电压反馈) → 鲁棒性更好 ✅✅
```

**经验总结**：
1. 弱磁控制对电感参数敏感
2. 内置式PMSM必须准确测量Ld, Lq
3. 推荐自适应弱磁控制策略
4. 电压极限圆和电流极限圆必须正确绘制

### 案例3：转矩常数不准——定义混淆

**项目背景**：
```
应用:工业机器人关节
电机:表贴式PMSM,额定转矩10Nm
控制器:STM32H7,FOC控制
问题:实际输出转矩与指令偏差大(约16%)
```

**诊断过程**：
```
步骤1:数据表Kt = 1.0 Nm/A,计算Te = 1.0 × 5 = 5Nm
步骤2:实测Te = 4.2Nm
步骤3:反推Kt_actual = 4.2 / 5 = 0.84 Nm/A
步骤4:发现数据表Kt与FOC公式Kt = 1.5 × P × ψf不一致!
```

**根本原因**：转矩常数定义混淆！
```
  数据表Kt = 1.0 Nm/A (基于峰值相电流)
  FOC中Kt = 1.5 × P × ψf (基于Iq电流)
  转换：Kt_FOC = Kt_datasheet / √(3/2) = 1.0 / 1.225 = 0.816 Nm/A
  与实测0.84接近！
```

**解决方案**：修正Kt_FOC = 0.84 Nm/A，转矩精度提升到±2% ✅

**经验总结**：
1. 注意转矩常数的定义(峰值电流 vs RMS电流)
2. FOC中转矩公式：Te = 1.5 × P × ψf × Iq
3. 推荐实测转矩常数
4. 不同工作点可能有偏差(饱和效应)

### 案例4：极对数配置错误

**项目背景**：
```
应用:无人机电机
电机:外转子PMSM,极数未知
控制器:STM32G4,FOC控制
问题:电机抖动,无法正常运转

步骤1:检查编码器 → 正常
步骤2:检查Park变换角度 → 角度变化异常

**经验总结**：
1. 极对数必须准确配置
2. 测量方法：手动旋转一圈，观察电角度变化
3. 注意：极数 = 2P(N+S)，极对数 = P

## 附录：快速计算公式汇总

### A. 电角度与机械角度
```
θe = P × θm
ωe = P × ωm
fe = P × n / 60
```

### B. 反电动势
```
E = Ke × ωm = ψf × P × ωm
Ke = ψf × P
```

### C. 转矩
```
SPM: Te = 1.5 × P × ψf × Iq
IPM: Te = 1.5 × P × [ψf × Iq + (Ld - Lq) × Id × Iq]
Kt = 1.5 × P × ψf
```

### D. 电流环带宽
```
ωc = (Rs/L) × (Kp/Ki)（简化）
或 ωc = 1 / τ（其中 τ = L/R）
```

### E. 基速
```
nb = Vdc / (√3 × Ke)
```

### F. 弱磁电流
```
Id_fw = -ψf / Ld（完全弱磁）
```
