## 一、参数计算题（★★☆☆☆）

### 题目1：角度精度与Park变换交叉耦合
**问题：** 一台PMSM参数：Kt = 0.6 N·m/A，Iq = 20A，Id = 0。角度传感器精度为±0.5°。请计算：
1. 角度误差导致的Id/Iq交叉耦合量
2. 转矩误差
3. 如果要求Id耦合 < 0.5%，需要什么角度精度？

**参考答案：**
1. Δθ = 0.5° = 0.00873 rad；Id耦合 = Iq × sin(Δθ) ≈ Iq × Δθ = 20 × 0.00873 = 0.175A（0.875%）；Iq损失 = Id × Δθ = 0（Id=0时无损失）
2. 转矩误差 ≈ Kt × Id耦合 = 0.6 × 0.175 = 0.105 N·m
3. Δθ < arcsin(0.005/20) ≈ 0.005/20 = 0.00025 rad ≈ 0.014°。需要极高精度传感器，或通过观测器插值提升等效精度

### 题目2：角度延迟与高速性能
**问题：** 一台PMSM极对数P = 4，最高转速6000rpm，使用旋变器+硬件RDC，总延迟td = 100μs。请计算：
1. 最高速时的角度偏差
2. 如果要求角度偏差 < 3°，最大允许延迟
3. 选择编码器（延迟5μs）后的角度偏差

**参考答案：**
1. ωe_max = P × 2π × 6000/60 = 4 × 628.3 = 2513 rad/s；Δθ = ωe × td = 2513 × 100μs = 0.2513 rad ≈ 14.4°
2. td_max = 3° / (2513 × π/180) = 0.0524 / 2513 = 20.8μs
3. 编码器延迟5μs：Δθ = 2513 × 5μs = 0.01257 rad ≈ 0.72° → 远小于3°，满足要求。结论：高速应用必须使用低延迟编码器，旋变器延迟太大

### 题目3：编码器分辨率与最低可控转速
**问题：** 一台PMSM使用2500PPR增量编码器（4倍频），控制频率10kHz，极对数P = 3。请计算：
1. 角度分辨率
2. 最低可控转速
3. 如果要求最低转速1rpm，需要什么分辨率？

**参考答案：**
1. 4倍频后步数 = 4 × 2500 = 10000步/转；1 LSB = 360°/10000 = 0.036°
2. n_min = (0.036°/360°) × 60 × 10000 = 60 rpm
3. 需要1 LSB < 1rpm × 360°/(60 × 10000) = 0.0006°；步数 > 360°/0.0006° = 600000 → 需要19.2位以上。实际方案：14位编码器(16384步) + PLL观测器插值，等效分辨率可提升10-100倍

## 二、电路设计题（★★★☆☆）

### 题目4：增量编码器接口电路设计
**问题：** 为一个2500PPR增量编码器设计MCU接口电路，编码器输出RS422差分信号（A+/A-/B+/B-/Z+/Z-），MCU为STM32G4。请给出：
1. 差分接收电路设计（含参数）
2. MCU定时器配置方案
3. 零点标定流程

**参考答案：**
1. 差分接收电路：每对信号使用AM26C32或MAX3095差分接收器。终端电阻120Ω（在接收端A+与A-之间），RC滤波：R = 1kΩ，C = 10nF，fc ≈ 16kHz
2. MCU定时器配置：TIM1/TIM8编码器模式 → 自动4倍频计数；A相→TI1，B相→TI2；计数器自动根据A/B相位关系增减；Z信号→外部中断，用于零点校准
3. 零点标定流程：
   - (a) 施加Id电流（Iq=0），将转子拉到d轴方向
   - (b) 等待1秒让转子稳定
   - (c) 读取编码器当前计数值作为零点偏移
   - (d) 将偏移值保存到Flash
   - (e) 验证：施加Iq电流，电机应平稳旋转

### 题目5：磁编码器选型与接口设计
**问题：** 需要为一个云台电机选择位置传感器，要求：角度精度<0.1°，分辨率14位以上，响应时间<100μs，成本敏感。请设计：
1. 传感器选型及理由
2. SPI接口时序设计
3. 角度零点标定方案

**参考答案：**
1. 选择AS5047P（14位磁编码器，精度±0.05°，SPI接口，响应时间100μs，成本适中）。理由：满足精度和分辨率要求，SPI接口简单，非接触式抗震动
2. SPI接口：CPOL=1, CPHA=1（Mode 1），时钟最高10MHz，16位帧（1位奇偶校验+1位错误标志+14位数据）。CS→CLK→MOSI(命令0x3FFF)→MISO(14位角度)。角度 = DATA × 360° / 16384
3. 零点标定：d轴对齐法 → 施加Id电流锁定转子 → 读取AS5047P角度值 → 计算偏移 = 读出角度 - 0°（电角度0°对应的机械角度）→ 保存偏移到Flash。每次上电后：电角度 = (读取角度 - 偏移) × P

## 三、故障诊断题（★★★☆☆）

### 题目6：电机启动抖动——零点标定错误
**问题：** 一台AGV驱动轮电机，使用2500PPR增量编码器，FOC控制。电机启动时剧烈抖动，有时能转起来有时不行。标定时使用1V电压对齐。请分析：
1. 故障根本原因
2. 如何验证？
3. 解决方案

**参考答案：**
1. 标定电压1V太低 → 转子未完全对齐到d轴方向 → 零点偏移误差大（可能>10°电角度）→ Park变换角度错误 → Id/Iq解耦失败 → 启动时电流方向错误 → 转子来回震荡
2. 验证方法：
   - (a) 施加Id电流，观察转子是否稳定锁定在固定位置（若抖动则对齐不充分）
   - (b) 标定后施加Iq电流，观察是否平稳旋转（若抖动则零点偏移大）
   - (c) 多次标定，比较零点值的一致性（偏差大说明对齐不稳定）
3. 解决方案：
   - (a) 增大标定电流到50%额定电流，确保转子完全对齐
   - (b) 多次标定取平均，减小随机误差
   - (c) 标定后验证：施加Iq=10%额定电流，电机应平稳旋转
   - (d) 保存标定值到Flash，避免每次上电重新标定

### 题目7：磁编码器受干扰诊断
**问题：** 一台外转子PMSM使用AS5048A磁编码器，静止时角度稳定，电机运行时角度信号出现周期性跳变，最大偏差5°。请分析：
1. 干扰来源
2. 诊断步骤
3. 解决方案（至少3种）

**参考答案：**
1. 干扰来源：电机绕组电流产生的磁场干扰磁编码器的霍尔元件阵列。外转子电机磁编码器安装在转子下方，距离绕组近，受漏磁影响大
2. 诊断步骤：
   - (a) 静止时角度稳定 → 排除传感器故障
   - (b) 电机通电但堵转，观察角度 → 确认是电流磁场干扰
   - (c) 改变电流大小，观察角度偏差与电流的关系 → 确认线性相关
   - (d) FFT分析角度误差频率 → 确认与电流频率一致
3. 解决方案：
   - (a) 增加磁屏蔽：在磁编码器和电机之间加坡莫合金屏蔽罩
   - (b) 增大磁体与编码器距离：减弱干扰磁场
   - (c) 软件滤波：低通滤波 + 陷波（去除电流频率分量）
   - (d) 离线校准：在不同电流下测量角度误差，查表补偿
   - (e) 改用光电编码器（完全不受磁场干扰，但成本和体积增加）

## 四、软硬件关联题（★★★★☆）

### 题目8：角度延迟补偿与观测器设计
**问题：** 一台高速主轴电机（最高20000rpm，P=2），使用旋变器+RDC，延迟200μs。请分析：
1. 最高速时的角度偏差（定量计算）
2. 简单前馈补偿的公式和局限性
3. PLL观测器的优势及参数设计

**参考答案：**
1. ωe_max = 2 × 2π × 20000/60 = 4189 rad/s；Δθ = 4189 × 200μs = 0.838 rad ≈ 48°！→ 严重失真，Park变换完全错误
2. 前馈补偿：θ_comp = θ_meas + ωe × td。局限性：(a) ωe的估计有误差→补偿不精确；(b) 延迟td不恒定（随温度、转速变化）；(c) 仅补偿稳态延迟，无法补偿动态变化
3. PLL观测器优势：(a) 自动跟踪角度和速度，无需精确知道延迟；(b) 滤波效果好，抑制噪声；(c) 提供平滑的角度插值。参数设计：
   ```
   θ_est += ω_est × dt
   e = sin(θ_meas - θ_est)
   ω_est += Kp_pll × e + Ki_pll × ∫e dt
   ```
   带宽选择：f_pll < 1/(4×td) = 1/(4×200μs) = 1.25kHz。Kp_pll = 2×ζ×ωn，Ki_pll = ωn²，其中ωn = 2π×500（取带宽的40%），ζ = 0.707

### 题目9：极对数倍增效应对传感器选型的影响
**问题：** 两台PMSM需要选择位置传感器，要求电角度精度<0.5°：
- 电机A：4极（P=2），机械角度精度需求？
- 电机B：16极（P=8），机械角度精度需求？
- 如果使用同一款14位磁编码器（精度±0.05°），是否都满足？

**参考答案：**
- 电机A：电角度精度 = P × 机械角度精度 → 机械角度精度 < 0.5°/2 = 0.25°
- 电机B：机械角度精度 < 0.5°/8 = 0.0625°
- 14位磁编码器精度±0.05°：
  - 电机A：电角度精度 = 2 × 0.05° = 0.1° < 0.5° ✓
  - 电机B：电角度精度 = 8 × 0.05° = 0.4° < 0.5° ✓（勉强满足）
- 结论：多极电机对传感器精度要求更高。电机B若要求更高精度（如<0.2°电角度），14位磁编码器不够，需要更高精度传感器或软件补偿

---

### 题目10：绝对编码器与增量编码器的系统级选择
**问题：** 为一个工业机器人关节电机选择位置传感器，需求：上电即知位置（无需回零），角度精度<0.1°电角度，P=4，最高转速3000rpm，通信延迟<50μs。请分析：
1. 增量编码器是否可行？为什么？
2. 绝对编码器的选型要求（分辨率、精度、接口）
3. 旋变器是否适合？为什么？

**参考答案：**
1. 增量编码器不可行。原因：增量编码器上电时不知道绝对位置，必须回零才能确定初始角度。工业机器人关节不允许每次上电回零（可能碰撞），必须使用绝对式传感器
2. 绝对编码器选型：
   - 分辨率：机械角度精度 < 0.1°/4 = 0.025° → 步数 > 360°/0.025° = 14400 → 需要14位以上
   - 精度：机械精度 < 0.025° → 选择±0.02°精度的绝对编码器
   - 接口：SSI或BiSS，通信时间 < 50μs → 25位数据 × (1/2MHz) = 12.5μs + 帧间隔20μs = 32.5μs ✓
   - 推荐：17位绝对编码器（如AS5047P的绝对模式或BiSS接口编码器）
3. 旋变器不太适合。原因：(a) 延迟大（RDC 100-500μs）> 50μs要求；(b) 高速3000rpm时P=4 → ωe = 1257 rad/s，延迟100μs → Δθ = 7.2°，超出0.1°要求太多；(c) 需要额外的RDC电路增加成本。旋变器适合恶劣环境（高温、强震），但本应用环境温和，编码器更优
