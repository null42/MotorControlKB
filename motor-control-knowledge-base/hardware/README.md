## 学习建议

### 零硬件基础入门路线
1. 先学HW-1（电机本体），建立对控制对象的基本认知
2. 再学HW-2（电流采样），理解FOC最关键的反馈通道
3. 然后学HW-3（位置传感器），理解角度反馈
4. 之后按需学习HW-4到HW-7

### 有硬件基础进阶路线
1. 直接从HW-5（功率器件）开始，深入理解开关特性
2. 学习HW-6和HW-7，掌握系统级设计能力
3. 回顾HW-1到HW-4，建立软硬件关联

### 每个模块的学习方法
1. 先读"问题引入"，建立学习动机
2. 再读"直观理解"，建立感性认识
3. 精读"技术原理"，掌握计算方法
4. 必读"交叉视角"，建立软硬件关联
5. 选读"工程案例"，积累实践经验
6. 完成"实践练习"，检验学习效果
