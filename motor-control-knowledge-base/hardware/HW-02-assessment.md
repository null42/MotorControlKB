## 一、参数计算题（★★☆☆☆）

### 题目1：采样电阻选型计算
**问题：** 设计一个电流采样电路，已知：额定电流I_rated = 15A，峰值电流I_peak = 30A，ADC参考电压Vref = 3.3V，ADC位数12位，放大器增益G = 20。请计算：
1. 采样电阻阻值（双向电流，中点偏置Vref/2）
2. 电流分辨率（1 LSB对应的电流值）
3. 采样电阻在峰值电流时的功耗

**参考答案：**
1. Vout_max = Vref / 2 = 1.65V（双向偏置）；Rshunt = Vout_max / (G × I_peak) = 1.65 / (20 × 30) = 0.00275Ω ≈ 2.75mΩ，选标准值3mΩ
2. 1 LSB = Vref / 2^12 = 3.3 / 4096 = 0.806mV；电流分辨率 = 0.806mV / (20 × 0.003) = 13.4mA
3. P_peak = I_peak² × Rshunt = 30² × 0.003 = 2.7W；P_rated = 15² × 0.003 = 0.675W。选择额定功率≥3W的合金箔电阻

### 题目2：采样延迟与电流环带宽
**问题：** 已知PWM频率10kHz（中心对齐模式），ADC采样时间1μs，ADC转换时间1μs，DMA传输0.5μs，FOC算法计算3μs。请计算：
1. PWM中心对齐模式下的总采样延迟
2. 电流环最大允许带宽
3. 如果改为边沿对齐模式，延迟和带宽如何变化？

**参考答案：**
1. 中心对齐：ADC在谷底触发，延迟 ≈ 0.25×Tpwm + 采样 + 转换 + DMA = 25 + 1 + 1 + 0.5 = 27.5μs
2. 带宽_max ≈ 1 / (4 × td) = 1 / (4 × 27.5μs) ≈ 9.1kHz。实际设计带宽应远小于此值，通常取1/10即约900Hz
3. 边沿对齐：延迟 ≈ 0.5×Tpwm + 2.5 = 50 + 2.5 = 52.5μs；带宽_max ≈ 1 / (4 × 52.5μs) ≈ 4.76kHz。中心对齐比边沿对齐延迟减半，带宽翻倍

---

### 题目3：过采样提升分辨率计算
**问题：** 系统使用12位ADC，PWM频率20kHz，每个PWM周期可采样4次。请计算：
1. 4倍过采样后的有效分辨率
2. 16倍过采样需要的条件
3. 过采样对采样延迟的影响

**参考答案：**
1. OSR = 4，增加位数 n = log4(4)/2 = 1位，有效分辨率 = 12 + 1 = 13位
2. OSR = 16，增加位数 n = log4(16)/2 = 2位，有效分辨率 = 14位。条件：信号中必须存在足够的热噪声（>0.5 LSB），使量化噪声近似白噪声
3. 4倍过采样：每个PWM周期采样4次，需在PWM周期内完成4次ADC转换，采样窗口更窄，对时序要求更严格。延迟基本不变（仍以最后一次采样为准），但数据处理量增大4倍

## 二、电路设计题（★★★☆☆）

### 题目4：电流采样电路完整设计
**问题：** 为一台5A额定/15A峰值的PMSM设计电流采样电路，控制器为STM32G4（12位ADC，3.3V参考），电流环带宽2kHz，PWM频率20kHz。请给出完整设计方案：
1. 采样拓扑选择及理由
2. 采样电阻参数
3. 放大器选型与增益设计
4. RC滤波器参数
5. 总精度估算

**参考答案：**
1. 双电阻采样（两相低端采样）：成本适中、算法简单、精度满足要求。第三相ic = -(ia + ib)
2. Rshunt = (1.65V) / (G × 15A)，若G=20：Rshunt = 5.5mΩ，选标准值5mΩ/1%/2W合金箔电阻
3. 选择INA240（CMRR=120dB，Vos=5μV，GBW=400kHz）。增益G=20：Vout_max = 15 × 0.005 × 20 = 1.5V < 1.65V ✓
4. fc > 10 × fsw = 200kHz，fc < fs/2；选fc = 200kHz；R = 470Ω，C = 1/(2π×470×200k) = 1.7nF，取C = 1.5nF；延迟 ≈ R×C = 0.7μs
5. 总精度(RSS) = √(1%² + 0.5%² + 0.05%² + 0.012%²) ≈ 1.12%。其中：电阻1%，放大器增益0.5%，失调0.05%，ADC量化0.012%

### 题目5：RC滤波器对采样精度的影响
**问题：** 一个电流采样电路的RC滤波器参数为R = 1kΩ，C = 10nF。请分析：
1. 截止频率和延迟
2. 该滤波器是否适合20kHz PWM系统？
3. 如果电流变化率di/dt = 50A/ms，滤波器延迟导致的电流误差是多少？

**参考答案：**
1. fc = 1/(2π×1k×10nF) = 15.9kHz；延迟 ≈ R×C = 1k×10nF = 10μs
2. fc = 15.9kHz < 10 × fsw(200kHz)，截止频率偏低，会衰减PWM开关频率的谐波成分，但也会引入较大延迟。10μs延迟对50μs PWM周期占20%，偏大。建议降低R或C，使fc > 100kHz
3. 电流误差 ≈ di/dt × 延迟 = 50A/ms × 10μs = 0.5A。对15A峰值电流，误差3.3%，不可忽略。优化：减小RC → R=470Ω, C=1.5nF → 延迟0.7μs → 误差0.035A

## 三、故障诊断题（★★★☆☆）

### 题目6：Id电流偏置诊断
**问题：** FOC控制电机，设置Id=0，但实际Id始终有0.5A偏置。温度升高时偏置增大到0.8A，重启后偏置可能变化。请分析：
1. 可能的原因（至少3个）
2. 诊断步骤
3. 解决方案（按优先级排列）

**参考答案：**
1. 可能原因：(a) 放大器失调电压导致的零漂；(b) ADC零点偏移；(c) PCB漏电流；(d) 共模抑制不足（CMRR不够）；(e) 增益不匹配（三相放大器增益不一致）
2. 诊断步骤：
   - 步骤1：关闭PWM，读取ADC原始值 → 检查零漂（若原始值偏离中点，确认零漂存在）
   - 步骤2：加热PCB，观察偏置变化 → 若增大，确认温漂
   - 步骤3：施加已知电流，比较三相增益 → 检查增益匹配
   - 步骤4：测量放大器输出共模抑制 → 检查CMRR
3. 解决方案（优先级）：
   - (a) 上电零漂校准：关闭PWM，采样256次取平均作为零点（成本零，效果显著）
   - (b) 在线零漂补偿：在下管全关时采样更新零点（自动温漂补偿）
   - (c) 增益校准：施加已知电流校准三相增益一致性
   - (d) 更换低失调放大器（如INA240，Vos=5μV）

### 题目7：高速电流失真诊断
**问题：** 一台PMSM低速运行电流波形正常，但高速（>5000rpm）时电流波形畸变，Iq波动大，转矩脉动增大。PWM频率10kHz，双电阻低端采样。请分析：
1. 可能的根本原因
2. 如何验证你的假设？
3. 给出至少2种解决方案

**参考答案：**
1. 根本原因：高速时PWM占空比大 → 下管导通时间短 → 采样窗口不足。10kHz PWM周期100μs，高速时占空比>90% → 下管导通时间<10μs，减去死区1μs和电流建立时间2μs，实际采样窗口仅7μs，可能不够ADC采样保持时间
2. 验证方法：
   - (a) 示波器观察PWM波形和ADC触发时刻，确认采样点是否在下管导通期间
   - (b) 降低转速，观察电流波形是否恢复正常
   - (c) 计算最小导通时间 = Tpwm × (1-Dmax) - Tdead - Tsettle，与ADC采样时间比较
3. 解决方案：
   - (a) 采用三电阻采样（相电流采样），不受下管导通时间限制
   - (b) 非对称PWM + 采样窗口扩展：在特定扇区人为扩展下管导通时间
   - (c) 降低PWM频率（如从10kHz到8kHz），增加下管导通时间（但开关损耗增大）

## 四、软硬件关联题（★★★★☆）

### 题目8：采样精度对FOC控制的影响链分析
**问题：** 一个电流采样系统的总精度为±2%（RSS），电机转矩常数Kt = 0.5 N·m/A，额定电流10A。请分析：
1. 采样精度导致的最大转矩误差
2. 采样精度对Id=0控制的影响（假设角度精确）
3. 如果ADC只有8位，对控制性能的影响

**参考答案：**
1. 电流误差 = 10A × 2% = ±0.2A；转矩误差 = Kt × 0.2 = 0.5 × 0.2 = ±0.1 N·m（额定转矩5 N·m的2%）
2. Id=0控制中，采样误差使实际Id ≠ 0：Id_error ≈ Iq × (采样误差比) = 10 × 0.02 = 0.2A。这0.2A的Id偏置导致：(a) 弱磁/增磁效应（改变气隙磁通）；(b) 额外铜损3 × 0.2² × Rs；(c) 转矩常数微调。对高性能伺服（转矩精度<0.5%），2%采样精度不够
3. 8位ADC：1 LSB = 3.3/256 = 12.9mV；电流分辨率 = 12.9mV/(20×5mΩ) = 129mA；量化精度 = 129mA/10A = 1.29%。Park变换后量化噪声放大√2倍 → Id/Iq量化噪声 ≈ 182mA → 对10A电流4.5%误差。这将导致：电流环抖动明显、转矩脉动大、低速性能差。8位ADC不适合FOC控制

---

### 题目9：单电阻采样的低速限制分析
**问题：** 一个空调压缩机采用单电阻FOC控制，PWM频率10kHz，母线电压310V，电机电感1mH。低速时电流波形严重失真。请分析：
1. 单电阻采样的基本原理和限制
2. 低速时失真的根本原因（定量分析）
3. 给出3种改善方案并比较优劣

**参考答案：**
1. 原理：不同开关状态组合下，母线电流=不同相电流。需要两个不同开关状态来重建两相电流。限制：每个开关状态需要足够的持续时间供ADC采样（建立时间≥2μs）
2. 低速时调制比低 → 有效向量时间短。例如：调制比0.1时，有效向量时间 = 0.1 × Tpwm/2 = 5μs，减去死区1μs和建立时间2μs，仅剩2μs → 勉强够采样。调制比0.05时，有效向量时间2.5μs → 不够！
3. 改善方案：
   - (a) PWM移相+注入：在有效向量中插入测量窗口 → 低速性能改善，但电流纹波增大
   - (b) 低速开环启动：V/F控制启动到一定速度后切换FOC → 启动平稳，切换有短暂波动
   - (c) 改用双电阻采样：成本略增（+1个电阻+1路ADC），但低速性能大幅提升，算法简单 → 推荐方案

---

### 题目10：共模抑制与Id零漂的定量分析
**问题：** 一个低端采样电路，放大器CMRR = 80dB，采样电阻5mΩ，增益G = 20，母线电压48V。请分析：
1. 母线电压变化48V时，放大器输入等效误差电压
2. 该误差导致的电流偏移
3. 对Id=0控制的影响
4. 如何改善？

**参考答案：**
1. CMRR = 80dB → 放大倍数 = 10^(-80/20) = 0.0001；输入等效误差 = 48V × 0.0001 = 4.8mV
2. 电流偏移 = 4.8mV / (G × Rshunt) = 4.8mV / (20 × 5mΩ) = 48mA
3. 48mA的Id偏置 → 在Iq = 10A时，Id/Iq = 0.48%，影响较小。但在Iq = 1A（轻载）时，Id/Iq = 4.8%，影响显著。轻载时Id偏置导致转矩方向偏移、效率下降
4. 改善方案：
   - (a) 选择高CMRR放大器：INA240（CMRR=120dB）→ 误差降低100倍至0.48mA
   - (b) 差分布线减少共模耦合
   - (c) 软件零漂补偿：在不同共模电压下校准偏移，运行时查表补偿
   - (d) 采用三电阻相电流采样（共模电压变化小）
