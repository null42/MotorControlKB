## 一、参数计算题（★★☆☆☆）

### 题目1：MOSFET功率损耗计算
**问题：** 已知MOSFET参数：RDS(on) = 80mΩ @ 100°C，电流15A有效值，占空比0.5，开关损耗能量Esw(on) = 0.5mJ，Esw(off) = 0.3mJ，fsw = 20kHz。请计算：
1. 导通损耗
2. 开关损耗
3. 总损耗
4. 哪种损耗是主要的？

**参考答案：**
1. Pcond = I² × RDS(on) × D = 15² × 0.08 × 0.5 = 9W
2. Psw = (Esw(on) + Esw(off)) × fsw = (0.5 + 0.3) × 10^-3 × 20000 = 16W
3. Ptotal = 9 + 16 = 25W
4. 开关损耗16W > 导通损耗9W → 开关损耗是主要的（占64%）。降低开关频率可显著减少总损耗，但会增大电流纹波

### 题目2：散热器选型计算
**问题：** 已知MOSFET损耗Ploss = 25W，设计结温Tj_design = 130°C，环境温度Ta = 45°C，Rth_JC = 0.8°C/W，Rth_CS = 0.2°C/W。请计算：
1. 允许的最大热阻Rth_JA
2. 所需散热器热阻Rth_SA
3. 选择合适的散热器类型

**参考答案：**
1. Rth_JA_max = (Tj_design - Ta) / Ploss = (130 - 45) / 25 = 3.4°C/W
2. Rth_SA_max = Rth_JA_max - Rth_JC - Rth_CS = 3.4 - 0.8 - 0.2 = 2.4°C/W
3. 选择铝型材散热器：Rth_SA < 2.4°C/W。估算面积：Rth_SA ≈ 50/(√A × √v)，自然对流v≈0.5 → A > (50/(2.4×0.707))² = (29.5)² = 870 cm² → 选择200mm×100mm×50mm铝型材散热器（表面积约800cm²，加风扇可满足）

### 题目3：热时间常数与瞬态温升
**问题：** 已知Rth_JA = 5°C/W，Cth = 100 J/°C，Ploss = 20W，Ta = 40°C。请计算：
1. 热时间常数
2. 稳态结温
3. 达到95%稳态的时间
4. 脉冲负载100W持续1秒时的峰值结温（Zth(1s) = 0.5°C/W）

**参考答案：**
1. τ = Rth × Cth = 5 × 100 = 500s ≈ 8.3分钟
2. Tj_ss = Ta + Ploss × Rth = 40 + 20 × 5 = 140°C
3. 达到95%稳态时间 = 3τ = 1500s = 25分钟
4. Tj_peak = Ta + Ppulse × Zth(1s) = 40 + 100 × 0.5 = 90°C → 短时脉冲安全（远低于稳态140°C）

## 二、电路设计题（★★★☆☆）

### 题目4：EMI滤波器设计
**问题：** 设计一个EMI滤波器，要求在150kHz处衰减40dB，开关频率20kHz。请给出：
1. 截止频率计算
2. 差模滤波器参数
3. 共模滤波器参数
4. 完整滤波器电路

**参考答案：**
1. 40dB衰减 → 衰减倍数 = 10^(40/20) = 100；100 = (f/fc)² → fc = 150kHz / 10 = 15kHz
2. 差模滤波器：选择Ldm = 470μH × 2，Cx = 1μF；fc = 1/(2π×√(470μ×2×1μ)) = 5.2kHz < 15kHz ✓
3. 共模滤波器：选择Lcm = 10mH，Cy = 4.7nF × 2；fc = 1/(2π×√(10m×4.7n)) = 23.2kHz；衰减@150kHz ≈ 20×log10(1+(150k/23.2k)²) ≈ 32dB → 需要增加一级
4. 完整滤波器（两级共模+差模）：
   ```
   L ──[470μH]──┬──[10mH]──┬──[10mH]──┬──[470μH]── L'
                 │          │          │
                1μF        4.7nF      4.7nF
                 │          │          │
   N ──[470μH]──┴──[10mH]──┴──[10mH]──┴──[470μH]── N'
                              │          │
                             PE         PE
   总衰减@150kHz ≈ 32 + 32 + 差模 ≈ 70dB ✓
   ```

### 题目5：PCB散热设计
**问题：** 一个D2PAK封装MOSFET，损耗5W，无外部散热器，环境温度50°C。请设计PCB散热方案：
1. 铜箔面积计算
2. 热过孔设计
3. 总热阻估算
4. 结温验证

**参考答案：**
1. 铜箔面积：Rth_copper ≈ 500/(A×√A)，设A=10cm² → Rth ≈ 500/(10×3.16) = 15.8°C/W
2. 热过孔：10个0.3mm直径过孔；Rth_via = 100/10 = 10°C/W
3. 总热阻：Rth_JA ≈ Rth_JC + Rth_copper ‖ Rth_via = 2.0 + (15.8×10)/(15.8+10) = 2.0 + 6.1 = 8.1°C/W
4. Tj = 50 + 5 × 8.1 = 90.5°C < 130°C ✓ → 安全

## 三、故障诊断题（★★★☆☆）

### 题目6：MOSFET过温保护频繁触发
**问题：** MOSFET在满载运行时温度持续升高直到触发过温保护，但选型时计算的热阻应该足够。请分析：
1. 可能的原因（至少4个）
2. 诊断步骤
3. 解决方案

**参考答案：**
1. 可能原因：
   - (a) 导热硅脂涂抹不均匀或太厚 → Rth_CS增大
   - (b) 散热器安装不紧密 → 接触热阻增大
   - (c) 实际损耗比计算值大（开关损耗被低估）
   - (d) 环境温度比预期高（密闭空间无通风）
   - (e) 散热器被遮挡或灰尘堵塞
   - (f) 多个器件热耦合（相邻MOSFET互相加热）
2. 诊断步骤：
   - (a) 红外热像仪测量散热器温度分布 → 确认热点位置
   - (b) 测量MOSFET壳温 → 反推实际热阻
   - (c) 示波器测量开关波形 → 计算实际开关损耗
   - (d) 测量环境温度和风速
   - (e) 检查导热硅脂和安装力矩
3. 解决方案：
   - (a) 重新涂抹导热硅脂（薄而均匀）
   - (b) 增大散热器或增加风扇
   - (c) 降低开关频率减少开关损耗
   - (d) 增加器件间距减少热耦合
   - (e) 改善通风条件

### 题目7：电流采样尖峰诊断
**问题：** 电流采样信号上有与PWM开关同步的尖峰，幅值约200mA。请分析：
1. 尖峰的来源
2. 从PCB布局到软件滤波的完整解决方案
3. 如何区分是布局问题还是采样时序问题？

**参考答案：**
1. 来源：(a) 功率回路辐射EMI耦合到采样走线；(b) 地线压降（功率地和信号地未分离）；(c) ADC采样时刻靠近PWM开关瞬间（未充分等待）
2. 完整解决方案：
   - PCB层面：功率回路面积最小化、采样差分走线、功率地与信号地单点连接、去耦电容靠近IC
   - 电路层面：RC滤波器、共模电感
   - 时序层面：ADC采样点远离PWM开关瞬间（延迟1-2μs）
   - 软件层面：数字低通滤波、中值滤波（去除尖峰）
3. 区分方法：
   - 改变ADC采样时刻（延迟1μs vs 5μs），若尖峰消失 → 时序问题
   - 若改变采样时刻无效 → 布局/EMI问题
   - 用示波器探头直接在采样电阻两端测量 → 若无尖峰则问题在走线耦合
   - 用示波器测量信号地和功率地的电位差 → 若有压降则地设计有问题

## 四、软硬件关联题（★★★★☆）

### 题目8：温升对控制参数的影响与补偿
**问题：** 一台PMSM在25°C和100°C下的参数变化：Rs从0.5Ω增至0.7Ω，ψf从0.1Wb降至0.092Wb，L从1mH降至0.95mH。请分析：
1. 各参数变化对电流环PI参数的影响
2. 对转矩常数和弱磁控制的影响
3. 设计软件补偿策略

**参考答案：**
1. PI参数影响：
   - Rs增大40% → Ki应增大40%（Ki = Rs × ωc）
   - L减小5% → Kp应减小5%（Kp = L × ωc）
   - 若不补偿：Ki偏小 → 稳态误差增大；Kp偏大 → 电流环可能震荡
2. 转矩常数：Kt = 1.5 × P × ψf，ψf减小8% → Kt减小8% → 同样Iq输出转矩减小8%
   - 弱磁影响：ψf减小 → 反电动势降低 → 弱磁进入点推迟 → 同一转速下弱磁电流需求减少
3. 软件补偿策略：
   ```c
   void Parameter_TempCompensation(float T_winding, float T_magnet) {
       // 电阻温度补偿
       float Rs_comp = Rs_25C * (1 + 0.00393 * (T_winding - 25));
       // 磁链温度补偿（-0.1%/°C典型）
       float psi_comp = psi_25C * (1 - 0.001 * (T_magnet - 25));
       // 电感温度补偿（-0.05%/°C近似）
       float L_comp = L_25C * (1 - 0.0005 * (T_winding - 25));
       
       // 更新PI参数
       Kp = L_comp * omega_c;
       Ki = Rs_comp * omega_c;
       
       // 更新转矩常数和观测器
       Kt = 1.5 * P * psi_comp;
   }
   ```

### 题目9：功率回路面积对EMI的定量影响
**问题：** 一个三相逆变器，优化前功率回路面积50cm²，优化后5cm²。请分析：
1. 回路面积减小对寄生电感的影响
2. 对开关尖峰电压的影响（di/dt = 5A/ns）
3. 对EMI辐射的影响
4. 对电流采样噪声的影响

**参考答案：**
1. 寄生电感近似公式：L ≈ μ0 × l × (ln(2l/w) + 0.5)。面积减小10倍 → 回路周长减小约√10≈3.16倍 → 寄生电感减小约3倍。典型值：50cm²回路Lpar ≈ 50nH，5cm²回路Lpar ≈ 15nH
2. 开关尖峰：Vspike = Lpar × di/dt。优化前：Vspike = 50nH × 5A/ns = 250V；优化后：Vspike = 15nH × 5A/ns = 75V。尖峰电压降低70%！
3. EMI辐射：辐射场强 ∝ 回路面积 × 电流变化率。面积减小10倍 → 辐射场强降低20dB（10倍）。实测数据：优化前EMI@150kHz = 95dBμV（超标），优化后 = 65dBμV（合格）
4. 电流采样噪声：辐射EMI降低 → 采样走线耦合噪声降低 → 采样噪声从200mA降至15mA → 转矩纹波从5%降至0.5%

### 题目10：热/EMC/PCB协同设计综合题
**问题：** 一个2kW电机控制器，6个TO-247 MOSFET，每个损耗20W，fsw = 20kHz，Vdc = 48V，Ta = 40°C。需要同时满足热设计和EMC要求。请给出：
1. 散热方案（散热器选型、安装方式）
2. PCB层叠设计（4层板）
3. 功率区布局方案
4. 热/EMC/PCB的冲突点和解决策略

**参考答案：**
1. 散热方案：
   - 6个MOSFET安装在同一铝型材散热器上
   - Rth_JA_max = (130-40)/20 = 4.5°C/W
   - Rth_SA_max = 4.5 - 0.7 - 0.2 = 3.6°C/W
   - 选择Rth_SA = 1.5°C/W的散热器（考虑6个器件热耦合）
   - 总损耗120W，散热器温升 = 120 × 1.5 = 180°C → 单个散热器不够！
   - 修正：每个MOSFET需要Rth_SA < 3.6°C/W，6个共用散热器需Rth_SA < 3.6/6 = 0.6°C/W → 需要大型散热器+风扇
2. 4层板层叠：
   - 顶层：功率器件、大电流走线（2oz铜厚）
   - 内层1：完整地平面（GND）
   - 内层2：电源平面（15V/5V/3.3V分割）
   - 底层：MCU、敏感信号、差分走线
3. 功率区布局：
   - 每相上下桥臂靠近放置
   - 去耦电容紧靠MOSFET（100nF+1μF）
   - 栅极驱动器距离MOSFET < 10mm
   - 电流采样差分走线远离功率走线
   - 功率地和信号地单点连接
4. 冲突与解决：
   - 冲突1：MOSFET靠近放置减小回路面积 vs 散热需要间距 → 优先保证回路小，散热靠散热器解决
   - 冲突2：散热片可能成为辐射天线 → 散热片接功率地
   - 冲突3：4层板增加成本 → 4层板是EMC和SI的最佳性价比，不可省
   - 冲突4：采样精度 vs 速度 → PWM同步采样+数字滤波折中
