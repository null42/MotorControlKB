### 题目1：Buck变换器电感计算
**问题：** 设计一个Buck变换器：输入48V，输出15V，负载2A，开关频率500kHz，要求电感电流纹波ΔIL = 0.5A（25%）。请计算：
1. 所需电感值
2. 输出电容值（要求纹波<50mV）
3. 电感饱和电流要求

**参考答案：**
1. L = (Vin - Vout) × Vout / (Vin × ΔIL × fsw) = (48-15) × 15 / (48 × 0.5 × 500k) = 495 / 12000000 = 41.25μH → 选择47μH
2. Cout = ΔIL / (8 × fsw × ΔVout) = 0.5 / (8 × 500k × 0.05) = 2.5μF → 选择10μF（留4倍裕度）
3. Isat > Iout + ΔIL/2 = 2 + 0.25 = 2.25A → 选择Isat > 3A

### 题目2：LDO功耗与温升计算
**问题：** 一个LDO稳压器：输入5V，输出3.3V，负载200mA，封装SOT-223（热阻RθJA = 100°C/W），环境温度50°C。请计算：
1. LDO功耗
2. 结温
3. 是否安全（最大结温125°C）
4. 如果不安全，给出改善方案

**参考答案：**
1. Pd = (Vin - Vout) × Iout = (5 - 3.3) × 0.2 = 0.34W
2. Tj = Ta + Pd × RθJA = 50 + 0.34 × 100 = 84°C
3. 84°C < 125°C → 安全，但裕度仅41°C，长期运行需注意
4. 改善方案（如果裕度不够）：
   - (a) 使用更低热阻封装（如TO-252，RθJA = 50°C/W）→ Tj = 50 + 17 = 67°C
   - (b) 增加散热铜箔面积
   - (c) 降低输入电压（如用4.2V输入）→ Pd = (4.2-3.3)×0.2 = 0.18W → Tj = 68°C
   - (d) 改用Buck变换器（效率>90%）

1. 四级保护的阈值和响应时间
2. 硬件实现方案
3. 软件实现方案

**参考答案：**

   | 级别 | 阈值 | 响应时间 | 动作 | 实现方式 |
   |------|------|---------|------|---------|
   | 电流限制 | 1.2×30 = 36A | 即时 | 限制电流给定 | 软件 |
   | 过流预警 | 1.5×30 = 45A | 即时 | 记录计数 | 软件 |

## 二、电路设计题（★★★☆☆）

### 题目4：48V系统电源架构设计
**问题：** 为一个48V/2kW电机控制器设计电源架构，需要提供：15V（驱动器，2W）、5V（编码器/CAN，1W）、3.3V（MCU/ADC，0.5W）。请给出：
1. 电源架构图
2. DC-DC和LDO芯片选型
3. 功率预算

**参考答案：**
1. 电源架构：
   ```
   48V DC ──┬──→ 逆变器功率级
            │
            └──→ Buck (LM5017) ──→ 15V ──┬──→ 栅极驱动器
                                         │
                                         └──→ LDO (AMS1117-5.0) ──→ 5V ──┬──→ 编码器/CAN
                                                                            │
                                                                            └──→ LDO (AMS1117-3.3) ──→ 3.3V ──→ MCU/ADC
   ```
2. 芯片选型：
   - Buck: LM5017 (100V输入, 600mA输出, 效率>85%) → 15V/133mA输出
   - LDO 5V: AMS1117-5.0 (15V输入, 5V输出, 1A) → 功耗(15-5)×0.2A = 2W
   - LDO 3.3V: AMS1117-3.3 (5V输入, 3.3V输出, 1A) → 功耗(5-3.3)×0.15A = 0.255W
3. 功率预算：
   - 15V负载：2W（驱动器）
   - 5V负载：1W（编码器/CAN）+ LDO损耗2W = 3W（从15V取）
   - 3.3V负载：0.5W + LDO损耗0.255W = 0.755W（从5V取）
   - 15V总负载：2 + 3 = 5W
   - 48V输入功率：5/0.85(Buck效率) ≈ 5.88W

### 题目5：防反接电路设计
**问题：** 为一个48V/50A系统设计防反接电路，要求压降<0.1V。请给出：
1. 方案选择及理由
2. 器件参数计算
3. 完整电路原理

**参考答案：**
1. 选择P-MOSFET防反接方案。理由：串联二极管压降0.7V太大（功耗35W），P-MOSFET压降<0.1V（功耗<5W），成本适中，可恢复
2. 器件参数：VDS > 1.5 × 48 = 72V → 选择100V器件；ID > 2 × 50 = 100A；RDS(on) < ΔV_max / I_max = 0.1V / 50A = 2mΩ
3. 电路原理：
   ```
   Vin+ ────┬──── 负载+
            │
            S(源极)
            │
        ┌───┴───┐
        │ P-MOS │  (如IRFP4468 × 2并联, RDS(on)≈1.3mΩ)
        │  Q1   │
        └───┬───┘
            │
            D(漏极)
            │
   Vin- ───┴──── 负载-
            │
           [Rgs] 10kΩ (G到S，确保无输入时MOSFET关断)
   ```
   工作原理：正确接法时体二极管先导通→VGS=-48V→MOSFET完全导通→压降=50A×1.3mΩ=0.065V ✓；反接时体二极管截止→MOSFET关断

## 三、故障诊断题（★★★☆☆）

### 题目6：3.3V电源电压偏低
**问题：** 系统上电后MCU不工作，测量3.3V电源只有2.1V。LDO输入5V正常。请分析：
1. 可能的原因（至少4个）
2. 诊断步骤
3. 解决方案

**参考答案：**
1. 可能原因：(a) LDO输出过载（负载电流过大）；(b) LDO损坏；(c) 输出电容短路或漏电；(d) 某个负载芯片短路（如MCU I/O口被静电击穿）；(e) PCB走线压降过大
2. 诊断步骤：
   - (a) 断开LDO输出所有负载，测量空载输出 → 若3.3V正常，说明过载
   - (b) 逐个恢复负载，找到导致电压下降的芯片
   - (c) 测量LDO输出电流 → 若远超200mA，确认过载
   - (d) 检查输出电容是否短路（万用表测电阻）
   - (e) 检查MCU是否异常发热（短路指示）
3. 解决方案：
   - (a) 若过载：增加LDO输出能力或减少负载
   - (b) 若芯片短路：更换损坏芯片
   - (c) 若电容问题：更换输出电容
   - (d) 预防措施：增加过流保护、上电时序控制

### 题目7：制动时过压保护频繁触发
**问题：** 一个变频器制动时频繁触发过压保护，但制动电阻已安装。测量发现制动电阻两端没有电压。请分析：
1. 可能的原因
2. 完整诊断流程
3. 解决方案

**参考答案：**
1. 可能原因：(a) 制动开关管未导通（驱动故障）；(b) 制动电阻开路（断线）；(c) 制动逻辑未使能（软件问题）；(d) 制动阈值设置过高；(e) 制动开关管驱动电路故障
2. 诊断流程：
   - (a) 测量制动电阻阻值 → 确认电阻完好
   - (b) 测量制动开关管栅极驱动信号 → 确认驱动是否正常
   - (c) 检查软件制动使能标志 → 确认制动逻辑是否激活
   - (d) 检查制动阈值设置 → 确认V_brake是否合理
   - (e) 测量开关管漏-源电压 → 确认开关管是否导通
3. 解决方案：
   - (a) 若驱动故障：检查驱动电路、栅极电阻、自举电路
   - (b) 若软件未使能：修改代码，在母线电压超过阈值时激活制动
   - (c) 若阈值过高：降低V_brake（如从750V降到700V）
   - (d) 若开关管损坏：更换并检查驱动电路

## 四、软硬件关联题（★★★★☆）

### 题目8：母线电压与弱磁控制的协调
**问题：** 一台PMSM参数：P = 4，ψf = 0.1Wb，母线电压48V（允许范围40V~60V）。请分析：
1. 不同母线电压下的基速
2. 欠压时对弱磁控制的影响
3. 软件如何协调母线电压变化与控制策略？

**参考答案：**
1. Vmax = Vdc / √3；Ke = P × ψf = 0.4 V·s/rad
   - 48V时：ωb = 48/(1.732×0.4) = 69.3 rad/s ≈ 662 rpm
   - 40V时：ωb = 40/(1.732×0.4) = 57.7 rad/s ≈ 551 rpm（基速降低16.7%）
   - 60V时：ωb = 60/(1.732×0.4) = 86.6 rad/s ≈ 827 rpm（基速提高25%）
2. 欠压影响：Vdc降低 → Vmax降低 → 弱磁提前进入 → 转矩能力下降 → 同一转速下需要更多弱磁电流 → 可用Iq减小 → 输出转矩降低
3. 软件协调策略：
   ```c
   // 根据母线电压动态调整
   float Vdc = ADC_ReadBusVoltage();
   float Vmax = Vdc / 1.732;
   float omega_base = Vmax / Ke;
   
   // 欠压降额
   if (Vdc < V_UV_warn) {
       I_limit = In * (Vdc - V_UVLO) / (V_UV_warn - V_UVLO);
   }
   
   // 弱磁控制使用实际Vdc
   if (V_ref > Vmax) {
       Id_ref = -(E - Vmax) / (omega_e * Ld);
   }
   ```

### 题目9：软硬件协同保护系统设计
**问题：** 设计一个完整的软硬件协同保护系统，要求：硬件保护响应<1μs，软件能区分过流和短路，过温降额策略，故障记录和恢复机制。请给出：
1. 保护架构图
2. 各级保护的阈值和响应
3. 故障代码定义
4. 恢复策略

**参考答案：**
1. 保护架构：
   ```
   硬件保护（最快、最粗糙）         软件保护（精确、灵活）
   ──────────────────────         ──────────────────────
   短路保护 3.0×In <1μs     ←→   电流限制 1.2×In 即时
   过流保护 2.0×In <10μs    ←→   过流预警 1.5×In 计数
   过压保护 硬件比较器       ←→   过压预警 软件检测
   欠压锁定 UVLO            ←→   欠压预警 降额运行
   过温保护 NTC+比较器       ←→   过温降额 线性降低
   ```
2. 各级保护（以In=20A为例）：
   - 电流限制24A（软件限幅）→ 过流预警30A（软件计数）→ 过流保护40A（软件+硬件）→ 短路保护60A（硬件<1μs）
3. 故障代码：
   ```c
   FAULT_OVERCURRENT_U = 0x01,  // U相过流
   FAULT_OVERCURRENT_V = 0x02,  // V相过流
   FAULT_SHORT_CIRCUIT  = 0x05, // 短路（硬件触发）
   FAULT_OVERVOLTAGE    = 0x10, // 过压
   FAULT_UNDERVOLTAGE   = 0x11, // 欠压
   FAULT_OVERTEMP_DRIVER= 0x21, // 驱动器过温
   FAULT_OVERTEMP_MOTOR = 0x20, // 电机过温
   ```
4. 恢复策略：
   - 致命故障（短路）：禁止自动恢复，需人工确认
   - 严重故障（过流/过压/过温）：延时5秒后自动尝试恢复，3次失败后锁定
   - 轻微故障（预警）：降额运行，故障消除后自动恢复

### 题目10：泵升电压与制动电阻设计
**问题：** 一个300V母线变频器，制动时再生功率5kW，制动时间2秒。请设计：
1. 制动电阻值和功率
2. 制动开关管选型
3. 制动控制逻辑
4. 如果制动电阻开路失效，系统如何保护？

**参考答案：**
1. 制动阈值Vbrake = 350V；Rbrake = Vbrake² / Pregen = 350² / 5000 = 24.5Ω → 选择25Ω；Pavg = 5000 × (2s/10s占空比) = 1000W → 选择1000W铝壳电阻（带散热）
2. 开关管：Imax = 350/25 = 14A；VDS > 500V → 选择500V/30A MOSFET（如FDPF3860T）或IGBT
3. 制动控制逻辑：
   ```c
   void Brake_Control(float Vdc) {
       if (Vdc > V_brake_on) {    // 如360V
           Brake_PWM_Enable();     // 导通制动开关管
           brake_active = true;
       } else if (Vdc < V_brake_off) {  // 如340V
           Brake_PWM_Disable();    // 关断制动开关管
           brake_active = false;
       }
       // 带迟滞防止频繁开关
   }
   ```
4. 制动电阻开路保护：
   - (a) 检测制动开关管导通时电流是否为零 → 判断开路
   - (b) 母线电压持续升高超过V_OV_trip → 过压保护停机
   - (c) 软件限制减速度，减少再生能量
   - (d) 故障记录：FAULT_BRAKE_RESISTOR_OPEN
