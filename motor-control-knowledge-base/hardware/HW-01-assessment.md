## 一、参数计算题（★★☆☆☆）

### 题目1：电感参数与电流环PI参数计算
**问题：** 已知一台表贴式PMSM参数：Ld = Lq = 0.8mH，Rs = 0.15Ω，极对数P = 4，磁链ψf = 0.12Wb。电流环设计带宽ωc = 1500 rad/s，请计算：
1. d轴和q轴电流环的Kp和Ki参数
2. 若实际电感为1.0mH（比标称值偏大25%），原PI参数会导致什么问题？

**参考答案：**
1. Kp_d = Ld × ωc = 0.8mH × 1500 = 1.2；Kp_q = Lq × ωc = 1.2；Ki = Rs × ωc = 0.15 × 1500 = 225
2. 实际电感偏大 → 实际需要Kp = 1.0mH × 1500 = 1.5 → 原Kp = 1.2偏小 → 电流环带宽不足 → 响应变慢、相位裕度降低 → 可能出现低频震荡

### 题目2：转矩常数与额定转矩计算
**问题：** 一台内置式PMSM参数：极对数P = 3，磁链ψf = 0.08Wb，Ld = 1.5mH，Lq = 3.0mH，额定Iq = 15A。请计算：
1. 仅考虑永磁转矩时的额定转矩
2. 在Id = -5A时，磁阻转矩贡献了多少额外转矩？
3. 总转矩和等效转矩常数

**参考答案：**
1. Te_pm = 1.5 × P × ψf × Iq = 1.5 × 3 × 0.08 × 15 = 5.4 N·m
2. Te_reluctance = 1.5 × P × (Ld - Lq) × Id × Iq = 1.5 × 3 × (1.5 - 3.0) × (-5) × 15 = 1.5 × 3 × (-1.5) × (-75) = 506.25 × 10^-3 = 0.506 N·m
3. Te_total = 5.4 + 0.506 = 5.906 N·m；Kt_eq = Te_total / Iq = 5.906 / 15 = 0.394 N·m/A（注意：磁阻转矩使总转矩增大，这是IPM电机MTPA控制的基础）

### 题目3：电阻温漂计算
**问题：** 一台PMSM在20°C时定子电阻Rs = 0.5Ω，铜的温度系数α = 0.00393/°C。电机运行后绕组温度升至120°C，请计算：
1. 120°C时的定子电阻
2. 电阻变化对铜损的影响（额定电流10A）
3. 对电流环Ki参数的影响

**参考答案：**
1. Rs(120°C) = 0.5 × [1 + 0.00393 × (120 - 20)] = 0.5 × 1.393 = 0.697Ω
2. 20°C铜损：Pcu = 3 × 10² × 0.5 = 150W；120°C铜损：Pcu = 3 × 10² × 0.697 = 209W；增加39.3%
3. Ki = Rs × ωc，Rs增大39.3% → Ki应相应增大39.3%，否则电流环积分增益不足，稳态误差增大

## 二、概念辨析题（★★☆☆☆）

### 题目4：SPM与IPM电机特性对比
**问题：** 以下关于表贴式PMSM（SPM）和内置式PMSM（IPM）的描述，哪些正确？请逐条判断并说明理由。
1. SPM的Ld ≈ Lq，因此不存在磁阻转矩
2. IPM的Ld < Lq，因为d轴方向永磁体磁阻大
3. SPM不适合弱磁控制，因为无法产生有效的弱磁电流
4. IPM的MTPA控制中Id为负值

**参考答案：**
1. ✅ 正确。SPM永磁体贴在转子表面，d轴和q轴磁路对称，Ld ≈ Lq，磁阻转矩项(Ld-Lq)×Id×Iq ≈ 0
2. ✅ 正确。d轴方向经过永磁体，永磁体磁导率≈空气，磁阻大→电感小；q轴方向经过铁芯，磁导率高→电感大
3. ❌ 不正确。SPM也可以弱磁控制，通过Id<0减小d轴磁链，只是弱磁范围比IPM小（因为Ld小，需要更大的弱磁电流）
4. ✅ 正确。MTPA要求Id为负值，利用(Ld-Lq)为负的特性，负Id与正Iq组合产生正的磁阻转矩

## 三、电路设计题（★★★☆☆）

### 题目5：电机参数测量方案设计
**问题：** 你拿到一台未知参数的PMSM，需要测量Ld、Lq、ψf、Rs四个关键参数。请设计完整的测量方案，包括：
1. 每个参数的测量方法
2. 所需仪器设备
3. 测量步骤和计算公式
4. 注意事项

**参考答案：**

| 参数 | 方法 | 仪器 | 公式 | 注意事项 |
|------|------|------|------|---------|
| Rs | 直流法 | 直流电源、电流表、电压表 | Rs = Vdc / Idc | 转子静止，避免旋转电动势干扰；多测取平均 |
| Ld | 高频注入法（锁定转子d轴方向） | 交流电源、示波器 | Ld = Vd_hf / (2πf × Id_hf) | 确保对齐d轴，频率选择1-10kHz |
| Lq | 高频注入法（锁定转子q轴方向） | 同上 | Lq = Vq_hf / (2πf × Iq_hf) | 旋转90°电角度后测量 |
| ψf | 空载反电动势法 | 示波器、转速计 | ψf = E_phase / (P × ωm) | 电机匀速旋转，测量相电压峰值 |

补充方法：
- Ld/Lq也可用LCR电桥离线测量（锁定转子不同位置）
- ψf也可用堵转法：ψf = T_measured / (1.5 × P × Iq)
- 极对数P可用反电动势法：P = 60 × fe / n

### 题目6：弱磁控制电流计算
**问题：** 一台IPM电机参数：P = 4，ψf = 0.15Wb，Ld = 1.2mH，Lq = 2.4mH，母线电压Vdc = 300V，电流极限Imax = 30A。请计算：
1. 基速（恒转矩区最高转速）
2. 完全弱磁时的Id电流
3. 2倍基速时的弱磁电流（近似）

**参考答案：**
1. Ke = P × ψf = 4 × 0.15 = 0.6 V·s/rad；Vmax = Vdc / √3 = 173.2V；基速ωb = Vmax / Ke = 173.2 / 0.6 = 288.7 rad/s ≈ 2757 rpm
2. Id_fw = -ψf / Ld = -0.15 / 0.0012 = -125A → 超出电流极限30A，说明该电机无法实现完全弱磁
3. 2倍基速时ωe = 2 × P × ωb = 2 × 4 × 288.7 = 2309.6 rad/s；弱磁约束：√(Vd² + Vq²) ≤ Vmax，近似Id ≈ -(ψf - Vmax/ωe) / Ld = -(0.15 - 173.2/2309.6) / 0.0012 = -(0.15 - 0.075) / 0.0012 = -62.5A → 仍超出电流极限，说明该电机2倍基速不可达。需重新评估电机设计或提高母线电压。

## 四、故障诊断题（★★★★☆）

### 题目7：电流环震荡——电感参数不准
**问题：** 一台PMSM采用FOC控制，电流环出现约200Hz的低频震荡。已知配置参数L = 0.5mH，Rs = 0.1Ω，Kp = 0.5，Ki = 100，ωc = 1000 rad/s。实测发现电感为0.8mH。请分析：
1. 震荡的根本原因
2. 修正后的PI参数
3. 如果无法修改电感参数，还有什么替代方案？

**参考答案：**
1. 电感实际值0.8mH > 配置值0.5mH → Kp偏小（应为0.8，实际0.5）→ 电流环实际带宽 = Kp/L = 0.5/0.8mH = 625 rad/s → 远低于设计值1000 rad/s → 相位裕度不足 → 震荡
2. 修正：Kp = L × ωc = 0.8mH × 1000 = 0.8；Ki = Rs × ωc = 0.1 × 1000 = 100（Ki不变）
3. 替代方案：(a) 降低电流环带宽至ωc = 600 rad/s，Kp = 0.8 × 0.6 = 0.48 ≈ 原0.5，震荡消除但响应变慢；(b) 在线电感辨识（高频注入法或RLS），实时更新PI参数；(c) 增加电流环前馈补偿Vff = ωe × (Lq × Iq + ψf)

### 题目8：极对数配置错误诊断
**问题：** 一台外转子PMSM无人机电机，铭牌标注"14极"，工程师配置极对数P = 7，但FOC控制时电机剧烈抖动无法正常运转。手动旋转电机一圈，编码器电角度变化了720°。请分析：
1. 极对数配置是否正确？
2. 正确的极对数是多少？
3. 极对数错误为什么会导致电机抖动？

**参考答案：**
1. 不正确。手动旋转一圈（360°机械角度），电角度变化720°，说明P = 720°/360° = 2
2. 正确极对数P = 2（即4极电机）。铭牌"14极"可能是14槽的误标，或者工程师将极数与极对数混淆
3. 极对数错误 → Park变换角度θe = P × θm计算错误 → 实际P=2但配置P=7 → 电角度被放大3.5倍 → Park变换后Id/Iq完全解耦失败 → d轴电流泄漏到q轴，q轴电流泄漏到d轴 → 电流环震荡 → 电机抖动

## 五、软硬件关联题（★★★★☆）

### 题目9：电机参数对FOC控制策略的影响
**问题：** 一台电机参数如下：P = 4，ψf = 0.1Wb，Ld = 0.8mH，Lq = 2.0mH，Rs = 0.2Ω，Vdc = 48V，In = 20A。请从软硬件协同角度分析：
1. 该电机应采用Id=0控制还是MTPA控制？计算两种策略在Iq=20A时的转矩差异
2. 弱磁控制的进入转速是多少？弱磁时对软件算法有什么特殊要求？
3. 温度从25°C升至100°C时，哪些参数会变化？软件应如何补偿？

**参考答案：**
1. Ld ≠ Lq（0.8 vs 2.0），是IPM电机，应采用MTPA控制。
   - Id=0时：Te = 1.5 × 4 × 0.1 × 20 = 12 N·m
   - MTPA时需优化Id，使dT/dId = 0，近似Id_opt ≈ -ψf/(2(Lq-Ld)) + ... ≈ -0.1/(2×1.2) ≈ -0.042A（此例中磁阻转矩贡献小，但Iq更大时差异显著）。若Id=-5A：Te = 1.5×4×[0.1×20 + (0.8-2.0)×(-5)×20] = 6×[2+1.2] = 19.2 N·m，比Id=0的12 N·m增大60%
2. 基速ωb = Vdc/(√3 × P × ψf) = 48/(1.732 × 4 × 0.1) = 69.3 rad/s ≈ 662 rpm。弱磁要求：(a) Id_ref从0向负值调节；(b) 电压反馈判断是否进入弱磁区；(c) 弱磁电流限幅不超过Imax；(d) 防止Id过负导致永磁体不可逆退磁
3. 温升影响：(a) Rs增大→Ki需增大；(b) ψf减小（约-0.1%/°C）→转矩常数减小→观测器参数需更新；(c) Ld/Lq略减小→PI参数微调。软件补偿：在线参数辨识+查表补偿，或基于温度传感器的参数前馈修正

### 题目10：转矩常数定义混淆分析
**问题：** 某工程师从电机数据表读取Kt = 1.0 N·m/A，在FOC代码中直接使用Te = Kt × Iq计算转矩，但实测转矩偏差约18%。已知电机参数：P = 4，ψf = 0.082Wb。请分析：
1. 偏差的根本原因
2. 正确的FOC转矩常数是多少？
3. 如何避免此类问题？

**参考答案：**
1. 数据表Kt通常基于峰值相电流定义，而FOC中Iq是旋转坐标系下的直流分量。转换关系：Kt_FOC = 1.5 × P × ψf = 1.5 × 4 × 0.082 = 0.492 N·m/A；Kt_datasheet / Kt_FOC = 1.0 / 0.492 = 2.03 ≈ √(3) × 某系数，说明数据表Kt基于线电流RMS值或峰值相电流定义，与FOC定义不一致
2. Kt_FOC = 1.5 × P × ψf = 0.492 N·m/A。验证：Iq = 10A时，Te = 0.492 × 10 = 4.92 N·m；若用数据表值：Te = 1.0 × 10 = 10 N·m（偏差103%！）。实际偏差18%说明工程师可能做了部分修正但不完整
3. 避免方法：(a) 不直接使用数据表Kt，而是从ψf和P计算；(b) 实测转矩常数：施加已知Iq，测量输出转矩；(c) 明确Kt的定义基准（峰值相电流/RMS/FOC-Iq）；(d) 在代码注释中标注Kt的来源和定义
