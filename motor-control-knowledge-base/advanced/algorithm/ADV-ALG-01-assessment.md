## 一、基础概念题

### 题目1：电流环带宽与PI参数的关系（★★☆☆☆）

**问题：** 使用零极点对消法设计电流环PI参数时，电流环闭环带宽 $\omega_{cl}$ 与 $K_p$、$K_i$ 的关系是什么？如果将带宽提高一倍，$K_p$ 和 $K_i$ 分别如何变化？

A. $\omega_{cl} = K_p / L_s$；$K_p$ 翻倍，$K_i$ 翻倍
B. $\omega_{cl} = K_i / K_p$；$K_p$ 翻倍，$K_i$ 不变
C. $\omega_{cl} = K_p \times L_s$；$K_p$ 减半，$K_i$ 减半
D. $\omega_{cl} = K_p / R_s$；$K_p$ 翻倍，$K_i$ 翻倍

**正确答案：** A

**详细解析：**
零极点对消法下，PI零点与被控对象极点对消（$K_i/K_p = R_s/L_s$），闭环传递函数简化为：

$$G_{cl}(s) = \frac{\omega_{cl}}{s + \omega_{cl}}$$

其中 $\omega_{cl} = K_p / L_s$。

从带宽反推PI参数：
- $K_p = \omega_{cl} \times L_s$
- $K_i = \omega_{cl} \times R_s$

带宽提高一倍：$K_p' = 2\omega_{cl} \times L_s = 2K_p$，$K_i' = 2\omega_{cl} \times R_s = 2K_i$，两者都翻倍。

注意：零极点对消条件 $K_i/K_p = R_s/L_s$ 在带宽变化时始终成立，因为 $K_i'/K_p' = 2K_i/(2K_p) = K_i/K_p = R_s/L_s$。

选项B的关系 $K_i/K_p$ 是PI零点频率，不是闭环带宽；选项C的公式方向错误；选项D的分母应为电感而非电阻。

### 题目2：三环带宽分配原则（★★☆☆☆）

**问题：** 电机控制系统中电流环、速度环、位置环的带宽分配原则是什么？以下哪个配置是合理的？

A. 电流环500Hz，速度环200Hz，位置环100Hz
B. 电流环2kHz，速度环200Hz，位置环20Hz
C. 电流环1kHz，速度环800Hz，位置环600Hz
D. 电流环3kHz，速度环2kHz，位置环1kHz

**正确答案：** B

**详细解析：**
三环带宽分配原则：$f_{BW,current} : f_{BW,speed} : f_{BW,position} \approx 100 : 10 : 1$

每级之间5~10倍的带宽衰减，保证：
1. 内环对外环的指令跟踪足够快，内环可近似为一阶惯性环节
2. 内环的动态特性不会与外环耦合
3. 系统稳定性有足够裕度

选项B：2kHz:200Hz:20Hz = 100:10:1，完美符合。

选项A：500Hz:200Hz:100Hz，电流环与速度环仅2.5倍，速度环与位置环仅2倍，带宽过于接近，内外环动态耦合严重。

选项C：1kHz:800Hz:600Hz，三环带宽几乎相同，系统必然不稳定。

选项D：3kHz:2kHz:1kHz，速度环与电流环仅1.5倍差距，电流环简化模型失效。

### 题目3：位置环控制器选择（★★☆☆☆）

**问题：** 位置环通常只使用P控制器而不加I，以下哪个**不是**正确原因？

A. 位置环被控对象已包含两个积分环节（速度→位置 + 速度环PI中的积分），系统型别足够，无需额外积分
B. 加I会导致位置超调，而位置伺服应用通常不允许超调
C. P控制器结构简单，调试方便，且不会引入积分饱和问题
D. P控制器无法消除稳态误差，必须加I才能实现零位置误差

**正确答案：** D

**详细解析：**
选项D是错误的——位置环使用P控制器**可以**实现零稳态位置误差。原因：位置环的开环传递函数中已包含积分环节（速度→位置的纯积分 + 速度环PI的积分），系统对阶跃输入的稳态误差为零。具体来说，位置环开环传递函数包含至少一个积分环节（$1/s$），对于阶跃给定（位置阶跃），稳态误差为零。

选项A正确——系统型别已足够（至少I型系统），P控制即可实现阶跃输入零稳态误差。
选项B正确——位置伺服（如CNC、机器人关节）通常不允许超调，加I会引入超调。
选项C正确——P控制器无积分项，不存在积分饱和和anti-windup问题。

### 题目4：电流环PI参数完整设计（★★★☆☆）

**问题：** 某SPMSM参数：$R_s = 0.5\Omega$，$L_s = 0.5\text{mH}$，$\psi_f = 0.0085\text{Wb}$，极对数 $p = 4$。PWM频率20kHz，最高转速3000 RPM。要求电流环带宽1500Hz，请计算PI参数并验证带宽约束。

A. $K_p = 4.71$，$K_i = 4712$；带宽满足约束（< f_PWM/10 = 2000Hz，> 3 x f_e_max = 600Hz）
B. $K_p = 0.471$，$K_i = 471$；带宽不满足约束
C. $K_p = 47.1$，$K_i = 47120$；带宽过高，不满足约束
D. $K_p = 4.71$，$K_i = 4712$；带宽不满足下限约束

**正确答案：** A

**详细解析：**
带宽选择：$f_{BW} = 1500\text{Hz}$，$\omega_{cl} = 2\pi \times 1500 = 9425\text{rad/s}$

PI参数计算：
$K_p = \omega_{cl} \times L_s = 9425 \times 0.5 \times 10^{-3} = 4.71$
$K_i = \omega_{cl} \times R_s = 9425 \times 0.5 = 4712$

验证零极点对消：$K_i/K_p = 4712/4.71 = 1000$，$R_s/L_s = 0.5/0.0005 = 1000$，对消条件成立。

带宽约束验证：
- 上限约束：$f_{BW} < f_{PWM}/10 = 2000\text{Hz}$，1500Hz < 2000Hz，满足
- 下限约束：$f_{e,max} = 3000/60 \times 4 = 200\text{Hz}$，$f_{BW} > 3 \times f_{e,max} = 600\text{Hz}$，1500Hz > 600Hz，满足

离散化（$T_s = 50\mu s$）：
$K_{p,disc} = 4.71$
$K_{i,disc} = 4712 \times 50 \times 10^{-6} = 0.2356$

### 题目5：速度环PI参数与对称最优法（★★★☆☆）

**问题：** 续上题，电机转动惯量 $J = 0.0001\text{kg}\cdot\text{m}^2$，转矩常数 $K_t = 0.051\text{N}\cdot\text{m/A}$，电流环带宽1500Hz。使用对称最优法设计速度环，选择速度环带宽150Hz，计算速度环PI参数。

A. $K_{p,s} = 1.847$，$K_{i,s} = 1740$
B. $K_{p,s} = 18.47$，$K_{i,s} = 174$
C. $K_{p,s} = 0.185$，$K_{i,s} = 17.4$
D. $K_{p,s} = 1.847$，$K_{i,s} = 174$

**正确答案：** A

**详细解析：**
速度环带宽：$f_{BW,s} = 150\text{Hz}$，$\omega_{BW,s} = 2\pi \times 150 = 942\text{rad/s}$

PI参数计算（简化方法）：
$K_{p,s} = J \times \omega_{BW,s} / K_t = 0.0001 \times 942 / 0.051 = 1.847$
$K_{i,s} = J \times \omega_{BW,s}^2 / K_t = 0.0001 \times 942^2 / 0.051 = 0.0001 \times 887364 / 0.051 = 1740$

验证带宽比：$f_{BW,current}/f_{BW,speed} = 1500/150 = 10$，满足5~10倍要求。

对称最优法验证：PI零点频率 $\omega_{z,s} = K_{i,s}/K_{p,s} = 1740/1.847 = 942\text{rad/s} = \omega_{BW,s}$，穿越频率 $\omega_c = \sqrt{\omega_{z,s} \times \omega_{cl}} = \sqrt{942 \times 9425} = 2979\text{rad/s}$，约为电流环极点的1/3，相角裕度合理。

离散化（速度环周期 $T_{s,spd} = 200\mu s$）：
$K_{i,s,disc} = 1740 \times 200 \times 10^{-6} = 0.348$

### 题目6：低通滤波器对相角裕度的影响（★★★☆☆）

**问题：** 电流环带宽1500Hz，在电流反馈通道添加一阶IIR低通滤波器。如果滤波器截止频率设为1500Hz（等于电流环带宽），在穿越频率处引入的相位延迟是多少？对相角裕度的影响如何？

A. -45度，相角裕度减少45度，系统很可能不稳定
B. -26.6度，相角裕度减少26.6度，需谨慎
C. -90度，相角裕度减少90度，系统必然不稳定
D. -18.4度，相角裕度减少18.4度，可接受

**正确答案：** A

**详细解析：**
一阶低通滤波器在截止频率处的相位延迟为-45度。当滤波器截止频率等于穿越频率（电流环带宽）时，在穿越频率处恰好引入-45度相位延迟。

相角裕度影响：$PM_{actual} = PM_{ideal} - 45度$

理想情况下（无滤波器、考虑1.5个采样周期延迟），电流环相角裕度约49.5度。减去45度后仅剩4.5度，远低于45度的良好设计标准，系统很可能不稳定（振荡或接近振荡边缘）。

这就是为什么滤波器截止频率必须满足 $f_{c,filter} \geq (2 \sim 3) \times f_{BW}$：
- $f_c = 2 \times f_{BW}$：相位延迟-26.6度，PM从49.5度降至22.9度，勉强但偏小
- $f_c = 3 \times f_{BW}$：相位延迟-18.4度，PM从49.5度降至31.1度，可接受
- $f_c = 5 \times f_{BW}$：相位延迟-11.3度，PM从49.5度降至38.2度，良好

### 题目7：Bode图分析——穿越频率与相角裕度（★★★☆☆）

**问题：** 某电流环开环传递函数 $G_{ol}(s) = K_p/(L_s \times s)$（零极点对消后），考虑1.5个采样周期延迟 $T_d = 75\mu s$。当 $K_p/L_s = 9425\text{rad/s}$（对应带宽1500Hz）时，穿越频率处的相角裕度是多少？

A. 约90度
B. 约49.5度
C. 约30度
D. 约60度

**正确答案：** B

**详细解析：**
开环传递函数（含延迟）：
$$G_{ol}(j\omega) = \frac{K_p/L_s}{j\omega} \times e^{-j\omega T_d}$$

穿越频率：$\omega_c = K_p/L_s = 9425\text{rad/s}$（即 $f_c = 1500\text{Hz}$）

在穿越频率处的相位：
- 积分环节贡献：-90度
- 延迟环节贡献：$-180度 \times 1.5 \times f_c / f_{PWM} = -180 \times 1.5 \times 1500/20000 = -40.5度$

总相位：$-90 - 40.5 = -130.5度$

相角裕度：$PM = 180 - 130.5 = 49.5度$

49.5度 > 45度，设计合理。如果带宽提高到2000Hz（f_PWM/10），延迟相位增加：$-180 \times 1.5 \times 2000/20000 = -54度$，总相位-144度，PM=36度，低于45度标准，系统稳定性裕度不足。这验证了 $f_{BW} < f_{PWM}/10$ 的工程经验法则。

### 题目8：滤波器+控制器联合设计（★★★★☆）

**问题：** 电流环被控对象含一阶低通滤波器：$G_{plant}(s) = \frac{1}{Ls+R} \times \frac{\omega_f}{s+\omega_f}$，其中 $\omega_f = 2\pi \times 4500\text{rad/s}$（滤波器截止频率4500Hz）。仍用零极点对消消去 $s=-R/L$ 极点后，闭环传递函数变为二阶系统。若要求阻尼比 $\zeta = 0.707$，$K_p$ 应为多少？与无滤波器时的 $K_p$ 相比如何？

A. $K_p = \omega_f L / 2 = 7.07$，比无滤波器时（4.71）更大
B. $K_p = \omega_f L = 14.14$，比无滤波器时更大
C. $K_p = 4.71$，与无滤波器时相同
D. $K_p = \omega_f L / 4 = 3.54$，比无滤波器时更小

**正确答案：** A

**详细解析：**
零极点对消后，开环传递函数：
$$G_{ol}(s) = \frac{K_p \omega_f / L}{s(s + \omega_f)}$$

闭环传递函数：
$$G_{cl}(s) = \frac{K_p \omega_f / L}{s^2 + \omega_f s + K_p \omega_f / L}$$

与标准二阶系统 $s^2 + 2\zeta\omega_n s + \omega_n^2$ 对比：
- $\omega_n = \sqrt{K_p \omega_f / L}$
- $2\zeta\omega_n = \omega_f$

取 $\zeta = 0.707$：
$$2 \times 0.707 \times \sqrt{K_p \omega_f / L} = \omega_f$$
$$\sqrt{K_p \omega_f / L} = \omega_f / 1.414$$
$$K_p \omega_f / L = \omega_f^2 / 2$$
$$K_p = \omega_f L / 2$$

代入数值：$\omega_f = 2\pi \times 4500 = 28274\text{rad/s}$，$L = 0.5\text{mH}$

$$K_p = 28274 \times 0.0005 / 2 = 7.07$$

无滤波器时 $K_p = 4.71$，联合设计后Kp更大（7.07 vs 4.71），因为需要"克服"滤波器引入的额外极点带来的相位延迟。代价是闭环带宽降低：$\omega_n = \omega_f / 1.414 = 19990\text{rad/s}$，对应约3180Hz——但这是自然频率，实际闭环带宽（-3dB点）约为 $\omega_n \times \sqrt{1-2\zeta^2+\sqrt{2-4\zeta^2+4\zeta^4}} \approx 0.99\omega_n$，约3148Hz，看似比1500Hz高很多，但这是因为联合设计改变了带宽定义——实际穿越频率和相角裕度才是关键指标。

### 题目9：速度环带宽与机械共振（★★★★☆）

**问题：** 某伺服电机系统通过联轴器连接负载，经测试发现机械系统最低共振频率为300Hz。速度环带宽设计为200Hz时，系统在特定转速下出现持续振荡。以下哪个分析和解决方案最合理？

A. 速度环带宽200Hz低于共振频率300Hz，不应激发共振，振荡可能是电流环问题
B. 速度环带宽200Hz虽低于共振频率，但闭环谐振峰值可能在共振频率处仍有增益，应将速度环带宽降至共振频率的1/3（100Hz）或加notch滤波器
C. 应提高速度环带宽到400Hz以"压制"共振
D. 机械共振无法通过软件解决，必须更换联轴器

**正确答案：** B

**详细解析：**
速度环带宽200Hz低于共振频率300Hz，但并不意味着安全。原因：

1. 闭环系统在带宽频率附近存在谐振峰值，即使带宽200Hz，在300Hz处闭环增益可能仍大于0dB（取决于阻尼比和谐振峰值大小）。

2. 速度环的闭环幅频特性在穿越频率附近有峰值，如果这个峰值延伸到300Hz处仍有正增益，机械共振就会被激发。

3. 经验法则：速度环带宽不应超过最低机械共振频率的1/3，即 $f_{BW,speed} < f_{res}/3 = 100\text{Hz}$。

解决方案：
- **方案1（降带宽）：** 将速度环带宽降至100Hz，确保300Hz处的闭环增益足够低（约-10dB），但牺牲动态响应。
- **方案2（notch滤波器，推荐）：** 在速度环中串联陷波滤波器，中心频率设为300Hz，陷波深度-20dB以上，带宽降至100Hz。这样可以在保留较高带宽的同时抑制共振频率处的增益。
- **方案3（前馈补偿）：** 不改变闭环带宽，通过速度和加速度前馈提高跟踪性能，闭环带宽可以适当降低。

选项A错误——200Hz带宽的闭环响应在300Hz处可能仍有正增益；选项C错误——提高带宽只会使共振更严重；选项D过于绝对——软件手段（notch滤波器、降带宽）可以有效抑制共振。

### 题目10：前馈补偿对位置跟踪性能的影响（★★★★☆）

**问题：** 某位置伺服系统，位置环带宽15Hz，速度环带宽150Hz。在跟踪10Hz正弦位置轨迹时，无前馈的位置跟踪误差为 $e_{no-ff}$。加入速度前馈后，跟踪误差降低到原来的多少？如果同时加入加速度前馈，误差进一步降低到多少？

A. 速度前馈后误差降至约1/10；加速度前馈后降至约1/100
B. 速度前馈后误差降至约1/2；加速度前馈后降至约1/4
C. 速度前馈后误差降至约1/3；加速度前馈后降至约1/10
D. 速度前馈后误差降至约1/5；加速度前馈后降至约1/25

**正确答案：** A

**详细解析：**
位置跟踪误差的来源分析：

**无前馈时：** 位置环P控制器输出 $= K_{p,pos} \times e_{pos}$，速度环需要速度指令 $= d\theta^*/dt$。由于位置环带宽仅15Hz，对于10Hz正弦轨迹，闭环增益约为：

$$|G_{cl}(j2\pi \times 10)| \approx \frac{1}{\sqrt{1 + (10/15)^2}} \approx 0.83$$

相位延迟约 $\arctan(10/15) \approx 33.7度$，跟踪误差较大。

**速度前馈：** $\omega_{ff} = d\theta^*/dt$，直接提供速度环所需的大部分指令，位置环P控制器只需补偿误差。速度前馈补偿了位置环的"速度需求"，跟踪误差主要取决于速度环的跟踪精度。速度环带宽150Hz，对10Hz信号的跟踪误差很小（约1/15的相对误差），因此位置跟踪误差降低约一个数量级（1/10）。

**加速度前馈：** $i_{ff} = (J/K_t) \times d^2\theta^*/dt^2$，进一步补偿速度环的"加速度需求"。加入加速度前馈后，速度环的跟踪误差也大幅降低，位置跟踪误差再降低约一个数量级（1/100）。

定量分析：无前馈时跟踪误差 $e \propto (f_{traj}/f_{BW,pos})^2$；速度前馈后 $e \propto (f_{traj}/f_{BW,speed})^2$；加速度前馈后 $e \propto (f_{traj}/f_{BW,current})^2$。

比例关系：$(f_{BW,pos}/f_{BW,speed})^2 = (15/150)^2 = 0.01 = 1/100$（速度前馈相对无前馈的改善），$(f_{BW,speed}/f_{BW,current})^2 = (150/1500)^2 = 0.01 = 1/100$（加速度前馈相对速度前馈的改善）。

因此：速度前馈后误差降至约1/10（保守估计），加速度前馈后进一步降至约1/100。前馈不改变闭环带宽和稳定性，但大幅提高跟踪精度——这是轨迹跟踪应用的核心技术。
