## 第1题：电压椭圆的几何特征

**题目：** 在dq平面上，PMSM的电压约束随转速升高而缩小。关于电压椭圆的几何特征，以下说法正确的是？

A. 椭圆中心位于原点 $(0, 0)$，随转速升高椭圆等比例缩小
B. 椭圆中心位于 $(-\psi_f/L_d, 0)$，随转速升高椭圆等比例缩小
C. 椭圆中心位于 $(-\psi_f/L_d, 0)$，随转速升高椭圆中心向右移动
D. 椭圆中心位于 $(\psi_f/L_d, 0)$，随转速升高椭圆等比例缩小

**正确答案：** B

**详细解析：**

电压椭圆的标准方程为：

$$\frac{(I_d + \psi_f/L_d)^2}{(V_{max}/(\omega_e L_d))^2} + \frac{I_q^2}{(V_{max}/(\omega_e L_q))^2} \leq 1$$

- 椭圆中心位于 $I_d = -\psi_f/L_d$，$I_q = 0$，即短路电流点。该中心**不随转速变化**。
- d轴半轴长 $a = V_{max}/(\omega_e L_d)$，q轴半轴长 $b = V_{max}/(\omega_e L_q)$，均与 $\omega_e$ 成反比。
- 因此随转速升高，椭圆**等比例缩小**（中心不动，半轴等比缩短）。

选项A错误：中心不在原点。选项C错误：中心不随转速移动。选项D错误：中心横坐标应为负值。

## 第2题：SPMSM与IPMSM的MTPA轨迹

**题目：** 关于SPMSM和IPMSM的MTPA轨迹，以下说法正确的是？

A. SPMSM的MTPA轨迹是 $I_d = 0$，因为SPMSM没有磁阻转矩
B. IPMSM的MTPA轨迹是 $I_d = 0$，因为负 $I_d$ 会减小总转矩
C. SPMSM的MTPA轨迹在 $I_d < 0$ 区域，因为负 $I_d$ 可以削弱反电动势
D. IPMSM的MTPA轨迹在 $I_d > 0$ 区域，因为正 $I_d$ 可以增大磁阻转矩

**正确答案：** A

**详细解析：**

SPMSM的 $L_d = L_q$，磁阻转矩项 $(L_d - L_q)I_d I_q = 0$，转矩仅由永磁转矩 $T_e = \frac{3}{2}p\psi_f I_q$ 决定。负 $I_d$ 只会减小d轴磁链（弱磁效果），不会增加转矩，反而浪费电流。因此SPMSM的MTPA轨迹就是 $I_d = 0$。

IPMSM的 $L_d < L_q$，$(L_d - L_q) < 0$。当 $I_d < 0$ 且 $I_q > 0$ 时，$(L_d - L_q)I_d I_q > 0$，磁阻转矩为正，增加了总转矩。因此IPMSM的MTPA轨迹在 $I_d < 0$ 区域（选项B、D错误）。选项C对SPMSM的描述错误——虽然负 $I_d$ 确实削弱反电动势，但这不是MTPA的考量，MTPA关注的是在相同电流幅值下最大化转矩。

## 第3题：弱磁区域划分

**题目：** 在弱磁I区（$\omega_{base} < \omega < \omega_{c1}$），关于电流和电压约束状态，以下描述正确的是？

A. 电压和电流同时受限，$I_d$ 和 $I_q$ 都由约束决定
B. 电压受限，电流有余量，$I_d$ 由电压约束决定，$I_q$ 仍由转矩需求决定
C. 电压有余量，电流受限，$I_d$ 由MTPA决定
D. 电压和电流都有余量，$I_d$ 由MTPA决定

**正确答案：** B

**详细解析：**

弱磁I区的特征是电压椭圆缩小到与电流圆相交，但交点处电流幅值仍小于 $I_{max}$。此时：
- **电压受限**：工作点必须在电压椭圆内
- **电流有余量**：电流圆内部还有空间
- $I_d$ 由电压约束决定（比MTPA的 $I_d$ 更负，以满足电压约束）
- $I_q$ 仍由转矩需求决定（只要不超过电流极限）

选项A描述的是弱磁II区（$\omega > \omega_{c1}$），此时电压椭圆完全在电流圆内部，电压和电流同时受限。选项C描述的是恒转矩区。选项D在弱磁区不可能出现。

## 第4题：前馈法弱磁电流计算

**题目：** 某IPMSM参数：$L_d = 0.5\text{mH}$，$L_q = 1.2\text{mH}$，$\psi_f = 8.5\text{mWb}$，$V_{dc} = 48\text{V}$。在电角速度 $\omega_e = 2184\text{rad/s}$（约2倍基速）、$I_q = 10\text{A}$ 时，忽略电阻，用前馈法计算弱磁电流 $I_d^{fw}$，结果约为？

A. $-3.2\text{A}$
B. $-8.7\text{A}$
C. $-17.0\text{A}$
D. $+5.4\text{A}$

**正确答案：** B

**详细解析：**

前馈法弱磁电流公式（忽略电阻）：

$$I_d^{fw} = -\frac{\psi_f}{L_d} + \frac{1}{L_d}\sqrt{\frac{V_{max}^2}{\omega_e^2} - (L_q I_q)^2}$$

计算步骤：

1. $V_{max} = V_{dc}/\sqrt{3} = 48/1.732 = 27.71\text{V}$

2. $V_{max}^2/\omega_e^2 = 27.71^2/2184^2 = 767.84/4769856 = 1.610 \times 10^{-4}$

3. $(L_q I_q)^2 = (0.0012 \times 10)^2 = (0.012)^2 = 1.44 \times 10^{-4}$

4. 根号内：$1.610 \times 10^{-4} - 1.44 \times 10^{-4} = 1.70 \times 10^{-5}$

5. $\sqrt{1.70 \times 10^{-5}} = 0.004123$

6. $I_d^{fw} = -0.0085/0.0005 + 0.004123/0.0005 = -17 + 8.25 = -8.75\text{A}$

结果约为 $-8.7\text{A}$，对应选项B。

**验证电流极限：** $I_s = \sqrt{8.7^2 + 10^2} = \sqrt{75.7 + 100} = 13.3\text{A}$。若 $I_{max} = 20\text{A}$，则电流有余量，处于弱磁I区。

**关键判断：** 如果根号内为负（即 $V_{max}^2/\omega_e^2 < (L_q I_q)^2$），说明在该转速下即使 $I_d$ 降到最低也无法满足电压约束，必须减小 $I_q$（降低转矩）。选项C的 $-17.0\text{A}$ 对应 $-\psi_f/L_d$（短路电流点），是最大弱磁极限。

## 第5题：电压反馈法弱磁的PI参数设计

**题目：** 弱磁PI控制器的带宽应远低于电流环带宽，推荐比例为？

A. 弱磁带宽 = 电流环带宽的 1/2
B. 弱磁带宽 = 电流环带宽的 1/5 ~ 1/10
C. 弱磁带宽 = 电流环带宽的 1/50 ~ 1/100
D. 弱磁带宽可以等于电流环带宽

**正确答案：** B

**详细解析：**

弱磁PI控制器的带宽必须远低于电流环带宽，原因有二：

1. **避免弱磁与电流环耦合振荡**：弱磁回路通过改变 $I_d$ 影响电压幅值，如果弱磁响应太快，会与电流环的调节产生交互振荡。
2. **弱磁是慢过程**：转速变化本身是机械时间常数级别的（ms级），而电流环是电气时间常数级别的（us级），弱磁不需要那么快。

推荐弱磁回路带宽为电流环带宽的1/5~1/10。例如电流环带宽1500Hz，弱磁带宽应设为150~300Hz。

选项A的1/2太接近，容易耦合。选项C的1/50~1/100太慢，弱磁响应迟缓，高速时可能来不及进入弱磁。选项D会导致严重振荡。

## 第6题：MTPA与弱磁的协调原则

**题目：** 在全速域控制中，MTPA计算的 $I_d^{MTPA}$ 和弱磁计算的 $I_d^{FW}$ 如何协调？以下代码实现正确的是？

A. `Id_ref = Id_MTPA + Id_FW;`
B. `Id_ref = (Id_MTPA + Id_FW) / 2;`
C. `Id_ref = min(Id_MTPA, Id_FW);`（取更负的值）
D. `if (Vs > Vmax) Id_ref = Id_FW; else Id_ref = Id_MTPA;`

**正确答案：** C

**详细解析：**

协调的核心原则：**电压约束优先于效率优化**。

- 电压约束是**硬约束**——违反会导致电流环饱和、控制失效
- MTPA是**软约束**——不满足只是效率降低，不会导致失稳

因此取 `min(Id_MTPA, Id_FW)`，即取更负的值：
- 在恒转矩区：$I_d^{FW} > I_d^{MTPA}$（前馈法计算出的弱磁Id为正值或不如MTPA负），取 $I_d^{MTPA}$
- 在弱磁区：$I_d^{FW} < I_d^{MTPA}$（弱磁Id更负），取 $I_d^{FW}$

选项A直接相加没有物理意义。选项B取平均也不正确。选项D是**硬切换**，会导致 $I_d$ 参考值在MTPA和弱磁之间跳变，引起转矩脉动。正确做法是用 `min()` 实现**连续过渡**。

## 第7题：六步方波的电压利用率

**题目：** 从SVPWM线性调制区过渡到六步方波，电压利用率提升约多少？

A. 约5.0%
B. 约10.3%
C. 约15.5%
D. 约27.3%

**正确答案：** B

**详细解析：**

SVPWM线性调制区电压极限：$V_{max,linear} = V_{dc}/\sqrt{3} \approx 0.577 V_{dc}$

六步方波电压极限：$V_{max,six-step} = 2V_{dc}/\pi \approx 0.637 V_{dc}$

电压利用率提升比：

$$\frac{V_{max,six-step}}{V_{max,linear}} = \frac{2/\pi}{1/\sqrt{3}} = \frac{2\sqrt{3}}{\pi} \approx \frac{3.464}{3.14159} \approx 1.103$$

即提升约10.3%，对应转速可提升约10.3%。

## 第8题：深度弱磁的稳定性问题

**题目：** 在深度弱磁区运行时，以下哪项不是主要的稳定性挑战？

A. 电流波形畸变，转矩脉动增大
B. 参数敏感性增大，$L_d$ 变化导致弱磁计算不准
C. 电流环稳定性下降，电压裕度极小
D. 永磁体温度升高导致电阻增大，铜损增加

**正确答案：** D

**详细解析：**

深度弱磁区的主要稳定性挑战包括：

- **A**：过调制谐波导致电流波形畸变，转矩脉动增大——这是深度弱磁的典型问题
- **B**：弱磁Id大，$L_d$ 随饱和变化，导致弱磁计算不准——参数敏感性增大
- **C**：电压裕度极小，电流环几乎没有调节空间，稳定性下降

选项D描述的"电阻增大导致铜损增加"虽然在高电流时确实存在，但这不是**稳定性**问题，而是**效率和热管理**问题。电阻增大不会直接导致控制系统失稳，而A、B、C都会直接影响控制的稳定性和动态性能。

## 第9题：电压反馈法 vs 前馈法的鲁棒性

**题目：** 当电机实际 $L_d$ 比标称值偏大20%时，以下哪种弱磁方法受影响更大？

A. 电压反馈法受影响更大，因为 $L_d$ 直接参与PI计算
B. 前馈计算法受影响更大，因为 $L_d$ 直接出现在弱磁电流公式中
C. 两种方法受影响程度相同
D. 两种方法都不受 $L_d$ 偏差影响

**正确答案：** B

**详细解析：**

前馈法弱磁电流公式：

$$I_d^{fw} = -\frac{\psi_f}{L_d} + \frac{1}{L_d}\sqrt{\frac{V_{max}^2}{\omega_e^2} - (L_q I_q)^2}$$

$L_d$ 直接出现在公式中，$L_d$ 偏差20%会导致 $I_d^{fw}$ 计算值显著偏差。具体来说，$L_d$ 偏大会导致弱磁Id计算偏小（弱磁不足），$L_d$ 偏小会导致弱磁Id计算偏大（过度弱磁）。

电压反馈法不依赖电机参数计算弱磁电流，而是通过电压幅值反馈自动调节 $I_d$。即使 $L_d$ 有偏差，PI控制器会根据实际电压误差自动修正，具有**自适应**能力。

文档中的定量分析也证实：参数偏差在20%以内时，电压反馈法仍有良好表现，而前馈法的弱磁Id误差可能非常显著（深度弱磁区 $\Delta I_d^{fw} \approx \psi_f / (10 L_d)$）。

## 第10题：退磁保护

**题目：** 某IPMSM的最大去磁电流 $I_{d,demag} = -25\text{A}$，最大工作电流 $I_{max} = 20\text{A}$。在弱磁控制中，$I_{d,min}$ 应设为多少？考虑高温下退磁电流绝对值减小10%。

A. $-20\text{A}$
B. $-22.5\text{A}$
C. $-25\text{A}$
D. $-27.5\text{A}$

**正确答案：** B

**详细解析：**

退磁保护的两个约束：

1. **电流极限约束**：$|I_d| \leq I_{max} = 20\text{A}$，即 $I_{d,min} \geq -20\text{A}$
2. **退磁约束**：$I_d \geq I_{d,demag}$，考虑高温修正后：$I_{d,demag,hot} = I_{d,demag} \times \text{temp\_coeff}$

高温下退磁电流绝对值减小10%，即更容易退磁：

$I_{d,demag,hot} = -25 \times (1 - 0.1) = -25 \times 0.9 = -22.5\text{A}$

取两个约束中更保守的（更接近0的）：

$\max(-20, -22.5) = -20\text{A}$

等等，这里需要仔细理解"更保守"的含义。退磁保护要求 $I_d$ 不能比退磁电流更负，即 $I_d \geq I_{d,demag,hot} = -22.5\text{A}$。电流极限要求 $I_d \geq -I_{max} = -20\text{A}$。

两个约束取交集：$I_d \geq \max(-22.5, -20) = -20\text{A}$

但题目问的是 $I_{d,min}$ 应设为多少。$I_{d,min}$ 是允许的最负Id值。

如果 $I_{d,min} = -20\text{A}$，则永远不会超过退磁极限 $-22.5\text{A}$，安全。
如果 $I_{d,min} = -22.5\text{A}$，则在 $-22.5 < I_d < -20$ 范围内，虽然不退磁，但电流可能超过 $I_{max}$（需要检查 $I_s = \sqrt{I_d^2 + I_q^2}$）。

实际上，$I_{d,min}$ 的设定需要同时考虑退磁和电流极限。如果仅从退磁角度，$I_{d,min} = -22.5\text{A}$。但实际中电流极限检查是另外做的（$\sqrt{I_d^2 + I_q^2} \leq I_{max}$），$I_{d,min}$ 主要由退磁约束决定。

所以答案应为 $-22.5\text{A}$，选项B。

注意：如果 $I_{d,min}$ 设为 $-20\text{A}$，虽然更安全，但在弱磁时可能无法充分利用弱磁能力。设为 $-22.5\text{A}$ 是在退磁安全边界上，配合电流极限检查可以安全运行。
