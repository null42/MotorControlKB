## 第1题：PFC与FOC双环结构类比

**题目：** Boost PFC的电压外环+电流内环与FOC的速度外环+电流内环在结构上高度相似。以下对应关系正确的是？

A. PFC电压环 ↔ FOC电流环；PFC电流环 ↔ FOC速度环

B. PFC电压环 ↔ FOC速度环；PFC电流环 ↔ FOC电流环；PFC输出电容C ↔ 电机惯量J

C. PFC电压环 ↔ FOC位置环；PFC电流环 ↔ FOC速度环

D. 两者结构完全不同，无法建立类比

**正确答案：** B

**详细解析：**

PFC与FOC的双环结构映射关系：

| PFC概念 | FOC概念 | 映射关系 |
|---------|---------|---------|
| 电压外环 → 控制$V_{dc}$ | 速度外环 → 控制$\omega$ | 外环都是慢变量 |
| 电流内环 → 控制$i_L$ | 电流内环 → 控制$i_d, i_q$ | 内环都是快变量 |
| 输出电容$C$ | 机械惯量$J$ | 外环被控对象都是积分器 |
| Boost电感$L$ | 电机电感$L_s$ | 内环被控对象都是L-R电路 |
| 电网电压$v_{in}$ | 转子角度$\theta_e$ | 同步信号源 |
| $V_{dc}$误差 → $I_{ref,raw}$ | $\omega$误差 → $I_{q,ref}$ | 外环输出是内环参考 |

**核心洞察：** 两者外环被控对象的传递函数都是积分器形式——PFC的$G_v(s) = 1/(Cs)$，FOC的$G_\omega(s) = K_t/(Js)$。内环被控对象都是一阶惯性——PFC的$G_i(s) = 1/(Ls+R)$，FOC的$G_i(s) = 1/(L_s s + R_s)$。这意味着PI参数整定方法完全相同！

## 第2题：PFC电流环PI整定

**题目：** Boost PFC的电感参数为$L = 1mH$，$R_L = 0.1\Omega$，开关频率65kHz。设电流环期望带宽5kHz，采用零极点对消法，PI参数应为？

A. $K_p = 31.4$, $K_i = 3140$

B. $K_p = 3.14$, $K_i = 314$

C. $K_p = 62.8$, $K_i = 6280$

D. $K_p = 0.314$, $K_i = 31.4$

**正确答案：** A

**详细解析：**

PFC电流环的PI整定方法与FOC电流环完全相同——零极点对消法。

**被控对象：** Boost电感的传递函数为：

$$G(s) = \frac{1}{Ls + R_L} = \frac{1/R_L}{(L/R_L)s + 1}$$

**零极点对消条件：**

$$\frac{K_i}{K_p} = \frac{R_L}{L} = \frac{0.1}{0.001} = 100$$

**期望带宽：** $f_{BW} = 5kHz$，$\omega_c = 2\pi \times 5000 = 31416 \text{ rad/s}$

**PI参数计算：**

$$K_p = \omega_c \cdot L = 31416 \times 0.001 = 31.4$$

$$K_i = \omega_c \cdot R_L = 31416 \times 0.1 = 3142$$

**验证：** $K_i/K_p = 3142/31.4 = 100.1 \approx R_L/L = 100$，零极点对消条件满足。

**与FOC的对比：** 如果电机参数为$L_s = 3.5mH$, $R_s = 0.85\Omega$，期望带宽2kHz：

$$K_p = 2\pi \times 2000 \times 3.5 \times 10^{-3} = 43.98$$
$$K_i = 2\pi \times 2000 \times 0.85 = 10681$$

方法完全相同，只是参数不同。

## 第3题：PFC电压环带宽约束

**题目：** 单相Boost PFC的电压环带宽为什么必须远低于2倍工频（100Hz或120Hz）？以下解释最准确的是？

A. 电压环响应太慢，无法跟踪100Hz信号

B. 如果电压环带宽高于2倍工频，电流参考会含有2倍工频分量，导致输入电流畸变，降低功率因数

C. 输出电容的ESR在100Hz时阻抗最低

D. ADC采样率不够，无法处理100Hz信号

**正确答案：** B

**详细解析：**

**PFC输出电压的2倍工频纹波：**

单相PFC的输入功率为：

$$P_{in}(t) = V_{in,peak}\sin(\omega t) \cdot I_{in,peak}\sin(\omega t) = \frac{V_{in,peak}I_{in,peak}}{2}(1 - \cos(2\omega t))$$

其中：
- $V_{in,peak}$：输入电压峰值 (V)
- $I_{in,peak}$：输入电流峰值 (A)
- $\omega$：电网角频率 (rad/s)

输入功率含有2倍工频（100Hz或120Hz）的脉动分量，这个脉动功率会导致输出电压出现2倍工频纹波：

$$V_{dc}(t) = V_{dc,avg} + \Delta V_{dc} \cdot \cos(2\omega t)$$

**电压环带宽的约束：**

如果电压环带宽高于2倍工频，电压环PI控制器会"看到"这个纹波并试图消除它。结果是：

$$I_{ref,raw} \propto (V_{dc,ref} - V_{dc}(t))$$

$I_{ref,raw}$将含有2倍工频分量，经过波形整形后：

$$I_{ref}(t) = I_{ref,raw}(t) \cdot \frac{|v_{in}(t)|}{V_{rms}^2}$$

2倍工频分量与基频相乘，产生3次谐波电流，导致输入电流畸变，THD增大，功率因数降低。

**工程准则：**

$$f_{BW,voltage} \leq \frac{f_{line}}{10} \approx 5 \sim 6 \text{ Hz}$$

这就是PFC电压环带宽远低于FOC速度环带宽（10~200Hz）的原因——PFC有2倍工频纹波的约束，而FOC速度环没有类似限制。

## 第4题：PFC电流参考生成

**题目：** PFC电流参考公式为$I_{ref} = I_{ref,raw} \times |v_{in}(t)| / V_{rms}^2$。其中除以$V_{rms}^2$的物理意义是？

A. 归一化电流参考到单位值

B. 补偿电网电压波动，保证输入功率恒定——当电网电压升高时电流参考自动降低

C. 计算瞬时功率

D. 补偿电感压降

**正确答案：** B

**详细解析：**

这是PFC最精妙的设计。分步分析：

**Step 1：乘以$|v_{in}(t)|$——波形整形**

将电压环输出的直流信号$I_{ref,raw}$调制为正弦半波，使电流参考跟踪电压波形。

**Step 2：除以$V_{rms}^2$——功率前馈**

当电网电压升高时：
- $|v_{in}(t)|$增大 → 电流参考增大（趋势正确，功率应增加）
- 但$V_{rms}^2$也增大 → 电流参考减小（补偿效果）
- 净效果：$I_{ref} \propto |v_{in}|/V_{rms}^2 = |v_{in}|/(V_{rms} \cdot V_{rms})$

输入功率：

$$P_{in} = V_{rms} \cdot I_{rms} = V_{rms} \cdot \frac{I_{ref,raw} \cdot V_{rms}}{V_{rms}^2} \cdot \frac{1}{\sqrt{2}} = \frac{I_{ref,raw}}{\sqrt{2}}$$

**与电网电压无关！** 这就是$V_{rms}^2$前馈的物理意义——保证在电网电压波动时，输入功率恒定，仅由电压环输出$I_{ref,raw}$决定。

**如果不除以$V_{rms}^2$：** 电网电压升高10%时，$|v_{in}|$增大10%，电流参考增大10%，功率增大约20%（电压×电流都增大），导致输出过压。

## 第5题：过零畸变

**题目：** 单相Boost PFC在输入电压过零附近电流波形出现"缺口"，即电流跟踪不上正弦参考。以下原因中，哪个不是导致过零畸变的原因？

A. 占空比在过零处接近1，受最大占空比限制而饱和

B. 轻载时电感电流在过零处进入断续模式（DCM），平均电流与占空比关系变为非线性

C. 电流环PI参数过大导致过零处振荡

D. 快恢复二极管的反向恢复电荷在过零处导致额外电流

**正确答案：** C

**详细解析：**

过零畸变是Boost PFC的固有问题，有四个主要成因：

**1. 占空比饱和（选项A）：** 过零处$v_{in} \approx 0$，稳态占空比$D = 1 - v_{in}/V_{dc} \to 1$。但实际最大占空比受限（通常$D_{max} < 0.95$，需要最小关断时间给ADC采样），控制器输出饱和，无法提供足够的占空比使电流跟踪参考。

**2. DCM模式（选项B）：** 轻载时电感电流在过零附近进入断续模式。DCM下平均电流与占空比的关系为：

$$I_{avg,DCM} = \frac{V_{dc} \cdot D^2 \cdot T_s}{2L}$$

而CCM下为线性关系$I_{avg,CCM} \propto D$。PI控制器基于CCM模型设计，在DCM区域输出不正确。

**3. 二极管反向恢复（选项D）：** 快恢复二极管在过零处（电流从正变负的瞬间）反向恢复电荷$Q_{rr}$导致额外电流尖峰，干扰电流采样。

**选项C为什么不是过零畸变的原因：** PI参数过大会导致整个周期的振荡（不仅在过零处），且振荡频率接近电流环带宽。过零畸变是过零处特有的现象，与PI参数大小无关——即使PI参数很小，过零处仍然会有畸变，因为根因是占空比饱和和DCM模式，而非控制器增益。

## 第6题：交错并联PFC

**题目：** 2相交错并联Boost PFC，每相开关频率65kHz，载波移相180度。输入电流纹波频率和最大纹波降低比例（在50%占空比附近）约为？

A. 纹波频率65kHz，纹波降低50%

B. 纹波频率130kHz，纹波降低约50%

C. 纹波频率130kHz，纹波完全消除

D. 纹波频率65kHz，纹波降低25%

**正确答案：** B

**详细解析：**

**纹波频率：** N相交错并联时，各相纹波相互叠加，输入电流纹波频率为$N \times f_{sw}$。2相交错时纹波频率为$2 \times 65 = 130kHz$。

**纹波降低效果：** 纹波抵消系数为：

$$\text{Ripple Reduction} = \frac{1}{N} \cdot \left| \frac{\sin(N\pi D)}{N\sin(\pi D)} \right|$$

2相（N=2）时：

$$\text{Ripple Reduction} = \frac{1}{2} \cdot \left| \frac{\sin(2\pi D)}{2\sin(\pi D)} \right| = \frac{1}{2} \cdot \left| \frac{2\sin(\pi D)\cos(\pi D)}{2\sin(\pi D)} \right| = \frac{|\cos(\pi D)|}{2}$$

在50%占空比（$D = 0.5$）时：

$$\text{Ripple Reduction} = \frac{|\cos(\pi \times 0.5)|}{2} = \frac{0}{2} = 0$$

即纹波完全抵消！但这是理想情况，实际中由于各相参数不完全一致，纹波降低约50%是更保守的估计。

**工程意义：** 纹波频率翻倍意味着输入EMI滤波器的截止频率可以提高，滤波器体积可以更小。纹波幅值降低意味着电感体积可以更小（每相只需处理1/2的功率）。

## 第7题：Totem-pole PFC

**题目：** Totem-pole PFC必须使用GaN或SiC器件，而不能使用普通Si MOSFET，根本原因是？

A. GaN的导通电阻更低

B. Si MOSFET的体二极管具有较大的反向恢复电荷$Q_{rr}$，在高速桥臂换流时产生巨大的反向恢复电流

C. GaN的耐压更高

D. Si MOSFET开关速度太慢

**正确答案：** B

**详细解析：**

**Totem-pole PFC的工作原理：**

高速桥臂（S1/S2）在高频切换，当S1关断、S2开通时，电流从S1的体二极管转移到S2。如果使用Si MOSFET，其体二极管的反向恢复电荷$Q_{rr}$可达数百nC到$\mu C$级别。

**反向恢复的问题：**

$$E_{sw,loss} = \frac{1}{2} C_{oss} \cdot V_{ds}^2 + Q_{rr} \cdot V_{ds}$$

其中：
- $E_{sw,loss}$：单次开关损耗能量 (J)
- $C_{oss}$：MOSFET输出电容 (F)
- $V_{ds}$：漏源极电压 (V)
- $Q_{rr}$：体二极管反向恢复电荷 (C)

反向恢复电流在S2开通瞬间流过S2，产生巨大的开关损耗和严重的EMI问题。在65kHz以上开关频率下，Si MOSFET的$Q_{rr}$问题使得Totem-pole PFC不可行。

**GaN HEMT的优势：**

- **零反向恢复电荷：** $Q_{rr} \approx 0$，因为GaN没有体二极管（是横向器件，电流双向流动）
- **极低$C_{oss}$：** 开关损耗低
- **高dv/dt能力：** 可达100V/ns

**选项A不准确：** GaN的导通电阻不一定比同规格Si MOSFET低，且导通电阻不是Totem-pole必须用GaN的原因。选项C不准确：GaN耐压不一定比Si高。选项D不完全：Si MOSFET开关速度不是根本问题，反向恢复才是。

## 第8题：PFC与FOC联合开发——中断优先级

**题目：** PFC+电机控制共用一个MCU，PFC开关频率100kHz，电机开关频率16.7kHz。以下中断优先级设计最合理的是？

A. 电机PWM中断 > PFC PWM中断 > 通信中断

B. PFC PWM中断 > 电机PWM中断 > 通信中断

C. 通信中断 > PFC PWM中断 > 电机PWM中断

D. PFC和电机使用相同优先级，轮流执行

**正确答案：** B

**详细解析：**

**中断优先级设计原则：**

1. **频率最高的控制环优先级最高** —— PFC 100kHz > 电机 16.7kHz
2. **安全关键的保护优先级最高** —— 过流硬件保护（通常不由软件中断处理）
3. **非实时任务优先级最低** —— 通信、显示

**为什么PFC优先级更高：**

- PFC开关频率100kHz意味着每个PWM周期仅10us，中断延迟必须极小
- 如果PFC中断被电机中断抢占，PFC电流环的执行延迟增大，可能导致电流跟踪误差增大
- 电机的16.7kHz周期约60us，即使被PFC中断抢占（PFC ISR约3~5us），仍有足够时间完成计算

**推荐的中断优先级设计：**

| 优先级 | 中断源 | 执行内容 | 频率 |
|--------|--------|---------|------|
| 最高 | PFC PWM中断 | PFC电流环 | 100kHz |
| 高 | 电机PWM中断 | FOC电流环 | 16.7kHz |
| 中 | 电机速度环 | 速度PI | 1~10kHz |
| 低 | PFC电压环 | 电压PI | 1~10kHz |
| 最低 | 通信/显示 | UI/协议 | 100~1000Hz |

**关键约束：** 两个ADC的采样时刻不能重叠。如果PFC和电机的ADC触发时刻冲突，需要调整PWM相位偏移，确保采样时刻错开。

## 第9题：从电控转PFC——知识迁移

**题目：** 一位FOC电机控制工程师转做PFC开发，以下哪项知识可以直接复用（无需修改）？

A. 电流环PI控制器的代码实现

B. SVPWM生成模块

C. Park变换模块

D. 编码器速度计算模块

**正确答案：** A

**详细解析：**

**70%可直接复用的知识：**

| 知识模块 | 可否复用 | 理由 |
|---------|---------|------|
| **PI控制器代码** | **完全复用** | PFC和FOC的PI控制器在数学上完全相同：误差计算→比例项→积分项→抗饱和→输出限幅 |
| PWM配置 | 高度复用 | 中心对齐模式、ADC触发相同，但PFC频率更高 |
| ADC采样+DMA | 完全复用 | 采样框架相同，只是通道不同 |
| 保护逻辑框架 | 框架复用 | OVP/OCP/UVP逻辑相同，阈值不同 |
| SVPWM | **不能复用** | PFC是单相单管，不需要SVPWM（三相空间矢量调制） |
| Park变换 | **不能复用** | PFC没有dq旋转坐标系，不需要Park变换 |
| 编码器速度计算 | **不能复用** | PFC不需要编码器，需要的是电网PLL |

**PI控制器复用的代码示例：**

```c
/* 以下代码在PFC和FOC中完全相同 */
float PI_Execute(PI_Controller_t *pi, float ref, float fdb)
{
    float err = ref - fdb;
    float p_out = pi->kp * err;
    pi->integral += pi->ki * err * pi->ts;
    pi->integral = CLAMP(pi->integral, pi->out_min, pi->out_max);
    float out = p_out + pi->integral;
    out = CLAMP(out, pi->out_min, pi->out_max);
    return out;
}
```

PFC调用：`duty = PI_Execute(&pfc_pi_current, i_ref, i_L);`
FOC调用：`v_d = PI_Execute(&foc_pi_id, i_d_ref, i_d);`

## 第10题：综合分析——PFC+电机联合系统

**题目：** 工业变频器系统中，前端Totem-pole PFC + 后端电机逆变器构成AC→DC→AC完整能量链。当电机从电动模式切换到制动模式时，以下关于PFC行为的描述正确的是？

A. PFC继续工作在整流模式，制动能量通过制动电阻消耗

B. PFC检测到母线电压升高后切换到逆变模式，将制动能量回馈电网

C. PFC关闭，制动能量充入母线电容

D. 电机制动时PFC不受影响

**正确答案：** B

**详细解析：**

**能量流分析：**

- **电动模式：** 电网 → PFC(AC→DC) → 母线电容 → 逆变器(DC→AC) → 电机
- **制动模式：** 电机 → 逆变器(AC→DC) → 母线电容 → PFC(DC→AC) → 电网

**Totem-pole PFC的双向能力：**

传统Boost PFC只能实现AC→DC（整流），制动能量只能通过制动电阻消耗。Totem-pole PFC天然支持双向能量流——只需改变电流参考的极性：

```c
if (power_direction == FORWARD) {
    i_ref = i_voltage_loop_output * |v_in| / v_rms_sq;  /* 整流模式 */
} else {
    i_ref = -i_voltage_loop_output * |v_in| / v_rms_sq; /* 逆变模式 */
}
```

**母线电压控制逻辑：**

1. 电动模式：PFC电压环维持$V_{dc}$在设定值
2. 制动模式：电机回馈能量导致$V_{dc}$升高
3. 当$V_{dc}$超过上限阈值时，PFC切换到逆变模式，将多余能量回馈电网
4. 当$V_{dc}$回到正常范围时，PFC切回整流模式

**这就是"主动前端(AFE)"的概念：** Totem-pole PFC作为主动前端，不仅实现功率因数校正，还实现了能量的双向流动，消除了制动电阻，提高了系统效率。

**其他选项的问题：**

- **选项A：** 使用制动电阻是传统方案，浪费制动能量，效率低。
- **选项C：** 母线电容容量有限，无法吸收大量制动能量，电压会持续升高触发过压保护。
- **选项D：** 电机制动时母线电压升高，PFC必须响应，否则会触发过压保护。
