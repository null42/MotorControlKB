## 一、参数计算题（★★★☆☆）

### 题目1：超前校正器参数计算
**问题：** 一台PMSM电流环在原剪切频率 800Hz 处相位裕度为 22°。需要设计超前校正器将相位裕度提升到 48°，剪切频率维持不变。计算：
1. 需要补充的相位角 φm（考虑5°余量）
2. 超前校正器的 α 和 T 参数
3. 校正后的Bode图幅值在800Hz处会升高多少dB？

**参考答案：**
1. φm = 48° - 22° + 5° = **31°**
2. α = (1-sin31°)/(1+sin31°) = (1-0.515)/(1+0.515) = 0.485/1.515 = **0.320**
   ωm = 2π × 800 = 5027 rad/s
   T = 1/(ωm√α) = 1/(5027×0.566) = **3.52×10⁻⁴**
   校正器：$G_c(s) = \frac{3.52\times10^{-4}s+1}{1.13\times10^{-4}s+1}$
3. 幅值升高 = 10log₁₀(1/α) = 10log₁₀(3.125) = **4.95 dB**

### 题目2：滞后校正器设计
**问题：** 某电机速度环P控制，Ksp=0.1，Kt/J=50。在给定1000rpm时，稳态误差2%（即20rpm）。要求将稳态误差降至0.2%。速度环剪切频率为30Hz。设计滞后校正器，给出β和T。

**参考答案：**
1. 需要增大的低频增益：ess从2%→0.2%，需10倍提升 → β = 10
2. T的选择：转角频率1/T应远低于剪切频率
   1/T = ωc/10 = 2π×30/10 = 18.85 → T = 0.053
3. 校正器：$G_{lag}(s) = \frac{0.053s+1}{0.53s+1}$
4. 验证：校正后在30Hz处相位滞后角=arctan(0.053×188.5)-arctan(0.53×188.5)=84.3°-89.4°=-5.1°，对相位裕度影响可接受

### 题目3：陷波滤波器参数计算
**问题：** 伺服系统在220Hz处发现机械谐振峰（+18dB）。要求设计陷波滤波器，使谐振峰衰减20dB以上。计算：
1. ω₀
2. 为达到26dB衰减的 ζz 和 ζp 组合（提供至少两种方案）
3. 讨论两种方案的优缺点

**参考答案：**
1. ω₀ = 2π × 220 = **1382 rad/s**
2. 衰减 = 20log₁₀(ζp/ζz) ≥ 26 → ζp/ζz ≥ 20
   - 方案A：ζz=0.01, ζp=0.2 → 衰减26dB，陷波窄（Q高）
   - 方案B：ζz=0.02, ζp=0.4 → 衰减26dB，陷波宽（Q低）
3. 优缺点：
   - 方案A：陷波深而窄，对精确匹配的频率效果好，但频率偏移5%效果减半
   - 方案B：陷波宽度大，对频率偏移不敏感（鲁棒性好），但在ω₀附近引入更大的相位滞后
   - **工程建议**：先用方案B保证鲁棒性，如果衰减不够再切换到方案A

## 二、分析设计题（★★★★☆）

### 题目4：超前校正 vs 增大Kp
**问题：** 一个电流环PI控制器，Kp=0.5, Ki=100。工程师发现响应太慢，有两个方案：
- 方案A：将Kp增大到0.8
- 方案B：保持Kp=0.5，串联超前校正器（α=0.2, T=2×10⁻⁴）

1. 分析两种方案对闭环极点的不同影响
2. 哪种方案的高频噪声更大？为什么？
3. 在电机控制中，你倾向哪种方案？

**参考答案：**
1. 方案A（增大Kp）：开环零极点比例改变，闭环极点向共轭方向移动，阻尼比减小，可能产生超调和振荡
   方案B（超前校正）：额外的零点吸引根轨迹向左偏移，改善阻尼；额外的极点限制高频增益。闭环极点可在更高阻尼比下获得更快响应。

2. 方案A：高频增益 = Kp = 0.8（比原来+4dB）
   方案B：高频增益 = Kp/α = 0.5/0.2 = 2.5（比原来+14dB！）
   **方案B的高频噪声放大更严重**

3. 在电机电流环中：
   - 如果电流采样噪声已较大 → 选方案A（增大Kp），避免超前校正放大噪声
   - 如果对振荡/超调敏感（如精密伺服）→ 选方案B，但要配合低通滤波抑制噪声
   - **实际中方案A更常用**，因为电机电流环通常噪声环境不理想

### 题目5：速度环陷波滤波器位置选择
**问题：** 陷波滤波器可以放置在以下三个位置：
- 位置1：速度环PI输出之后（转矩给定之前）
- 位置2：速度反馈通道中
- 位置3：电流给定之后（电流环前）

分析三种位置对谐振抑制效果的异同，及各自的适用场景。

**参考答案：**

| 位置 | 抑制对象 | 优点 | 缺点 |
|------|---------|------|------|
| 位置1 | 转矩给定中的谐振分量 | 直接切断谐振激励源 | 不影响速度反馈测量精度 |
| 位置2 | 速度反馈中的谐振分量 | 防止谐振进入控制器 | 可能掩盖真实的机械振动信息 |
| 位置3 | 电流环输入中的谐振分量 | 电流环带宽内有效 | 谐振频率>电流环带宽时无效 |

**最佳实践：**
- 一般选**位置1**——在转矩给定后加陷波，抑制谐振力矩分量
- 位置2用于谐振频率较低（<10Hz）且速度传感器带宽足够的情况
- 位置3用于谐振频率远低于电流环带宽的情况
- **伺服驱动器通常将陷波滤波器放在速度环输出端（位置1）**

### 题目6：多谐振模态的校正策略
**问题：** 某大型六轴机器人关节电机在频谱分析中发现三个谐振峰：42Hz、156Hz、580Hz。速度环带宽目标50Hz。
1. 设计校正策略（需要几个陷波滤波器？放在什么位置？）
2. 42Hz谐振（低于速度环带宽）与156Hz（高于速度环带宽）的处理方式有何不同？
3. 580Hz谐振是否需要处理？为什么？

**参考答案：**
1. 策略：
   - 42Hz和156Hz：各需要一个陷波滤波器，串联在速度环输出端
   - 多陷波需注意级联后的相位滞后累积效应
   - 陷波顺序：先低频后高频（低频陷波对高频影响小）

2. 42Hz < 50Hz（速度环带宽内）：
   - 速度环本身会"看到"这个谐振并试图补偿 → 可能正反馈加剧
   - 必须用陷波滤波器抑制
   - 陷波会降低该频率附近的速度环增益 → 可能降低速度环带宽
   
   156Hz > 50Hz：
   - 速度环本身对该频率的响应已衰减
   - 但如果谐振增益极高（>20dB），仍可能穿透
   - 陷波对此频率的抑制不影响速度环带宽

3. 580Hz谐振：
   - 远高于速度环带宽 → 速度环基本不响应
   - 可能由电流环的高频分量激励
   - **如果580Hz处谐振峰增益<6dB → 可不处理**
   - 如果>10dB → 需在电流环中加陷波或低通滤波
   - 580Hz对应的转速很高（如四对极电机：580Hz→8700rpm），需确认是否在运行范围内

### 题目7：数字实现中的校正器离散化
**问题：** 超前校正器 $G_c(s) = \frac{2\times10^{-4}s+1}{4\times10^{-5}s+1}$ 需要在16kHz的DSP中实现。
1. 用Tustin变换离散化，推导差分方程
2. 计算离散化后的实际频率响应与连续域的偏差（在800Hz处）
3. 如果因为离散化导致相位超前角偏小5°，如何补偿？

**参考答案：**
1. Ts = 62.5μs, Tustin: $s = \frac{2}{Ts}\frac{z-1}{z+1} = 32000\frac{z-1}{z+1}$
   代入得：$G_c(z) = \frac{(2\times10^{-4}\times32000+1)z + (2\times10^{-4}\times32000-1)}{(4\times10^{-5}\times32000+1)z + (4\times10^{-5}\times32000-1)} = \frac{7.4z + 5.4}{2.28z + 0.28}$
   差分方程：$y[k] = (7.4x[k] + 5.4x[k-1] - 0.28y[k-1])/2.28$

2. 在800Hz处，Tustin变换存在频率翘曲（frequency warping）。
   预翘曲频率：$\omega_a = \frac{2}{Ts}\tan(\omega_d Ts/2)$
   ωd = 5027, ωa = 32000×tan(5027×62.5×10⁻⁶/2) = 32000×tan(0.1571) = 5080 rad/s
   偏差 = (5027-5080)/5027 = **-1.05%**（接受范围内）

3. 补偿方法：
   - 在连续域设计时将目标相位裕度提高5°
   - 或使用预翘曲Tustin变换（pre-warping）
   - 或提高采样频率降低离散化误差

## 三、综合应用题（★★★★★）

### 题目8：完整的速度环校正设计
**问题：** 某PMSM伺服系统速度环相关参数：J=0.001 kg·m², Kt=0.5 N·m/A, 电流环等效时间常数 τc=0.5ms, 速度反馈滤波器截止频率500Hz。性能要求：速度环带宽≥40Hz，相位裕度≥45°，稳态误差<0.1%。
1. 分析原系统（纯P控制，Ksp=0.04）是否满足要求
2. 如不满足，设计完整的校正方案（可能包含超前/滞后/陷波组合）
3. 给出最终的离散化实现伪代码

**参考答案：**
1. 速度环开环：$G(s) = 0.04 \times \frac{0.5}{0.001s} \times \frac{1}{0.0005s+1} \times \frac{1}{0.000318s+1}$
   简化：$G(s) = \frac{20}{s(0.0005s+1)(0.000318s+1)}$
   
   在ω=2π×40=251 rad/s处：
   幅值≈20/251=0.0797=**-22dB**（远未到达0dB，带宽不足）
   相位≈-90°-arctan(0.126)-arctan(0.08)=-90°-7.2°-4.6°=-101.8°
   **不满足带宽要求**

2. 需大幅提高增益。尝试Ksp=1.0：
   $G(s) = \frac{500}{s(0.0005s+1)(0.000318s+1)}$
   在251rad/s处幅值≈500/251=**+6dB**→剪切频率约350rad/s(56Hz)→带宽够，但需验证相位裕度
   在350rad/s处：相位=-90°-arctan(0.175)-arctan(0.111)=-90°-9.9°-6.3°=-106.2°
   相位裕度=180°-106.2°=**73.8°**（满足！）
   
   但纯P控制稳态误差：Kv=500/1=500, ess=1000/500=2rpm=0.2%（不满足0.1%）
   → 需要积分或滞后校正
   
   **方案：PI + 滞后校正**
   PI：Ksp=1.0, Ksi=50（加积分消除稳态误差）
   验证后如相位裕度不足→叠加超前校正

3. 离散化伪代码（略）

### 题目9：参数不确定性下的陷波鲁棒设计
**问题：** 机械谐振频率标称值185Hz，但实际可能因负载变化在170~200Hz间漂移。要求至少有15dB的谐振抑制。
1. 设计单一陷波滤波器（给出ζz、ζp），覆盖整个不确定范围
2. 如果单一陷波无法满足，提出替代方案
3. 计算该陷波在170Hz和200Hz处的衰减量

**参考答案：**
1. 中心频率取中间值：ω₀ = 2π×185 = 1162 rad/s
   需要覆盖170~200Hz范围（±8.1%），要求在最差频率处衰减≥15dB
   
   最差情况在170Hz（ω=1068）：频率偏差=(1068-1162)/1162=-8.1%
   陷波在偏离ω₀处的衰减公式（近似）：$A(\omega) \approx K \cdot \frac{(\omega-\omega_0)^2 + (\zeta_z\omega_0)^2}{(\omega-\omega_0)^2 + (\zeta_p\omega_0)^2}$
   
   试探ζz=0.03, ζp=0.25：中心衰减=18.4dB，170Hz处≈12dB（不足）
   试探ζz=0.02, ζp=0.30：中心衰减=23.5dB，170Hz处≈16dB（满足！）
   
   最终：**ζz=0.02, ζp=0.30, ω₀=1162**

2. 替代方案：自适应陷波或双陷波级联
   - 双陷波：一个中心175Hz，一个195Hz，各负责一半范围

3. 验算：
   - 170Hz处：Δω=-55rad/s，衰减≈15.8dB
   - 200Hz处：Δω=+94rad/s，衰减≈17.2dB
   均≥15dB ✓

### 题目10：FOC中陷波滤波器的实时性分析
**问题：** 陷波滤波器的二阶IIR实现每次调用需要几次乘法和加法？在16kHz控制频率下，单个陷波滤波器占用多少CPU时间（假设Cortex-M4, FPU, 单周期浮点乘法）？如果级联4个陷波滤波器，对控制环路延迟有何影响？如何优化？

**参考答案：**
1. 每次调用：$y = b_0x + b_1x_1 + b_2x_2 - a_1y_1 - a_2y_2$
   = **5次乘法 + 4次加法** + 4次状态更新（赋值）

2. 单个陷波：约9条指令 × 1周期 = **9个CPU周期**（近似）
   16kHz → 周期=62.5μs → 占用率=9/62500 ≈ 0.014%（微不足道）

3. 4个陷波级联：
   - 计算量：4×9=36周期（仍可忽略）
   - 组延迟（group delay）：每个陷波在ω<ω₀区域引入约 Ts×(a1+a2)/(1+a1+a2) 的延迟
   - 4个级联 ≈ 4×0.5Ts = 2Ts = **125μs @ 16kHz**
   - 这对电流环（几百Hz带宽）影响不大，但对速度环（~50Hz）有轻微影响

4. 优化策略：
   - 用SIMD指令（如CMSIS-DSP的arm_biquad_cascade_df1_f32）
   - 将系数预计算并存储在Flash中
   - 如果多个陷波的ω₀接近，可考虑合并为一个高阶滤波器
