## 一、概念判断题（★★★☆☆）

### 题目1：根轨迹条件判断
**问题：** 某电机电流环开环传递函数为 $G(s) = \frac{K}{s(s+500)(s+2000)}$，判断以下各点哪些在根轨迹上：s₁ = -100, s₂ = -250 + j433, s₃ = -1000 + j1000。请说明判断依据。

**参考答案：**
根据相角条件 $\angle G(s) = (2k+1)\pi$：
- s₁ = -100：$\angle G = -0° - 0° - 0° = 0°$ → **不在根轨迹上**
- s₂ = -250 + j433：$\angle G = -120° - 60° - 13.7° = -193.7° ≈ -180°$ → **在根轨迹上**
- s₃ = -1000 + j1000：$\angle G = -135° - 63.4° - 45° = -243.4°$ → **不在根轨迹上**

## 二、参数计算题（★★★☆☆）

### 题目2：电流环PI控制的阻尼比计算
**问题：** 一台PMSM电流环参数：L = 0.5mH，R = 0.1Ω，PI控制器 Kp = 0.6，Ki = 150。
1. 计算闭环系统的阻尼比 ζ
2. 该系统是过阻尼、临界阻尼还是欠阻尼？
3. 若希望 ζ = 0.707，Kp 应调整为多少？（Ki 保持不变）

**参考答案：**
1. 特征方程：$0.0005s^2 + (0.1+0.6)s + 150 = 0$，即 $0.0005s^2 + 0.7s + 150 = 0$
   标准化：$s^2 + 1400s + 300000 = 0$
   $\omega_n = \sqrt{300000} = 547.7$ rad/s
   $2\zeta\omega_n = 1400$ → $\zeta = 1400/(2 \times 547.7) = 1.278$
2. ζ = 1.278 > 1 → **过阻尼**，系统无超调，响应较慢
3. ζ = 0.707 → $2 \times 0.707 \times 547.7 = 774.6$ → 新系数 0.1 + Kp = 0.0005 × 774.6 = 0.387 → **Kp = 0.287**

### 题目3：根轨迹分离点计算
**问题：** 某简化电机速度环开环传递函数 $G(s) = \frac{K}{s(s+2)(s+5)}$。
1. 计算根轨迹在实轴上的分离点
2. 该分离点对应的增益 K 是多少？
3. 该分离点在电机控制中的物理意义是什么？

**参考答案：**
1. 分离点方程：$\frac{1}{s} + \frac{1}{s+2} + \frac{1}{s+5} = 0$
   即 $3s^2 + 14s + 10 = 0$ → $s = \frac{-14 \pm \sqrt{196 - 120}}{6} = \frac{-14 \pm 8.718}{6}$

   s₁ = -0.880（在根轨迹区段 [0, -2] 上）→ **分离点**
   s₂ = -3.787（在 [-2, -5] 区间，是会合点）

2. 幅值条件：$K = |s| \cdot |s+2| \cdot |s+5|$
   s₁ = -0.880: K = 0.880 × 1.12 × 4.12 = **4.06**

3. 物理意义：当速度环增益 K < 4.06 时，系统为过阻尼（三个实极点）；K = 4.06 时临界阻尼；K > 4.06 时出现一对共轭极点→速度响应出现振荡。**这就是为什么速度环增益设置过大会导致电机啸叫。**

## 三、分析设计题（★★★★☆）

### 题目4：根轨迹与Bode图的对应关系
**问题：** 对电流环 $G(s) = \frac{K_p s + K_i}{s(Ls + R)}$，L=0.8mH, R=0.15Ω, Ki=120。
1. 画出 Kp 从 0.1 到 2.0 的根轨迹草图，标注分离点和 ζ=0.707 点
2. 分别用根轨迹法和Bode图法解释：为什么 Kp 过大会导致振荡？
3. 这两种分析方法给出的结论是否一致？如果不一致，原因是什么？

**参考答案：**
1. （略，要求学生在草图上标注开环极点 (0, 0) 和 (-187.5, 0)，开环零点 = -Ki/Kp 随 Kp 变化，分离点计算...）

2. **根轨迹法：** Kp 增大→共轭极点虚部增大→阻尼比 ζ 减小→超调增大→振荡
   **Bode图法：** Kp 增大→剪切频率升高→相位裕度减小→当相位裕度<0时系统不稳定

3. 两种方法在二阶系统中结论一致。但对于高阶系统（如考虑数字延迟的速度环），Bode图可能显示出更保守的稳定条件（因为Bode图考虑的是频率响应而非极点位置）。**在电机控制中，通常先根轨迹设计极点位置，再用Bode图验证相位裕度。**

### 题目5：速度环根轨迹与临界增益
**问题：** 一台永磁同步电机的速度环开环传递函数为：
$$G_{ol}(s) = \frac{K(0.05s + 1)}{s(0.001s + 1)(0.01s + 1)}$$
其中 K = Ksp·Kt/J，是速度环总增益。
1. 计算系统稳定的 K 值范围（用Routh判据）
2. 当 K = K_critical 时，振荡频率是多少？
3. 在电机速度环调试中，如何利用这一信息避免振荡？

**参考答案：**
1. 特征方程：$s(0.001s+1)(0.01s+1) + K(0.05s+1) = 0$
   展开：$10^{-5}s^3 + 0.011s^2 + (1+0.05K)s + K = 0$
   Routh表：
   ```
   s³ | 10⁻⁵         1+0.05K
   s² | 0.011        K
   s¹ | (0.011(1+0.05K)-10⁻⁵K)/0.011 = (0.011+0.00055K-10⁻⁵K)/0.011
   s⁰ | K
   ```
   第一列全正 → $0.011 + 0.00054K > 0$ 且 $K > 0$
   → **K < 20.37**（0.011/0.00054 ≈ 20.37）

2. 临界时 s¹ 行=0，辅助方程 $0.011s^2 + 20.37 = 0$
   → $s = \pm j\sqrt{20.37/0.011} = \pm j43.0$ rad/s
   → 振荡频率 f = 43.0/(2π) = **6.85 Hz**

3. 调试策略：
   - 逐步增大 Ksp，同时监测速度反馈信号
   - 当检测到约 7Hz 的速度振荡时，说明已接近临界增益
   - 将 Ksp 回调至 0.5×K_critical 以获得足够稳定裕度
   - **实际经验：速度环振荡频率通常在 5~30 Hz 范围，可用频谱分析仪确认**

### 题目6：电流环PI参数的根轨迹法整定
**问题：** 已知 PMSM 参数 L = 1.2mH, R = 0.25Ω, 期望电流环阻尼比 ζ = 0.707，调节时间 ts ≤ 5ms。
1. 分别计算 Kp 和 Ki（假设采用零极点对消法）
2. 如果不采用对消法，用根轨迹法在复平面上选择满足要求的极点位置，给出对应的 Kp, Ki
3. 比较两种方法得到的PI参数，分析各自的优缺点

**参考答案：**
1. **零极点对消：** Ki/Kp = R/L = 0.25/0.0012 = 208.33
   ts = 4/(ζωn) ≤ 5ms → ωn ≥ 4/(0.707 × 0.005) = 1132 rad/s
   Kp = L·ωc = 0.0012 × 1132 = 1.36, Ki = Kp × 208.33 = 283

2. **根轨迹法：** 特征方程 $0.0012s^2 + (0.25+K_p)s + K_i = 0$
   要求 ζ = 0.707, σ ≥ 800 (ts ≤ 5ms)
   极点位于 s = -σ ± jσ (45°线)
   代入特征方程匹配实部虚部：
   实部：$0.0012(-2σ²) + (0.25+K_p)(-σ) + K_i = 0$
   虚部：$2×0.0012×σ² = 0$（矛盾，二阶系统ζ=0.707时虚部不为零）
   正确解法：$(s - s₁)(s - s₂) = s^2 + 2ζω_n s + ω_n² = 0$
   其中 $s² + (0.25+K_p)/0.0012·s + K_i/0.0012 = 0$
   匹配：$2ζω_n = (0.25+K_p)/0.0012$, $ω_n² = K_i/0.0012$
   选 ωn = 1132, ζ = 0.707 → Kp = 2ζωn·L - R = 2×0.707×1132×0.0012 - 0.25 = 1.67
   Ki = ωn²·L = 1132² × 0.0012 = 1538

3. **比较：**
   - 对消法：Kp=1.36, Ki=283 → 带宽低，保守，适合参数准确的情况
   - 根轨迹法：Kp=1.67, Ki=1538 → Ki 远大于对消法，响应更快但可能引起积分饱和
   - **工程建议：** 先用根轨迹法确定理论最优值，实际调试时 Ki 适当折衷（取理论值的 50~70%）避免积分饱和，然后用对消法的 Kp/Ki 比例关系作为下限约束。

### 题目7：数字延迟对根轨迹的影响
**问题：** 电机FOC控制中，电流采样到PWM更新存在一拍延迟 Td = 1/fs（fs为PWM频率）。该延迟近似为 $e^{-T_d s} \approx \frac{1}{1+T_d s}$。
1. 写出考虑延迟的电流环开环传递函数
2. 分析延迟对根轨迹的影响（fs = 10kHz vs 20kHz）
3. 给出考虑延迟时的最大允许 Kp

**参考答案：**
1. $G_{ol}(s) = \frac{K_p s + K_i}{s} \cdot \frac{1}{Ls+R} \cdot \frac{1}{1+T_d s}$

2. 延迟引入了一个额外的极点 s = -1/Td：
   - fs = 10kHz: Td = 0.1ms, 极点 s = -10000
   - fs = 20kHz: Td = 0.05ms, 极点 s = -20000

   该额外极点使根轨迹在右偏转——原本稳定的高增益在引入延迟后可能不稳定。fs越低，额外极点越靠近虚轴，对稳定性的影响越大。

3. 特征方程变为三阶：
   $T_d L s^3 + (L + T_d R)s^2 + (R + K_p)s + K_i = 0$

   用Routh判据：$K_p < \frac{L}{T_d}$ （近似，忽略小项）
   fs=10kHz: $K_p < 0.0012/0.0001 = 12$ → 安全范围很大，**正常PI参数(0.5~2.0)不会受延迟影响**
   但当 Kp 极高（如无差拍控制）时，延迟必须考虑

## 四、综合应用题（★★★★☆）

### 题目8：级联控制的根轨迹设计
**问题：** 典型的电机三环级联控制中，内环（电流环）带宽通常是外环（速度环）的5~10倍。请用根轨迹方法解释：
1. 为什么电流环带宽必须远大于速度环带宽？
2. 如果电流环带宽仅3倍于速度环，根轨迹会如何变化？
3. 给出基于根轨迹的电流环-速度环带宽比设计准则

**参考答案：**
1. 速度环设计时，电流环被近似为一阶惯性环节 $1/(τ_c s + 1)$。只有当电流环极点 τ_c 远小于速度环主导极点时，这一近似才成立。根轨迹上表现为：电流环极点远离速度环区域，不发生交互。

2. 带宽比仅3倍 → 电流环极点与速度环极点接近 → 根轨迹上两条支路互相"排斥" → 原本的共轭极点对阻尼减小 → 速度环超调增大，甚至不稳定。

3. **设计准则：**
   - 带宽比 ≥ 5: 电流环极点与速度环极点完全解耦，速度环设计可忽略电流环动态
   - 带宽比 3~5: 需将电流环极点纳入速度环根轨迹分析，适当降低速度环增益
   - 带宽比 < 3: 不可接受，必须提高电流环带宽或降低速度环带宽

## 五、前沿综合题（★★★★★）

### 题目9：参数不确定性下的根轨迹鲁棒设计
**问题：** 电机运行中，电感 L 因磁饱和可能变化 ±30%，电阻 R 因温升可能变化 +50%。电流环采用PI控制。
1. 画出 L 和 R 在不确定范围内的根轨迹族（示意即可）
2. 设计 Kp, Ki 使得在所有参数组合下，闭环极点ζ≥0.5 且 ts≤10ms
3. 讨论：追求鲁棒性的代价是什么？

**参考答案：**
1. L变化主要影响 ωn（极点离原点距离），R变化主要影响阻尼（极点向实轴靠拢或远离）。L_min + R_max 是最差工况（阻尼最小）。

2. 按最差工况设计（L=0.7×L_nom, R=1.5×R_nom）：
   ζ_min ≥ 0.5 且 σ_min = ζωn ≥ 400
   解得：Kp ≥ 2ζωn·L_min - R_max
   正常工况下（L_nom, R_nom）：ζ 会偏大，响应偏保守。

3. **代价：** 鲁棒设计以最差工况为准，正常工况下性能不是最优（响应偏慢、带宽偏低）。这是性能与鲁棒性的经典trade-off。

### 题目10：基于根轨迹的电流环自适应增益调度设计
**问题：** 设计一个基于根轨迹的自适应PI参数更新方案：
1. 假设在线辨识出当前电机参数 L_est, R_est，写出PI参数的自适应更新公式（使闭环极点保持在ζ=0.707, ωn=1500）
2. 画出自适应PI的软件实现流程图（伪代码级）
3. 讨论：在什么工况下这种自适应最有价值？什么工况下不必要？

**参考答案：**
1. 更新公式：
   ```
   Kp = 2 × 0.707 × 1500 × L_est - R_est
   Ki = 1500² × L_est
   ```

2. 伪代码：
   ```
   if parameter_update_flag:
       L_est = rls_identify_L()
       R_est = rls_identify_R()
       if L_est > 0.5*L_nom && L_est < 2.0*L_nom:  // 合理性检查
           Kp_new = 2*0.707*1500*L_est - R_est
           Ki_new = 1500² * L_est
           apply_lowpass_filter(Kp, Kp_new, 0.95)  // 平滑过渡
           apply_lowpass_filter(Ki, Ki_new, 0.95)
   ```

3. **最有价值的工况：**
   - 宽调速范围（弱磁区电感显著变化）
   - 大负载变化（温度导致R大幅偏移）
   - 高动态伺服（需要始终保持最优响应）
   
   **不必要的工况：**
   - 恒速恒载运行（参数基本不变）
   - 对动态响应要求不高的风机泵类负载
   - 参数辨识精度不足以支持自适应（辨识噪声>参数变化）
