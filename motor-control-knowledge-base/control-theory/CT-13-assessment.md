## 一、概念计算题（★★★★☆）

### 题目1：一阶LQR的解析解
**问题：** 一阶电流环：A=-500, B=1000, Q=[100], R=[0.01]。
1. 写出Riccati方程并求解P
2. 计算最优增益K
3. 闭环极点是多少？与期望的极点配置结果比较

**参考答案：**
1. ARE: $2AP - B^2P^2/R + Q = 0$
   代入：$-1000P - 10^6P^2/0.01 + 100 = 0$
   → $-1000P - 10^8P^2 + 100 = 0$
   → $10^8P^2 + 1000P - 100 = 0$
   
   $P = \frac{-1000 + \sqrt{10^6 + 4\times10^8\times100}}{2\times10^8} = \frac{-1000 + \sqrt{10^6+4\times10^{10}}}{2\times10^8}$
   
   注意运算：4×10⁸×100 = 4×10¹⁰，√(4.0001×10¹⁰) ≈ 200002.5
   
   P = (-1000 + 200002.5)/(2×10⁸) = 199002.5/(2e8) = **9.95×10⁻⁴**
   
   取正根（P正定）。

2. $K = R^{-1}B^TP = (1/0.01)\times1000\times9.95\times10^{-4} = 100\times0.995 = **99.5**$

3. $A-BK = -500 - 1000\times99.5 = -100000$
   闭环极点：s = **-100,000**（时间常数10μs！）
   
   对比：极点配置若选ωc=10000（√(q/r)=√(10000)=100），极点=-10000
   LQR给出的带宽与√(q/r)一致。这里Q/R=10000→ωc=10000→极点=-10000左右。

### 题目2：PMSM电流环LQR的K矩阵
**问题：** PMSM在ωe=500rad/s, L=0.5mH, R=0.1Ω。LQR设计：Q=diag(1000, 5000), R=diag(0.01, 0.01)。
1. 解释为什么q₂>q₁（iq权重大于id）
2. 求解ARE得到K矩阵（使用简化近似公式）
3. 该K对应的闭环极点

**参考答案：**
1. q₂=5000 > q₁=1000 → iq误差的惩罚是id的5倍 → iq响应优先级高于id。
   这符合FOC的id=0策略：id偏离零不是灾难（弱磁需要负id），但iq偏离参考直接导致转矩误差。

2. 近似公式（忽略ωe对ARE的影响）：
   k₁₁ ≈ √(q₁/r + (R/L)²) - R/L = √(1000/0.01 + 200²) - 200 = √(100000+40000) - 200 = √140000 - 200 = 374.2 - 200 = **174.2**
   k₂₂ ≈ √(5000/0.01 + 200²) - 200 = √(500000+40000) - 200 = √540000 - 200 = 734.8 - 200 = **534.8**
   k₁₂ = ωeL = 500×0.0005 = **0.25**
   k₂₁ = -ωeL = **-0.25**

   $$K = \begin{bmatrix} 174.2 & 0.25 \\ -0.25 & 534.8 \end{bmatrix}$$

3. 完整求解需MATLAB。近似闭环极点：
   - id环：s₁≈-L⁻¹(Bk₁₁+R)=-(174.2×2000+200)≈**-348,600**
   - iq环：s₂≈-(534.8×2000+200)≈**-1,069,800**
   这是极高的带宽（~55kHz和~170kHz），说明Q/R=500000配置过于激进→实际中应降低Q或增大R。

### 题目3：Bryson法则设计Q和R
**问题：** 一台PMSM：id_max=10A, iq_max=15A, ωm_max=300rad/s, ud_max=uq_max=200V。
1. 用Bryson法则给出Q和R的对角元素
2. 计算对应的等效带宽（近似）
3. Bryson法则的优势和局限

**参考答案：**
1. q₁ = 1/id_max² = 1/100 = **0.01**
   q₂ = 1/iq_max² = 1/225 = **0.00444**
   q₃ = 1/ωm_max² = 1/90000 = **1.11×10⁻⁵**
   r₁ = r₂ = 1/u_max² = 1/40000 = **2.5×10⁻⁵**

2. 电流环等效带宽：ωc_id ≈ √(q₁/r₁) = √(0.01/2.5e-5) = √400 = **20 rad/s ≈ 3.2Hz**
   这极低！因为Bryson法则给出的Q/R太小→保守设计。
   
   实际需要调整：将q₁放大1000倍→q₁=10→ωc≈√(10/2.5e-5)=√400000=632rad/s≈100Hz。

3. 优势：提供"不会太差"的初始值，有物理依据。
   局限：结果通常过于保守（带宽太低）。需要经验性放大Q（10~1000倍）才能获得实用性能。

## 二、分析设计题（★★★★★）

### 题目4：LQR的相位裕度保证
**问题：** LQR理论上保证至少60°相位裕度和无穷大幅值裕度（在输入端）。
1. 这个性质是如何推导的？（简述证明思路）
2. 在PMSM电流环LQR中，这一保证的实际意义是什么？
3. 当存在数字延迟（一拍延迟）时，60°相位裕度保证还成立吗？

**参考答案：**
1. 证明思路（回路传函法）：
   输入端的回路传函 L(s)=K(sI-A)⁻¹B
   可以证明 1+L(jω) 的Nyquist曲线不会进入以(-1,0)为圆心、半径为1的圆内。
   这意味着：增益裕度=∞（Nyquist曲线不穿越负实轴），相位裕度≥60°。
   证明用到Riccati方程的性质和Return Difference不等式。

2. 实际意义：
   - 即使电机参数有一定误差（±30%），闭环仍稳定
   - K矩阵"自带鲁棒性"，不需要额外设计鲁棒控制器
   - 这解释了为什么LQR调参比极点配置更安全——选择"差"的Q/R只会降低性能，不会引起不稳定

3. **不成立！** 数字延迟引入了e^(-Td·s)因子，改变了回路传函的相位特性。
   对于Ls+Rs的一阶对象：
   - 无延迟：LQR保证60°+相位裕度
   - Ts=62.5μs (16kHz)：延迟在ω=10000rad/s处引入-36°相位→相位裕度剩余约24°
   - Ts=125μs (8kHz)：在ω=5000rad/s处引入-36°→相位裕度约24°
   
   **对策：** 在离散域直接求解DARE（离散LQR）→离散LQR在z平面保证类似性质。

### 题目5：LQG的分离原理
**问题：** LQG=LQR+Kalman。分离原理说两者可独立设计。
1. 写出LQG的完整闭环状态方程（含观测器误差的动态）
2. 证明：闭环极点是LQR极点（A-BK的特征值）和Kalman极点（A-LC的特征值）的并集
3. 分离原理成立的条件是什么？在电机控制中是否满足？

**参考答案：**
1. 真实系统：ẋ=Ax+Bu+w
   观测器：x̂̇=Ax̂+Bu+L(y-Cx̂)
   控制律：u=-Kx̂
   
   状态估计误差：e=x-x̂, ė=(A-LC)e+w-Lv
   真实状态：ẋ=Ax-BKx̂+w = (A-BK)x+BKe+w

2. 闭环系统：
   $$\begin{bmatrix} \dot{\mathbf{x}} \\ \dot{\mathbf{e}} \end{bmatrix} = \begin{bmatrix} \mathbf{A-BK} & \mathbf{BK} \\ \mathbf{0} & \mathbf{A-LC} \end{bmatrix} \begin{bmatrix} \mathbf{x} \\ \mathbf{e} \end{bmatrix} + \begin{bmatrix} \mathbf{w} \\ \mathbf{w-Lv} \end{bmatrix}$$
   
   特征值：det(sI-(A-BK))·det(sI-(A-LC))=0 → 极点=并集。✓

3. 成立条件：
   - 系统线性（PMSM是线性的吗？→在工作点线性化后近似成立）
   - 无模型不确定性（实际中电机参数随时间/温度/磁饱和变化）
   - 噪声为高斯白噪声（实际中电流采样噪声近似满足）
   
   在电机控制中**近似满足**，分离原理提供了良好的设计指导，但需注意：
   - 参数变化时LQR和Kalman性能都退化，退化可能互相耦合
   - 为保证鲁棒性，通常观测器比LQR快5倍以上

### 题目6：LQR vs PI的能耗对比
**问题：** 同一台电机，PI级联和LQR分别达到相同的电流环带宽（1000Hz）。
1. 对比两种方法在阶跃响应中的电压使用情况（峰值和RMS）
2. LQR的R矩阵如何影响电压使用？
3. 如果目标是"最小化铜损"，Q应如何设计？

**参考答案：**
1. PI（Kp=6, Ki=1200, +前馈解耦 ωeL=0.5）：
   阶跃iq 0→10A：uq峰值≈60V（Kp×10） + 解耦≈5V = 65V
   稳态uq≈R×10=1.5V（仅电阻压降，反电动势被前馈补偿）
   
   LQR（Q=diag(1,1000), R=diag(0.01,0.01)）：
   K≈[1.4 0.5; -0.5 14]
   阶跃iq：uq峰值≈K₂₂×10 = 140V（更激进！）
   但iq响应时间：LQR约0.5ms vs PI约1ms
   
   **LQR更快但需要更高电压**——这是Q/R权衡的体现。

2. R增大→惩罚控制能量→K变小→电压使用降低→响应变慢
   R减小→鼓励使用控制能量→K变大→电压需求增大→响应更快

3. 最小化铜损 = 最小化 i²R = 最小化 xᵀQ_cu·x, Q_cu=diag(R, R, 0)
   同时不希望控制电压过大 → R适中
   
   总代价：J = ∫(R·id²+R·iq² + r·ud²+r·uq²)dt
   Q应设：q₁=q₂=R（电阻值），R=diag(r,r)
   
   物理意义：电阻压降的功率损失和控制电压的"代价"在同一个量纲中比较。

## 三、综合编程题（★★★★★）

### 题目7：DARE的迭代求解
**问题：** 离散ARE（DARE）没有像连续ARE那样的闭式解。在DSP上需要数值迭代求解。
1. 写出迭代求解DARE的算法伪代码（基于Riccati差分方程迭代）
2. 迭代初值P₀如何选择？
3. 在16kHz控制频率下，能否每个周期重新求解DARE？（计算量分析）

**参考答案：**
1. 迭代算法（Kleinman法或直接迭代）：
   ```
   P = Q  // 初值
   for iter = 1 to max_iter:
       P_prev = P
       K = (R + B'*P*B) \ (B'*P*A)
       P = A'*P*A - A'*P*B*K + Q
       if norm(P - P_prev) < tol:
           break
   K_final = (R + B'*P*B) \ (B'*P*A)
   ```

2. P₀的选择：
   - P₀=Q（最简单，保证收敛到稳定解）
   - P₀=I（单位阵，对等权初始猜测）
   - P₀=0（从零开始，保守但收敛）

3. 计算量：
   - 每次迭代：矩阵乘法4×4×4 ≈ 64次浮点乘法 + 矩阵求逆（2×2）≈6次乘法
   - 通常需10~20次迭代收敛 → 约1000~2000次浮点乘法
   - Cortex-M4 @168MHz: ~6~12μs
   - 16kHz周期62.5μs → 占用率10~19%
   
   **不建议每周期求解DARE！** 应在参数变化时求解（如每100ms），平时查表或使用固定K。

### 题目8：有限时域LQR用于位置伺服
**问题：** 点对点位置伺服（如转台从0°转到90°）可以用有限时域LQR。
1. 与无限时域LQR的区别是什么？
2. 写出有限时域LQR的迭代Riccati差分方程
3. 在电机位置伺服中，何时使用有限时域LQR优于无限时域？

**参考答案：**
1. 区别：
   - 无限时域：K是常数，P由ARE解出，适用于调节问题（保持状态在原点附近）
   - 有限时域：K是时变的（K₀, K₁, ..., K_{N-1}），P由Riccati差分方程反向递推，适用于跟踪问题（从起点到终点有确定的时间要求）

2. 反向递推（从N到0）：
   P_N = P_f（终端权重矩阵）
   for k = N-1, N-2, ..., 0:
       K_k = (R + B'P_{k+1}B)⁻¹(B'P_{k+1}A)
       P_k = Q + A'P_{k+1}A - A'P_{k+1}B(R+B'P_{k+1}B)⁻¹B'P_{k+1}A

3. 适用场景：
   - 转台快速定位（0→90°，100ms内完成）
   - 有限时域可设计"先加速后减速"的最优轨迹
   - 终端权重P_f可惩罚终端位置误差（保证精确停在90°）
   
   无限时域LQR更适合：
   - 持续跟踪（如速度控制、恒速运行）
   - 调节问题（保持位置不变）
   
   **实用方案：** 在线计算有限时域LQR的时变增益序列{K₀,...,K_{N-1}}，离线存储在FLASH中，执行时按时间步查表使用。

### 题目9：LQR与MPC的平滑切换
**问题：** 设计一个LQR→MPC的混合控制策略：
1. 正常工况使用LQR（计算量小）
2. 当状态接近约束边界时（电压/电流即将饱和），切换到MPC
3. 写出切换条件和切换时的bumpless transfer方案

**参考答案：**
1-2. 混合策略架构：
   ```
   if (|u_lqr| < 0.8*Vmax && |x| < 0.9*Imax):
       u = -K_lqr * x  // 正常工况，快速LQR
   else:
       u = solve_mpc(x, constraints)  // 临近约束，带约束MPC
   ```

3. Bumpless transfer：
   从LQR切换到MPC：
   - MPC的初始解 u₀ 设为当前LQR输出 → 热启动QP求解器
   - 第一个MPC控制周期用LQR的输出作为参考轨迹
   
   从MPC切换回LQR：
   - 不直接切，而是线性插值过渡5个周期：
   ```
   for i = 0 to 4:
       alpha = i/4.0
       u = (1-alpha)*u_mpc + alpha*u_lqr
   ```

   更精细的方案：始终运行MPC（1步预测≈LQR的退化形式），根据约束状态调整预测步长N。
   - 无约束时：N=0 → 退化为LQR
   - 约束激活时：N=3~5 → 完整MPC

### 题目10：Q和R的自动调参策略
**问题：** 为LQR电流环设计基于性能指标的自动调参策略：
1. 定义性能指标：超调量PO、调节时间ts、电压峰值Vpeak
2. 如果PO>5%，应如何调整Q和R？
3. 写出自动调参的迭代伪代码

**参考答案：**
1. 性能采集（从阶跃响应）：
   - PO = (i_peak - i_ref)/i_ref × 100%
   - ts = 电流进入±2%稳态带的时间
   - Vpeak = max(|ud|, |uq|) in transient

2. 调整规则：
   - PO>5%：q过大/r过小→系统过于激进→①增大r（降低控制能量）或②降低q
     建议：r_new = r_old × 1.5（保守调整）
   - ts>目标值：q过小→响应太慢→增大q
     建议：q_new = q_old × 2.0
   - Vpeak>0.9×Vmax：接近饱和→增大r
     建议：r_new = r_old × 2.0

3. 自动调参伪代码：
   ```
   def auto_tune_lqr(A, B, Q_init, R_init, target_ts, target_PO, Vmax):
       Q, R = Q_init, R_init
       for epoch in range(10):
           K = solve_are(A, B, Q, R)
           y = run_step_response(K)  // 硬件在环测试
           PO = measure_overshoot(y)
           ts = measure_settling_time(y)
           Vpeak = measure_voltage_peak()
           
           if PO > target_PO * 1.2:
               R *= 1.5
           elif ts > target_ts * 1.2:
               Q *= 2.0  // for iq channel
           elif Vpeak > 0.9 * Vmax:
               R *= 2.0
           else:
               break  // 满足要求
       return Q, R, K
   ```
   
   **注意：** 在实际电机上自动调参需要注意安全——每次迭代前应验证K不会导致不稳定（先在小信号/低电压下测试）。
