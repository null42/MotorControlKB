# CT-09: 系统校正设计

**副标题：超前/滞后/陷波——用频率整形重塑电机控制动态**
**难度：** ★★★★★ 专家级
**适用对象：** 电机控制工程师、伺服驱动开发者
**前置知识：** Bode图与频率响应、根轨迹分析、PID控制、复变函数

---

## 1. 📌 核心摘要

**一句话：** 系统校正（补偿）是通过在控制回路中串联超前、滞后、陷波等校正网络，对系统频率特性进行"外科手术式"整形——在需要的频段加增益/相位，在不需要的频段减增益——从而同时满足稳定性、响应速度和抗扰能力的要求。

**认知挂钩：** 如果说PID是"通用扳手"，那么串联校正器就是"专用手术刀"。PID调三个参数Kp/Ki/Kd，只能调整增益和积分/微分。但当系统有特定频段的谐振峰、或者相位滞后严重时，PID的"自由度"不够——这时需要专门的校正网络。

**核心流程：**
```
确定性能指标（相位裕度、带宽、稳态误差）
    ↓
分析原系统Bode图，找出不足
    ↓
选择校正器类型（超前/滞后/陷波）
    ↓
按设计步骤计算校正器参数
    ↓
验证校正后的Bode图/根轨迹/时域响应
```

**三种校正器对比：**
| 校正器 | 作用 | 适用场景 | 副作用 |
|--------|------|---------|--------|
| 超前校正 | 增加相位裕度、提高带宽 | 相位裕度不足、响应慢 | 放大高频噪声 |
| 滞后校正 | 减小稳态误差、压低高频增益 | 稳态精度不够 | 降低相位裕度（低频段） |
| 陷波滤波器 | 消除特定频率谐振 | 机械谐振、电流谐波 | 引入额外相位滞后 |

## 2. 🤔 问题引入

**场景1：** 你的伺服驱动器电流环调试好了，带宽也够。但接上负载后，电机在某个转速（比如1200rpm）开始剧烈抖动——转速计显示 ±50rpm 的振荡。频谱分析仪显示振荡频率约 185Hz。排查发现传动轴系存在机械谐振模态，共振频率恰好 185Hz。降低速度环增益能缓解，但响应变慢了。

**问题：** 能否在"不牺牲响应速度"的前提下消除这个机械谐振？

**场景2：** 某台大惯量电机的速度环，稳态速度误差有 3rpm（给定1000rpm），这在一些精密应用中不可接受。增大 Ki 可以消除误差，但增大到一定程度后系统开始振荡。

**问题：** 如何在不引起振荡的前提下消除稳态误差？

**答案都是"校正网络"：** 场景1用陷波滤波器，场景2用滞后校正。

## 3. 💡 直观理解

### 3.1 校正器的频域直觉

把Bode图想象成一块"橡皮泥"：

- **超前校正** = 在中频段"推高"相位曲线 → 增加相位裕度 → 系统更稳定、响应更快
- **滞后校正** = 在低频段"抬高"增益曲线 → 增加低频增益 → 减小稳态误差；同时"压低"高频段 → 不恶化稳定性
- **陷波滤波器** = 在特定频率"挖一个坑" → 抑制该频率分量 → 消除谐振

### 3.2 传递函数形态

**超前校正：**
$$
G_c(s) = K_c \cdot \frac{s + z}{s + p}, \quad z < p
$$

- 零点z在极点p的左边 → 先"超前"增加相位，后增益回落

**滞后校正：**
$$
G_c(s) = K_c \cdot \frac{s + z}{s + p}, \quad z > p
$$

- 极点p在零点z的左边 → 先"滞后"降低相位，但增加了低频增益

**陷波滤波器：**
$$
G_c(s) = \frac{s^2 + 2\zeta_z \omega_n s + \omega_n^2}{s^2 + 2\zeta_p \omega_n s + \omega_n^2}, \quad \zeta_z < \zeta_p
$$

- 在 ω=ωn 处增益骤降

### 3.3 电机控制的校正直觉

| 电机问题 | 校正对策 |
|---------|---------|
| 电流环超调大 | 超前校正（增大相位裕度） |
| 速度环稳态有差 | 滞后校正（提升低频增益） |
| 特定转速振动 | 陷波滤波器（抑制谐振频率） |
| 高频噪音放大 | 滞后校正（压低高频增益） |
| 动态响应慢 + 稳态有差 | 滞后-超前复合校正 |

## 4. 🔬 技术原理

### 4.1 超前校正器（Lead Compensator）

#### 4.1.1 传递函数

$$
G_{lead}(s) = K_c \cdot \frac{Ts + 1}{\alpha Ts + 1}, \quad 0 < \alpha < 1
$$

其中 T 是时间常数，α 决定最大相位超前角。

#### 4.1.2 Bode图特征

- 低频增益：$K_c$
- 高频增益：$K_c / \alpha > K_c$（所以超前校正也放大了高频噪声！）
- 最大相位超前角发生的频率：

$$
\omega_m = \frac{1}{T\sqrt{\alpha}}
$$

- 最大相位超前角：

$$
\phi_m = \arcsin\left(\frac{1-\alpha}{1+\alpha}\right)
$$

- 对应的幅值增益：$20\log_{10}(1/\sqrt{\alpha})$ dB

#### 4.1.3 设计步骤（Bode图法）

1. 确定期望的相位裕度 $PM_{des}$
2. 计算需要补充的相位：$\phi_{need} = PM_{des} - PM_{orig} + 5° \sim 15°$（补偿超前校正引入的增益变化导致剪切频率偏移）
3. 由 $\phi_m = \phi_{need}$ 计算 α：

$$
\alpha = \frac{1-\sin\phi_m}{1+\sin\phi_m}
$$

4. 在Bode图上找到 $-10\log_{10}(1/\alpha)$ dB 对应的频率，设为新的剪切频率 $\omega_c'$，即 $\omega_m$
5. 计算 T：

$$
T = \frac{1}{\omega_m\sqrt{\alpha}}
$$

6. 确定 Kc 使低频增益满足稳态误差要求

### 4.2 滞后校正器（Lag Compensator）

#### 4.2.1 传递函数

$$
G_{lag}(s) = K_c \cdot \frac{Ts + 1}{\beta Ts + 1}, \quad \beta > 1
$$

#### 4.2.2 Bode图特征

- 低频增益：$K_c$
- 高频增益：$K_c / \beta < K_c$（滞后校正压低高频增益）
- 最大相位滞后角发生的频率：$\omega_m = \frac{1}{T\sqrt{\beta}}$
- 该相位滞后是**不需要的副作用**——设计时应使 $\omega_m$ 远小于剪切频率，使滞后角不影响稳定性

#### 4.2.3 设计步骤（Bode图法）

1. 确定需要增加的稳态增益：$K_v = \frac{1}{e_{ss}}$
2. 设 $\beta = K_v$（近似）
3. 选择 T 使得转角频率 $1/T$ 比剪切频率低至少一个十倍频程：

$$
\frac{1}{T} \approx \frac{\omega_c}{10}
$$

4. 验证校正后相位裕度是否满足要求

### 4.3 滞后-超前校正器（Lag-Lead Compensator）

结合两者优势：低频段提供滞后校正（提升稳态精度），高频段提供超前校正（改善动态）。

$$
G_c(s) = K_c \cdot \frac{(T_{lag}s + 1)(T_{lead}s + 1)}{(\beta T_{lag}s + 1)(\alpha T_{lead}s + 1)}
$$

设计方法：先设计超前部分满足动态要求，再设计滞后部分满足稳态要求。

### 4.4 陷波滤波器（Notch Filter）

#### 4.4.1 传递函数

$$
G_{notch}(s) = \frac{s^2 + 2\zeta_z \omega_0 s + \omega_0^2}{s^2 + 2\zeta_p \omega_0 s + \omega_0^2}
$$

- $\omega_0$：陷波中心频率（等于谐振频率）
- $\zeta_z$：零点阻尼（决定陷波深度，越小越深）
- $\zeta_p$：极点阻尼（决定陷波宽度，越大越宽）

#### 4.4.2 频率特性

- 在 $\omega = \omega_0$ 处，增益为 $20\log_{10}(\zeta_z/\zeta_p)$ dB
- 陷波深度 = $-20\log_{10}(\zeta_p/\zeta_z)$ dB
- 相位在 $\omega_0$ 附近急剧变化（从0°降到-180°再回到0°）

#### 4.4.3 设计要点

1. $\omega_0$ 必须精确匹配机械谐振频率——偏移5%就会使抑制效果减半
2. $\zeta_p / \zeta_z$ 决定陷波深度：典型值 $\zeta_z = 0.01$, $\zeta_p = 0.1 \to$ 20dB衰减
3. 陷波滤波器在 $\omega < \omega_0$ 区域引入相位滞后——可能降低相位裕度 → 需验证

### 4.5 根轨迹设计法

校正器还可以在根轨迹上设计：

**超前校正 = 增加一个零点 + 一个更远的极点**
- 零点吸引根轨迹向左偏转 → 改善阻尼
- 极点防止增益无限增大

**滞后校正 = 增加一个靠近原点的零极点对（偶极子）**
- 偶极子几乎不改变根轨迹形态
- 但提高稳态增益（零极点比）

## 5. 🔗 交叉视角：校正器与电机控制的桥梁

### 5.1 电流环超前校正：提升相位裕度

**问题：** 电流环在高带宽（>2kHz）设计时，数字延迟和滤波器延迟导致相位滞后严重，PI控制器可能无法满足相位裕度要求。

**方案：** 在PI控制器输出串联超前校正器

$$
G_c(s) = \frac{T_{lead}s + 1}{\alpha T_{lead}s + 1}
$$

**参数设计：**
- 测量电流环开环在期望剪切频率处的相位
- 计算需要补偿的相位 $\phi_{need}$
- 设计 $\alpha$ 和 $T_{lead}$

**实际效果：** 某伺服驱动器电流环，带宽从 1.5kHz 提升到 2.8kHz，相位裕度保持 >45°。

### 5.2 速度环滞后校正：消除稳态误差

**问题：** 纯比例速度控制存在稳态误差 $e_{ss} = \omega_{ref} / (1+K_v)$。增大 Ksp 受限于稳定性。

**方案：** 在速度环串入滞后校正器

$$
G_{lag}(s) = \frac{T_{lag}s + 1}{\beta T_{lag}s + 1}, \quad \beta > 1
$$

低频增益提高 β 倍 → 稳态误差降为原来的 1/β。

**注意：** 滞后校正引入的极点-零点对在低频段，不影响中高频稳定性。但如果 β 过大，可能放大低频扰动（如负载转矩波动）。

### 5.3 陷波滤波器：机械谐振抑制

**问题背景：** 伺服系统中，电机通过联轴器、丝杠等传动机构连接负载。整个轴系在特定频率下出现扭转谐振：

$$
f_{res} = \frac{1}{2\pi}\sqrt{\frac{K_{shaft}}{J_{load}}}
$$

当速度环带宽接近 f_res 时，电流环的转矩输出在谐振频率处被放大 → 机械振动 → 速度反馈振荡 → 控制系统正反馈 → 加剧振荡。

**陷波滤波器解决方案：**
1. 频谱分析确定谐振频率 f_res
2. 设计陷波滤波器，$\omega_0 = 2\pi f_{res}$
3. 将陷波滤波器串联在速度环输出（转矩/电流给定之前）
4. 验证陷波后开环Bode图：谐振峰消失，相位裕度恢复

**工程实例：**
- 谐振频率：185 Hz
- 陷波参数：$\omega_0 = 1162$ rad/s, $\zeta_z = 0.05$, $\zeta_p = 0.5$ → 20dB 抑制
- 结果：速度振荡消除，速度环带宽从 20Hz 恢复到 50Hz

### 5.4 多陷波：多个谐振模态

大型伺服系统可能有多个谐振模态（如两质量系统有低频和高频两个谐振峰）。需串联多个陷波滤波器分别抑制。

**FOC代码实现：** 陷波滤波器通常实现为二阶IIR数字滤波器：

```c
float notch_filter(notch_t *nf, float input) {
    float output = nf->b0 * input + nf->b1 * nf->x1 + nf->b2 * nf->x2
                   - nf->a1 * nf->y1 - nf->a2 * nf->y2;
    nf->x2 = nf->x1; nf->x1 = input;
    nf->y2 = nf->y1; nf->y1 = output;
    return output;
}
```

系数由双线性变换从 s 域映射到 z 域得到。

## 6. 🎯 工程案例

### 案例1：伺服驱动器电流环超前校正

**电机参数：** L=0.5mH, R=0.08Ω, fs=16kHz

**原系统：** PI控制，Kp=0.4, Ki=80，电流环带宽约900Hz，相位裕度约38°（偏低）

**目标：** 带宽提升至1500Hz，相位裕度>50°

**设计过程：**

1. 分析原系统在期望剪切频率 $\omega_c' = 2\pi \times 1500 = 9425$ rad/s 处的相位
2. 原系统相位滞后约 -170° → 相位裕度仅 10°（不足！）
3. 需要补充相位约 $\phi_{need} = 50° - 10° + 10° = 50°$
4. $\alpha = \frac{1-\sin 50°}{1+\sin 50°} = \frac{1-0.766}{1+0.766} = 0.132$
5. $\omega_m = \omega_c' = 9425$ rad/s
6. $T = \frac{1}{\omega_m\sqrt{\alpha}} = \frac{1}{9425 \times 0.363} = 2.92 \times 10^{-4}$
7. 超前校正器：

$$
G_{lead}(s) = \frac{2.92 \times 10^{-4}s + 1}{0.132 \times 2.92 \times 10^{-4}s + 1} = \frac{2.92 \times 10^{-4}s + 1}{3.86 \times 10^{-5}s + 1}
$$

**验证：** 校正后相位裕度 52°，带宽 1520Hz，电流阶跃响应调节时间从 1.8ms 缩短到 0.9ms。

### 案例2：陷波滤波器抑制185Hz机械谐振

**系统：** 750W伺服电机 + 滚珠丝杠，联轴器刚度有限
**现象：** 电机在1200rpm附近（对应185Hz的力矩脉动频率）出现剧烈振动
**诊断：** 速度反馈频谱分析显示185Hz处能量急剧放大

**陷波设计：**
- $\omega_0 = 2\pi \times 185 = 1162$ rad/s
- $\zeta_z = 0.02$（深陷波，40dB抑制）
- $\zeta_p = 0.3$（适中宽度，覆盖±5%不确定范围）

**DSP离散化实现（Tustin变换）：**

采样周期 Ts = 62.5μs (16kHz)

```c
typedef struct {
    float b0, b1, b2, a1, a2;
    float x1, x2, y1, y2;
} notch_filter_t;

void notch_init(notch_filter_t *nf, float w0, float zeta_z, float zeta_p, float Ts) {
    float w0Ts = w0 * Ts;
    float den = 4.0f + 4.0f * zeta_p * w0Ts + w0Ts * w0Ts;
    nf->b0 = (4.0f + 4.0f * zeta_z * w0Ts + w0Ts * w0Ts) / den;
    nf->b1 = (2.0f * w0Ts * w0Ts - 8.0f) / den;
    nf->b2 = (4.0f - 4.0f * zeta_z * w0Ts + w0Ts * w0Ts) / den;
    nf->a1 = (2.0f * w0Ts * w0Ts - 8.0f) / den;
    nf->a2 = (4.0f - 4.0f * zeta_p * w0Ts + w0Ts * w0Ts) / den;
}
```

**效果：** 185Hz分量衰减33dB，速度振荡消失，同时速度环带宽从被迫降低的15Hz恢复到50Hz。

## 8. 🚀 前沿拓展

### 8.1 自适应陷波滤波器（ANF）

机械谐振频率可能随负载惯量、温度、磨损而变化。固定频率的陷波滤波器可能失效。

**自适应陷波方案：**
- 在线FFT分析速度/电流频谱
- 检测谐振峰频率
- 自动调整陷波滤波器中心频率
- 平滑过渡陷波参数

算法可以用梯度下降法或锁相环结构实时跟踪谐振频率。

### 8.2 鲁棒校正设计（H∞方法）

传统频域校正假设模型精确。实际电机参数存在不确定性。H∞校正通过求解优化问题，设计使系统在最差工况下仍保持稳定的校正器。

对于电机控制，可以将电机参数不确定性建模为乘性摄动，设计鲁棒校正器保证所有工况下相位裕度 >30°。

### 8.3 基于非线性校正的摩擦力补偿

伺服系统低速运行时，摩擦力（Stribeck效应）导致速度爬行。传统线性校正器难以处理这种非线性。

**方案：** 在速度环PI输出叠加摩擦力前馈补偿项——基于速度的摩擦力模型（静摩擦+库仑摩擦+粘滞摩擦），实质是一种"非线性校正"。

### 8.4 分数阶校正器

传统校正器是整数阶的（s的一阶零点/极点）。分数阶校正器（如 $s^{0.5}$）提供更灵活的频率整形能力，在特定应用中可能以更低阶数达到相同效果。

**电机应用：** 分数阶PI^λD^μ控制，相比传统PID，在电流环中可以获得更好的鲁棒性。
