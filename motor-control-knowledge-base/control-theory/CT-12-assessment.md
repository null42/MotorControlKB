## 一、参数计算题（★★★★☆）

### 题目1：Ackermann公式计算状态反馈增益
**问题：** 某简化电机电流环（一阶）：A=[-500], B=[1000]，期望闭环极点为 s=-2000。
1. 用Ackermann公式计算K
2. 验证 A-BK 的特征值是否等于 -2000
3. 写出控制律 u = -Kx + Nr，计算N（使id→id_ref）

**参考答案：**
1. 期望特征多项式：φ_c(s)=s+2000 → a₀=2000
   能控性矩阵：C=[B]=[1000]→C⁻¹=[0.001]
   φ_c(A)=A+2000I=-500+2000=1500
   K = [1]·C⁻¹·φ_c(A) = 1×0.001×1500 = **1.5**

2. A-BK = -500-1000×1.5 = -2000 → 特征值 = **-2000** ✓

3. N = [C(-(A-BK))⁻¹B]⁻¹ = [1×(2000)⁻¹×1000]⁻¹ = [0.5]⁻¹ = **2.0**
   控制律：u = -1.5·id + 2.0·id_ref

### 题目2：PMSM二阶状态反馈K矩阵计算
**问题：** 一台PMSM在ωe=800rad/s工作点，参数L=0.6mH, R=0.12Ω。设计状态反馈电流环，期望带宽1500Hz（临界阻尼）。
1. 计算K矩阵
2. 验证解耦效果：K₁₂和K₂₁的物理意义
3. 比较与"PI(Kp=1.8, Ki=360)+前馈解耦(±ωe·L·iq/d)"的异同

**参考答案：**
1. ωc = 2π×1500 = 9425 rad/s
   k₁₁ = k₂₂ = 2ωc·L - R = 2×9425×0.0006 - 0.12 = 11.31 - 0.12 = **11.19**
   k₁₂ = ωe·L = 800×0.0006 = **0.48**
   k₂₁ = -ωe·L = **-0.48**

   $$K = \begin{bmatrix} 11.19 & 0.48 \\ -0.48 & 11.19 \end{bmatrix}$$

2. K₁₂=0.48 → ud中包含+0.48·iq → 这就是前馈解耦项 -ωeL·iq 的反号！
   状态反馈公式：ud = -11.19id - 0.48iq → 展开恰好等于 -(11.19id + ωeL·iq)
   **状态反馈天然包含前馈解耦**

3. PI+前馈方案：
   - ud_PI = Kp(id_ref-id) + Ki∫, Kp=1.8, Ki=360
   - ud_dec = -ωeL·iq = -0.48iq
   - ud_total = 1.8(id_ref-id)-0.48iq（忽略积分项）
   
   状态反馈方案：
   - ud = -11.19id - 0.48iq + 11.19id_ref
   
   差异：状态反馈的"P增益"（11.19）远大于PI的Kp（1.8）→响应更快！

### 题目3：三阶状态反馈的前馈增益N计算
**问题：** PMSM三阶系统（id, iq, ωm），A(3×3), B(3×2), C(3×3)=I（全状态可测）。设计K后闭环矩阵A_cl=A-BK。输出要求跟踪参考r=[0, iq_ref, ωm_ref]ᵀ。
1. 写出N的表达式
2. N是多少维？哪个元素对应"从ωm_ref到uq的增益"？
3. N与传统的"速度环PI输出→iq_ref→电流环PI→uq"的区别是什么？

**参考答案：**
1. N = (C(-A_cl)⁻¹B)⁻¹ = ( -A_cl⁻¹ B )⁻¹（3×2）

2. N为3×2矩阵。N(2,2)对应"ωm_ref→uq"。
   具体地：输出y=[id,iq,ωm]ᵀ，参考r=[0,iq_ref,ωm_ref]ᵀ
   u = -Kx + Nr, u=[ud,uq]ᵀ
   uq = -k₂₁id - k₂₂iq - k₂₃ωm + N₂₁·0 + N₂₂·iq_ref + N₂₃·ωm_ref
   
   N₂₃就是"ωm_ref→uq"增益。

3. 传统级联：
   位置误差→位置PI→ωref→速度PI→iq_ref→电流PI→uq
   是三段串行传递。
   
   状态反馈N：
   ωm_ref 直接通过N₂₃影响uq（不需要经过速度PI和电流PI！）
   同时K₂₃提供转速阻尼反馈
   
   两种路径并行工作（前馈+反馈），响应更快。

## 二、分析设计题（★★★★☆）

### 题目4：极点选择对K矩阵的影响
**问题：** 二阶电流环状态反馈，在ωe=600rad/s工作点。分析三种极点选择对K的影响：
- 方案A：s₁=s₂=-2000（临界阻尼，快）
- 方案B：s₁,₂=-1000±j1000（ζ=0.707，中等阻尼）
- 方案C：s₁=-500, s₂=-8000（一快一慢）

**参考答案：**

| 方案 | k₁₁=k₂₂ | k₁₂ | k₂₁ | 特点 |
|------|---------|-----|-----|------|
| A | 2ωcL-R=2.4-R | ωeL=0.36 | -0.36 | 无超调，最快响应 |
| B | 2ζωnL-R=0.85-R | 0.36 | -0.36 | 约4.3%超调，综合好 |
| C | (λ₁+λ₂)L-R=-5.1-R | 0.36 | -0.36 | k₁₁为负（不稳定！） |

方案C的k₁₁=-5.1-R < 0 → 正反馈 → 闭环系统不稳定。
**教训：** 不是任意极点组合都能实现——极点和的符号决定了K的符号和大小。选择极点时需保证K能使系统稳定。

### 题目5：状态反馈 vs 级联PI的带宽突破
**问题：** 传统级联要求电流环带宽≥5×速度环带宽。状态反馈可以突破这一限制。
1. 用根轨迹或特征值解释：为什么状态反馈不需要5倍带宽比？
2. 在状态反馈中，电流环和速度环"带宽"的概念还存在吗？
3. 如果状态反馈的电流环极点=速度环极点，系统还能稳定吗？

**参考答案：**
1. 级联结构：速度环设计时把电流环近似为一阶惯性→要求电流环极点远小于速度环极点→5倍规则。
   状态反馈：A-BK的特征值包含所有闭环极点→电流和速度极点同时被K矩阵配置→不存在"先设计电流环再设计速度环"的分离过程→不需要5倍规则。

2. "带宽"概念弱化。传统"电流环带宽=ωc_cur"和"速度环带宽=ωc_spd"在状态反馈中被替代为"系统所有模态的阻尼和自然频率"。但仍可定义输入到某个输出的带宽。

3. 可以稳定！只要A-BK的特征值都在左半平面即可。例如三个极点都配置在s=-1000 → 系统稳定（但物理上电流和速度模态被强行统一，可能要求不现实的电压/电流）。

## 三、综合应用题（★★★★★）

### 题目6：四阶位置伺服状态反馈设计
**问题：** PMSM四阶模型（id,iq,ωm,θe）。设计全状态反馈实现位置伺服控制：
1. 选择四个期望极点（位置环30Hz, 速度环150Hz, 电流环900Hz×2）
2. 写出K矩阵中K₂₄（θe→uq）的物理意义
3. 讨论：状态反馈能替代三环级联吗？优缺点是什么？

**参考答案：**
1. 期望极点：
   - s₁,₂ = -2π×900 = -5655（电流，临界阻尼，τ=0.18ms）
   - s₃ = -2π×150 = -942（速度，τ=1.06ms）
   - s₄ = -2π×30 = -188（位置，τ=5.3ms）

2. K₂₄（θe→uq）的作用：
   - θe误差 → 通过K₂₄→uq → 产生iq→转矩→ωm→θe补偿
   - 等价于位置环P控制+速度环+电流环的集成效果
   - K₂₄>0表示正的角度误差产生正转矩（试图纠正位置）

3. 替代三环级联：
   **优点：**
   - 一个K矩阵统一优化，避免逐环调试
   - 无带宽瓶颈（电流环不需要比速度环快5倍）
   - 天然处理状态耦合
   
   **缺点：**
   - 需要全状态可测（实际中角度有编码器，但id,iq有噪声）
   - K矩阵参数不直观→调试困难（不像"调个Kp"那么简单）
   - 电机参数不准→K不准→极点不准→性能偏离
   - 抗饱和（anti-windup）需要额外设计
   
   **工程建议：** 电流环用状态反馈（参数少、效果显著），速度/位置环保留PI级联（调试直观、鲁棒可靠）。

### 题目7：增益K的物理可实现性约束
**问题：** 状态反馈计算出的K矩阵可能包含过大的元素，在物理上无法实现。
1. K₁₁=100意味着什么？对DSP的16位定点实现有何影响？
2. 电压限幅Vmax=300V时，K元素的上限是多少？
3. 如何在极点配置中考虑输入约束？

**参考答案：**
1. K₁₁=100意味着：1A的id误差 → 100V的ud输出。
   16位定点（Q12格式，范围±8）：
   id_max≈±8A → K·id_max=800V → 超出Vmax(300V) → **饱和是常态**
   
   但这不是bug——在正常工作点（id≈0），误差小，输出正常。K大只是意味着"即使小误差也会强力纠正"。

2. 粗略上限：K_max ≈ Vmax/x_max
   如果id_max=5A：ud_max=300V → k₁₁_max=300/5=60
   但这是静态约束，动态中K可超过此值（电压限幅会自然限制）。

3. 考虑约束的设计方法：
   - **约束极点配置**：将极点配置在确保K元素合理的区域
   - **Anti-windup**：K计算的控制量超出Vmax时，削减积分/状态
   - **模型预测控制（MPC）**：天然处理约束（见CT-13）
   - **饱和非线性Lyapunov分析**：确保限幅不破坏稳定性

### 题目8：积分增强型状态反馈
**问题：** 纯状态反馈 u=-Kx+Nr 不能消除稳态误差。增加积分状态 xI=∫(r-y)dt。
1. 写出扩展系统的A_e, B_e（原系统为三阶PMSM）
2. 积分状态的期望极点如何选择？
3. 写出最终控制律中积分项对应的增益

**参考答案：**
1. 原系统：ẋ=Ax+Bu, y=Cx=Ix（全状态可测）
   扩展状态：xₑ=[x; xI], ẋI=r-Cx=r-x
   
   $$A_e = \begin{bmatrix} A & 0 \\ -I & 0 \end{bmatrix}, \quad B_e = \begin{bmatrix} B \\ 0 \end{bmatrix}$$
   
   维度：A_e为6×6（原3阶×2=6维：id,iq,ωm, ∫id_err, ∫iq_err, ∫ωm_err）

2. 积分极点应**慢于**主状态极点，避免积分"抢跑"引起超调。
   通常积分极点选为主极点的1/5~1/10：
   如电流主极点s=-3000 → 积分极点s=-300
   速度主极点s=-100 → 积分极点s=-20

3. 扩展K_e=[K_original | K_integral]，是2×6矩阵。
   u = -[K₀ | Kᵢ]·[x; xI] = -K₀x - KᵢxI
   
   Kᵢ的物理意义类似于传统PI中的Ki，但：
   - 有3个积分状态（id_err, iq_err, ωm_err）→ Kᵢ有2×3个增益
   - 每个控制输入可参考所有积分状态（而传统PI中每个环只有自己的积分）
   - 这是更一般的"多变量积分控制"

### 题目9：状态反馈在参数变化时的鲁棒性
**问题：** 电感L从0.8mH变为1.2mH（+50%），原K=[1.92 0.4; -0.4 1.92]不变。
1. 计算实际闭环极点
2. 系统是否还能稳定？性能退化多少？
3. 与相同参数变化下的PI控制器对比鲁棒性

**参考答案：**
1. 实际A=-R/L_actual=-0.15/0.0012=-125
   B=1/L_actual=833
   
   A_cl = A-BK = -125 - 833×1.92 = -1725
   **实际极点s=-1725**（期望-2000）
   偏差=(2000-1725)/2000=**13.75%**（与L变化+50%不完全成比例，因R的影响）

2. 系统稳定（s=-1725<0）。带宽从318Hz降到274Hz（-14%）。
   L增大→B减小→A-BK的模减小→极点右移→带宽降低。
   **不引起振荡（无虚部），只是变慢。**

   更精确地（二阶dq模型）：
   特征多项式：|sI-(A-BK)| 具体计算
   共轭极点可能出现→如果ωe≠0，L变化导致解耦不完全→可能引入振荡

3. PI对比：
   PI参数：Kp=L_nom·ωc=0.8mH×12000=9.6, 实际需要Kp=1.2mH×12000=14.4
   PI的Kp偏差=(14.4-9.6)/9.6=+50%（完全正比于L）
   
   状态反馈中L出现在B中，B偏差对K的影响通过A-BK传递，偏差比例小于PI。
   **状态反馈对L参数的鲁棒性通常优于PI**（因为K的非对角项补偿了部分参数误差效应）。

### 题目10：离散状态反馈的时延补偿
**问题：** DSP实现中，采样到PWM更新存在一拍延迟Td。设计考虑延迟的状态反馈。
1. 写出含延迟的增广状态空间模型（需要增加几个状态？）
2. 设计考虑延迟的K矩阵
3. 与"连续域设计+离散实现"方法的精度对比

**参考答案：**
1. 一拍延迟 → 当前u[k]要到下一拍才生效 → 增加m个状态u_delayed
   
   增广系统：
   ```
   x[k+1] = Ad·x[k] + Bd·u_delayed[k]
   u_delayed[k+1] = u[k]
   ```
   状态：[x; u_delayed] = [id,iq,ωm; ud_delayed, uq_delayed]
   → 增广2个状态（5阶系统）

2. 对5阶增广系统设计K（2×5），前3列→x反馈，后2列→u_delayed反馈
   u[k] = -K₁₋₃·x[k] - K₄₋₅·u_delayed[k]
   
   实际控制律考虑了"上一拍输出还没生效"→避免重复补偿→防止过冲。

3. 连续域设计+离散实现：
   - 连续域K → 离散化直接使用 → 忽略延迟
   - 在Ts较小时（10~20kHz控制频率），误差可接受
   - 在Ts较大时（<5kHz），差异显著→离散域设计更精确

   离散域设计的K自动补偿了ZOH效应和延迟效应→极点位置准确。
