## 一、参数计算题（★★☆☆☆）

### 题目1：Nyquist判据基本应用
**问题：** 电流环开环传递函数 $L(s)=1200/s$（零极点对消后）。请：
1. 写出 Nyquist 轨迹 $L(j\omega)$ 的表达式
2. 计算Nyquist轨迹在复平面上的位置（ω=600, 1200, 2400 rad/s 三点）
3. 判断轨迹是否包围 $(-1, j0)$ 点，计算闭环极点数Z

**参考答案：**
1. $L(j\omega) = 1200/(j\omega) = -j(1200/\omega)$
2. ω=600: L=-j2；ω=1200: L=-j1；ω=2400: L=-j0.5
3. 整个轨迹在负虚轴上（从-j∞到0），不包围(-1,j0)。开环P=0→Z=N+P=0+0=0→稳定。Nyquist轨迹距(-1,j0)最近在ω=1200处：距离=|-1+j|=√2≈1.414。

### 题目2：延迟裕度计算
**问题：** 电流环 ωc=1800 rad/s，实测 PM=65°。Ts=80μs（含计算+ADC延迟）。
1. 计算延迟裕度 DM（ms）
2. 当前系统的总数字延迟约120μs。DM是否足够？
3. 如果要将 ωc 提升到 3000 rad/s，需要将延迟降至多少？

**参考答案：**
1. DM = PM/(ωc×180/π) = 65°×(π/180)/1800 = 1.134/1800 = 0.63 ms = 630 μs
2. 120 μs << 630 μs → 裕度充足，延迟不是瓶颈。
3. 在ωc=3000下维持相同的相对稳定性：DM_new = PM/(3000×π/180) = 1.134/3000 = 0.378 ms = 378 μs。需要数字延迟 ≤ 378 μs。当前120μs < 378μs → 可行。若需更高ωc=5000: DM=1.134/5000=227μs，仍>120μs。但需额外考虑滤波器等其他延迟叠加。

## 二、概念辨析题（★★★☆☆）

### 题目3：Nyquist图 vs 伯德图——互补关系
**问题：** 以下哪种情况伯德图可能「误判」稳定但奈奎斯特图能正确诊断？请说明原理。
1. 开环传递函数含有RHP极点（P>0）
2. 相位在高频段第二次穿越-180°且增益>1
3. 系统是条件稳定的

**参考答案：**
1. ❌ 伯德图不「误判」。伯德图的Nyquist稳定判据形式（Bode稳定判据）明确要求P=0。当P>0时，Bode判据需额外条件（相位穿越次数≥P/2），只是多数工程师忽略了。Nyquist判据的Z=N+P对此自动处理。
2. ✅ 这种情伯德图容易误判。工程师通常只看第一次-180°穿越的GM。如果第二次穿越（更高频率）增益>1而第一次<1，伯德图仅报告安全的GM，但Nyquist图显示包围了(-1,j0)→不稳定。
3. ✅ 条件稳定系统在增益减小时失稳——伯德图只看单一增益曲线的PM/GM，不会提示「增益降低会失稳」。Nyquist图通过分析不同增益下的包围关系揭示条件稳定。

### 题目4：条件稳定的本质
**问题：** 某系统 Nyquist 图显示：在 ω1=50 rad/s 处相位=-180°, |L|=2.5（增益>1）；在 ω2=500 rad/s 处相位再次=-180°, |L|=0.4（增益<1）。该系统当前增益K=1。请：
1. 判断当前稳定性（P=0）
2. 如果增益K变为0.3，稳定性如何？
3. 如果增益K变为4，稳定性如何？

**参考答案：**
1. Nyquist轨迹在ω=ω1处穿越负实轴于点-2.5（位于-1左侧）→净包围？需跟踪全轨迹。设其他频率无额外穿越：轨迹在低频→高增益→穿越-1左侧一次→顺时针。Z=N+P。如果只有一次顺时针包围(-1,j0)且在ω1处→Z≠0→不稳定！但如果轨迹在低频更多穿越则需要完整图。简化判断：在增益>1的-180°穿越中，包围关系决定稳定性。此处P=0，有两个-180°穿越点，|L|=2.5>1和|L|=0.4<1→净包围=1→Z=1→不稳定。
2. K=0.3→缩放L→ω1处|L|=0.75<1, ω2处|L|=0.12<1→无包围→Z=0→稳定（但这正好印证了条件稳定——增益减小使系统变稳定）。
3. K=4→ω1处|L|=10>1, ω2处|L|=1.6>1→两次包围→Z=2? N=2→Z=2→不稳定。

## 三、设计应用题（★★★☆☆）

### 题目5：延迟对Nyquist图的影响——数值仿真
**问题：** $L(s) = 2000 \cdot e^{-sT_s} / s$，Ts=100μs。
1. 计算前三个相位=-180°的频率点
2. 计算每个穿越点的|L|
3. 判断稳定性

**参考答案：**
1. 相位条件：$-90° - \omega T_s\times180°/\pi = -180° + k\times360°$ → $\omega T_s = \pi/2 + 2k\pi$ (k=0,1,2,...)
   k=0: ω1 = π/(2Ts) = π/(2×0.0001) = 15708 rad/s
   k=1: ω3 = (π/2+2π)/Ts = 78540 rad/s
   k=2: ω5 = (π/2+4π)/Ts = 141372 rad/s
2. |L(ω1)| = 2000/15708 = 0.127 < 1✅；|L(ω3)| = 2000/78540 = 0.025 < 1✅；|L(ω5)| = 2000/141372 = 0.014 < 1✅
3. 所有穿越点|L|<1→无包围→Z=0→稳定。ωc=2000远小于临界值π/(2×0.0001)=15708，延迟安全裕度极大。

## 四、综合案例题（★★★★☆）

### 题目6：FOC电流环条件稳定诊断
**问题：** 电流环包含：PI控制器 + RL模型(Rs=0.2Ω, Ls=3mH) + RC电流采样滤波器(τf=30μs, 即ωf=33333 rad/s)。PI参数Kp=4.5, Ki=300。系统出现小幅震荡。
1. 写出完整开环传递函数L(s)
2. 绘制Nyquist图渐近线，找出所有-180°穿越点
3. 判断是否为条件稳定，给出解决方案

**参考答案：**
1. $L(s) = (K_p + K_i/s) \cdot \frac{1}{R_s+L_s s} \cdot \frac{1}{1+\tau_f s} = \frac{K_p s + K_i}{s(R_s+L_s s)(1+\tau_f s)}$
   代入：$L(s) = \frac{4.5s + 300}{s(0.2+0.003s)(1+0.00003s)}$
2. 相位=分子相位-90°-arctan(Lsω/Rs)-arctan(τfω)。求解∠L(jω)=-180°：三个环节相位和=−90°。数值求解：ω≈1500 rad/s附近（PI零点转折区）→相位=arctan(4.5ω/300)-90°-arctan(0.003ω/0.2)-arctan(0.00003ω)≈−180°时|L|需数值计算。
3. 判断：若该-180°穿越点的|L|>1→不稳定。条件稳定分析：降低Kp→增加Kp是否都稳定？需要完整Nyquist分析。解决方案：提高滤波器转折频率(减小τf)，或降低ωc来确保在滤波器相位显著贡献前已达到足够的增益衰减。

### 题目7：传感器噪声在Nyquist图中的表现
**问题：** 电流传感器噪声通过反馈路径进入Nyquist分析需要考虑噪声传递函数。实际中，电流环在极高频率处（>10kHz）的Nyquist轨迹受噪声影响可能呈现出随机扰动。请分析：
1. Nyquist判据在高频噪声区域的适用性
2. 如何在线测量Nyquist图（扫频法）并处理噪声

**参考答案：**
1. Nyquist判据基于确定性LTI模型，噪声不被建模。在极高频率（远大于ωc和fs/2），L(jω)→0, Nyquist轨迹趋近原点。噪声引入的随机扰动不会使Nyquist轨迹「包围」(-1,j0)点（因为噪声均值为零）。但强噪声下的非线性效应（如ADC饱和）可能改变有效L(jω)。对策：在Nyquist分析中关注fs/2以下的频率范围，高频噪声通过滤波抑制。
2. 在线扫频：注入正弦扰动到Iq_ref，同步解调Iq_fb（用Lock-in Amplifier技术或Goertzel算法）→在每个频率点获得|L|和∠L。噪声处理：a) 每个频率点采集多个周期→平均降噪；b) 剔除SNR<10dB的频率点；c) 相关法：$|L| = \frac{\sum I_{fb}\cdot I_{ref}}{\sum I_{ref}^2}$有效抑制不相关噪声。

### 题目8：多轴联动耦合的MIMO Nyquist分析
**问题：** 两台电机驱动的XY平台，X轴运动通过机械耦合影响Y轴（耦合传递函数$G_{xy}(s)$）。写出2×2传递函数矩阵，简述广义Nyquist判据的应用。

**参考答案：**
传递函数矩阵：
$$\mathbf{G}(s) = \begin{bmatrix} G_{xx}(s) & G_{xy}(s) \\ G_{yx}(s) & G_{yy}(s) \end{bmatrix}$$

控制器矩阵 $\mathbf{K}(s)=diag(K_x(s), K_y(s))$。

开环：$\mathbf{L}(s)=\mathbf{G}(s)\mathbf{K}(s)$。

广义Nyquist：计算 $\mathbf{L}(j\omega)$ 的特征值 $\lambda_1(j\omega), \lambda_2(j\omega)$ → 两条特征轨迹（Characteristic Loci）。如果两条特征轨迹的净包围(-1,j0)次数之和等于开环不稳定极点数（取负号）→ 闭环稳定。

工程简化：如果耦合$G_{xy}$在关心的频率范围内远小于$G_{xx}$（如<-20dB），可近似解耦→分别对每轴进行SISO Nyquist分析。当耦合显著时（如高速高加速XY平台），必须进行MIMO分析。

### 题目9：基于Nyquist的电流环带宽极限推导
**问题：** FOC电流环受以下因素限制：a) 数字延迟Ts；b) PWM更新延迟（0.5Ts）；c) 电流采样滤波器τf。请：
1. 写出包含所有延迟和滤波器的完整L(s)
2. 推导ωc的上限表达式（保证PM≥45°）

**参考答案：**
1. 延迟总计≈1.5Ts（1拍计算延迟 + 0.5拍PWM更新延迟）。$L(s) = \frac{\omega_c}{s} \cdot e^{-1.5T_s s} \cdot \frac{1}{1+\tau_f s}$
2. 相位：∠L(jωc) = -90° - 1.5Tsωc×180°/π - arctan(τfωc)
   要求PM = 180° + ∠L(jωc) ≥ 45°
   即 90° - 1.5Tsωc×180°/π - arctan(τfωc) ≥ 45°
   1.5Tsωc×180°/π + arctan(τfωc) ≤ 45°
   对于小τf，近似：$1.5T_s\omega_c \times 180/\pi + \tau_f\omega_c \le 45°$
   $\omega_c \le \frac{45°\times\pi/180}{1.5T_s + \tau_f} = \frac{0.785}{1.5T_s + \tau_f}$

   数值：Ts=62.5μs, τf=8μs → ωc ≤ 0.785/(93.75+8)×10⁻⁶ = 7716 rad/s（约1228Hz）。

### 题目10：无传感器FOC中滑模观测器的Nyquist稳定性验证
**问题：** 滑模观测器估计的反电动势含有高频切换分量，其等效闭环传递函数可能存在RHP极点。请设计Nyquist稳定性验证的HIL测试方案：
1. 如何从HIL仿真中提取观测器的等效开环传递函数L_obs(s)
2. 如何判断L_obs(s)是否有RHP极点
3. 完整验证流程

**参考答案：**
1. 在HIL中注入小信号正弦扰动到观测器输入端（如注入角度误差Δθ），记录观测器输出（如估计反电动势$\hat{E}$）。扫频获取$L_{obs}(j\omega)=Δ\hat{E}/Δθ$的频率响应数据。
2. 判断RHP极点：a) 如果L_obs(s)的Nyquist图在ω→∞时趋于原点，且相位在高频段收敛→检查是否存在某频率段相位突变180°（共振反共振对）；b) 更可靠：拟合传递函数模型→检查极点位置。$P$ = RHP极点数。
3. 验证流程：a) 注入扫频信号→获取L_obs(jω)数据点；b) 绘制Nyquist图→统计包围(-1,j0)的圈数N；c) 从拟合模型或独立稳定性测试确定P；d) 若Z=N+P=0→观测器与被控对象构成的闭环稳定；e) HIL中实际在多种工况（不同ωe、负载）下重复验证。
