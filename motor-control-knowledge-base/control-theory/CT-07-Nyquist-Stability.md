# CT-07: 奈奎斯特稳定判据

**副标题：从奈奎斯特围线到条件稳定——为什么电流环在低频安定、在高频可能不安定，以及数字延迟如何通过 Nyquist 图吃掉稳定裕度**
**难度：** ★★★★☆ 专业级
**适用对象：** 高级电机控制工程师、控制算法研究者
**前置知识：** 伯德图与频率响应（CT-03）、传递函数与极点、复变函数基础

---

## 1. 📌 核心摘要

**一句话讲清楚**：奈奎斯特稳定判据是对闭环稳定性最精确的频域判据——通过观察开环传递函数 $L(j\omega)$ 在复平面上的轨迹（Nyquist图）包围 $(-1, j0)$ 点的圈数来判断闭环极点是否全部在左半平面。它能揭示伯德图「看不清」的现象——如条件稳定（Conditional Stability）和延迟诱发的失稳。

**认知挂钩**：伯德图告诉我们PM和GM，但有一个死角：**如果系统的相位多次穿越-180°呢？** 伯德图只记录第一次-180°穿越对应的GM，但 Nyquist 图能看到全貌。电机控制中典型的例子：电流环在低频段 PM=90°（稳定），但在更高频率因数字延迟和谐振可能出现第二次-180°穿越且增益>1——Nyquist图显示包围了(-1,j0)→闭环不稳定。

**与FOC算法的关联**：
- 🔗 **电流环数字延迟稳定性**：Nyquist图量化 $e^{-sT_s}$ 延迟对稳定性的影响
- 🔗 **条件稳定**：某些电机+滤波器组合在增益减小或增大时都稳定，但在中间增益区域不稳定
- 🔗 **无传感器观测器**：高频注入法观测器在某些频率下相频多次穿越-180°，Nyquist判据是唯一能完整判断稳定性的工具

问题现象:
- Kp=3时稳定运行

- PM>0仍震荡 → 多次穿越-180°→Nyquist判据才能发现
- 增益降低反而震荡 → 条件稳定（Conditional Stability）
- 高带宽失稳 → 延迟效应在Nyquist图上表现为螺旋式逼近(-1,j0)

## 3. 💡 直观理解

### 类比1：Nyquist图 = 飞机飞行轨迹的雷达图

**生活场景**：空中交通管制员看雷达屏幕，关注飞机的飞行轨迹是否会经过禁飞区（$(-1,j0)$点）。

- 飞机 = $L(j\omega)$的轨迹（从ω=0到ω=∞）
- 禁飞区 = $(-1, j0)$点
- 包围禁飞区的圈数 = 不稳定闭环极点数

**电机对应**：电流环的$L(j\omega)$轨迹从ω=0（低频高增益，远离原点）开始，随着ω增大沿顺时针螺旋（受延迟影响）逐渐趋近原点。如果轨迹包围了$(-1,j0)$点→闭环不稳定。

### 类比2：条件稳定 = 保险丝的手感

**生活场景**：一个特殊的插头——插得太紧（增益太大）会短路，插得太松（增益太小）会接触不良，只有在「刚刚好」的中间范围才正常工作。

**电机对应**：条件稳定系统——Kp太小或太大都不稳定，只有中间范围稳定。

## 4. 🔬 技术原理

### 4.1 奈奎斯特判据

**Cauchy幅角原理** → **奈奎斯特判据**：

$$Z = N + P$$

其中：
- $Z$：闭环系统在右半平面的极点数（$Z=0$ → 稳定）
- $N$：Nyquist曲线顺时针包围 $(-1, j0)$ 点的净圈数
- $P$：开环传递函数 $L(s)$ 在右半平面的极点数

**对于FOC电流环**：$L(s)=\omega_c/s$（纯积分器），$P=0$（无极点在右半平面），Nyquist曲线是负虚轴上的直线（从$-j\infty$沿虚轴到原点），不包围$(-1,j0)$ → $N=0$ → $Z=0$ → 稳定。

### 4.2 纯积分型电流环的Nyquist图

$$L(j\omega) = \frac{\omega_c}{j\omega} = -j\frac{\omega_c}{\omega}$$

Nyquist图：整个负虚轴。

| ω | L(jω) |
|---|-------|
| ω→0 | $-j\infty$ |
| ω=ωc | $-j$ (即0-j1) |
| ω→∞ | 0 |

**结论**：轨迹完全在负虚轴上，与实轴无交点，距$(-1,j0)$的最小距离为$|(-1)-(-j)|=| -1+j |=\sqrt{2}$→ PM=90°。

### 4.3 数字延迟的Nyquist图分析

加入数字延迟 $e^{-sT_s}$：

$$L(s) = \frac{\omega_c}{s} \cdot e^{-sT_s}$$

$$L(j\omega) = \frac{\omega_c}{j\omega} \cdot e^{-j\omega T_s} = \frac{\omega_c}{\omega} \cdot e^{-j(\pi/2 + \omega T_s)}$$

在 $\omega = \pi/(2T_s)$ 处：相位 = $-90° - 90° = -180°$，Nyquist轨迹首次穿越负实轴。

$$|L(j\pi/(2T_s))| = \frac{\omega_c}{\pi/(2T_s)} = \frac{2\omega_c T_s}{\pi}$$

**稳定条件**：$|L| < 1$ 在相位=-180°处（即轨迹不包围-1点）：
$$\frac{2\omega_c T_s}{\pi} < 1 \quad \Rightarrow \quad \omega_c < \frac{\pi}{2T_s}$$

**数值示例**：
- $T_s=62.5\mu$s (16kHz)：$\omega_c < 25133$ rad/s → 远大于工程取值，延迟单独不构成威胁
- $T_s=250\mu$s (4kHz)：$\omega_c < 6283$ rad/s → 若ωc>6283，Nyquist轨迹包围-1点→不稳定

**但多次穿越**：随着ω增大，$e^{-j\omega T_s}$周期旋转，Nyquist图在$-1$附近螺旋。需检查所有穿越点的$|L|$是否<1。

### 4.4 条件稳定——增益减小反而震荡

条件稳定系统的Nyquist特征：相位在增益>1的区间内两次穿越-180°。

```
增益 K1 (大): |L(ω180,1)| > 1, |L(ω180,2)| < 1 → 包围-1点0次? 需计算
增益 K2 (中): |L(ω180,1)| = 1, |L(ω180,2)| = 1 → 临界
增益 K3 (小): |L(ω180,1)| < 1, |L(ω180,2)| > 1 → 包围-1点≠0? → 不稳定！
```

**电机控制中的条件稳定场景**：
- 电流传感器RC滤波 + 电机RL模型 → 两个极点 → 相位可能多次穿越-180°
- 电流环+速度环级联共振 → 在某些频率下出现条件稳定

### 4.5 延迟裕度（Delay Margin）

系统还能承受多少额外延迟而不失稳：

$$DM = \frac{PM}{\omega_c}$$

其中PM以弧度计。

**手推示例**：
```
电流环 ωc=2000 rad/s, PM=60°=1.047 rad
DM = 1.047/2000 = 0.524 ms

这意味着：如果数字延迟或滤波器延迟增加超过0.524ms，
系统PM将从60°降至0°以下→失稳。
```

## 5. 🔗 交叉视角

### 5.1 电流环数字延迟的完整稳定性分析

| 频率区域 | 主导因素 | Nyquist图行为 |
|---------|---------|-------------|
| 低频 (ω<ωc) | 积分器$1/s$ | 沿负虚轴→高增益、-90° |
| 中频 (ω≈ωc) | 交叉 | 轨迹接近(-1,j0)区域 |
| 高频 (ω≈1/Ts) | 延迟$e^{-sT_s}$ | 螺旋逼近原点, 多次穿越 |

### 5.2 无传感器观测器的Nyquist稳定性

以滑模观测器为例，其等效闭环传递函数在某些工作点可能出现RHP极点($P>0$)，此时需要Nyquist图包围-1点$P$次才能保证闭环稳定（$Z=N+P=0$ → $N=-P$）。

## 6. 🎯 工程案例

### 案例1：条件稳定导致「Kp减小反而震荡」

**背景**：
```
电机：Ls=5mH, Rs=0.5Ω, 电流采样RC滤波τf=50μs (ωf=20000 rad/s)
滤波器带来额外极点→系统为三阶
Kp=4.0时稳定，Kp=2.5时反而震荡
```

**Nyquist分析**：
- Kp=4.0：三个穿越-180°的频率点中，两个 |L|<1 → 净包围0次 → 稳定
- Kp=2.5：增益的缩放改变了穿越顺序，使其中一个|L|>1 → 净包围1次 → 不稳定

**解决**：Kp取5.0（更高增益）反而稳定，但需权衡超调量。

### 案例2：高带宽电流环的数字延迟失稳

**背景**：ωc=3000 rad/s, Ts=100μs (10kHz电流环), 有1拍计算延迟。
延迟裕度DM=PM/ωc ≈ 60°×π/(180×3000)=0.35ms。系统实际延迟约150μs(计算+ADC转换)→DM足够→稳定。
若ωc=5000：DM=60°×π/(180×5000)=0.21ms < 150μs→失稳风险。

### 案例3：传感器故障时的Nyquist诊断

编码器信号丢失→角度估计器发散→等效$L(s)$出现RHP极点($P=1$)→Nyquist图需包围-1点1次才能稳定→实际由于饱和限幅等非线性无法满足→系统必然不稳定。此分析帮助确定「为什么传感器故障后FOC完全不可控」。

## 8. 🚀 前沿拓展

### 8.1 多输入多输出（MIMO）Nyquist判据

FOC的dq双轴耦合构成MIMO系统。广义Nyquist判据通过特征轨迹（Characteristic Loci）分析MIMO稳定性——是FOC在高转速、深度弱磁工况下稳定性分析的前沿工具。

### 8.2 μ分析与鲁棒稳定性

μ分析是Nyquist判据在结构不确定性下的推广——可以定量回答：给定电机参数的不确定范围（Rs±20%, Ls±15%），闭环系统是否对所有可能的参数组合都稳定？
