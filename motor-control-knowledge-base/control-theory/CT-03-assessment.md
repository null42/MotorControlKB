## 一、参数计算题（★★☆☆☆）

### 题目1：穿越频率和相位裕度计算
**问题：** 某FOC电流环开环传递函数 $L(s) = 1200/s$（零极点对消后）。请计算：
1. 穿越频率 $\omega_c$
2. 相位裕度 PM（忽略非理想因素）
3. 在 $\omega = 120$ rad/s 处的环路增益（dB）
4. 该电流环对应的闭环带宽约多少 Hz？

**参考答案：**
1. $|L(j\omega_c)| = 1200/\omega_c = 1$ → $\omega_c = 1200$ rad/s
2. $\angle L(j\omega_c) = \angle(1200/j1200) = -90°$，$PM = 180° + (-90°) = 90°$
3. $|L(j120)| = 1200/120 = 10$ → $20\log_{10}10 = 20$ dB
4. 闭环带宽 ≈ $\omega_c = 1200$ rad/s ≈ 191 Hz

### 题目2：非理想因素对相位裕度的侵蚀
**问题：** 电流环 $L(s) = 1500/s$，存在数字控制延迟 $e^{-sT_s}$（$T_s=62.5\mu$s）。请计算：
1. 在穿越频率 $\omega_c=1500$ rad/s 处，延迟引入的相位滞后
2. 扣除延迟后的有效相位裕度 PM_eff
3. 如果要求 PM_eff ≥ 60°，$\omega_c$ 最大能取多少？

**参考答案：**
1. $\phi_{delay} = -\omega_c T_s \times 180/\pi = -1500 \times 62.5\times10^{-6} \times 57.3 = -5.37°$
2. PM_eff = 90° - 5.37° = 84.63°
3. 设 $\omega_c$ 未知，要求 $90° - \omega_c T_s \times 57.3° \ge 60°$ → $\omega_c \le 30°/(T_s \times 57.3) = 30/(62.5\times10^{-6} \times 57.3) \approx 8375$ rad/s

---

### 题目3：滤波器对电流环伯德图的影响
**问题：** 电流环 $L(s) = 2000/s$，电流采样前端有RC低通滤波器 $1/(1+\tau_f s)$，$\tau_f=10\mu$s。请计算：
1. 滤波器的转折频率 $\omega_f$
2. 在穿越频率 $\omega_c=2000$ rad/s 处，滤波器引入的相位滞后
3. 综合延迟($T_s=50\mu$s)和滤波器后的 PM_eff

**参考答案：**
1. $\omega_f = 1/\tau_f = 1/(10\times10^{-6}) = 100000$ rad/s ≈ 15.9 kHz
2. $\phi_f = -\arctan(\omega_c/\omega_f) = -\arctan(2000/100000) = -\arctan(0.02) \approx -1.15°$
3. $\phi_{delay} = -2000 \times 50\times10^{-6} \times 57.3 = -5.73°$；PM_eff = 90° - 5.73° - 1.15° = 83.12°

## 二、概念辨析题（★★★☆☆）

### 题目4：PM与超调量的关系
**问题：** 以下关于相位裕度 PM 和时域超调量 $M_p$ 关系的描述，请判断正误：
1. PM 越大，闭环系统越稳定，超调越小
2. PM = 90° 意味着闭环为一阶系统，阶跃响应零超调
3. PM = 45° 的电流环阶跃超调约20%，不适合伺服应用
4. PM = 30° 的电流环仍可稳定运行，只是超调较大

**参考答案：**
1. ✅ 正确。PM 越大 → ζ 越大 → 超调越小。
2. ✅ 正确（对纯积分型开环 $L(s)=\omega_c/s$）。若存在非理想因素（延迟/极点），即使PM=90°，闭环也可能不是精确一阶，但超调极小。
3. ✅ 正确。PM=45° → $\zeta\approx0.45$ → $M_p\approx20\%$。20%的电流超调对于伺服应用不可接受。
4. ⚠️ 理论上正确但冒险。PM=30°理论可稳定但极其脆弱——电机参数微小变化（温升→Rs变化→零极点对消破坏）、负载扰动都可能使其失稳。工程中PM不应低于45°。

### 题目5：增益裕度 GM 与 相位裕度 PM 的区别
**问题：** 某电流环的伯德图显示 PM=50°, GM=12dB。请解释：
1. PM 和 GM 分别衡量什么？
2. 为什么 PM=50° 足够而 GM=12dB 也足够？
3. 是否存在 PM 足够但 GM 不足的情况？在电机控制中可能出现吗？

**参考答案：**
1. PM衡量在穿越频率（增益=0dB）处相位离-180°的裕度；GM衡量在相位=-180°处增益离0dB的裕度。两者互补：PM反映$\omega_c$附近的稳定性，GM反映高频区的稳定性。
2. 工程准则：PM ≥ 45°且 GM ≥ 6dB 即可接受。PM=50°对应$\zeta\approx0.5$，超调约16%；GM=12dB意味着高频区即使增益因参数偏差增大4倍，系统仍稳定。
3. 存在。常见于具有谐振峰的系统（如柔性耦合负载）。谐振峰在相位穿越频率处产生高增益→GM不足→即使PM正常也可能震荡。在电机控制中，传动轴弹性谐振可能导致此现象（双惯量系统）。

## 三、设计应用题（★★★☆☆）

### 题目6：电流环带宽设计——多约束综合
**问题：** 电机：Ls=2.5mH, Rs=0.35Ω。约束：开关频率12kHz(Ts≈83μs)，电流采样滤波$\tau_f=8\mu$s，要求 PM_eff ≥ 65°。请：
1. 列出所有约束条件并计算$\omega_c$的上限
2. 选择合理的$\omega_c$并计算Kp、Ki
3. 验证选定的$\omega_c$下闭环带宽是否满足5倍电频率要求（最高转速6000rpm, P=3）

**参考答案：**
1. 约束条件：
   - 开关频率：$\omega_c < 2\pi\times12000/10 = 7540$ rad/s
   - PM：$90°-\omega_c T_s\times57.3°-\arctan(\omega_c\tau_f)\times57.3° \ge 65°$
   - 即 $\omega_c(83\times10^{-6} + 8\times10^{-6})\times57.3° \le 25°$ → $\omega_c \le 25/(91\times10^{-6}\times57.3) \approx 4792$ rad/s
   （取近似 $\arctan(\omega_c\tau_f)\approx\omega_c\tau_f$ 对小值成立）

2. 取 $\omega_c = 2000$ rad/s（保守）。Kp = 2.5mH × 2000 = 5.0；Ki = 0.35 × 2000 = 700

3. 最高电频率 $\omega_{e,max} = P \times 2\pi \times 6000/60 = 3 \times 628.3 = 1885$ rad/s
   $\omega_c/\omega_{e,max} = 2000/1885 \approx 1.06 < 5$ → 不满足！重新取$\omega_c$。
   需要 $\omega_c \ge 5 \times 1885 = 9425$ rad/s → 但开关频率约束上限7540 → 矛盾！
   **结论**：开关频率12kHz对P=3、6000rpm的电机不足。需提高开关频率至≥16kHz，或降低最高转速要求。

### 题目7：速度环带宽设计——基于电流环伯德图
**问题：** 已知电流环 $\omega_c^{cur}=1800$ rad/s。设计速度环，要求 PM_spd ≥ 50°，速度环开环传递函数 $L_{spd}(s) = (K_p^{spd}+K_i^{spd}/s) \cdot \frac{1}{1+s/1800} \cdot \frac{K_t}{Js}$，其中 J=0.004, Kt=0.7。
1. 确定 $\omega_c^{spd}$ 的最大值（仅考虑电流环相位贡献）
2. 取 $\omega_c^{spd}=200$ rad/s，计算Kp_spd和Ki_spd
3. 验证PM_spd

**参考答案：**
1. 电流环在$\omega_c^{spd}$处贡献相位 $-\arctan(\omega_c^{spd}/1800)$。PI积分为-90°。要求PM≥50° → $-90°-\arctan(\omega_c^{spd}/1800)+180° \ge 50°$ → $\arctan(\omega_c^{spd}/1800) \le 40°$ → $\omega_c^{spd} \le 1800\times\tan40° \approx 1510$ rad/s（仅考虑电流环贡献时理论值；实际还需考虑机械共振等限制）。
2. 在$\omega_c^{spd}$处设$|L_{spd}(j\omega_c^{spd})|=1$。忽略摩擦B，近似：
   $|K_p^{spd} \cdot \frac{K_t}{J\cdot j\omega_c^{spd}}| \approx 1$ → $K_p^{spd} \approx J\omega_c^{spd}/K_t = 0.004\times200/0.7 = 1.14$
   $K_i^{spd} = K_p^{spd}\omega_c^{spd}/4 = 1.14\times200/4 = 57$（对称最优法）
3. 相位：$-90°-\arctan(200/1800)+\arctan(K_p^{spd}\omega_c^{spd}/K_i^{spd}) ≈ -90°-6.3°+76° = -20.3°$ → PM ≈ 159.7°（对称最优法的相位在$\omega_c^{spd}$处有一个峰值，实际上PM可能更低，需更精确计算。简化估计：积分为-90°+76°=-14°，电流环-6.3°，总-20.3°，PM=159.7°——远高于50°要求。）

## 四、综合案例题（★★★★☆）

### 题目8：电流环震荡根因分析——伯德图诊断法
**问题：** 一台FOC驱动的PMSM，电流环不断震荡（频率约400Hz）。PI参数：Kp=6.0, Ki=900。电机：Ls=3mH, Rs=0.4Ω。开关频率8kHz(Ts=125μs)，电流采样$\tau_f=15\mu$s。请：
1. 计算理论穿越频率$\omega_c$和理想PM
2. 计算非理想因素吃掉的相位裕度
3. 判断震荡根因并给出解决方案

**参考答案：**
1. $\omega_c = K_p/L_s = 6.0/0.003 = 2000$ rad/s（约318Hz）。理想PM=90°（零极点对消：$K_i/K_p=900/6=150$，$R_s/L_s=0.4/0.003=133.3$，有偏差→非理想对消）。
2. 非理想相位损失：
   - 数字延迟：$\phi_d = -2000\times125\mu s\times57.3° = -14.3°$
   - 滤波器：$\phi_f = -\arctan(2000\times15\mu s)\times57.3° = -1.7°$
   - 零极点失配：额外相位弯曲，近似-5°~-10°
   总相位损失≈21°~26°。PM_eff=90°-21°~26°=64°~69°→理论上仍稳定。
3. 但400Hz≈2513 rad/s > $\omega_c$=2000 rad/s，震荡频率高于穿越频率→可能非PM问题。进一步排查：a) 开关频率8kHz的低频混叠效应（400Hz=8kHz/20→可能是谐波耦合）；b) 死区效应带来的非线性畸变；c) 反电动势解耦不完全。解决方案：提高开关频率至16kHz，优化死区补偿，验证解耦前馈。

### 题目9：电流环带宽对速度环稳定性的级联影响
**问题：** 一台伺服系统，电流环$\omega_c^{cur}=2500$ rad/s（PM_cur=72°），速度环$\omega_c^{spd}=500$ rad/s。系统在高速轻载时稳定，低速重载时出现低频振荡（~30Hz）。请从伯德图角度分析：
1. 速度环PM在低速重载时可能降低的原因
2. 如何通过伯德图诊断级联稳定性问题
3. 给出解决方案

**参考答案：**
1. 低速重载时：a) 电流环解耦前馈依赖$\omega_e$，低速时$\omega_e$小→前馈弱→耦合效应相对增强→电流环实际PM降低、相位滞后增大；b) 重载时摩擦力（特别是静摩擦）引入非线性，线性伯德图分析失效；c) 母线电压纹波在低速重载时占比大→电流环增益周期波动。
2. 分别在轻载和重载工况下注入扫频信号测量速度环闭环频率响应。对比伯德图→若重载时在30Hz处出现谐振峰→说明速度环实际PM严重不足。也可以离线分别在额定和重载电流下测量电流环伯德图，发现耦合导致的PM损失。
3. 解决方案：a) 完善解耦前馈（特别是在低速区域也保证足够精度）；b) 增加电流环前馈系数；c) 降低速度环带宽（如$\omega_c^{spd}$从500降至300 rad/s）；d) 增加陷波滤波器抑制30Hz振荡。

### 题目10：基于实测伯德图的PI参数在线校准
**问题：** 你需要在嵌入式DSP上实现电流环伯德图测量功能。给出：
1. 扫频信号的生成方法（嵌入式环境）
2. 幅频/相频的在线计算方法
3. 如何从测得的伯德图反推最优PI参数

**参考答案：**
1. 扫频信号生成：使用DDS（直接数字频率合成）生成对数扫频的正弦Iq_ref信号。频率从$\omega_{min}$到$\omega_{max}$对数递增，每个频率点持续N个周期（确保稳态）。伪代码：
```c
for(f=f_min; f<f_max; f*=1.1) { // 对数扫频
    for(i=0; i<N_cycles; i++) {
        Iq_ref = A*sin(2*PI*f*t);
        // 运行FOC，记录Iq_fb
    }
    // 计算该频率点的幅值比和相位差
}
```
2. 在线计算：对每个频率点，用Goertzel算法或DFT提取Iq_ref和Iq_fb的基波幅值和相位：
   $|G(j\omega)| = |Iq_{fb}|/|Iq_{ref}|$，$\angle G(j\omega) = \angle Iq_{fb} - \angle Iq_{ref}$
3. PI参数反推：从伯德图读取穿越频率$\omega_c$（$|L|=0dB$处）和PM。若目标PM=70°，实际PM=55°→降低$\omega_c$。根据新的$\omega_c$重新计算Kp=Ls×$\omega_c$，Ki=Rs×$\omega_c$。反复迭代或在DSP中实现自动优化（如单纯形法搜索最优$\omega_c$使PM目标函数最小）。
