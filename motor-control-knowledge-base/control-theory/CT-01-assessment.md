## 一、参数计算题（★★☆☆☆）

### 题目1：开环/闭环传递函数计算
**问题：** 已知电机电流环被控对象传递函数 $G(s) = \frac{1}{0.3 + 0.002s}$，PI控制器 $C(s) = 2 + \frac{300}{s}$，请计算：
1. 开环传递函数 $L(s) = C(s)G(s)$
2. 闭环传递函数 $T(s) = L(s)/(1+L(s))$
3. 判断闭环系统是否为I型系统（含一个积分器），对阶跃输入有无稳态误差？

**参考答案：**
1. $L(s) = (2 + 300/s) \cdot \frac{1}{0.3+0.002s} = \frac{2s+300}{s(0.3+0.002s)}$
2. $T(s) = \frac{2s+300}{0.002s^2 + (0.3+2)s + 300} = \frac{2s+300}{0.002s^2 + 2.3s + 300}$
3. 开环含一个积分器（$1/s$）→ I型系统，对阶跃输入 $e_{ss}=0$ ✅

### 题目2：灵敏度函数计算
**问题：** 某电流环开环传递函数 $L(s) = 1500/s$（零极点对消后）。请计算：
1. 灵敏度函数 $S(s)$ 和互补灵敏度函数 $T(s)$
2. 在 $\omega = 150$ rad/s 处的 $|S(j\omega)|_{dB}$ 和 $|T(j\omega)|_{dB}$
3. 说明此时闭环对参数变化的抑制能力

**参考答案：**
1. $S(s) = 1/(1+1500/s) = s/(s+1500)$，$T(s) = 1500/(s+1500)$
2. $L(j150)=1500/(j150)=10/j$，$|L|=10$（20dB）；$|S|=1/|1+10/j|≈0.0995≈-20dB$；$|T|≈0.995≈0dB$
3. 在 $\omega=150$ rad/s（约24Hz）处，参数变化对输出的影响被衰减到约10%（-20dB）→ 闭环鲁棒性良好

### 题目3：闭环对扰动的抑制计算
**问题：** 一台PMSM的电流环开环传递函数 $L(s) = 1000/s$。负载电流扰动 $I_d=2A$ 通过传递函数 $G_d(s)=1/(0.1+0.001s)$ 作用于系统。求 $\omega=50$ rad/s 时：
1. 扰动到输出的传递函数 $Y(s)/D(s)$
2. 扰动衰减倍率

**参考答案：**
1. $Y/D = G(s)/(1+L(s)) = \frac{1/(0.1+0.001s)}{1+1000/s} = \frac{s}{(0.1+0.001s)(s+1000)}$
2. 在 $\omega=50$ rad/s：$|1+L|=|1+1000/(j50)|=|1-20j|≈20.02≈26dB$ → 扰动被衰减约26dB（约20倍）

## 二、概念辨析题（★★★☆☆）

### 题目4：V/f开环 vs FOC闭环——多维度对比
**问题：** 以下关于电机驱动中开环V/f控制和闭环FOC控制的描述，请逐条判断正误并说明理由：
1. V/f控制在低速时性能差，主要因为低速时反电动势太低，电阻压降占比大且无法自动补偿
2. FOC的电流闭环使Id、Iq跟踪误差趋近于零，不受电机参数偏差影响
3. 闭环控制总是优于开环控制，任何电机驱动场景都应优先选择闭环

**参考答案：**
1. ✅ 正确。低速时 $\omega_e\psi_f$ 很小，$R_s I_q$ 主导电压方程，电阻温漂30%导致电压估计偏差大，开环无法补偿。
2. ⚠️ 不完全正确。闭环能消除稳态误差（通过积分），但在动态过程中仍受参数偏差影响（如Kp不准导致带宽变化）。此外，闭环稳定性受参数偏差影响（当前参数偏差过大可能导致失稳）。
3. ❌ 错误。闭环控制需要电流传感器（增加成本、PCB面积、故障点），对于成本敏感且对动态性能要求不高的场景（如风机负载），V/f开环可能是更优解。

### 题目5：灵敏度函数的物理意义
**问题：** 已知 $S(s) + T(s) = 1$，请解释：
1. 为什么 $S(s)$ 被称为「灵敏度函数」，而 $T(s)$ 被称为「互补灵敏度函数」？
2. 在电机电流环中，$|S(j\omega)|$ 在低频区应该大还是小？为什么？
3. 如果电机电感参数误差+30%，PI参数按错误值计算，$|S(j\omega)|$ 会如何变化？

**参考答案：**
1. $S(s) = \partial T_{cl}/\partial G \cdot G/T_{cl}$ 衡量闭环传递函数对被控对象参数变化的敏感度。$T(s) = 1 - S(s)$，两者之和为1，互为补充。
2. 低频区 $|S(j\omega)|$ 应尽可能小（<<1）。原因：低频区工作频率覆盖负载扰动和设定值变化的主要频谱，需要强鲁棒性。$|S|$小意味着参数偏差对闭环输出的影响被大幅衰减。
3. 电感错误→PI参数Kp计算错误→$L(s)$增益变化→$|S(j\omega)|$增大（抑制能力下降）。例如Kp偏低30%，$|L|$降低30%，在$\omega=\omega_c/10$处$|S|$从0.1升至约0.13。

## 三、设计应用题（★★★☆☆）

### 题目6：从开环V/f升级到闭环FOC的系统设计
**问题：** 一台额定功率2kW的PMSM（参数：Rs=0.5Ω, Ls=3mH, ψf=0.1Wb, P=3），目前使用V/f开环控制。用户要求最低运行转速从200rpm降至20rpm，且转矩脉动<5%。请：
1. 分析V/f在20rpm时无法运行的根本原因
2. 设计FOC电流环架构，给出必要的硬件增加清单
3. 计算电流环PI参数的初始值（取ωc=1200 rad/s）

**参考答案：**
1. 20rpm时电频率 $f_e = P \times 20/60 = 1$Hz，反电动势 $E = \psi_f \times 2\pi \times 1 \times P = 0.1 \times 6.28 \times 3 = 1.88$V。电阻压降 $I_q \times R_s = 5 \times 0.5 = 2.5$V（假设5A电流）。电阻压降占主导，电阻温漂±30%导致电压误差约±0.75V，反电动势仅1.88V，相对误差可达40%。开环无法感知和修正此误差。
2. 硬件增加：2个/3个电流传感器（霍尔效应或采样电阻+运放）、ADC调理电路、转子位置传感器（编码器或霍尔）。软件增加：Clarke变换、Park变换、电流PI控制器(d轴+q轴)、解耦前馈、角度估计。
3. Kp = Ls × ωc = 3mH × 1200 = 3.6；Ki = Rs × ωc = 0.5 × 1200 = 600

### 题目7：灵敏度约束下的带宽设计
**问题：** 一台伺服电机电流环要求：在负载扰动频率范围内（0~50Hz），参数偏差导致的电流误差<2%。已知电机参数测量精度为±15%（影响开环增益），请：
1. 确定所需的最小开环增益（在50Hz处）
2. 推算电流环带宽ωc的最小值
3. 考虑16kHz开关频率限制，这个带宽是否可行？

**参考答案：**
1. 参数偏差15%→开环增益偏差15%。要求闭环输出误差<2%→$|S| \le 0.02/0.15/|G|$。保守取$|S|<0.1$(-20dB)在50Hz处→需要$|L(j314)|>10$（50Hz=314 rad/s）。
2. 若$L(s)=\omega_c/s$，$|L(j314)|=\omega_c/314>10$→$\omega_c>3140$ rad/s（约500Hz）。
3. 开关频率16kHz→采样频率16kHz→Nyquist频率8kHz。按$\omega_c < 2\pi f_s/10 ≈ 10000$ rad/s的工程准则，500Hz带宽完全可行。

## 四、综合案例题（★★★★☆）

### 题目8：从开环抖动到闭环稳定——完整故障分析
**问题：** 某风扇驱动系统，V/f开环控制在600rpm以上正常，500rpm以下出现电流失控、电机抖动。电机参数：Rs=1.2Ω, Ls=5mH, ψf=0.06Wb, P=6, Vdc=310V。请：
1. 计算500rpm时反电动势大小，分析为什么在这个转速以下出问题
2. 如果改用FOC闭环，设计完整方案（硬件变更+软件架构+PI参数初始值，取ωc=800 rad/s）
3. 估算FOC方案增加的BOM成本

**参考答案：**
1. 500rpm→电频率 $f_e=P×500/60=50$Hz→$\omega_e=314$ rad/s。反电动势 $E=\psi_f×\omega_e=0.06×314=18.8$V。假设额定电流5A，电阻压降$=5×1.2=6$V。反电动势虽然仍大于电阻压降，但V/f控制需要精确估计反电动势才能给出正确电压。500rpm以下，V/f斜率偏差和电阻补偿不精确导致电流失控。
2. 硬件：2个ACS712霍尔电流传感器（±5A量程）；磁编码器AS5047（14位）；ADC采样。软件：FOC主循环（Clarke→Park→PI→InvPark→SVPWM）、电流环PI Kp=5mH×800=4.0，Ki=1.2×800=960。架构：10kHz电流环+1kHz速度环。
3. BOM增加：ACS712×2≈$4，AS5047≈$8，运放+阻容≈$2，合计约$14。

### 题目9：二自由度控制在电机电流环中的应用设计
**问题：** 在标准一自由度PI电流环中，快速设定值跟踪和强扰动抑制之间存在矛盾（大Kp利于跟踪但不利于噪声抑制）。请设计一个二自由度电流环结构：
1. 画出框图，标注前馈通道和反馈通道
2. 说明如何实现「跟踪性能」和「抗扰性能」的解耦设计
3. 给出前馈通道的一种实现方式

**参考答案：**
1. 框图：Iq_ref → [前馈Cf(s)] → Vq_ff ↘  Vq_sum → 电机
Iq_ref → ⨁(-) → [反馈Cb(s)] → Vq_fb ↗
         Iq_fb ←──────────────────┘
2. 解耦设计：前馈Cf(s)负责设定值跟踪（可设计为逆模型$1/G(s)$的近似），反馈Cb(s)负责扰动抑制和鲁棒稳定。两者独立调节，无折衷问题。
3. 前馈实现：$C_f(s) = \frac{L_s s + R_s}{\tau_f s + 1}$（电机逆模型+低通滤波，$\tau_f$为滤波器时间常数，抑制高频噪声放大）。DSP离散化实现：
```c
Vq_ff = (Ls/Ts)*(Iq_ref - Iq_ref_prev) + Rs*Iq_ref; // 欧拉离散
```

### 题目10：数字控制计算延迟对闭环性能的影响
**问题：** 在DSP实现的FOC电流环中，存在一拍计算延迟（ADC采样→PWM更新≈一个PWM周期Ts）。该延迟可建模为 $e^{-sT_s}$。已知Ts=62.5μs（16kHz），电流环带宽设计ωc=2000 rad/s：
1. 计算在ωc处，计算延迟引入的相位滞后
2. 分析该相位滞后对闭环稳定性和动态响应的影响
3. 给出两种补偿方法

**参考答案：**
1. 延迟环节的相位：$\angle e^{-j\omega T_s} = -\omega T_s$（弧度）。在$\omega=\omega_c=2000$处：相位滞后 = $2000 \times 62.5\times10^{-6} = 0.125$ rad = 7.16°
2. 7.16°的相位滞后会直接吃掉相位裕度。若原设计PM=45°，延迟后PM≈37.8°。此时闭环超调增大（ζ降低），甚至可能接近失稳。动态响应：超调量增大，调节时间延长。
3. 方法一（角度前向补偿）：Park变换时使用 $\theta_e + \omega_e T_s$ 替代 $\theta_e$，提前补偿一个采样周期的角度变化。方法二（Smith预估器）：在控制器中增加延迟模型，使反馈信号等效为无延迟信号。方法三：适当降低带宽设计值（如从2000降至1600 rad/s）。
