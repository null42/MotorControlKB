## 一、LADRC电流环实现（★★★★☆）

### 题目1：电流环LADRC的完整参数设计
**问题：** PMSM电流环参数：Ls=2.5mH, Rs=0.4Ω, Ke=0.053V/rpm, fs_pwm=16kHz（Ts=62.5μs），Vdc=48V。
1. 设计LADRC电流环：确定 b₀、ωc、ωo 及对应的所有增益
2. 与标准PI（零极点对消法，相同带宽）对比参数数量
3. 估算两者的计算量差异（以浮点运算次数计）

**参考答案：**
1. **LADRC电流环参数设计**：
   
   **步骤1：确定b₀**
   $$b_0 = \frac{1}{L_s} = \frac{1}{0.0025} = 400 \quad [\text{V}/(\text{A}\cdot\text{s})]$$
   
   **步骤2：确定控制器带宽 ωc**
   - 目标电流环带宽：fc = 500 Hz → ωc = 2π × 500 = **3142 rad/s**
   - （或按经验取 ωc = (1/3~1/5) × 2π × fpwm = 3350~2010 rad/s，取 **3000 rad/s**）
   
   **步骤3：确定观测器带宽 ωo**
   - 推荐 ωo = (3~5) × ωc = **9000~15000 rad/s**
   - 取 ωo = **10000 rad/s**（注意：需满足 ωo < π/Ts = 50265 rad/s 以保证离散化稳定）
   
   **步骤4：计算LESO增益**（二阶LESO，极点在-ωo）：
   - β₁ = 2ωo = **20000**
   - β₂ = ωo² = **100,000,000 = 1×10⁸**
   
   **步骤5：计算PD增益**：
   - kp = ωc² / b₀ = 9×10⁶ / 400 = **22500**
   - kd = 2ωc / b₀ = 6000 / 400 = **15**

2. **参数数量对比**：
   
   | 控制器 | 参数 | 数量 |
   |--------|------|------|
   **PI（零极点对消）** | Kp, Ki (+ 需要知道 Rs, Ls) | **2个控制参数 + 2个对象参数** |
   **LADRC** | ωc, ωo, b₀ | **3个（全部是整定参数）** |
   
   LADRC优势：**不需要知道Rs和Ls的精确值**（它们的变化被ESO当作扰动估计）

3. **计算量估算**（每次中断调用）：
   
   **PI（增量式）**：
   ```
   e = ref - fb;           // 1次减法
   u = Kp*(e-e_prev) + Ki*Ts*e + u_prev;  // 2次乘法 + 2次加法 + 1次乘法
   u = clamp(u, ...);      // 2次比较
   总计: ~5次浮点运算
   ```
   
   **LADRC（一阶对象+二阶LESO）**：
   ```
   e = z1 - fb;            // 1次减法
   z1 += Ts*(z2 - beta1*e + b0*u);  // 3次乘法 + 3次加法
   z2 += Ts*(-beta2*e);    // 1次乘法 + 1次加法
   u0 = kp*(ref - z1) - kd*z2;  // 2次乘法 + 2次加减法
   u = u0 - z2/b0;         // 1次除法 + 1次减法
   u = clamp(u, ...);      // 2次比较
   总计: ~15次浮点运算
   ```
   
   **结论：LADRC计算量约为PI的3倍**，但在Cortex-M4+FPU上仍远小于中断预算（62.5μs内轻松完成）。

### 题目2：电流环LADRC的抗扰性能验证
**问题：** 同一台电机，分别用PI和LADRC控制电流环。测试条件：突加反电动势扰动（模拟电机从静止突然以3000rpm旋转）。
1. 分析反电动势对电流环的影响机制
2. 分别推导PI和LADRC对该扰动的响应
3. 定量计算：反电动势从0阶跃到E=40V时的Iq跌落和恢复时间

**参考答案：**
1. **反电动势影响机制**：
   - 反电动势 E = Ke × ω = 0.053 × 3000 = **159V**（线电动势）→ 相电动势 ≈ **92V**
   - 等效为：电流环被控对象上叠加了一个阶跃扰动 f = -E/Ls = -92/0.0025 = **-36800 A/s**
   - PI的反应：I项需要时间累积误差来产生补偿电压
   - LADRC的反应：ESO的z₂直接估计该扰动→立即补偿

2. **响应推导**：
   
   **PI（零极点对消，ωc=3000 rad/s）**：
   - 闭环传递函数对扰动的灵敏度：$|\frac{I_q(s)}{E(s)}| \approx \frac{1}{Ls \cdot (s+\omega_c)} = \frac{400}{s(s+3000)}$
   - 阶跃扰动 E(s)=92/s 下的响应：
   - 稳态误差：ess = 0（PI有积分项，最终完全补偿）
   - 最大跌落（近似）：ΔIq_peak ≈ E/(Ls·ωc) = 92/(0.0025×3000) = **12.3A**
   - 恢复时间（±5%）：ts ≈ 4/ωc = **1.33ms**
   
   **LADRC（ωc=3000, ωo=10000）**：
   - ESO估计时间常数：τ_est ≈ 4/ωo = 0.4ms
   - 扰动估计后立即补偿 → 残余误差取决于ESO估计精度
   - 最大跌落：ΔIq_peak ≈ E/(Ls·ωo) = 92/(0.0025×10000) = **3.68A**（ESO越快，跌落越小）
   - 恢复时间：ts ≈ 4/ωo = **0.4ms**（由ESO带宽决定，而非控制器带宽）

3. **定量对比**：
   
   | 指标 | PI | LADRC | 改善 |
   |------|-----|-------|------|
   最大电流跌落 | 12.3A | 3.68A | **70%↓** |
   恢复时间 | 1.33ms | 0.4ms | **70%↓** |
   稳态误差 | 0 | 0 | 持平 |

### 题目3：LADRC电流环的离散化细节
**问题：** LADRC电流环在DSP上以16kHz运行。讨论几个关键的实现细节。
1. LESO状态的初始化值应该如何设？
2. 当电流环输出Uq达到限幅时，LADRC该如何处理？
3. 如何验证LADRC电流环的正确性？

**参考答案：**
1. **LESO状态初始化**：
   ```c
   // 推荐初始化
   z1 = 0;    // 电流估计初始值为0（电机静止时）
   z2 = 0;    // 扰动估计初始值为0（无先验信息时）
   
   // 更好的做法（如果已知工作点）：
   z1 = Iq_bias;    // 如有偏置电流
   z2 = E_bemf_est; // 如果已知当前转速的反电动势等效扰动
   ```

2. **输出限幅（饱和）处理**：
   - **问题**：标准LADRC没有anti-windup。当u被限幅时，LESO仍在用「未被限幅的u」更新状态→估计偏离
   - **方案A（简单）**：限幅后将实际输出值反馈给LESO（用clamp后的u更新z1）
     ```c
     u_raw = u0 - z2/b0;
     u_sat = clamp(u_raw, -UQ_MAX, UQ_MAX);
     
     // LESO用限幅后的u更新（关键！）
     e = z1 - iq_fb;
     z1 += Ts*(z2 - beta1*e + b0*u_sat);  // 用u_sat而非u_raw
     z2 += Ts*(-beta2*e);
     ```
   - **方案B（进阶）**：加入条件积分式anti-windup

3. **正确性验证步骤**：
   ```
   Step 1: 开环验证
   - 断开电流环反馈，注入正弦Iq_ref
   - 检查z1是否跟踪Iq_fb（应一致）
   - 检查z2是否估计出反电动势（应有直流分量）
   
   Step 2: 闭环阶跃响应
   - Iq_ref阶跃 0→5A
   - 检查超调<5%，上升时间符合预期（~2.2/ωc）
   - 检查z2是否有明显的估计值（不应为0）
   
   Step 3: 抗扰测试
   - 保持Iq_ref=5A恒定
   - 手动施加外部扰动（如机械堵转一下再释放）
   - 观察Iq恢复速度和z2的估计波形
   
   Step 4: 参数扫描
   - ωc从1000变到5000，观察响应变化趋势
   - ωo从3ωc变到10ωc，观察抗扰性能变化
   - b₀从0.5×nominal变到2×nominal，验证鲁棒性
   ```

## 二、LADRC速度环实现（★★★★★）

### 题目4：速度环LADRC的ESO阶数选择
**问题：** 速度环的被控对象：$J\dot{\omega} = K_t I_q - T_L - B\omega$（含粘滞摩擦B）。
1. 应该用几阶ESO？为什么？
2. 写出三阶LESO的状态方程
3. 三阶LESO的带宽参数化公式是什么？

**参考答案：**
1. **ESO阶数选择：三阶（LESO-3）**
   
   速度环对象本质是**一阶**（J·ω̇ = Te - TL），需要估计**负载转矩扰动TL**。
   
   扩展状态：x₁=ω（速度）, x₂=扰动f, x₃=扰动导数ḟ
   
   选择**三阶LESO**的原因——为了更好地估计变化的扰动及其变化率

2. **三阶LESO状态方程**：
   $$\begin{cases}
   e = z_1 - \omega \\
   \dot{z}_1 = z_2 - \beta_1 e + b_0 u \\
   \dot{z}_2 = z_3 - \beta_2 e \\
   \dot{z}_3 = -\beta_3 e
   \end{cases}$$

3. **带宽参数化（极点在 -ωo，三重极点）**：
   - **β₁ = 3ωo**
   - **β₂ = 3ωo²**
   - **β₃ = ωo³**

### 题目5：速度环LADRC的完整C代码
**问题：** 基于三阶LESO，写出完整的速度环LADRC的C代码实现。

**参考答案：**
```c
typedef struct {
    float z1, z2, z3;
    float beta1, beta2, beta3;
    float b0, kp, kd;
    float out_min, out_max;
    float output;
} LadrcSpeed_t;

void ladrc_speed_init(LadrcSpeed_t *ladrc, float omega_c, float omega_o, 
                       float Kt, float J_total, float iq_max) {
    ladrc->b0 = Kt / J_total;
    ladrc->beta1 = 3.0f * omega_o;
    ladrc->beta2 = 3.0f * omega_o * omega_o;
    ladrc->beta3 = omega_o * omega_o * omega_o;
    ladrc->kp = omega_c * omega_c / ladrc->b0;
    ladrc->kd = 2.0f * omega_c / ladrc->b0;
    ladrc->out_min = -iq_max;
    ladrc->out_max = iq_max;
    ladrc->z1 = ladrc->z2 = ladrc->z3 = ladrc->output = 0.0f;
}

float ladrc_speed_update(LadrcSpeed_t *ladrc, float spd_ref, 
                          float spd_fb, float ts) {
    float e = ladrc->z1 - spd_fb;
    
    ladrc->z1 += ts * (ladrc->z2 - ladrc->beta1 * e 
                        + ladrc->b0 * ladrc->output);
    ladrc->z2 += ts * (ladrc->z3 - ladrc->beta2 * e);
    ladrc->z3 += ts * (-ladrc->beta3 * e);
    
    float u0 = ladrc->kp * (spd_ref - ladrc->z1) - ladrc->kd * ladrc->z2;
    ladrc->output = u0 - ladrc->z2 / ladrc->b0;
    
    ladrc->output = fmaxf(ladrc->out_min, fminf(ladrc->output, ladrc->out_max));
    return ladrc->output;
}
```

## 三、系统集成与优化（★★★★★）

### 题目7：LADRC三环级联系统设计
**问题：** 用LADRC构建完整的三环伺服系统：电流环LADRC + 速度环LADRC + 位置环P控制。
1. 各环的带宽如何分配？与传统PI三环有何不同？
2. 位置环能否也用LADRC？有什么优劣？

**参考答案：**
1. **带宽分配对比**：
   
   | | PI三环 | LADRC三环 |
   |---|--------|-----------|
   **电流环** | ωc_i = 2000~4000 rad/s | 基本相同 |
   **速度环** | ωc_v = ωc_i / 5~10 | 可略高（LADRC抗扰更强） |
   **位置环** | ωc_p = ωc_v / 5~10 | P控制，带宽不宜过高 |
   
   关键差异：LADRC三环的带宽比可以比PI更紧凑（如4:1而非10:1）

2. **位置环用LADRC**：
   - 优势：减少一个控制环；统一架构
   - 劣势：位置信号经两次微分→噪声极度敏感；需23位以上编码器
   
   结论：**位置环用P控制 + 速度环LADRC是更实用的方案**

### 题目8：LADRC的参数自整定工具设计
**问题：** 为LADRC设计参数自动配置工具——用户只需输入电机参数，工具自动计算出所有增益。

**参考答案：**
```
输入: Ls, Rs, Kt, J_motor, J_load_ratio, fpwm, 目标带宽
输出: 各环的 b0, ωc, ωo, β1~β3, kp, kd

核心算法:
  电流环: b0=1/Ls, ωo=(3~5)×ωc, β1=2ωo, β2=ωo², kp=ωc²/b0, kd=2ωc/b0
  速度环: b0=Kt/J_total, ωo=(3~5)×ωc, β1=3ωo, β2=3ωo², β3=ωo³, kp=ωc²/b0, kd=2ωc/b0

校验规则:
  - ωo > π/(2Ts) → 警告: 降低ωo或提高PWM频率
  - ωo/ωc < 3 或 > 10 → 建议: 调整到3~5范围
  - 速度环ωc > 电流环ωc/3 → 警告: 违反带宽级联约束
```
