### 题目1：Luenberger观测器增益设计
**问题：** PMSM电流-反电动势四阶模型（αβ坐标系），A矩阵已知，C=[I₂ₓ₂ 0₂ₓ₂]（只测电流）。要求观测器极点配置在 s=-3000（四重根）。
1. 计算观测器增益矩阵L（4×2）
2. 解释为什么观测器极点要比电流环极点快5倍以上
3. 如果实际L增大了20%，观测器还能稳定吗？收敛速度会如何变化？

**参考答案：**
1. 期望特征多项式：(s+3000)⁴，使用Ackermann公式或直接求解L使|sI-(A-LC)|=(s+3000)⁴
   计算得L≈[5800, ωe·const; -ωe·const, 5800; -9×10⁶, 0; 0, -9×10⁶]（具体值取决于电机参数）

2. 观测器是"控制器之前的控制器"：
   - 控制器用估计状态做决策
   - 如果观测器收敛慢→估计状态有较大偏差→控制器决策基于"错误信息"
   - 5倍规则确保观测器误差在控制器做出反应前已经收敛

3. L增大20%：
   - A-LC的特征值会改变→但稳定性不一定丧失（只要|sI-(A-LC)|的根仍在左半平面）
   - 一般来讲L增大相当于"更相信测量"，观测器更快但噪声放大
   - 对于Luenberger观测器，L增大不破坏稳定性（只要(A,C)可观）但可能放大测量噪声

### 题目2：SMO增益k的选择
**问题：** PMSM SMO的开关增益k需要满足滑模存在条件。
1. 写出滑模存在条件k的表达式（包含反电动势幅值）
2. 一台电机 ψf=0.08Wb, ωe_max=2000rad/s, L=0.8mH，计算最小k值
3. k选择过大会有什么问题？

**参考答案：**
1. 滑模存在条件：$k > |e_{\alpha,\beta}|_{max} = \psi_f \cdot \omega_{e,max}$（或更保守地k>ψf·ωe_max+裕度）

2. k_min = ψf·ωe_max = 0.08×2000 = **160 V**
   加上50%裕度：k = 240 V
   在SMO方程中 k/L = 240/0.0008 = 300000 A/s

3. k过大 → 抖振（chattering）加剧 →
   - 等效控制信号z_eq的高频分量更大 → 需要更低截止频率的LPF → 更大的相位滞后
   - 角度估计的延迟增大 → Park变换角度不准 → 转矩脉动
   - 极限情况下可能激发机械谐振

## 二、分析设计题（★★★★☆）

### 题目3：EKF的Q和R矩阵设计
**问题：** 一台PMSM的EKF，状态x=[id,iq,ωm,θe]ᵀ。
1. 如果电流传感器噪声variance=0.01A²，写出R矩阵
2. 转速变化剧烈（高动态伺服），Q₃₃应如何设置？给出具体值并说明理由
3. Q₄₄=0可能吗？为什么？

**参考答案：**
1. R = diag(0.01, 0.01) = [0.01 0; 0 0.01]

2. 高动态伺服：转速变化率可达10000rpm/s=1047rad/s²
   在一个Ts=62.5μs周期内，转速变化≈0.065rad/s
   Q₃₃反映转速模型的不确定性→设为(Δω)²=(0.065)²≈**0.004**
   但实际中为提高动态跟踪，通常设得更大：**Q₃₃≈1~10**

3. Q₄₄=0 **不应该设为0，但不影响数值稳定性**。
   设Q₄₄=0意味着"角度模型完全精确"（θ̇=p·ωm无误差）。
   但实际上ωm有估计误差→θ̇=p·ω̂m也有误差→Q₄₄应>0
   实践中Q₄₄=0.1~1是常见范围，越小→角度估计越平滑（但可能滞后）

### 题目4：无传感器零速启动的问题
**问题：** EKF在零速时为什么无法收敛？
1. 从能观性角度解释
2. 从信息论角度解释
3. 提出至少两种零速/极低速的解决方案

**参考答案：**
1. 能观性：EKF基于反电动势信息估计转速。零速时反电动势e=ψf·ωe=0，电流中不包含转速信息→能观性矩阵降秩→无法从电流唯一确定转速。

2. 信息论：反电动势在电流信号中的"信噪比"正比于ωe/|i|。零速时SNR=0，没有任何关于转速的信息可用→任何估计都是"猜测"。

3. 解决方案：
   - **高频注入法（HFI）：** 注入高频电压/电流（500Hz~2kHz），利用转子的凸极性（Ld≠Lq）检测角度。IPM电机效果好，SPM电机效果差（凸极比小）。
   - **开环IF启动：** 用固定电流幅值、逐渐增加频率的开环方式启动，类似步进电机。切换到闭环需要平滑过渡。
   - **初始位置检测：** 注入脉冲电压检测初始角度，避免启动反转。
   - **混合策略：** 零速用HFI→低速用HFI+EKF融合→中高速纯EKF。

### 题目5：SMO vs EKF 在参数失配下的表现
**问题：** 电机电阻R因温升从0.2Ω增至0.3Ω（+50%），ψf因温升从0.08Wb降至0.072Wb（-10%）。分别分析SMO和EKF的角度估计误差。
1. 对SMO的影响（定性分析）
2. 对EKF的影响（定性分析）
3. 哪种方法更鲁棒？为什么？

**参考答案：**
1. SMO：
   - R增大50%：SMO中R出现在线性项-Rîα/L中，影响电流估计的收敛速度，但不直接影响滑模面到达条件。滑模面上的等效控制信号z_eq仍主要取决于反电动势→**角度误差约2~5°**
   - ψf减小10%：不影响角度提取（角度来自z_eq的方向，不是幅值）。但转速估计ω̂=√(eα²+eβ²)/ψf会偏小10%

2. EKF：
   - R增大50%：EKF的F矩阵（Jacobian）中-R/L项不准确→预测不准确→估计偏离。但K矩阵会通过测量（电流）自动修正→**角度误差约3~8°**
   - ψf减小10%：转速和角度方程中ψf项不准确→角度估计可能产生相位滞后→**角度误差约1~3°**

3. **SMO更鲁棒！** SMO的滑模切换机制天然对参数不敏感（只要k足够大，滑模总能到达），参数变化只影响收敛速度而不影响稳态精度。EKF虽然是"最优"的，但"最优"建立在模型准确的前提下——模型不准时，最优性丧失。

## 三、编程实现题（★★★★★）

### 题目6：EKF代码中的数值稳定性
**问题：** EKF中协方差矩阵P必须保持对称正定，但数值计算可能破坏这一性质。
1. P矩阵的Joseph稳定形式是什么？写出更新方程
2. 在定点DSP上（16位），如何避免P矩阵的数值问题？
3. 如果检测到P矩阵非正定，应急处理方案是什么？

**参考答案：**
1. Joseph形式（更数值稳定）：
   P_k = (I-KH)P_{k|k-1}(I-KH)ᵀ + KRKᵀ
   比标准形式 P=(I-KH)P 多一次矩阵运算，但保证P的对称性和正定性。

2. 定点DSP策略：
   - 使用UD分解（U-D factorization）：P=UDUᵀ，U是单位上三角阵，D是对角阵。更新U和D而非P，天然保持正定性。
   - 或用平方根滤波（Square-root Kalman）：传播P的Cholesky分解S（P=SSᵀ）。
   - 使用32位累加器（如Cortex-M4的FPU）。

3. 应急处理：
   ```
   if (P[0] < 0 || P[5] < 0 || P[10] < 0 || P[15] < 0) {
       // 对角线元素为负 → 非正定
       reset_covariance(P);  // 重置为初始P0
   }
   // 或强制对称
   for (i=0; i<4; i++)
       for (j=i+1; j<4; j++)
           P[i*4+j] = P[j*4+i] = (P[i*4+j] + P[j*4+i]) / 2;
   ```

### 题目7：观测器在中断中的执行时间
**问题：** EKF在Cortex-M4 @168MHz上的执行时间估算：
1. 四阶EKF一次完整迭代需要多少次浮点乘法？
2. 在16kHz PWM频率下，EKF占用CPU的百分比？
3. 如果同时运行EKF和FOC，CPU负载是否可接受？

**参考答案：**
1. 矩阵运算统计：
   - F*P*Fᵀ: 4×4矩阵乘法×2 ≈ 64次乘法
   - P+Q: 16次加法
   - H*P*Hᵀ+R: ~16次乘法
   - 矩阵求逆（2×2）: ~6次乘法
   - K*(y-Hx): ~12次乘法
   - P更新（Joseph）: ~80次乘法
   
   Jacobian计算：~20次乘法
   状态预测（非线性）：~16次乘法
   
   总计：**约220次浮点乘法 + 若干加法**

2. Cortex-M4 FPU单周期浮点乘法 → 220周期
   加上控制逻辑和加法 ≈ 400~500周期
   168MHz → 500/168M = **3.0μs**
   16kHz周期=62.5μs → 占用率=3.0/62.5=**4.8%**
   
3. CPU负载分析：
   - EKF: ~5%
   - FOC (Clarke/Park/PI/SVPWM): ~5%
   - ADC处理 + 通信: ~3%
   - 总计: **~13%** → **完全可接受，仍有大量余量**

## 四、综合应用题（★★★★★）

### 题目8：全速域无传感器控制策略设计
**问题：** 设计一套覆盖0~6000rpm的PMSM无传感器控制策略，要求：
1. 划分速度区间及各区间使用的观测器/方法
2. 设计区间切换的平滑过渡策略
3. 讨论在重载启动（带满载从零速启动）时的特殊挑战

**参考答案：**
1. 速度区间划分（以4对极电机为例）：
   - **区间I（0~150rpm）：** 高频注入（HFI）+ 初始位置检测
     原因：反电动势太低，EKF/SMO不可用
   - **区间II（150~500rpm）：** HFI+EKF混合，逐渐过渡
     原因：反电动势开始可用但信噪比低
   - **区间III（>500rpm）：** 纯EKF或SMO
     原因：反电动势信噪比足够

2. 平滑过渡策略：
   ```
   混合权重 α = clamp((ω_est - 150) / (500 - 150), 0, 1)
   θ_final = (1-α) * θ_HFI + α * θ_EKF
   ω_final = (1-α) * ω_HFI + α * ω_EKF
   ```

3. 重载启动挑战：
   - 初始位置检测时，负载转矩会拖动转子→检测结果不准
   - 开环IF启动时，需要足够大的电流克服负载→可能引起过流
   - 切换到闭环时，负载导致转速跌落→EKF收敛困难
   
   **应对策略：**
   - 初始位置检测时施加制动电流锁定转子
   - IF启动阶段电流给定 = max(固定值, 负载转矩/Kt)
   - 闭环切换时，先用PI控制器稳定转速再投入EKF
   - 考虑负载转矩观测器（扩张状态观测器ESO）辅助

### 题目9：EKF在MCU上的定点优化
**问题：** 在低成本Cortex-M0（无FPU）上实现EKF。
1. 设计定点数格式（Q格式选择）
2. 矩阵求逆的定点实现（2×2矩阵）
3. 避免除法和开方的策略

**参考答案：**
1. Q格式：
   - 电流id/iq（±数A）：Q20（范围±2048A，精度4.8×10⁻⁷）
   - 转速ωm（0~6000rpm→628rad/s）：Q22（范围±512，精度2.4×10⁻⁷）
   - 角度（0~2π）：Q14（[0, 2π]→[0, 65536]，精度9.6×10⁻⁵rad）
   - 协方差P：Q24（大动态范围）

2. 2×2矩阵求逆（S = [a b; b c]）：
   ```
   det_Q = a*c - b*b  // Q48累加后右移
   if (det_Q < 1) det_Q = 1  // 防止除零
   inv_a = (c << 24) / det_Q
   inv_b = (-b << 24) / det_Q
   inv_c = (a << 24) / det_Q
   ```

3. 避免除法和开方：
   - 用倒数查找表代替除法（256点线性插值）
   - 用CORDIC或查表+牛顿迭代计算反电动势的√(eα²+eβ²)
   - atan2用CORDIC或查表实现

### 题目10：观测器故障诊断
**问题：** 在实际运行中，EKF可能出现发散。设计观测器健康监测机制：
1. 发散的特征有哪些（至少3个可检测的指标）？
2. 设计一个发散自动检测与恢复的状态机
3. 当检测到发散时，如何"优雅降级"而非直接停机？

**参考答案：**
1. 检测指标：
   - **新息检验（Innovation test）：** |y-Hx̂| > 3√(HPHᵀ+R) → 统计异常
   - **协方差发散：** trace(P)持续增大 → P矩阵爆炸
   - **角度跳变：** |θ̂[k]-θ̂[k-1]| > p·ω̂m·Ts×1.5 → 角度变化超过物理可能
   - **电流残差：** RMS(id-îd, iq-îq) > 阈值（如0.5A）持续超过10ms

2. 状态机：
   ```
   State_NORMAL → (innovation > 5σ for 5ms) → State_SUSPECT
   State_SUSPECT → (innovation > 10σ) → State_DIVERGED
   State_SUSPECT → (innovation < 5σ for 10ms) → State_NORMAL
   State_DIVERGED → (reinitialize EKF) → State_RECOVERING
   State_RECOVERING → (innovation < 5σ for 20ms) → State_NORMAL
   ```

3. 优雅降级策略：
   - **不是立即停机！** 而是：
   - 阶段1：切换到开环IF控制（基于最后的ω̂），角度用开环积分
   - 阶段2：重置EKF协方差为P0，用开环角度重新初始化
   - 阶段3：EKF在新条件下重新收敛，检测稳定后切回闭环
   - 如果3次恢复尝试均失败→触发安全停机
