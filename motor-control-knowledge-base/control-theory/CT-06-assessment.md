## 一、参数计算题（★★☆☆☆）

### 题目1：反电动势解耦前馈电压计算
**问题：** 一台PMSM参数：Ld=1.8mH, Lq=2.2mH, ψf=0.09Wb, P=3。运行工况：n=4000rpm, Id=0A, Iq=12A。请：
1. 计算电角速度ωe
2. 计算d轴前馈电压Vd_ff
3. 计算q轴前馈电压Vq_ff
4. 若取消前馈，PI需要独自产生多少Vq电压来抵消反电动势？

**参考答案：**
1. ωe = P × 2π × n/60 = 3 × 2π × 4000/60 = 3 × 418.9 = 1256.6 rad/s
2. Vd_ff = -ωe × Lq × Iq = -1256.6 × 0.0022 × 12 = -33.2 V
3. Vq_ff = ωe × (Ld × Id + ψf) = 1256.6 × (0 + 0.09) = 113.1 V
4. 若无前馈，PI需产生Vq_PI=113.1V来抵消反电动势。如果Vdc=200V, PI预留空间仅约200/√3-113.1=2.4V→几乎无动态调节余量→极易饱和。

### 题目2：速度前馈的跟踪误差分析
**问题：** 位置伺服系统跟踪斜坡指令 θ_ref(t)=100t (°/s)。位置环PI参数Kp_pos=50, Ki_pos=200。系统速度误差常数Kv=Kp_pos×Kt/(J×Ki_pos)≈25。请：
1. 计算无速度前馈时的稳态跟踪误差
2. 若加入理想速度前馈(Kvff=1.0)，稳态跟踪误差为多少？
3. 若前馈增益偏差-10%(Kvff=0.9)，稳态误差为多少？

**参考答案：**
1. 斜坡跟踪误差 ess = 斜坡速率 / Kv = 100 / 25 = 4°（反馈PI对斜坡输入有有限稳态误差）
2. 理想前馈：前馈完全预置了100°/s的速度命令 → 反馈PI不需要产生稳态输出来维持速度 → ess ≈ 0（仅剩摩擦力等未建模扰动引起的微小误差）
3. 偏差10%：前馈提供90°/s的速度命令 → 反馈需补足10°/s → ess = 10/25 = 0.4° → 虽非零但仍比无前馈(4°)好10倍

## 二、概念辨析题（★★★☆☆）

### 题目3：前馈不影响稳定性——正误辨析
**问题：** 「前馈控制不影响闭环系统的稳定性」这句话在什么条件下正确？以下情况是否仍然成立？
1. 前馈和反馈的输出之和进入同一个饱和限幅环节
2. 前馈模型含有不稳定零点
3. 前馈信号通过与反馈信号不同的路径进入被控对象

**参考答案：**
1. ❌ 不成立。前馈和反馈共享饱和限幅时，前馈过大会提前推动系统进入饱和区→反馈的积分器在饱和期间的行为受anti-windup影响→间接影响闭环行为。从而前馈的幅值间接影响稳定性。
2. ✅ 理论成立但不可实现。前馈的传递函数 $C_f(s)$ 不进入闭环特征方程，不改变极点位置。但若 $C_f(s)$ 含有不稳定极点（RHP极点），前馈输出自身发散→饱和→间接影响。工程中前馈必须自身稳定。
3. ✅ 成立。只要前馈路径不与反馈路径交汇（即前馈是纯粹的开环注入），闭环特征方程不变。多路径注入不影响此结论。

### 题目4：逆模型前馈与低通滤波
**问题：** 电机RL逆模型 $G^{-1}(s)=R_s+L_s s$（分子阶次>分母阶次→非因果→无法直接实现）。请解释：
1. 为什么在DSP上可以实现含s的传递函数（虽然是离散的）但纯G^{-1}(s)仍被认为不可实现？
2. 设计可实现的近似前馈 $C_f(s) = \frac{R_s+L_s s}{\tau_f s+1}$，选择τf的原则是什么？

**参考答案：**
1. DSP可实现的是因果系统（输出不超前于输入）。$G^{-1}(s)=R_s+L_s s$意味着输出含有输入的微分→在连续域这需要未来的输入值→严格意义下不可物理实现。DSP上虽可通过差分近似微分（如$s\approx(z-1)/T_s$），但差分放大了高频噪声且相位响应在高频处错误。引入低通滤波使传递函数严格因果可实现。
2. τf选择原则：a) τf << 1/ωc（否则前馈在高频处增益被过度衰减→失去前馈意义）；b) τf >> Ts（否则数字实现不稳定）；c) τf不小于L/R（否则噪声放大严重）。工程取值：τf = (0.1~0.5) × (L_s/R_s)。

## 三、设计应用题（★★★☆☆）

### 题目5：完整FOC电流环前馈+反馈设计
**问题：** 电机参数：Ls=2mH, Rs=0.3Ω, ψf=0.1Wb, P=4。设计电流环：ωc=1500 rad/s。
1. 计算PI参数(Kp, Ki)和反电动势前馈系数
2. 画出完整的控制框图（标注每个信号）
3. 写出DSP实现的伪代码

**参考答案：**
1. Kp=2mH×1500=3.0；Ki=0.3×1500=450。前馈系数：K_ff_d=-Lq=-2mH（负号）；K_ff_q1=Ld=2mH（Id耦合补偿）；K_ff_q2=ψf=0.1（反电动势补偿）。
2. 框图：Id_ref→PI_d→Vd_PI; Vd_ff=-ωe·Lq·Iq→Vd=Vd_PI+Vd_ff；Iq_ref→PI_q→Vq_PI; Vq_ff=ωe·(Ld·Id+ψf)→Vq=Vq_PI+Vq_ff。
3. ```c
void CurrentLoop(float Id_ref, float Iq_ref, float we, float *Vd, float *Vq) {
    float Id_err = Id_ref - Id_fb;
    float Iq_err = Iq_ref - Iq_fb;
    // PI
    float Vd_pi = Kp*Id_err + Id_integral;
    float Vq_pi = Kp*Iq_err + Iq_integral;
    // 前馈
    float Vd_ff = -we * Lq * Iq_fb;
    float Vq_ff = we * (Ld * Id_fb + psi_f);
    // 总输出
    *Vd = Vd_pi + Vd_ff;
    *Vq = Vq_pi + Vq_ff;
    // anti-windup更新积分
    Id_integral += Ki*Ts*Id_err + Kb*(clamp(*Vd)-*Vd);
    Iq_integral += Ki*Ts*Iq_err + Kb*(clamp(*Vq)-*Vq);
}
```

### 题目6：加速度前馈的设计与限制
**问题：** 位置伺服加速度前馈 $I_q^{ff} = J \cdot \ddot{\theta}_{ref} / K_t$。J=0.005 kg·m², Kt=0.6 N·m/A, Imax=20A。轨迹规划最大加速度=5000 °/s²（≈87.3 rad/s²）。请：
1. 计算最大加速度前馈电流
2. 如果J估计偏大30%，前馈电流会怎样？
3. 前馈电流和反馈电流之和超过Imax时应该怎么处理？

**参考答案：**
1. $I_q^{ff}_{max} = 0.005 \times 87.3 / 0.6 = 0.728$ A → 远小于Imax=20A → 无问题
2. J偏大30%→前馈电流偏大30%→前馈过补偿→反馈PI需要产生反向电流来纠正→增加PI负担但仍在可控范围(0.728的30%仅0.22A)。但如果同时有其他前馈(速度FF+摩擦FF)→叠加后可能过补偿严重。
3. 前馈和反馈之和超Imax时：优先保证前馈通过（前馈是预测性的、不可延迟），限制反馈输出为Imax-Iq_ff。或采用前馈优先+反馈限幅策略。不应对前馈也限幅→前馈限幅会破坏其预测性，降低跟踪效果。

## 四、综合案例题（★★★★☆）

### 题目7：前馈参数偏差的完整影响链
**问题：** 反电动势解耦前馈中Lq估计值偏差对电流环的影响链。Lq_real=3mH, Lq_est=2mH(-33%)。ωe=2000 rad/s, Iq=10A。请：
1. 计算d轴前馈残差
2. 分析该残差在频域的等效扰动，PI在ωc=1200时能否抑制？
3. 推导d轴残余耦合电流的稳态表达式

**参考答案：**
1. Vd_ff_ideal = -2000×0.003×10 = -60V；Vd_ff_actual = -2000×0.002×10 = -40V；残差ΔVd = 20V（d轴存在20V未被补偿的耦合电压）。
2. 残差等效为d轴的阶跃扰动D=20/s。扰动到Id的传递函数 $G_{dist}(s) = \frac{1}{R_s+L_s s} \cdot \frac{1}{1+L(s)}$。在ωc=1200处|S(j1200)|≈0.707(-3dB)→残差的30%可通过→Id出现稳态耦合。在低频处PI积分可抑制，但接近ωc处抑制能力下降。
3. 稳态残差Id_couple由PI积分消除：$I_d^{ss} = 0$（因PI含积分器）。但在动态中残余Id存在：$I_d(t) \approx \frac{\Delta V_d}{R_s}(1-e^{-t/\tau_{cl}})$，其中τ_cl=1/ωc。即瞬时最大Id耦合≈ΔVd/Rs=20/0.2=100A（假设Rs=0.2Ω）→严重！必须修正Lq或降低ωe。

### 题目8：基于前馈的速度环自抗扰设计
**问题：** 速度环PI在负载突变时恢复慢。设计一个负载转矩观测器作为扰动前馈：
1. 写出降阶负载观测器的状态方程
2. 观测的负载转矩如何注入为前馈？
3. 分析观测器带宽对前馈效果的影响

**参考答案：**
1. 速度环机械方程：$J\dot{\omega} = K_t I_q - T_L$。令$T_L$为扩张状态，观测器：
   $\dot{\hat{\omega}} = (K_t I_q - \hat{T}_L)/J + l_1(\omega - \hat{\omega})$
   $\dot{\hat{T}}_L = l_2(\omega - \hat{\omega})$
   $l_1, l_2$为观测器增益。特征方程$s^2+l_1 s-l_2/J=0$，设计带宽$\omega_o$: $l_1=2\omega_o, l_2=-J\omega_o^2$。
2. 前馈：$I_q^{ff} = \hat{T}_L / K_t$。当负载增大→$\omega$下降→观测器感知$\hat{T}_L$增大→$I_q^{ff}$增大→抵消负载→$\omega$几乎不变。反馈PI只处理观测误差。恢复时间从纯PI的数百ms降至数十ms。
3. $\omega_o$越高→观测器收敛越快→前馈补偿越及时→但噪声放大越严重（观测器本质是高频放大器）。$\omega_o$通常设为速度环带宽的3~5倍。

### 题目9：柔性负载的谐振频率前馈抑制
**问题：** 电机+弹性联轴器+负载构成双惯量系统，谐振频率f_res=80Hz。速度环PI在80Hz附近产生谐振峰。设计陷波滤波器前馈抑制方案。

**参考答案：**
前馈通道中加入陷波滤波器：
$$H_{notch}(s) = \frac{s^2 + 2\zeta_z\omega_n s + \omega_n^2}{s^2 + 2\zeta_p\omega_n s + \omega_n^2}$$
取$\omega_n = 2\pi\times80 = 503$ rad/s，$\zeta_z=0.01$（零点窄），$\zeta_p=0.2$（极点宽）。陷波器放置在速度环前馈路径（Iq_ff通道），阻止80Hz的前馈分量激励机械谐振。同时反馈PI也需相应调整（降低80Hz附近的增益）。

DSP实现（Tustin离散化）：

y(n) = b0*x(n) + b1*x(n-1) + b2*x(n-2) - a1*y(n-1) - a2*y(n-2);

### 题目10：自抗扰ESO vs 传统前馈——综合对比
**问题：** 在FOC电流环中，反电动势前馈($V_q^{ff}=\omega_e\psi_f$)和ADRC的ESO(扩张状态观测器)都能补偿反电动势。从以下维度对比：
1. 对电机参数的依赖程度
2. 相位特性（延迟）
3. DSP计算量
4. 适用场景建议

**参考答案：**
1. **参数依赖**：传统前馈直接依赖$\psi_f$（±10%精度影响10%前馈精度）。ESO理论上不依赖精确参数，通过实时观测补偿「总扰动」（含反电动势+参数偏差+未建模动态），但ESO的收敛速度受$\omega_o$约束。
2. **相位**：传统前馈是即时前馈（零延迟），反电动势补偿无相位滞后。ESO是观测器→有固有的观测延迟（带宽$\omega_o$决定），对反电动势的估计有$\tau_o=1/\omega_o$的延迟。
3. **计算量**：传统前馈：2次乘法/轴（$\omega_e\times\psi_f$和$\omega_e\times L_q\times I_q$）→约4次乘法→极轻。ESO：2×2状态更新（2轴）→约16次乘法+非线性函数→中等（约为PI的3~5倍）。
4. **建议**：电机参数已知且变化不大→传统前馈（简单、高效、无延迟）。电机参数完全未知或时变剧烈（如高温应用、磁饱和严重）→ESO可提供更好的鲁棒性。多数工业伺服选择传统前馈+在线参数辨识的混合方案。
