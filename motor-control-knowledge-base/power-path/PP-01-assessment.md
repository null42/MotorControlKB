## 一、参数计算题（★★★☆☆）

### 题目1：Buck 电感电流纹波与临界模式
**问题：** 设计一个 Buck 变换器用于电机驱动栅极电源：Vin = 48V，Vout = 15V，fs = 250kHz，L = 33μH，输出电流 Iout_max = 3A，Iout_min = 0.2A。请计算：
1. 满载时的电感电流纹波 ΔIL 和峰值电流 Ipeak
2. CCM/DCM 临界电流 Iout_crit
3. 最小负载时（0.2A），该 Buck 工作于什么模式？如果是 DCM，输出电压会如何变化（定性分析）？

**参考答案：**
1. D = 15/48 = 0.3125；ΔIL = 48 × 0.3125 × (1-0.3125) / (250000 × 33e-6) = 10.31 / 8.25 = 1.25A；Ipeak = 3 + 1.25/2 = 3.625A
2. Iout_crit = ΔIL/2 = 0.625A
3. Iout_min = 0.2A < Iout_crit = 0.625A → DCM。DCM 下 Buck 增益大于 D × Vin，输出电压会偏高。如果反馈环路未考虑 DCM 特性，可能导致轻载时输出过压（15V 可能漂到 18V 以上），触发负载端过压保护

### 题目2：Boost RHPZ 重载限制
**问题：** 一台 Boost 变换器：Vin = 36V，Vout = 72V，L = 47μH，Cout = 470μF。负载从 50W 到 500W 变化。请计算：
1. 满载 500W 时的等效负载电阻 Rload 和占空比 D
2. 满载和轻载（50W）时的 RHPZ 频率
3. 若电压环路设计为 fc = 3kHz，该设计在满载时是否稳定？在轻载时呢？

**参考答案：**
1. Rload_full = 72²/500 = 10.37Ω；D = 1-36/72 = 0.5
2. f_RHPZ_full = 10.37×(1-0.5)²/(2π×47e-6) = 10.37×0.25/(2.95e-4) = 8788Hz。Rload_light = 72²/50 = 103.7Ω；f_RHPZ_light = 103.7×0.25/(2.95e-4) = 87880Hz
3. 满载：fc = 3kHz, f_RHPZ = 8.8kHz, fc/f_RHPZ ≈ 0.34 < 1/3（刚好在边界），稳定性可接受但裕量很小。轻载：f_RHPZ = 87.9kHz >> fc = 3kHz，稳定。结论：满载是稳定性最恶劣的工况 → **Boost 的环路校验必须在满载下进行**

### 题目3：Buck 多路损耗与效率
**问题：** 电机驱动辅助电源 Buck：Vin = 310V（整流后母线），Vout = 15V，Iout = 2A，fs = 100kHz，L = 1mH（DCR = 0.5Ω）。MOSFET：STP6N120K3（1200V/6A, Rdson = 2.2Ω, tr = 15ns, tf = 10ns）；二极管：STTH112（1200V/1A, VF = 1.2V, trr = 75ns, Irr = 5A）。请计算：
1. 占空比 D
2. 各元件损耗（MOSFET 导通、开关、二极管导通、反向恢复、电感铜损）
3. 总效率和主要损耗来源

**参考答案：**
1. D = 15/310 = 0.0484（极低占空比！高压 Buck 的典型特点）
2. MOSFET 导通：Irms = 2 × √0.0484 = 0.44A, Pcond = 0.44² × 2.2 = 0.426W。开关损耗：Psw = 0.5 × 310 × 2 × (15+10)×1e-9 × 100000 = 0.5 × 310 × 2 × 25e-9 × 1e5 = 0.775W。二极管导通：PD = 1.2 × 2 × (1-0.0484) = 2.28W。反向恢复：Prr = 0.5 × 310 × 5 × 75e-9 × 100000 = 5.81W！电感铜损：PL = 2² × 0.5 = 2W。总损耗 = 0.426 + 0.775 + 2.28 + 5.81 + 2 = 11.3W
3. 输出功率 = 15×2 = 30W。效率 = 30/(30+11.3) = 72.6%。**二极管反向恢复损耗占总损耗的 51%！** → 高压 Buck 必须用 SiC 肖特基二极管（零反向恢复）或准谐振/QR 模式降低开关损耗

## 二、概念辨析题（★★★☆☆）

### 题目4：CCM vs DCM 特性对比
**问题：** 以下关于 Buck 的 CCM 和 DCM 工作模式的描述，请逐条判断正误并说明理由：
1. CCM 下输出电压仅取决于占空比和输入电压，与负载无关
2. DCM 下输出电压随负载减小而升高
3. DCM 下 MOSFET 开通时电流从零开始，开关损耗更小
4. 同一 Buck 电路，CCM 和 DCM 的环路补偿网络可以相同
5. DCM 下右半平面零点消失

**参考答案：**
1. ✅ 正确（理想 Buck CCM：Vout = D × Vin，负载不影响稳态增益）
2. ✅ 正确（DCM 增益高于 CCM 且随负载变化：负载越轻 → K 越小 → Vout 越高）
3. ✅ 正确（DCM 下 MOSFET 零电流开通，无反向恢复问题，开通损耗小；但关断时仍有峰值电流关断损耗）
4. ❌ 错误。DCM 下功率级传递函数从 CCM 的二阶系统变为一阶系统，LC 双极点消失，补偿网络必须不同。如果用 CCM 的 Type III 补偿跑 DCM，会导致带宽过低或振荡
5. ✅ 正确（DCM 下电感电流在每个周期归零，不存在"先跌后升"的非最小相位行为，RHPZ 移到高频，对环路设计不再是限制因素。这是 DCM 的一大优势）

## 三、设计应用题（★★★★☆）

### 题目5：电机驱动栅极电源 Buck 完整设计
**问题：** 为 48V/500W 伺服驱动器设计栅极驱动电源 Buck。要求：Vin = 36~72V（宽范围电池），Vout = 15V ± 2%，Iout = 0.1~2A，fs = 400kHz，输出纹波 < 50mV。请设计：
1. 电感值 L（取纹波率 r = ΔIL/Iout_max = 0.3）
2. 电感饱和电流要求（考虑最恶劣输入电压 72V）
3. 输出电容选择（给出类型、容量、理由）
4. MOSFET 电压和电流定额

**参考答案：**
1. 最恶劣输入：Vin_max = 72V，D_min = 15/72 = 0.208。L = Vin × D × (1-D) / (r × Iout × fs) = 72 × 0.208 × 0.792 / (0.3 × 2 × 400000) = 11.87 / 240000 = 49.5μH → 选 47μH 标称值。验证纹波：ΔIL = 72×0.208×0.792/(400000×47e-6) = 11.87/18.8 = 0.631A，r_actual = 0.631/2 = 0.316 ≈ 0.3 ✅
2. Ipeak = 2 + 0.631/2 = 2.32A。Isat 取 1.5 倍 → 3.5A。注意是硬饱和电流（电感量下降 < 10%）而非软饱和（下降 30%）
3. 纹波限 50mV：容性纹波 < 25mV, ESR 纹波 < 25mV。C ≥ 0.631/(8×400000×0.025) = 7.89μF；ESR < 0.025/0.631 = 39.6mΩ。方案：22μF/25V MLCC（X7R）×2 并联 → C_total=44μF, ESR≈1.5mΩ。ΔVC=0.631/(8×400000×44e-6)=4.5mV；ΔVESR=0.631×0.0015=0.95mV；总纹波≈5.4mV ✅。MLCC 是唯一能做到 ESR < 40mΩ 的合理选择
4. VDS_max = 72V，取 1.5 倍 → 100V MOSFET，Id > 2.5A（峰值），Rdson < 50mΩ。推荐：SQ2362ES（100V/5.5A, Rdson=35mΩ）或类似

### 题目6：Boost PFC 电感设计
**问题：** 为 2kW 电机驱动 PFC 前级设计 Boost 电感。Vin_ac = 220V ± 20%，Vout = 400V，fs = 65kHz，要求电感电流纹波率 r = 0.2。计算：
1. 最低输入电压和最恶劣占空比（Vin_ac_min × 1.414 → Vin_dc_min，考虑整流压降）
2. 满载输入电流有效值和峰值
3. 电感值 L
4. 电感峰值电流和饱和电流要求

**参考答案：**
1. Vin_ac_min = 220 × 0.8 = 176V。Vin_dc_min = 176 × 1.414 - 2（整流压降）= 246V。最恶劣 D_max = 1 - 246/400 = 0.385
2. Iin_rms = 2000/246 = 8.13A（PF=1 时）。Iin_peak = 8.13 × 1.414 = 11.5A（正弦波峰值）
3. ΔIL_peak = r × Iin_peak = 0.2 × 11.5 = 2.3A。L = Vin_dc_min × D_max / (ΔIL_peak × fs) = 246 × 0.385 / (2.3 × 65000) = 94.7 / 149500 = 633μH → 选 680μH 标称值。实际 ΔIL_peak = 246×0.385/(680e-6×65000) = 2.14A，r_actual = 2.14/11.5 = 0.186 ≈ 0.2 ✅
4. IL_peak_total = Iin_peak + ΔIL_peak/2 = 11.5 + 2.14/2 = 12.57A。Isat > 12.57 × 1.3 = 16.3A（取 17A 以上）。磁芯可选铁硅铝（Kool Mμ）或铁氧体+气隙，线径需考虑集肤效应（65kHz 集肤深度 ≈ 0.26mm，需用多股线或利兹线）

## 四、故障诊断题（★★★★☆）

### 题目7：Buck 输出过压——DCM 模式惹的祸
**问题：** 一台 Buck 电源：Vin = 24V，Vout_nom = 5V，fs = 500kHz，L = 4.7μH，Cout = 100μF。轻载（10mA）时测得 Vout = 6.8V，远高于设计的 5V。但满载（2A）时 Vout = 5.02V，准确。请分析：
1. 为什么轻载时输出电压偏高？
2. 计算满载和轻载的 CCM/DCM 边界电流
3. 给出至少两种解决方案

**参考答案：**
1. 因为轻载时 Buck 进入 DCM 模式，DCM 下的电压增益高于 CCM 的 D × Vin。反馈环路虽在工作，但最小占空比受限（控制器有最小导通时间限制），导致无法进一步降低输出电压
2. D = 5/24 = 0.208。ΔIL = 24×0.208×(1-0.208)/(500000×4.7e-6) = 3.96/2.35 = 1.685A。Iout_crit = 1.685/2 = 0.843A。满载 2A > 0.843A → CCM；轻载 0.01A << 0.843A → 深度 DCM
3. 方案A：增加假负载（如 100Ω → 50mA 预负载，但仍 < 0.843A，不足以进入 CCM；需 > 0.843A 的假负载，但会严重降低效率 ❌）。方案B：选用支持强制 CCM 或脉冲跳跃模式（PSM）的控制器，在轻载时间歇工作降低等效频率 ✅。方案C：增大电感使临界电流降低：若 L=33μH，ΔIL=24×0.208×0.792/(500000×33e-6)=0.24A，Iout_crit=0.12A → 10mA 仍为 DCM 但程度更轻 ✅。方案D：使用同步整流 Buck（强制 CCM 模式），允许电感电流反向 → 即使在轻载也保持 CCM ✅✅

### 题目8：高频振荡——电流模式次谐波振荡
**问题：** 一台电流模式控制 Buck：Vin = 12V，Vout = 5V，fs = 1MHz。当负载 > 1A 时正常工作；负载降到 0.3A 后，电感电流出现每周期交替大小不一的振荡（次谐波振荡），SW 节点波形变得混乱。已知 D ≈ 0.42（5/12），看起来 D < 0.5 不应该振荡。请分析可能原因并给出解决方案。

**参考答案：**
虽然 D = 0.42 < 0.5，但如果存在以下情况仍会发生次谐波振荡：(a) 芯片内部电流采样有延迟（如前沿消隐 LEB 时间 50~100ns），等效 D 增大；(b) 电感电流纹波叠加采样噪声，每个周期的峰值比较基准有小幅变化；(c) 芯片内部的斜率补偿不足或根本没有。
解决方案：(a) 检查 datasheet，确认 IC 是否有内置斜率补偿。如果是电压模式 IC 误标为电流模式，实际没有斜率补偿。改用真正的电流模式 IC（如 TPS5430、MP2315）。(b) 在电流采样引脚增加 RC 滤波，但注意不要引入过大延迟。(c) 如果 IC 支持外部斜率补偿（如 UC384x 系列），在 Isense 引脚外加斜坡电压。(d) 检查 PCB 布局——电流采样走线必须 Kelvin 连接，远离 SW 节点的 dv/dt 干扰

## 五、综合案例题（★★★★★）

### 题目9：Boost + Buck 级联系统——电机驱动辅助电源架构
**问题：** 电池供电 AGV 电机驱动系统：电池电压范围 18~50.4V（12 串锂电）。需要产生 15V/2A（栅极驱动）、5V/1A（MCU）、3.3V/0.5A（传感器）。请设计最优辅助电源架构，考虑：
1. 当电池电压 > 15V 时和 < 15V 时的不同情况
2. 是否需要升降压拓扑
3. 给出完整方案框图和各级拓扑选择

**参考答案：**
电池 18~50.4V 范围，15V 在此范围之内 → 当电池 > 15V 需降压，当电池 < 15V（18V 远 > 15V，实际上始终 > 15V）...

重新分析：18V > 15V，所以 15V 始终需要降压。选择 Buck：Vin=18~50.4V → Vout=15V/2A，占空比最大 15/18=0.833（高压差工况，对 Buck MOSFET 选型要求高）。

5V 从 15V 取电：Buck 15V→5V/1A，占空比 5/15=0.333。

3.3V 从 5V 取电：LDO 5V→3.3V/0.5A，压差 1.7V×0.5A=0.85W（可接受）。

架构：电池 → 15V Buck（宽输入，如 LMR16006/LM5160 等支持 60V+ 输入）→ 5V Buck（如 TPS562201）→ 3.3V LDO（如 AMS1117-3.3）。三级级联，总效率 ≈ 0.88×0.92×0.66=53%（LDO 拉低了整体效率）。如需更高效率，可将 3.3V 也改为 Buck（15V→3.3V 或 5V→3.3V Buck）。

关键设计考量：(a) 18V→15V 压差仅 3V，D=0.833，高端 MOSFET 导通时间很长；(b) 应选用支持 100% 占空比的 Buck 控制器；(c) 或者改用 SEPIC/Buck-Boost 在电池电压低时也能稳压（但本场景始终 Vin>Vout，Buck 足够）

### 题目10：Buck-Boost 四开关变换器——电池供电电机驱动全范围供电
**问题：** 设计一个 4-Switch Buck-Boost 变换器为电机驱动控制板供电：Vin = 5~24V（USB-C PD 或电池），Vout = 12V/3A，fs = 400kHz。请分析：
1. 纯 Buck 模式（Vin > Vout）、纯 Boost 模式（Vin < Vout）、过渡模式（Vin ≈ Vout）下的开关管工作状态
2. 选择电感 L（取 r = 0.4），计算最恶劣工况下的纹波
3. 该变换器的 RHPZ 行为何时出现？如何影响环路设计？
4. 推荐市售控制器 IC（至少列出两个型号）

**参考答案：**
1. Buck 模式（Vin > 12V）：Q_A/Q_B 作 Buck 开关管，Q_C 常通、Q_D 常断。Boost 模式（Vin < 12V）：Q_C/Q_D 作 Boost 开关管，Q_A 常通、Q_B 常断。过渡模式（Vin ≈ 12V，如 11.5~12.5V）：四个开关管全部以 PWM 方式工作，效率最低但实现平滑过渡
2. 最恶劣纹波工况：Vin=24V（Buck, D=12/24=0.5, ΔIL_max_Buck=24/(4×400k×L)）或 Vin=5V（Boost, D=1-5/12=0.583, ΔIL_Boost=5×0.583/(400k×L)）。计算：Buck D=0.5 → 取 r=0.4, Iout=3A, ΔIL=1.2A → L=24×0.5×0.5/(400k×1.2)=6/480k=12.5μH。Boost D=0.583 → ΔIL=5×0.583/(400k×12.5e-6)=2.915/5=0.583A, r=0.583/3=0.194（Boost 侧纹波更小 ✅）。电感 12μH，Isat > 3+1.2/2=3.6A × 1.5 = 5.4A
3. RHPZ 仅存在于 Boost 模式下：f_RHPZ = Rload×(1-D)²/(2π×L)。Vin=5V 时：Rload=12²/36=4Ω, D=0.583, f_RHPZ=4×(1-0.583)²/(2π×12e-6)=4×0.174/(7.54e-5)=9233Hz。fc < 9233/3 ≈ 3kHz。Buck 模式下无 RHPZ，环路可更快
4. 推荐 IC：Texas Instruments LM5175/LM5176（4-Switch Buck-Boost, 3.5~42V, 集成驱动器）；Analog Devices LTC3789（4-Switch Buck-Boost, 4~38V, 高效）；MPS MP8859（2.8~22V, 集成 MOSFET, 更小体积）

---

> ⚠️ 工程注：4-Switch Buck-Boost 在过渡区（Vin≈Vout）时四个开关管高频切换，效率最低。系统设计时需确保热设计覆盖此工况。部分 IC 支持"降频过渡"策略，在 Vin≈Vout 时降低开关频率以减少损耗。
