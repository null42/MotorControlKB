## 一、参数计算题（★★★★☆）

### 题目1：电感完整设计
**问题：** 设计 Buck 电感：L = 22μH，Idc = 10A，ΔI = 3A（峰峰值），fs = 300kHz。选用铁硅铝磁环 MS-106125-2（AL = 86nH/T², Ae = 0.654cm², le = 6.35cm, Ve = 4.15cm³）。计算：
1. 所需匝数 N
2. 满载时的 Bmax（DC偏置 + AC纹波）
3. 磁芯损耗（Steinmetz: k=12.4, α=1.46, β=2.0, Bpk=ΔB/2）
4. 若 Bmax 超标，给出修改建议

**参考答案：**
1. N = √(22e-9/86e-9) = √0.256 = 16.0T → 16T。实测 L = 86×16² = 22.0μH ✅
2. Ipk = 10 + 1.5 = 11.5A。Bdc = L×Idc/(N×Ae) = 22e-6×10/(16×0.654e-4) = 2.2e-4/(1.046e-3) = 0.210T。ΔB = L×ΔI/(N×Ae) = 22e-6×3/(16×0.654e-4) = 6.6e-5/1.046e-3 = 0.063T。Bmax = 0.210 + 0.063/2 = 0.242T
3. Bpk = 0.063/2 = 0.0315T。Pv = 12.4 × 300000^1.46 × 0.0315^2.0 = 12.4 × (3e5)^1.46 × 9.92e-4。先算 f^α：(3e5)^1.46。ln(3e5)=12.61, ×1.46=18.41, exp=9.85e7。Pv = 12.4 × 9.85e7 × 9.92e-4 = 12.4 × 9.77e4 = 1.21e6 W/m³ = 1.21 W/cm³ → 这样太大了。用对数：Pv = 12.4 × (3e5)^1.46 × 0.0315^2.0。近似：log10(Pv) = log10(12.4) + 1.46×log10(3e5) + 2.0×log10(0.0315) = 1.093 + 1.46×5.477 + 2.0×(-1.502) = 1.093 + 7.997 - 3.004 = 6.086 → Pv = 10^6.086 = 1.22×10^6 mW/cm³ = 1220 mW/cm³。P_core = 1220 × 4.15 = 5063 mW = 5.06W（非常高！）
4. Bpk 虽小(0.0315T)，但 300kHz 下铁硅铝损耗极大。建议：降低 fs 到 100kHz 或换用更低损耗材质（如非晶磁环 AMCC-16，k=0.38/α=1.51/β=1.74）

### 题目2：集肤效应—利兹线设计
**问题：** Flyback 变压器：fs = 200kHz，初级 Irms = 3A，次级 Irms = 15A。计算：
1. 200kHz 时铜的集肤深度
2. 初级和次级推荐的利兹线规格
3. 若初级用单根 AWG24 线（0.511mm 直径），AC/DC 电阻比

**参考答案：**
1. δ = 66/√200000 = 66/447.2 = 0.148mm
2. 初级：3A/4A/mm² = 0.75mm²。推荐 0.1mm × 100 股利兹线（0.785mm²）。次级：15A/4A/mm² = 3.75mm²。推荐 0.1mm × 500 股（3.93mm²）或 0.05mm × 1000 股
3. AWG24 直径 0.511mm = 2×0.2555mm。d/δ = 0.511/0.148 = 3.45 > 2 → 集肤效应显著。Rac/Rdc ≈ d²/(4δ(d-δ)) ≈ 0.511²/(4×0.148×(0.511-0.148)) = 0.261/(0.592×0.363) = 1.22。实际多层绕组的邻近效应会使 Rac/Rdc > 5！

## 二、概念辨析题（★★★★☆）

### 题目3：磁芯材质选型
**问题：** 为以下应用选择最优磁芯材质并说明理由：(a) 50Hz 工频变压器, 1kVA；(b) 100kHz Forward 变压器, 200W；(c) 65kHz CCM PFC 电感, 1kW, 大 DC 偏置；(d) EMI 共模扼流圈, 150kHz~30MHz

**参考答案：**
(a) 硅钢片（Si-Steel）：Bsat 高达 1.5~2.0T，适合低频。铁氧体在 50Hz 需要极大截面积。(b) 铁氧体 PC40/3C90：高频损耗低，Bsat 适中，成本低，适合纯交流励磁。(c) 铁硅铝 Kool Mμ 或铁镍钼 MPP：分布式气隙，DC Bias 特性好，μ 随 DC 偏置变化平缓。(d) 纳米晶（Nanocrystalline）或高 μ 铁氧体（μr>10000）：极高初始磁导率 → 共模阻抗大。注意纳米晶的截止频率通常在几十 kHz~几百 kHz

## 三、设计应用题（★★★★★）

### 题目4：Flyback 变压器——气隙放错位置
**问题：** 某 Flyback 变压器用 EE25 磁芯，在中柱磨了 1mm 气隙。满载时效率 85%，但放在骨架中的热电偶测得磁芯中柱温度 95°C，靠近气隙的绕组局部温度 115°C。解释为什么气隙附近的绕组温度远高于磁芯？

**参考答案：**
中柱气隙处的扩散磁通（Fringing Flux）在绕组导线上感应出涡流，导致该区域局部过热。解决方案：(a) 在气隙处绕一层铜箔屏蔽（短路环），涡流在屏蔽环而非绕组中产生 → 屏蔽环发热但保护了绕组；(b) 改用边柱气隙（EE 磁芯在两个边柱磨气隙，中柱完整），中柱处无扩散磁通 → 绕组不受影响 ✅；(c) 改用铁硅铝磁环（分布式气隙），没有集中的扩散磁通区域

## 四、故障诊断题（★★★★★）

### 题目5：高温下 Bsat 衰减——铁氧体的致命弱点
**问题：** PFC 电感用铁氧体 ETD44 + 气隙设计，Bmax(25°C) = 0.32T。满载运行后电感温度升至 95°C，效率从 95% 降到 90%。铁氧体 Bsat(25°C)=0.40T, Bsat(100°C)=0.30T。诊断原因并给出解决方案。

**参考答案：**
室温下 Bmax=0.32T < Bsat=0.40T → 安全 ✅。95°C 时 Bsat≈0.31T → Bmax=0.32T > Bsat → 磁芯接近/进入饱和 → 电感量下降 → 纹波电流增大 → 磁芯损耗进一步增大 → 温度进一步升高 → 恶性循环 → 最终效率崩溃。

这是铁氧体最致命的问题——Bsat 的负温度系数（NTC）。解决方案：(a) 改用铁硅铝或铁镍钼磁环（Bsat > 1T，温度系数小）；(b) 如果坚持用铁氧体，将 Bmax(25°C) 设计到 0.25T，确保 100°C 时也有 0.25 < 0.31T ✅；(c) 增强散热降低温升

## 五、综合案例题（★★★★★）

### 题目6：电机驱动 EMI 共模扼流圈设计
**问题：** 为 5kW 电机驱动设计 DC 母线共模扼流圈。母线电流 15A DC，目标：150kHz~30MHz 共模衰减 > 20dB。计算共模电感量需求并选择磁芯。

**参考答案：**
共模扼流圈：两个绕组反向串联，差模电流磁通抵消（不饱和），共模电流磁通叠加。

所需共模阻抗：X_L > 10 × 线路阻抗 50Ω（LISN）= 500Ω@150kHz → L = 500/(2π×150000) = 531μH → 取 1mH。

选择纳米晶磁环（AMCC-25, AL≈30μH/T², μr≈30000@10kHz）。N = √(1e-3/30e-6) = √33.3 = 5.8T → 6T。

差模偏置检查：15A 差模时磁芯中 H_dm = N×(I1-I2)/le ≈ 0（差模抵消）。但漏感（差模电感）：L_leak ≈ 0.5~2% × L_cm = 5~20μH，对差模纹波有额外滤波效果。

绕制注意：(a) 双线并绕确保对称；(b) 注意绝缘耐压（母线到地 3kV+）；(c) 纳米晶磁环脆，需外壳保护

### 题目7：AP 法磁芯选型与优化
**问题：** 用 AP 法为 500W CCM PFC 电感选磁芯。L=470μH, Ipk=12A, Bmax=0.25T（考虑 DC Bias 后）, J=4A/mm², Ku=0.35。计算所需 AP 值并从以下磁芯中选择：EE40(Aw=1.74cm²,Ae=1.27cm²,AP=2.21)、ETD44(Aw=2.14,Ae=1.73,AP=3.70)、ETD49(Aw=2.76,Ae=2.11,AP=5.82)。

**参考答案：**
AP = L×Ipk²/(Bmax×J×Ku) = 470e-6×144/(0.25×4e6×0.35) = 0.0677/(350000) = 1.93e-7 m⁴ = 1.93 cm⁴。

EE40 AP=2.21 > 1.93 → 够用 ✅。ETD44 AP=3.70 → 余量充足 ✅（可选为减小温升）。

选 EE40：匝数 N = L×Ipk/(Bmax×Ae) = 470e-6×12/(0.25×1.27e-4) = 5.64e-3/3.175e-5 = 178T。太多匝数！验证窗口：线径取 0.6mm（12A/4=3mm² → 约 11 股 0.6mm），178T × 11 × 0.283mm² = 554mm²，EE40 窗口面积 174mm² → 554 > 174 ❌ 放不下！

选 ETD49：N = 470e-6×12/(0.25×2.11e-4) = 5.64e-3/5.275e-5 = 107T。线径同上 ≈ 11×0.283=3.11mm² × 107T = 333mm²，ETD49 窗口 276mm² → 333 > 276 ❌ 还是放不下！

需要降低电流密度：J=6A/mm²（短时工作）→ 线面积 2mm²，改用铜箔 0.2×10mm 绕制（扁平线，窗口利用率更好）→ 重新校验

### 题目8：变压器绕组交流电阻估算
**问题：** Forward 变压器：Np=40T（3 层），Ns=5T（1 层），fs=100kHz，初级用 0.4mm 导线。估算初级的 Rac/Rdc 比值并解释多层绕组交流电阻的快速增加。

**参考答案：**
d/δ = 0.4/0.209 = 1.91（在集肤效应临界点）。

单层时 Rac/Rdc ≈ 1.3（集肤效应为主）。但 3 层时邻近效应占主导：M层 ≈ (M²-1)/3 × (层间邻近效应因子)。近似：Rac/Rdc_3layer ≈ 1.3 + (3²-1)/3 × 2 ≈ 1.3 + 8/3×2 ≈ 1.3+5.3 ≈ 6.6。

这意味着 3 层绕组的交流电阻是直流电阻的 6 倍以上！这就是为什么多层变压器绕组需要用利兹线或 PCB 绕组交错排列（减少有效层数）。

**设计要点**：变压器设计中尽量每层次级交错（Np/2→Ns→Np/2 三明治绕法）→ 初级的有效层数从 3 层降为 1.5 层 → Rac/Rdc 大幅降低

### 题目9：电机驱动母线共模电感 vs 差模电感
**问题：** 电机驱动 DC 母线（300VDC/20A）需要同时抑制共模噪声（150kHz~30MHz）和差模纹波（PWM 谐波 20kHz~500kHz）。设计一体化的 EMI 滤波器磁元件方案。

**参考答案：**
方案：共模扼流圈 + 差模电感分离设计。

共模扼流圈：纳米晶磁环 AMCC-32（高 μr），N=8T×2 双线并绕，L_cm ≈ 2mH。共模阻抗@150kHz = 2π×150000×2mH = 1884Ω → 衰减 > 30dB。差模漏感 ≈ 0.5%×L_cm = 10μH → 对差模纹波有一些衰减。

差模电感：铁硅铝磁环 MS-157125（低 μ，不饱和），N=25T, L_dm ≈ 100μH。额定差模电流 20A (DC) 不会饱和（铁硅铝 DC Bias 特性好）。

组合：CM Choke → DM Inductor → X 电容 (1μF) → CM Choke → Y 电容 (4.7nF×2)。两级滤波，共模和差模各有针对性

### 题目10：电机电感与功率电感的设计异同
**问题：** PMSM 电机绕组电感 Ld=2mH 和 PFC 电感 L=2mH 在磁路设计上有哪些本质共同点和关键区别？

**参考答案：**
共同点：(a) 都遵循 L=N²/R_mag；(b) 都有铁芯（硅钢片 vs 铁氧体/铁硅铝），都需要校核 Bmax；(c) 铜损计算相同(I²R)；(d) 气隙都是储能主体。

关键区别：(a) 电机电感的磁路是旋转的（转子相对定子运动），PFC 电感磁路静止 → 电机有旋转铁损和永磁体工作点变化；(b) 电机电感的 Ld/Lq 随转子角度和电流变化（凸极性 + 饱和），PFC 电感只是电流的函数；(c) 电机电感的工作频率是基波（几十~几百 Hz）+ PWM 谐波，PFC 电感主要是 fs 和 2fs；(d) 电机电感的磁路由硅钢片叠成（低频 < 400Hz），PFC 电感用铁氧体/铁粉芯（高频 > 20kHz）；(e) 电机电感是三相耦合的，PFC 电感是单相的
