## 一、参数计算题（★★★★☆）

### 题目1：Type II K-Factor 设计
**问题：** 电流模式 Buck：Vin=24V, Vout=5V, L=15μH, Cout=100μF(ESR=5mΩ), fs=500kHz, Rload=1Ω。功率级简化为单极点：fp=1/(2π×Rload×Cout)=1592Hz, DC增益=Rload/Rsense(假设Rsense=0.1Ω)=10(20dB)。设计 fc=50kHz, PM=60°，用 K-factor 法求 Type II 补偿器的零极点频率和 fc 处增益。

**参考答案：**
功率级@50kHz：|Gvd|=10/|1+j×50000/1592|=10/√(1+987)=10/31.4=0.318(-9.95dB)。∠Gvd=-arctan(50000/1592)=-88.2°。

需要 θ_boost = 60 - 90 - (-88.2) = 58.2°。
K = tan(58.2/2+45) = tan(74.1°) = 3.51。
fz = 50000/3.51 = 14.2kHz。fp = 50000×3.51 = 176kHz。
|Gc(50k)| = 1/0.318 = 3.14(+9.95dB)。

验证：补偿器零点 14.2kHz 提升相位，极点 176kHz 高频衰减。fc=50kHz=fs/10 ✅

### 题目2：数字延迟下 PI 参数修正
**问题：** PMSM 电流环：Ld=2mH, Rs=0.5Ω, fs=16kHz。数字控制总延迟 Td=80μs。设计 ωc=1000rad/s (fc≈159Hz)。计算理想 PI 参数，然后计算数字延迟对 PM 的影响，给出修正后的 PI 参数建议。

**参考答案：**
理想 PI：Kp=L×ωc=0.002×1000=2.0, Ki=R×ωc=0.5×1000=500。
但 RL 极点 f_RL = R/(2πL) = 0.5/(2π×0.002) = 39.8Hz。
穿越频率 fc=159Hz >> f_RL=40Hz（理想零极点对消：Ki/Kp=R/L）

数字延迟@159Hz：Δ∠ = -360°×159×80e-6 = -4.58°。影响较小。

但如果 ωc=3000rad/s(fc=477Hz)：延迟相位损失 = -360°×477×80e-6 = -13.7°。原 PM=90°(纯积分器+一阶滞后) → PM=90-13.7-atan(477/40)=90-13.7-85.2=-8.9° ❌ 不稳定！

修正：降低 ωc 到 2000rad/s(318Hz)。Kp=0.002×2000=4.0, Ki=0.5×2000=1000。延迟损失=-360×318×80e-6=-9.2°。PM=90-9.2-atan(318/40)=90-9.2-82.8=-2.0°仍不够。需在 PI 基础上加低通滤波（牺牲带宽换 PM）或使用 Smith 预估器补偿延迟

## 二、概念辨析题（★★★★☆）

### 题目3：环路特性判断
**问题：** 判断以下说法：
1. Type III 补偿器的相位提升能力来自两个零点
2. 穿越频率越高，动态响应越快，所以 fc 越大越好
3. RHPZ 增加增益同时消耗相位，因此对环路稳定性极其不利
4. 数字控制的零阶保持（ZOH）等效引入 T/2 的延迟
5. 电流模式控制消除了输出 LC 滤波的其中一个极点

**参考答案：**
1. ✅ 正确。两个零点各提供 +90° 相位提升（在各自频率之后）
2. ❌ 错误。fc 受限于开关频率(fc≤fs/10~fs/5)、RHPZ(fc<f_RHPZ/3)、数字延迟等，必须折中
3. ✅ 正确。RHPZ 是 Boost/Buck-Boost 稳定性设计的首要限制因素
4. ✅ 正确。ZOH 的频率响应是 sinc 函数，等效延迟 ≈ T/2
5. ✅ 正确。电流内环将电感电流等效为受控电流源 → LC 双极点降为输出 RC 单极点

## 三、设计应用题（★★★★★）

### 题目4：Boost 电压模式 vs 电流模式——环路设计对比
**问题：** Boost：Vin=12V, Vout=24V, L=33μH, C=220μF(ESR=30mΩ), fs=200kHz, Rload=2Ω。分别设计电压模式和电流模式（峰值电流）的环路，对比 fc 上限和补偿复杂度。

**参考答案：**
电压模式：f_LC(考虑D=0.5，等效R=(1-D)²Rload=0.5Ω)：f₀=1/(2π√(LC))×√(1-D)²... 直接分析：f_RHPZ=2×(0.5)²/(2π×33e-6)=0.5/(2.07e-4)=2412Hz。fc_max ≤ f_RHPZ/3 = 804Hz。在 804Hz 处需补偿 LC 双极点(-180°)的相位损失 → Type III 必要但复杂。

电流模式：功率级降为单极点(Rload×Cout)+ESR零点。内环→外环传递函数在约 f_RHPZ 处仍有影响（RHPZ 未消除，只是被推高）。保守 fc ≤ 1500Hz。Type II 即可，设计简单。

结论：电流模式允许更高的 fc（~2×）且补偿更简单（Type II vs Type III），是 Boost 的推荐方案

### 题目5：FOC 电流环 PI 与 DC-DC 环路——统一频域设计方法
**问题：** 基于频域方法统一设计：(a) PMSM 电流环 PI (L=1mH, R=0.2Ω, fs=16kHz)；(b) Buck 电流模式补偿 (L=10μH, Cout=47μF, Rload=5Ω, fs=500kHz)。都要求带宽 2000rad/s, PM > 60°，使用零极点对消方法。

**参考答案：**
(a) 电机电流环：PI = Kp + Ki/s。零极点对消：Ki/Kp = R/L = 200。ωc=2000 → Kp=L×ωc=2.0, Ki=400。验证 PM：极点@R/L=200rad/s被零点对消，剩余 PM=90°（纯积分行为）但数字延迟需扣除。Ts=62.5μs, Td≈80μs → Δ∠@318Hz=-360×318×80e-6=-9.2° → PM=80.8° ✅

(b) Buck 电流模式（单极点模型）：fp=1/(2π×5×47e-6)=677Hz=4255rad/s。fc_target=2000rad/s=318Hz。由于 fp >> fc，功率级在 fc 处增益平坦。用 Type II：θ_boost=60-90-(-atan(2000/4255))=60-90-(-25.2)=-4.8°（几乎不需要提升）。简化为 PI（即 Type II 的变体）✅

统一框架：零极点对消让 PI/Type II 的零点消除功率级的主极点 → 剩余 PM≈90°（扣除延迟和额外极点）

## 四、故障诊断题（★★★★★）

### 题目6：环路增益测量——注入点选择错误
**问题：** 工程师用频率响应分析仪（FRA）测量 Buck 环路增益。注入信号通过 10Ω 电阻注入反馈分压器和误差放大器之间的节点，测得 PM=30°（期望 60°），但实际上输出波形稳定。诊断测量错误。

**参考答案：**
环路的断开点必须在高阻抗节点（电流驱动）和低阻抗节点（电压接收）之间。反馈分压器输出到误差放大器输入端之间是理想注入点（分压器输出阻抗 R1||R2 约几 kΩ 是高阻端，误差放大器输入也是高阻 → 注入信号无法有效驱动！）。

正确注入点：在输出电压分压后的节点与误差放大器之间串联注入电阻（10~50Ω），FRA 的源端接误差放大器侧（高阻），接收端接分压器侧（低阻）。如果注入点在分压器之前（输出端），则 FRA 的源端驱动低阻抗输出 → 信号被衰减。

工程师测得的低 PM 可能是因为注入点阻抗不匹配导致的测量误差（注入信号的一部分被衰减），而非真实环路不稳定

## 五、综合案例题（★★★★★）

### 题目7：PFC 电压环——为什么这么慢？
**问题：** PFC 电压环穿越频率通常设计在 10~20Hz，而电流环在 5~10kHz。解释为什么差距如此之大（近千倍），并与电机控制的速度环/电流环（通常仅差 5~10 倍）对比。

**参考答案：**
PFC 的电压环必须极慢的根本原因是输出有 100Hz（或 120Hz）的大幅值纹波。如果电压环穿越频率太高：(a) 电压环试图"纠正"100Hz 纹波 → 误差放大器的输出被 100Hz 调制 → 乘法器输出的电流基准被调制 → 输入电流畸变 → THD 恶化、PF 降低；(b) 输出 100Hz 纹波的幅值通常 5~10V，即使经过衰减仍有几百 mV 进入误差放大器 → 这是一个不可忽略的"干扰"

电机控制没有这种低频大幅值脉动——母线电压是 DC（无 100Hz 脉动），所以速度环可以做到电流环的 1/5~1/10 而不会有类似问题。PFC 的 100Hz 纹波是一个"硬约束"，迫使电压环极慢

---

### 题目8：LLC 的频率-电压环设计
**问题：** LLC 的输出电压通过调频控制。功率级传递函数（频率→输出电压）在 fr1 附近有谐振峰。设计 LLC 电压环补偿器时，与 PWM DC-DC 的最大区别是什么？

**参考答案：**
最大区别：LLC 的功率级在 fr1 附近有显著的谐振峰（由输出滤波器和 LLC 谐振网络共同决定）。这个谐振峰的频率通常在 1~5kHz（取决于 Lf×Cf），必须在穿越频率之前用补偿器极点衰减。

PWM DC-DC 类型：功率级在 f_LC 处有一个简单的双极点（或电流模式的单极点），特性相对简单。
LLC 类型：功率级在 fr1 处有复杂的增益峰（Q 值 > 1），且相位在穿越 fr1 时剧烈变化。

LLC 补偿设计要点：(a) 穿越频率必须低于输出滤波谐振频率（避免激励输出滤波器振荡）；(b) 补偿器需要在输出滤波器谐振频率处提供衰减（Type III 的一个极点专门用于此）；(c) 由于频率范围可能跨越升压区和降压区，功率级的传递函数在不同负载下变化剧烈 → 需在满载和轻载都做验证

### 题目9：电流环次谐波振荡——斜率补偿的频域解释
**问题：** 峰值电流模式控制在 D>0.5 时需加斜率补偿，否则会发生次谐波振荡。从频域角度（不是从时域几何分析）解释这一现象，并与数字控制延迟相类比。

**参考答案：**
频域解释：峰值电流模式控制中，在 D>0.5 时，采样-保持效应（离散时间行为）导致内环的等效采样增益产生一个复平面上的极点进入不稳定区域。这小信号不稳定性表现为采样频率的一半（fs/2）处的振荡——即次谐波振荡。

斜率补偿相当于在电流环引入一个阻尼项，将 fs/2 处的极点拉回稳定区域。补偿斜率 = 电感电流下降斜率的一半时恰好临界稳定。

与数字控制的对比：数字控制的延迟也是一个离散时间效应，会导致额外的相位滞后。在 fs/2 频率处，数字控制环路也可能出现条件不稳定。事实上，峰值电流模式 + D>0.5 的次谐波振荡与数字控制中 Nyquist 频率附近的不稳定本质上都是离散时间系统的采样效应

### 题目10：全数字 PFC 的级联环路——电流环-电压环-前馈协同设计
**问题：** 设计数字 PFC 的完整控制架构：电流内环（fs=100kHz, fc_current=10kHz）+ 电压外环（fc_voltage=15Hz）+ 输入电压前馈。给出设计流程并分析三个回路的交互。

**参考答案：**
设计流程：
1. 电流内环：PFC Boost 电流环控制电感电流跟踪正弦基准。fc_current ≤ fs/10 = 10kHz。电流环 PI 参数需要足够快以跟踪 50Hz 正弦波（基波）→ 实际上电流环带宽 > 2kHz 即可很好地跟踪 50Hz。取 fc=5kHz（折中速度和噪声）
2. 电压外环：fc_voltage=15Hz，远低于 100Hz → Type I 或 PI 即可
3. 输入电压前馈：Vin² 前馈项消除输入电压变化对环路增益的影响 → 使环路增益在全输入电压范围内基本恒定

交互分析：(a) 电压环的输出（误差放大器）调制电流环的基准幅度 → 如果电压环有 100Hz 纹波，则被调制到电流环输出 → THD 增加；(b) 输入电压前馈必须精确——如果 Vin 采样有延迟或误差，前馈反而引入失真；(c) 电流环必须在满载时保持稳定，同时还要在过零点（Iref≈0）时保持稳定——这是一个极端工作点变化

数字实现建议：电压环使用抗混叠滤波器（fc < 50Hz）后再 ADC 采样以避免 100Hz 混叠（ADC 采样率通常 10~20kHz）。电流环使用逐周期采样（与 PWM 同步）以最小延迟
