## 一、参数计算题（★★★☆☆）

### 题目1：母线电容纹波电流与配置
**问题：** 5kW 逆变器：Vdc = 650V，Iphase_rms = 22A，fs = 12kHz。现有 470μF/450V 电解电容：ESR = 0.25Ω@120Hz，纹波电流额定 2.8A@120Hz/105°C。请：
1. 估算母线总纹波电流有效值
2. 设计电容串并联方案满足纹波电流要求（考虑 12kHz 频率修正因子 0.7）
3. 计算配置后的 ESR 总损耗和每个电容温升

**参考答案：**
1. Ic_rms ≈ 0.6 × 22 = 13.2A
2. 每个电容等效允许纹波 = 2.8 × 0.7 = 1.96A。需并联数 ≥ 13.2/1.96 = 6.7 → 7 个。耐压：650/450=1.44 → 需两串。总配置：2 串 × 7 并 = 14 个电容。若用 2 串 × 8 并 = 16 个（对称布局更好）。等效容量 = 470 × 8/2 = 1880μF
3. ESR_total = 0.25 × 2/8 = 0.0625Ω。P_ESR_total = 13.2² × 0.0625 = 10.9W。每电容 P = 10.9/16 = 0.681W。Rth 约 25°C/W → 温升 17°C。环境 45°C → 核心温度 62°C < 105°C 额定 ✅

### 题目2：制动斩波器完整设计
**问题：** 10kW PMSM 驱动：J_total = 0.3 kg·m²，ω_max = 3000rpm(314rad/s)，制动时间 2s。Vdc_nom = 540V，允许 Vdc_max = 600V。设计制动电阻和 IGBT，考虑绕线电阻寄生电感（约 30μH）并给出关断电压应力。

**参考答案：**
E = ½ × 0.3 × 314² = 14789J。P_brake = 14789/2 ≈ 7395W。R_brake = 600²/7395 = 48.7Ω → 47Ω。I_brake = 600/47 = 12.77A。

IGBT 关断时寄生电感反冲：V_spike = L_para × dI/dt。dI/dt ≈ 12.77A/200ns = 63.9A/μs。V_spike = 30e-6 × 63.9e6 = 1917V → 总 Vce = 600 + 1917 = 2517V → 选 3.3kV IGBT（如 IXGH32N170, 1700V 不够！）。

解决方案：(a) 电阻两端并联续流二极管 → 消除尖峰。二极管选 1200V/20A 快恢复。(b) 电阻改用无感厚膜电阻（寄生电感 < 50nH）。V_spike = 50e-9 × 63.9e6 = 3.2V（可忽略）。IGBT 选 1200V/25A

制动电阻功率：7395W脉冲, 假设制动周期 5% → 平均 370W。47Ω/400W 铝壳电阻。或 4 × 47Ω(12Ω串联/4组并联，每组 200W) → 800W 额定

## 二、概念辨析题（★★★☆☆）

### 题目3：电机驱动功率链路判断
**问题：** 判断以下说法正误：
1. 母线电容的容量越大越好
2. Bootstrap 自举电路可以无限期维持上管导通
3. EMI 滤波器中的 Y 电容容量受安规漏电流限制
4. 逆变器输出到长电缆时，电机端电压可能超过母线电压的两倍
5. 预充电电阻在正常运行时应被继电器短路

**参考答案：**
1. ❌ 错误。容量只需满足纹波和瞬态约束，过大浪费体积和成本且增加浪涌电流
2. ❌ 错误。Bootstrap 电容通过下管导通充电，上管持续导通 → 电容逐渐放电 → 栅极电压不足
3. ✅ 正确。Y 电容的漏电流 = 2πf × Cy × Vline。安规要求 I_leak < 0.75mA(可插拔设备) → Cy_max ≈ 4.7nF@230V/50Hz
4. ✅ 正确。反射波效应在长电缆(> λ/4)的电机端产生 2×Vdc 的电压，更长的电缆 + 更快 dv/dt → 甚至 > 2 倍
5. ✅ 正确。预充电完成后必须短路预充电电阻以减少正常运行损耗和发热

## 三、设计应用题（★★★★☆）

### 题目4：辅助电源完整架构
**问题：** 设计 7.5kW 伺服驱动器辅助电源。输入 310VDC 母线。需输出：3 路 15V/0.3A 隔离（上桥臂 U/V/W）、1 路 15V/0.5A（下桥臂地）、1 路 5V/1.5A（MCU + 传感器）、1 路 3.3V/0.5A。给出完整架构、各级拓扑选择和上电时序方案。

**参考答案：**
架构：主 Flyback（310V→15V/2.5A 非隔离 + 5V/1.5A 隔离）→ 后级分发。

更优：310VDC → Flyback（VIPer26K, 内置 800V MOSFET）→ 变压器多路输出：
- Winding U: 15V/0.3A (隔离, 三层绝缘线, 3000VAC 耐压)
- Winding V: 15V/0.3A (隔离, TIW)
- Winding W: 15V/0.3A (隔离, TIW)
- Winding Lower: 15V/0.5A + 5V/1.5A(堆叠绕组, 与下桥臂共地)
- 15V 主反馈（加权：U/V/W 各 0.2 + Lower 0.4）

5V→3.3V：LDO TPS73733（1A, 低噪声, 带 PG）

上电时序：Flyback 启动 → 15V 建立 → EN_5V Buck 使能 → 5V 建立 → 5V PG → EN_3.3V LDO。3.3V 最后建立（DSP 内核先于 IO 的要求相反？需要确认 DSP 手册。通常 C2000 要求 3.3V 先于 1.8V。若仅需 3.3V，则 3.3V 最后建立 → 外设先有电 → IO 口可能输出高电平到未上电的 MCU → 风险。修正：5V→3.3V→MCU。或 3.3V 和 5V 同时建立）

总 Flyback 功率：3×15×0.3 + 15×0.5 + 5×1.5 = 13.5+7.5+7.5 = 28.5W。频率 65kHz，变压器 EF25，效率 ≈ 82%

## 四、故障诊断题（★★★★☆）

### 题目5：母线电容 ESR 老化——温水煮青蛙
**问题：** 一台 5 年的伺服驱动器，用户反馈"越来越容易触发过压保护"。现场测试：加速时母线电压跌落 25%（新机仅跌 8%），但空载电压正常。分析原因。

**参考答案：**
母线电解电容经过 5 年使用 → 电解液干涸 → ESR 增大（可能增大 3~5 倍）+ 容量下降（下降 20~30%）。

容量下降 → ΔV = Ipeak × Δt/Cdc → 瞬态电压跌落增大 → 原有 8% 跌落 × (C_old/C_new) ≈ 8% × (1/0.75) = 10.7%，尚不足以解释 25% 跌落（说明容量降幅可能 > 50% 或 ESR 影响）。

更关键：ESR 增大 → 纹波电流流过 ESR 时压降增大 → 母线电压最低值 = Vdc_avg - ΔV_ESR - ΔV_capacitive。ESR × 3 → ΔV_ESR × 3 → 额外的电压跌落。同时 ESR 增大的恶性循环：ESR↑ → 发热↑ → 温度↑ → 电解液干涸加速 → ESR↑↑。

验证方法：用 LCR 电桥在 120Hz 下测量拆下的电容 → ESR 应该 < 0.5Ω/个（规格值 0.15Ω × 3~5 = 0.45~0.75Ω）。如果 > 2Ω → 确认老化

## 五、综合案例题（★★★★★）

### 题目6：10kW 电机驱动系统完整功率链路设计
**问题：** 全面设计 10kW/380VAC 电机驱动的功率链路。包括：(a) EMI 滤波器；(b) 整流 + 软启动；(c) PFC 或直接整流选择；(d) 母线电容；(e) 辅助电源；(f) 制动斩波器。给出每级的参数、拓扑选择和关键设计考量。

**参考答案：**
(a) EMI：两级滤波。第一级 Lcm1=8mH(纳米晶)+Cx1=1μF+Cy1=4.7nF×2；第二级 Lcm2=2mH(NiZn 铁氧体)+Cx2=0.47μF+Cy2=1nF×2。目标 EN 55011 Class A

(b) 三相整流桥（SKD 62/16, 1600V/62A）+ 预充电。R_precharge=33Ω/50W×3(各相)。三个继电器或一个三相接触器(带辅助触点)。当 Vdc>350V 时接触器吸合

(c) 10kW → 建议 PFC。三相 Vienna 整流器：650VDC 母线。或三相六开关 AFE 如果需要能量回馈。若成本敏感 → 直接三相整流 → 约 540VDC 母线（380V×√2×1.35）→ 省去 PFC 成本但谐波超标需申请豁免。工业级 10kW 推荐带 PFC

(d) 母线电容：Ic_rms≈0.55×Iphase_rms。Iphase_rms=10000/(√3×380×0.85)=17.9A。Ic_rms≈9.8A。若用 12kHz 电解需 9.8/2=4.9 个并联（每电容 2A 有效）→ 2 串 × 5 并 = 10 个 470μF/450V。推荐混合方案：5×470μF 电解(处理低频) + 6×40μF 薄膜(处理高频) → 总容量 2350μF(电解)+240μF(薄膜)

(e) 主 Flyback 310V→15V/2A+5V/1A+±15V/0.1A(28.5W)。用 NCP1075(700V 内置 MOSFET) 或分立 VIPER26。三路上桥臂 15V 从主 Flyback 次级提供(三个 TIW 绕组)

(f) 制动斩波器：估算 J_total≈0.5 kg·m², ω_max=3000rpm, t_brake=3s → E=½×0.5×314²=24649J, P=8216W, R=650²/8216=51.4Ω→47Ω, I=13.8A。用 Semikron SKKH 57/16 (1600V/50A 晶闸管模块可做斩波) 或 1200V/40A IGBT(MBG40T120)。电阻 47Ω/1kW(铝壳强制风冷)。

软启动注意：母线电容充电时 PFC 尚未启动 → PFC 输出为 0 → 母线电容从 0 开始充电 → 浪涌由预充电电路限制。PFC 启动后从 540V 升压到 650V

### 题目7：电动汽车电机驱动的功率架构
**问题：** EV 电机驱动（400V 母线平台）的功率架构包括：HV Battery(250~450V) → Inverter → Motor，以及 HV→LV DC-DC（14V 辅助电池）、OBC（AC→HV DC）。请解释为什么 EV 通常不需要 PFC，但需要 HV-LV DC-DC，并比较工业驱动和 EV 驱动在功率架构上的关键区别。

**参考答案：**
EV 不需要 PFC 的原因：电池本身是 DC 源，没有交流整流环节 → 没有功率因数和谐波问题（PFC 是解决 AC-DC 整流的谐波问题的）。OBC（车载充电机）内部有 PFC，但那是充电时才用的。

EV 需要 HV-LV DC-DC 的原因：高压电池(250~450V)驱动牵引逆变器，但车辆的 12V/24V 辅助系统（灯光、ECU、冷却泵、风扇）不能直接从高压电池取电。需要一个隔离的 DC-DC 变换器将 400V 降到 14V。功率约 1~3kW（视车辆配置）。

工业驱动 vs EV 驱动的关键区别：
1. 输入源：AC 电网(工业) vs DC 电池(EV) → 工业需要整流+PFC，EV 不需要
2. 母线电压：固定(工业, PFC 输出) vs 宽范围(250~450V, 电池 SOC 相关) → EV 逆变器需要适应宽母线范围
3. 能量回馈：工业制动是偶尔的 → 制动电阻最常见；EV 制动是频繁的（再生制动）→ 能量回收到电池（天然双向）
4. 辅助电源架构：工业从 AC 取 → 需要 Flyback 直接从母线取；EV 有低压电池 → 辅助电源从 12V 取即可
5. 安规/功能安全：EV 有 ISO 26262 ASIL 要求 → 高压互锁(HVIL)、绝缘监测(IMD)、主动放电(Active Discharge)要求
6. 散热方式：工业用风冷；EV 用水冷/油冷（集成到整车热管理）

### 题目8：SiC/GaN 对电机驱动功率链路的重塑
**问题：** 分析 SiC MOSFET 和 GaN HEMT 对电机驱动功率链路设计的影响。覆盖：母线电容选型、EMI 滤波器、辅助电源、开关频率选择、散热设计。

**参考答案：**
SiC MOSFET（1200V/650V）：开关速度极快(tr<20ns) + 低导通电阻。影响：(a) 可提高 fs 到 50~100kHz → 电流纹波减小 → 母线电容容量需求降低 → 可用更小/更少的电容。但高频纹波的频谱更宽 → EMI 滤波更挑战（高频 dv/dt 更强）；(b) 开关损耗大幅降低 → 散热器体积减小 → 功率密度提升；(c) 栅极驱动要求更严：Vgs_on=+18~20V, Vgs_off=0~-5V, 门极振荡抑制（栅极电阻和 layout 关键）；(d) 辅助电源需要能处理高 dv/dt(>50kV/μs)下的共模噪声

GaN HEMT（650V/100V）：极快开关(<5ns) + 零反向恢复。影响：(a) fs 可到 200~500kHz → 电机电流接近纯正弦波 → 几乎不需要输出滤波器 → 电机铁损降低；(b) 无反向恢复 → EFficiency 再提升 1~2%；(c) 极快开关 → 寄生电感的影响被放大 → PCB Layout 和封装必须极低寄生 → 分立封装的局限性 → 集成模块的趋势；(d) EMI 更严峻——极高 dv/dt + 高频 → 共模噪声频谱延伸到 100MHz → EMI 滤波器需要 special 高频磁芯

母线电容：高频化 → 电解电容可被 MLCC/薄膜完全替代 → 长寿命令。EMI：高频化 → EMI 滤波器挑战增大 → 需要从"滤波"转向"从源头抑制"（减缓 dv/dt、屏蔽、Layout 优化）

### 题目9：电机驱动功率系统的故障保护层次
**问题：** 设计电机驱动功率链路的完整故障保护层次架构。覆盖过流/短路、过压、欠压、过温、缺相、接地故障。说明哪些是硬件保护（不可绕过）、哪些是软件保护（可配置）。

**参考答案：**
保护分层（响应速度从快到慢）：

Level 1 — 硬件保护（不可绕过，响应 < 1μs）：
(a) desat（退饱和检测）：IGBT/MOSFET 专用驱动 IC 内置，Vce/Vds 过高时立即关断
(b) 硬件过流（OCP）：比较器直接检测 shunt 或电流传感器，过阈值 → 硬件关断 PWM
(c) 硬件过温：NTC/PTC 比较器或温度开关直接关断

Level 2 — 逐周期软件保护（响应 1 PWM 周期 ~ T_pwm）：
(a) ADC 软件过流：每个 PWM 周期在 ISR 中检查电流
(b) 逐周期限流：修改 PWM 占空比限制电流
(c) Bus 电压监控：过压/欠压在 PWM ISR 中检查

Level 3 — 慢速软件保护（响应 ms 级）：
(a) RMS 过流保护（平均热保护 I²t）：积分 I² 判断热积累
(b) 缺相检测：监测电流不平衡度
(c) 接地故障检测：漏电流传感器 + ADC
(d) 模块温度监测：NTC 经 ADC → vs 温度限值表

Level 4 — 系统级保护（响应 > 100ms）：
(a) 看门狗：MCU 死机 → 硬件关断
(b) 通信丢失保护：CAN/RS485 超时 → 安全状态
(c) 制动电阻过热保护：热开关或温度传感器

所有 Level 1 保护都应有独立于 MCU 的硬件路径（即使 MCU 死机也能关断）；所有保护触发后的状态机（锁存/自动重试/需人工复位）应在需求中明确定义

### 题目10：从分立到集成——电机驱动功率链路的演进趋势
**问题：** 分析 IPM（智能功率模块）、GaN/SiC 功率模块、和 SiP（系统级封装）如何改变电机驱动的功率链路设计。从 500W 到 50kW，给出推荐的集成度方案。

**参考答案：**
< 1kW（低压）：全集成方案。如 TI DRV8305（三相栅极驱动 + Buck 辅助电源 + 电流检测运放） + 外部 MOSFET → 一个 IC + 6 MOSFET = 完整驱动。或 ST SpinFamily 全集成模块（MCU + 栅极驱动 + MOSFET 全部在一个封装）。功率链路极简化

1~5kW（中低压/低压中功率）：IPM（如 Infineon CIPOS, Mitsubishi DIPIPM）。集成 6 IGBT/MOSFET + 栅极驱动 + 自举二极管 + 保护 → 只需 PCB + 散热器。辅助电源仍需分立 Flyback。功率链路中等复杂度

5~20kW（工业标准功率）：分立 TO-247/TOLL 封装 SiC MOSFET + 隔离栅极驱动 IC + 分立辅助电源。功率链路以 PCB 设计为核心挑战（Layout、散热、EMI）。高集成度 IPM 仍可用（如 Vincotech flowPIM）

20~50kW（大功率）：功率模块（如 Infineon EconoPIM, Semikron SEMITRANS）+ 专用驱动板 + 模块化辅助电源。功率模块内部有优化的寄生参数和热路径，功率链路以系统集成为核心。

集成趋势：(a) 辅助电源由分立 Flyback → 集成隔离 DC-DC 模块 → 全集成 PMIC；(b) 栅极驱动 + 保护由分立 IC → 集成到 IPM/功率模块 → 带诊断功能；(c) 电流传感器从 shunt + 运放 → 集成霍尔传感器(如 Melexis MLX91220) → 更简单更隔离

> ⚠️ 工程注：集成度越高，灵活性越低，故障排查越难。在工业级高可靠性应用中，适当的分立度带来更透明的故障模式和更好的可维修性
