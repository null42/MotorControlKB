## 一、参数计算题（★★★☆☆）

### 题目1：Flyback 关键参数计算
**问题：** 某电机驱动辅助电源 Flyback：Vin = 280~340VDC，Vout = 15V/1A，fs = 65kHz，D_max = 0.45（Vin_min 时），η = 0.82。请计算：
1. 变压器匝比 N = Np/Ns
2. 最小输入电压时的反射电压 VOR
3. MOSFET 的电压应力（含 30% 漏感尖峰余量）
4. 最大占空比时 MOSFET 的占空比在 Vin_max 时是多少？

**参考答案：**
1. N = Vin_min × D_max / ((Vout+VF) × (1-D_max)) = 280 × 0.45 / (15.7 × 0.55) = 126 / 8.635 = 14.6 → 取 15
2. VOR = (15 + 0.7) × 15 = 235.5V
3. VDS_theory = Vin_max + VOR = 340 + 235.5 = 575.5V。含 30% 余量：575.5 × 1.3 = 748V → 选 800V MOSFET
4. Vin_max 时：D = VOR / (Vin_max + VOR) = 235.5 / (340 + 235.5) = 0.409。即在最高输入电压时占空比自动减小 → Flyback 具有天然的占空比自适应特性

### 题目2：Forward 变压器匝比与占空比约束
**问题：** Forward 变换器：Vin = 36~72V，Vout = 12V，fs = 250kHz，使用第三绕组复位（Nr = Np）。请：
1. 计算满足 D_max ≤ 0.5 的最小匝比 N
2. 若取 N = 4，验证各输入电压下的占空比是否合规
3. 计算 MOSFET 电压应力（Vin_max 时）

**参考答案：**
1. N需满足 Vin_min × D_max / N = Vout → D = Vout × N / Vin_min ≤ 0.5 → N ≤ 0.5 × Vin_min / Vout = 0.5 × 36 / 12 = 1.5。取 N = 1.5（即 Np:Ns = 3:2），D_36V = 12×1.5/36 = 0.5 ✅
2. N=4 → D_36V = 12×4/36 = 1.33 > 0.5 ❌ 违规！D_72V = 12×4/72 = 0.667 > 0.5 ❌。这个匝比不合理——占空比在全输入范围都超过 0.5
3. 修正：取 N = 1.5，D_72V = 12×1.5/72 = 0.25。VDS_max = Vin_max × (1+Np/Nr) = 72 × 2 = 144V → 选 200V MOSFET

## 二、概念辨析题（★★★☆☆）

### 题目3：Flyback 变压器 vs 普通变压器
**问题：** 以下关于 Flyback 变压器与普通变压器的对比描述，请逐条判断正误：
1. Flyback 变压器磁芯中储存的能量远大于普通变压器
2. Flyback 变压器初级的电流波形是纯交流，没有直流分量
3. Flyback 变压器必须开气隙，因为需要磁芯储能
4. Flyback 变压器的初级电感量越大越好
5. 普通变压器可以直接用于 Flyback 拓扑，只需调整匝比

**参考答案：**
1. ✅ 正确。Flyback 工作于储能模式，磁芯储能 E = ½ × Lp × Ipk²；普通变压器仅传输能量，储能极小（仅磁化能量）
2. ❌ 错误。CCM Flyback 初级电流是带有直流偏置的锯齿波，I_DC = Pout/(η × Vin_min × D_max)。DCM 下平均电流也非零
3. ✅ 正确。气隙存储了绝大部分能量（气隙储能占总储能的 90%+）。无气隙的 Flyback 变压器饱和电流极小，无法传输功率
4. ❌ 错误。Lp 需要精确设计——太小则纹波/峰值电流大、损耗大；太大则变压器体积大、DCM→CCM 转换点提前。MCU 根据功率和 DCM/CCM 目标计算 Lp
5. ❌ 错误。普通变压器无气隙 → 磁芯很快饱和 → 无法储能 → MOSFET 烧毁

## 三、设计应用题（★★★★☆）

### 题目4：Flyback RCD 钳位设计
**问题：** 一台 60W Flyback：Vin = 310V，Vout = 24V/2.5A，N = 10，Ipk = 2.2A，变压器漏感 Lleak = 12μH，fs = 100kHz。请设计 RCD 钳位电路：
1. 计算 VOR 和建议的 Vclamp
2. 计算 RCD 电阻功耗和阻值
3. 选择钳位电容（允许 10% 纹波）
4. 验证 MOSFET 电压应力并推荐额定电压

**参考答案：**
1. VOR = (24+0.7) × 10 = 247V。Vclamp = 1.5 × 247 = 370.5V → 取 370V
2. P_RCD = ½ × 12e-6 × 2.2² × 100000 + 370² / Rclamp。前半部分（漏感能量）= ½ × 12e-6 × 4.84 × 1e5 = 2.904W。Rclamp = 370² / 2.904 ≈ 47.2kΩ → 取 47kΩ/5W。验证 P = 2.904 + 370²/47000 = 2.904 + 2.913 = 5.82W → 选 47kΩ/10W 更安全
3. ΔVclamp = 0.1 × 370 = 37V。Cclamp = Vclamp / (ΔVclamp × Rclamp × fs) = 370 / (37 × 47000 × 100000) = 370 / 1.739e11 ≈ 2.13nF → 选 2.2nF/630V 薄膜电容
4. VDS_max = Vin_max + Vclamp + Vspike_remaining ≈ 310 + 370 + 30 = 710V。取 800V MOSFET（如 STP8N80K5, 800V/6A, Rdson=2.0Ω）

### 题目5：多路输出 Flyback 交叉调整率改善
**问题：** 一个三路输出 Flyback 供电电机驱动控制板：15V/0.5A（栅极驱动，反馈控制路），5V/1A（MCU），-15V/0.1A（运放负电源）。现有设计使用简单绕法（各绕组独立叠绕），交叉调整率约 15%。请设计：
1. 改善后的绕组结构（堆叠绕法）
2. 加权反馈网络（给出分压电阻值）
3. 评估改进后的预期交叉调整率

**参考答案：**
1. 堆叠绕法：15V 绕组在底层（N_15V = 8T），5V 绕组堆叠在 15V 之上（N_5V = 3T，从 15V 终端的抽头开始），-15V 绕组独立反向绕制（N_neg15V = 8T，与 15V 反向）。这样 5V 绕组与 15V 绕组完美耦合（同柱共绕），交叉调整率大幅改善
2. 加权反馈：15V 路径 R1(上)=10kΩ, R2(下)=1kΩ → VFB=15×1/(10+1)=1.36V。5V 路径增加一路馈入：R3=33kΩ 从 5V 接到 FB 节点 → 当 5V 偏高时多注入电流 → 反馈网络"感知"到 5V 变化。参考电压若 2.5V → 需要 15V 和 5V 同时满足：设 15V 贡献 1.8V, 5V 贡献 0.7V
3. 堆叠绕法 + 加权反馈 → 预期交叉调整率 < 3%~5%。15V 控制路满载 ±2%，5V 可控制在 ±5%（即 4.75~5.25V），满足 MCU 供电要求（通常 4.5~5.5V）。-15V 变化约 ±8%，但对运放供电足够

## 四、故障诊断题（★★★★☆）

### 题目6：Forward MOSFET 重复烧毁——磁芯饱和链式反应
**问题：** 一台 200W Forward 变换器（第三绕组复位），上电后几秒钟内 MOSFET 烧毁，更换后再次烧毁。已知设计参数 Np=40T, Ns=5T, Nr=40T，磁芯 TDK PC40 ETD39，fs=100kHz, Vin=310V, Vout=24V。请逐步诊断并找出根因。

**参考答案：**
1. 验证 D = 24×(40/5)/310 = 192/310 = 0.62 → 超过 0.5！第三绕组复位（1:1）要求 D_max ≤ 0.5
2. 当 D=0.62 时，导通时间 = 6.2μs，关断时间 = 3.8μs。复位需要 3.8μs 内完成，但磁化电流在关断期间需 (40T) 的伏秒来复位，可用的复位伏秒只有 VDS_clamp × 3.8μs/40T。VDS_clamp 最大约 620V（初级电压翻转到 -310V + Vin → 理论 -310V + 310V... 实际因钳位 ≈ Vin → 复位伏秒 = 310 × 3.8μs = 1178V·μs，但需要的复位伏秒 = 310 × 6.2μs = 1922V·μs → 不够！）
3. 剩余磁通累积到下一周期 → 逐周期偏磁 → 磁芯饱和 → 励磁电流急剧上升 → MOSFET 瞬间过流失效
4. 根因：匝比选择错误导致 D > 0.5。修正：N = Np/Ns = 3（D=24×3/310=0.232<0.5 ✅）或改用双管正激/有源钳位

### 题目7：Push-Pull 变压器异常发热——偏磁
**问题：** 48V→12V/20A Push-Pull，使用 ETD44 铁氧体磁芯，Np_half=6T × 2，Ns=3T × 2（全波整流），fs=100kHz。运行 30 分钟后变压器温度 110°C。测量发现两个 MOSFET 的源极电流波形正半周峰值 45A，负半周峰值 38A。分析原因并给出解决方案。

**参考答案：**
正负半周电流不对称（45A vs 38A） → 偏磁（Flux Walking）。可能原因：
1. MOSFET A Rdson = 4.8mΩ, MOSFET B Rdson = 3.5mΩ → 导通压降差：20A × (4.8-3.5) = 26mV → 小但逐周期累积
2. PCB 走线不对称 → 回路寄生电感不同 → 关断尖峰不同 → 等效占空比不同
3. 栅极驱动延时不一致

核心问题：伏秒积不对称 → 磁芯单向偏磁 → DC 偏置使磁芯进入饱和区 → Bmax 超限 → 磁芯损耗飙升。

解决方案：(a) 在变压器初级串联隔直电容（如 10μF/100V 薄膜电容 × 2 并联），隔直电容阻断直流偏磁 → 正负半周伏秒自动平衡 ✅；(b) 改用峰值电流模式控制 → 逐周期平衡；(c) MOSFET 严格配对，PCB 对称布局

隔直电容设计：C ≥ I_dc_imb × T / ΔV。I_dc_imb_max ≈ 20A × 0.1 = 2A（假设 10% 不平衡），T = 10μs，允许 ΔV = 1V → C ≥ 2×10e-6/1 = 20μF → 选 22μF/100V

## 五、综合案例题（★★★★★）

### 题目8：电机驱动全隔离辅助电源方案设计
**问题：** 设计 3kW 伺服驱动器辅助电源方案：310V 母线输入，需要 6 路隔离输出：3 路 15V/0.3A（U/V/W 上桥臂栅极驱动，各隔离）、1 路 15V/0.5A（下桥臂栅极驱动，可共地）、1 路 5V/1A（MCU 隔离供电）、1 路 ±15V/0.1A（运放）。请：
1. 给出完整的电源架构框图
2. 选择主变换器拓扑并说明理由
3. 估算变压器总功率和磁芯尺寸

**参考答案：**
1. 架构：310VDC → Flyback → 多路隔离输出。一台 Flyback 变压器提供全部 6 路（但上桥臂的 3 路 15V 需要相互隔离 → 需要 3 个独立次级绕组 + 3 个独立次级绕组（下桥臂+5V）→ 共 6 个次级绕组）

更优架构：主 Flyback 产生 15V/2A（非隔离）和 5V/1A（隔离）→ 再用 3 个小型隔离 DC-DC 模块或推挽变换器将 15V 转换为 3 路隔离 15V

2. 最能实现多路隔离的拓扑就是 Flyback（每个次级绕组天然隔离）✅。总输出功率 = 3×15×0.3 + 15×0.5 + 5×1 + 2×15×0.1 = 13.5 + 7.5 + 5 + 3 = 29W。但 6 路隔离绕组的变压器绕制极其困难（窗口不够，安规距离难以满足）

推荐：Flyback(310V→15V/3A 非隔离) + 3×SN6505 Push-Pull(15V→15V 隔离×3) + Buck(15V→5V) + 电荷泵(15V→-15V)
- Flyback 主变压器：功率 15×3+5+3=53W，考虑效率 → 65W 设计
- SN6505 推挽：每路 15V/0.3A=4.5W，小型变压器（EP7 或 EE8.3 磁芯）

3. Flyback 变压器：Pout≈53W，AP≈(Lp×Ipk²)/(Bmax×J×Ku)。Lp≈1.5mH, Ipk≈1.5A, Bmax=0.25T, J=4A/mm², Ku=0.25。AP=(1.5e-3×2.25)/(0.25×4e6×0.25)=3.38e-3/(250000)=1.35e-8 m⁴=0.0135 cm⁴ → EE25/EE28 足够。SN6505 变压器各用 EE8.3 或 EP7

### 题目9：DCM vs CCM Flyback——电机驱动辅助电源选型
**问题：** 为 10kW 电机驱动设计隔离辅助电源 Flyback：310V→24V/3A，fs=65kHz。请从以下方面对比 DCM 和 CCM 设计，并给出最终推荐：
1. 变压器尺寸（初级电感量）
2. MOSFET 峰值电流和导通损耗
3. 次级二极管应力
4. 环路补偿复杂度
5. EMI 特性

**参考答案：**
1. DCM Lp = Vin_min²×D²/(2×Pout×fs×η) = 260²×0.45²/(2×72×65000×0.85) = 13689/7956000 = 1.72mH。CCM (r=0.4): Lp = Vin_min×D/(r×Iav×fs) = 260×0.45/(0.4×0.35×65000) = 117/9100 = 12.9mH。CCM 变压器是 DCM 的 7.5 倍大！
2. DCM: Ipk = 2×Pout/(η×Vin_min×D) = 2×72/(0.85×260×0.45) = 144/99.45 = 1.45A。CCM: Iav = 72/(260×0.45) = 0.615A, Ipk = Iav+ΔIL/2 = 0.615+0.123 = 0.738A。CCM 峰值电流更低 → 导通损耗更低 ✅
3. DCM: 次级二极管零电流关断，反向恢复无。CCM: 次级二极管硬关断，反向恢复损耗大 ❌（SiC 肖特基可缓解）
4. DCM: 功率级为一阶系统，Type II 补偿即可 ✅。CCM: 存在 RHPZ 和 LC 谐振，需 Type III 补偿 ❌
5. DCM: 峰值电流高 → di/dt 大 → 差模噪声大。CCM: 电流连续 → di/dt 较小 → EMI 较好 ✅

最终推荐：72W Flyback 建议 **CCM 或 BCM（临界模式）**。72W 用 DCM 的变压器的确更小，但峰值电流大 + 开关损耗大。72W 用 CCM（准谐振/QR 模式）可在谷底开通降低开关损耗。如果选 DCM，建议用 QR Flyback IC（如 NCP1342）实现谷底开关

### 题目10：有源钳位 Forward（ACF）——突破 D_max 限制
**问题：** 分析有源钳位 Forward 如何突破传统 Forward 的 D_max = 0.5 限制。包括：(1) 电路拓扑和工作原理；(2) 为什么 D 可以超过 0.5；(3) 与双管正激的优劣势对比；(4) 适用于电机驱动的哪些场景？

**参考答案：**
1. ACF 拓扑：在传统 Forward 的变压器初级并联一个辅助 MOSFET(Q2) + 钳位电容(Cclamp) 串联支路。Q1（主开关）关断时 Q2 导通 → 磁化电流对 Cclamp 充电 → 形成谐振复位。Q2 关断时 Cclamp 通过变压器放电，实现零电压开关(ZVS)
2. 传统 Forward D_max=0.5 是因为固定复位电压（=Vin,Nr=Np）→ 放电时间=导通时间。ACF 的钳位电容电压 Vclamp = Vin×D/(1−D)（可变！）。D=0.6 时 Vclamp=1.5Vin，复位电压更大 → 复位更快 → 在(1-D)T 时间内即可完成复位。D 越大 → Vclamp 自动升高 → 复位伏秒自动匹配 导通伏秒 ✅
3. ACF 优势：D_max 可 > 0.5（通常 0.6~0.7），变压器匝比可以更大，ZVS 降低开关损耗，更高频率。劣势：需要两个 MOSFET 和高侧驱动，控制逻辑更复杂(需要互补 + 死区)
4. 适用场景：大功率电机驱动（>5kW）需要高功率密度辅助电源时，ACF 可在小磁芯条件下输出更大功率（D_max 大 → 匝比大 → 初级电流小 → 铜损小）。同时 ZVS 允许更高开关频率 → 变压器更小

---
