## 一、参数计算题（★★★★☆）

### 题目1：LLC 完整参数设计
**问题：** 设计一台 1kW 半桥 LLC：Vin = 370~410VDC，Vout = 48V/21A，fr1 = 150kHz，m = 6。请计算：
1. 变压器匝比 n
2. 满载等效交流电阻 Rac
3. 取 Q = 0.5，计算 Cr、Lr、Lm
4. 验证 Vin_min=370V 时的最大所需增益和 Vin_max=410V 时的最小所需增益
5. 频率范围（估算）

**参考答案：**
1. n = (400/2)/(48+0.6) = 200/48.6 = 4.12 → 取 4
2. Rload = 48²/1000 = 2.304Ω。Rac = 8×4²/π² × 2.304 = 128/9.87 × 2.304 = 29.88Ω
3. Cr = 1/(2π×150000×0.5×29.88) = 1/(14.14e6) = 70.7nF → 68nF。Lr = 1/[(2π×150000)² × 68e-9] = 1/(8.88e11 × 68e-9) = 1/60384 = 16.56μH → 16.5μH。Lm = 6×16.5 = 99μH → 100μH
4. M_max = 4×48/(370/2) = 192/185 = 1.038。M_min = 4×48/(410/2) = 192/205 = 0.937。M_min(Q=0, m=6) = 1/(1+1/6) = 0.857 < 0.937 → 空载需 Burst Mode
5. 满载 M_max=1.038 → fn≈0.85（fs≈128kHz）。空载 M_min≈1.0 → fn≈1.0（fs=150kHz）+ Burst。高频段 M_min=0.937 → fn≈1.6（fs≈240kHz）。频率范围：128~240kHz（+Burst Mode）

---

### 题目2：ZVS 条件与死区设计
**问题：** 续上题。MOSFET 选用 IPW65R041CFD（650V, Coss_eff≈300pF@400V），半桥 LLC 死区时间 300ns。验证 fr1 满载时和下频率边界 (128kHz) 时的 ZVS 条件。

**参考答案：**
fr1=150kHz 时：Im_pk = n×Vout/(4×Lm×fs) = 4×48/(4×100e-6×150000) = 192/60 = 3.2A。死区平均电流 ≈ Im_pk×2/π = 2.038A。Q_provided = 2.038 × 300e-9 = 611nC。Q_required = 2×300e-12×410 = 246nC。611 >> 246 ✅ ZVS 充足。

128kHz 时（低频边界）：Im_pk = 192/(4×100e-6×128000) = 192/51.2 = 3.75A。平均电流 = 2.389A。Q_provided = 717nC > 246nC ✅。ZVS 余量在低频时反而更大——因为励磁电流随频率降低而增大！

## 二、概念辨析题（★★★★☆）

### 题目3：LLC 工作区域判断
**问题：** 以下关于 LLC 工作状态的描述，判断正误并说明理由：
1. 开关频率低于 fr1 时，次级整流二极管实现 ZCS
2. 开关频率高于 fr1 时，LLC 工作在升压模式
3. Q 值越大（重载），增益曲线越扁平，最大增益越低
4. 空载时 LLC 无法通过调频降到任意低的输出电压
5. 全桥 LLC 的变压器初级电压幅值是半桥的两倍

**参考答案：**
1. ✅ 正确。fs < fr1 → 次级二极管电流自然断续 → ZCS 关断，无反向恢复
2. ❌ 错误。fs > fr1 → 增益 < 1 → 降压模式。升压模式是 fr2 < fs < fr1
3. ✅ 正确。Q 大 → 负载阻尼大 → 谐振峰被压低 → 最大增益降低。重载时最大增益可能不足以维持输出
4. ✅ 正确。空载（Q=0）时增益最小值 = 1/(1+1/m)，不可能降到 0。需要 Burst Mode 或加假负载进一步降压
5. ✅ 正确。半桥：方波幅值 Vin/2；全桥：方波幅值 Vin。变压器设计时匝比差一倍

## 三、设计应用题（★★★★★）

### 题目4：轻载 Burst Mode 设计
**问题：** 一台 LLC 空载时最小连续增益为 0.88，但需要 0.82 才能稳压。输出电容 2000μF，输出电压 48V，允许纹波 ±1V。请设计 Burst Mode 参数：
1. Burst 等效占空比 D_burst
2. Burst 频率 f_burst（避免音频噪声）
3. 每个 Burst 周期的导通脉冲数

**参考答案：**
1. D_burst = 0.82/0.88 = 0.932。即开关时间占总时间的 93.2%
2. f_burst 应 > 20kHz → 选 25kHz（周期 40μs）。可选 30kHz（周期 33μs）更安全
3. 若 LLC 开关频率在 Burst 期间为 240kHz（周期 4.17μs），每 Burst 周期导通时间 = 33×0.932 = 30.8μs，导通脉冲数 = 30.8/4.17 ≈ 7.4 → 8 个脉冲。Burst 时间：8×4.17 = 33.3μs ON，0μs OFF（填满）→ 需调整 f_burst 到 25kHz：40×0.932=37.3μs → 9 个脉冲。纹波验证：ΔV = I_load×Toff/Cout。空载时 I_load≈0 → 纹波极小。轻载（如 0.5A）时 ΔV=0.5×(40-37.3)×1e-6/2000e-6=0.68mV << 1V ✅

### 题目5：集成变压器 LLC（磁集成）
**问题：** 说明如何利用变压器的漏感作为 Lr（磁集成 LLC），省去独立的谐振电感。给出设计要点和潜在风险。

**参考答案：**
磁集成方法：在变压器磁芯中增加一个附加磁路（如在中柱和边柱之间开气隙），使初级漏感精确地等于所需的 Lr。实际操作中更常用的是：在变压器骨架的初级-次级之间插入磁片（如 PC40 材质的薄片），人为增大漏感。

设计要点：(a) Lr 由漏感提供 → 不需要独立谐振电感 → PCB 面积减小、成本降低；(b) 变压器设计时同时优化 Lm 和 Lr（而非分开设计）→ 需要电磁仿真或多次打样；(c) 匝比和绕组结构影响漏感 → 样机阶段需精细调整

潜在风险：(a) 漏感受绕制工艺影响大 → 量产一致性差；(b) 漏感本身的交流电阻（集肤+邻近效应）导致额外损耗；(c) 漏磁通可能在屏蔽层和磁芯中引起局部过热；(d) 难以精确控制 Lr 的值（通常 ±15% 公差）。如果对批量一致性有要求，建议保留独立谐振电感（如 PQ 磁芯分立 Lr）

## 四、故障诊断题（★★★★★）

### 题目6：LLC 瞬态响应过慢
**问题：** 一台 48V/500W LLC，设计 fr1=100kHz。负载从 5% 跳变到 100% 时，输出电压跌落到 42V，200ms 后才恢复到 48V。这个恢复时间远超设计要求（< 20ms）。请诊断并给出解决方案。

**参考答案：**
LLC 的频率调制本质上带宽较窄，尤其以下因素恶化瞬态响应：

1. 频率→增益的非线性关系在远离 fr1 时斜率很平，需要很大的频率变化才能获得小量增益调整 → 靠近 fr1 工作时瞬态最优
2. 输出 LC 滤波的极点频率：fp = 1/(2π√(Lf×Cf)) 决定了可达到的最大闭环带宽
3. LLC 的小信号传递函数在 fr1 附近有一个显著的谐振峰（由输出滤波器引起），限制了穿越频率

诊断步骤：检查频率控制器（VCO）的增益 → 检查补偿器是否为 Type II/III → 确认穿越频率是否低于 fr1/10

解决方案：(a) 将 fr1 设计得更高（如 200kHz）→ 更高带宽可能；(b) 使用电荷泵式补偿器(Charge Pump Compensator) 提高带宽；(c) 次级使用同步整流 + 数字控制实现"调频 + 移相"混合控制 → 更快响应；(d) 增加输出电容以"硬"吸收瞬态 → 增加物理尺寸

---

### 题目7：LLC 同步整流 MOSFET 烧毁
**问题：** LLC 次级采用同步整流（而非二极管）。正常工作时高效率 (> 96%)，但在输入电压突然升高时（370V→410V 跳变），同步整流 MOSFET 频繁烧毁。分析原因。

**参考答案：**
同步整流的时序控制是关键。输入电压突升 → 频率必须快速响应升高以维持稳压 → 但频率控制环路通常较慢（几 ms）→ 在频率响应完成前，有效占空比过大 → 输出电压短暂过冲 → 以下机制导致 SR MOSFET 烧毁：

1. **反向电流**：输出电压过冲期间，SR MOSFET 仍在导通，电流可能反向流入输出 → SR MOSFET 承受反向电流应力
2. **直通风险**：如果 SR 导通信号基于固定延时，在频率突变时 SR 关断时序可能出错 → 输出电容通过 SR MOSFET 和变压器短路
3. **体二极管反向恢复**：高频下 SR MOSFET 的体二极管反向恢复损耗可能成为主导热源

解决方案：(a) SR 控制必须具有逐周期自适应死区（基于 VDS 检测，而非固定延时）；(b) 频率前馈——输入电压变化时立即预调频率（类似电机控制的电压前馈）；(c) 选择 Qrr 小的 SR MOSFET（如具有低 Qrr 体二极管的专用 SR FET）

## 五、综合案例题（★★★★★）

### 题目8：LLC vs PSFB——大功率电机驱动辅助电源选型
**问题：** 为 20kW 电机驱动设计 2kW 隔离辅助电源（800VDC 输入→48VDC 输出）。请在 LLC 全桥和相移全桥（PSFB）之间选择，给出详细理由。

**参考答案：**
LLC 全桥优势：全负载范围 ZVS（原边）+ ZCS（副边），效率峰值 > 97%，EMI 低，适合固定输入或窄输入范围。
PSFB 优势：固定频率（变压器优化简单），宽输入范围能力，控制简单（仅需移相），短路/过流保护天然效果好。

本场景：2kW、800V 高压输入、固定母线（PFC 输出 800V 稳定）→ 输入范围窄。这是 LLC 的理想场景 ✅。

推荐：LLC 全桥。理由：(a) 固定母线 → LLC 频率范围小 → m 可以取 6~8，效率高；(b) 2kW 时 LLC 全桥原边可以用 SiC MOSFET（如 C3M0016120K, 1200V/16mΩ）实现极低损耗；(c) 副边用 GaN FET 同步整流（如 EPC2024, 40V/1.5mΩ）

如果输入电压范围很宽（如电池 450~800V），则 PSFB 更合适——LLC 在宽范围的频率变化太大，磁元件难设计。

### 题目9：三相 LLC——超高功率电机驱动应用
**问题：** 对于 > 10kW 的电机驱动辅助电源，三相 LLC 相对于单相 LLC 的优势是什么？给出三相 LLC 的基本工作原理和设计注意事项。

**参考答案：**
三相 LLC 用三个相差 120° 的半桥/全桥驱动三相变压器（或三相变压器），输出并联。

优势：(a) 功率均分 → 每相功率降低 → MOSFET 电流应力减小；(b) 输出电流纹波频率 = 6×fs（三相交错）→ 很小的输出滤波需求 → 可用 MLCC 替代大电解；(c) 三相自动交错 → EMI 频谱分散 → EMI 滤波器小型化；(d) 变压器体积总和可以更小（高频变压器功率密度高）

工作原理：三个 LLC 半桥的开关时序各差 T/3（120°），变压器次级输出通过整流后并联到公共输出电容。三相交错的净效果：相当于一个 LLC 以 3× 频率运行，但每个开关管仍以原始频率工作。

设计注意：(a) 三相必须严格均流——变压器参数和 PCB 走线阻抗必须精确对称；(b) 三相的谐振频率必须一致（Lr 和 Cr 的公差控制在 ±2%）；(c) 三相 LLC 的控制通常用数字控制器集中管理（DSP 或专用三相 LLC IC）；(d) 任何一相故障时需能隔离并降功率运行（容错设计）

### 题目10：LLC 在 EV 电机驱动 OBC 中的系统级设计
**问题：** 电动汽车 OBC 典型规格：220VAC→400VDC/6.6kW。设计 PFC + LLC 两级架构。给出 PFC 输出母线电压选择（380V vs 400V vs 420V）的理由，以及 LLC 的关键参数设计决策。

**参考答案：**
PFC 母线电压选 400V：(a) 足够高于 220VAC 峰值(311V) 以实现 PFC；(b) 400V 是 EV 电池满电电压，OBC 需要输出略高于 400V（如 420V）充电；(c) 420V 的 PFC 母线 → 电解电容需要 500V 耐压 → 成本和 ESR 都不利；(d) 380V 不足以给 400V 电池充电。折中选 400V。

但 LLC 需要输出升压能力才能从 400V 升到 420V → LLC 必须工作在升压区(fs < fr1)。Vout=420V, Vin_nom=400V, M_max=420/400=1.05。全桥 LLC 的匝比 n=1（或接近 1）。fr1 选 150kHz，m=6。

关键设计决策：(a) 为 420V 最大输出时的 M=1.05 留足增益余量（M_available > 1.1@满载）；(b) 同时在 Vin_max=420V 输出 250V（电池最低电压）时 M_min=250/420=0.595 → m=6 时 M_min(Q=0)=0.857，不够低！→ 需要 Burst Mode + 较大的频率范围 → 或改用 CLLC（双向 LLC，原副边都串联电容）→ 电池侧和母线侧双向优化

CLLC 拓扑在 OBC 中很常见，因为支持双向能量流动（V2G, Vehicle-to-Grid）

---

> ⚠️ 工程注：EV OBC 的 LLC 设计还要考虑：(a) 宽温度范围(-40~85°C)的 Cr 温漂（推荐 C0G 材质）；(b) 冷却液温度对磁芯特性的影响；(c) 高压安规距离（800V+ 加强绝缘）
