## 一、参数计算题（★★★☆☆）

### 题目1：PFC 功率因数与 THD 计算
**问题：** 测得某电机驱动输入侧：Vin = 230VAC，Iin_rms = 8.7A，有功功率 Pin = 1920W，基波电流 I1 = 8.5A。请计算：
1. 视在功率 S 和功率因数 PF
2. 总谐波失真 THD
3. 位移因数 DPF

**参考答案：**
1. S = 230 × 8.7 = 2001VA。PF = P/S = 1920/2001 = 0.959
2. THD = √(Iin_rms²/I1² - 1) = √(8.7²/8.5² - 1) = √(75.69/72.25 - 1) = √0.0476 = 0.218 = 21.8%
3. PF = DPF/√(1+THD²) → DPF = PF × √(1+THD²) = 0.959 × √(1+0.218²) = 0.959 × 1.023 = 0.981。即基波电流与电压基本同相（DPF≈1），功率因数下降主要由谐波引起

### 题目2：PFC 输出纹波与电压环设计
**问题：** 一台 PFC：Pout = 500W，Vout = 400V，Cout = 330μF（两串两并，等效 330μF/450V），输入 50Hz。计算：
1. 输出 100Hz 纹波电压峰峰值
2. 如果电压环穿越频率 fc = 15Hz，纹波被衰减多少？
3. 判断此设计是否合理

**参考答案：**
1. ΔV_pkpk = Pout / (2π × 2f_line × Cout × Vout) = 500 / (2π × 100 × 330e-6 × 400) = 500 / 82.94 = 6.03V
2. 单极点 -20dB/dec，15Hz → 100Hz：A = 100/15 = 6.67，衰减 20×log10(6.67) = 16.5dB。残留纹波 = 6.03/6.67 = 0.904V。相对 Vout = 0.904/400 = 0.226%
3. 残留调制 0.226% 很小，对电流基准的调制可忽略 ✅。设计合理

## 二、概念辨析题（★★★☆☆）

### 题目3：PFC 关键概念判断
**问题：** 判断以下描述的正误：
1. PFC 的输入电流必须与输入电压同相位，且波形一致
2. PFC Boost 的电压环和电流环都很快
3. CCM PFC 比 DCM PFC 更适合大功率
4. 交错 PFC 的两个相位相差 90°
5. PFC 可以完全消除输入电流谐波

**参考答案：**
1. ✅ 正确。PFC 目标就是输入电流 = 纯正弦波 × 与电压同相位
2. ❌ 错误。电流环快（穿越频率 > 5kHz），但电压环极慢（10~20Hz），因为必须抑制 100Hz 纹波
3. ✅ 正确。CCM 峰值电流低，MOSFET 和电感损耗小，适合大功率
4. ❌ 错误。两相交错相差 180°，三相交错相差 120°。90° 是四相交错
5. ❌ 错误。PFC 可以大幅降低谐波（THD < 5~10%），但不可能完全消除。开关频率和 EMI 耦合总会引入一些谐波

## 三、设计应用题（★★★★☆）

### 题目4：PFC 电感完整设计
**问题：** 为 1kW/400V CCM PFC 设计电感。Vin_rms = 176~264VAC，fs = 100kHz，r = 0.3。计算：
1. 最低输入时的最大输入电流峰值
2. 最恶劣占空比下的纹波电流
3. 电感值
4. 推荐磁芯和匝数（选铁硅铝磁环 77439-A7, Ae=1.99cm², le=9.84cm, AL=89nH/T²）

**参考答案：**
1. Pin_max = 1000/0.95 = 1053W。Iin_rms_max = 1053/176 = 5.98A。Iin_pk_max = √2 × 5.98 = 8.46A
2. Vrect_pk_min = √2 × 176 = 249V。最恶劣 D = 1 - 249/2/400 = 1 - 0.311 = 0.689。但实际最恶劣 D=0.5 在 Vrect=200V 的点处 → ΔIL_max = 400/(4×100000×L)
3. r = 0.3, Iin_pk_max × 0.3 = 8.46 × 0.3 = 2.54A。L = 400/(4×100000×2.54) = 400/1016000 = 394μH → 选 390μH
4. N = √(L/AL) = √(390e-9/89e-9) = √4.38 = 19.7T → 20T。B_max = L×Ipk/(N×Ae) = 390e-6×(8.46+2.54/2)/(20×1.99e-4) = 390e-6×9.73/(3.98e-3) = 0.953T → 铁硅铝 Bsat≈1.05T，但 0.95T 太高了！需增大匝数或换大磁芯。修正：用 40T, B_max=0.476T ✅（铁硅铝在 DC 偏置 0.5T 时电感量保持约 70%，需查 DC bias 曲线验证）

## 四、故障诊断题（★★★★☆）

### 题目5：PFC 效率低下——反向恢复与开关损耗
**问题：** 一台 2kW CCM PFC 实测效率仅 92%（预期 97%）。已知：SiC 二极管 C4D10120D（1200V/10A, Qc=33nC），MOSFET C3M0065090J（900V/65mΩ/36A），fs = 100kHz，L = 300μH。诊断效率低下的原因。

**参考答案：**
开关损耗分析：MOSFET Coss=80pF, Eoss=11μJ@400V。Ps_oss = 11e-6 × 100000 = 1.1W。导通损耗：Irms ≈ 8A, Pcond = 8² × 0.065 × 0.4 = 1.66W。栅极驱动损耗：Qg=60nC, Vgs=15V, Pg = 60e-9 × 15 × 100000 = 0.09W。二极管反向恢复（SiC 零反向恢复！）：可忽略。

总估算开关损耗 ~ 3~4W。但实测效率 92% 对应总损耗 2000×(1/0.92-1) = 174W → 远大于计算值！

可能原因：(a) MOSFET 硬开关——PFC 是硬开关应用，Coss 放电损耗 Eoss 只是一部分，更大的损耗来自 VDS 和 ID 交叉期间的功率（VI 重叠损耗），建模为 Psw = ½ × Vds × Id × (tr+tf) × fs。没有 ZVS！(b) 电感的磁芯损耗被忽略——高频纹波在铁硅铝磁芯中产生显著的涡流和磁滞损耗；(c) 输入整流桥损耗：2 × 1.1V × 8A = 17.6W

核心：PFC 是硬开关拓扑，不能与 LLC 的高效率类比。94~96% 对 CCM PFC 属于正常范围，98% 需要交错 + SiC MOSFET + 同步整流。

## 五、综合案例题（★★★★★）

### 题目6：三相 PFC (Vienna 整流器) 电机驱动应用
**问题：** 为 15kW 三相电机驱动设计前级。380VAC×3 → 650VDC。讨论三相 VIENNA 整流器 vs 传统三相六开关 Boost PFC 的优劣。

**参考答案：**
Vienna 整流器：三电平拓扑，每相 1 个 MOSFET + 6 个二极管。MOSFET 电压应力 = Vout/2 = 325V（可用 600V MOSFET）。三电平 → 更小电感 + 更小 EMI。效率 > 98%。缺点：只能单向功率流（不能回馈）。

三相六开关 Boost PFC：用 6 个 MOSFET 组成三相逆变器拓扑，反向作 PFC。电压应力 = Vout = 650V（需 1200V SiC MOSFET）。可以双向功率流（AFE）。缺点：成本高、控制复杂（需三相 SVPWM + PFC 算法）。

推荐：如果不需要制动能量回馈电网，Vienna 整流器是 15kW 级别最优解。如果需要在电网侧回收制动能量（如电梯、起重机），用 AFE。

三相 PFC 的输出无 100Hz 纹波（三相对称功率恒定），电压环带宽可以显著提高（50~100Hz），瞬态响应远优于单相 PFC

### 题目7：无桥 PFC（Bridgeless PFC）
**问题：** 传统 PFC 的整流桥损耗占总损耗的 20~30%。解释无桥 PFC 如何消除整流桥，并分析其优缺点。

**参考答案：**
传统：AC → 整流桥（每个半周期 2 个二极管导通，VF×2×I_loss）→ PFC Boost。
无桥：去掉整流桥，用两个 MOSFET 和各一个二极管组成。正半周：Q1+D1 组 Boost；负半周：Q2+D2 组 Boost。交流直接输入。

优点：(a) 每个半周仅 1 个二极管导通（而非 2 个），导通损耗减半；(b) 少了整流桥元件 → PCB 面积小、成本低。

缺点：(a) 输出地对交流输入变化（浮动地），EMI 差——共模噪声难以滤波（无桥 PFC 的共模 EMI 是传统 PFC 的 3~5 倍！）；(b) 输入电压检测复杂（浮动地）；(c) 需要两个电感（或耦合电感）

变体：图腾柱无桥 PFC（Totem-Pole PFC）用 GaN/SiC MOSFET 实现高频臂 + 工频臂 → 效率 > 99%！适合 > 3kW 的电机驱动。但共模 EMI 仍是最大挑战

---

### 题目8：PFC 母线电压选择对电机驱动性能影响
**问题：** 220VAC 输入，电机额定电压 380V/640V（基速/弱磁）。比较母线电压选择 400V 和 650V 对系统性能的影响。包括功率器件选型、电机弱磁范围、效率和成本。

**参考答案：**
400V 母线：逆变器用 600V IGBT（如 IKW40N60），低成本。电机在 400V 母线下的最大基波相电压（SVPWM）= 400/√3 × 1.15 = 266Vrms（线电压 460Vrms）。电机额定 380V → 略高于额定，弱磁起始点约在 380V 反电动势对应转速。弱磁范围约 640/380 = 1.68 倍基速。

650V 母线：逆变器需 1200V SiC MOSFET（如 C3M0040120K），成本高 3~5 倍。但最大基波相电压 = 650/√3 × 1.15 = 431Vrms（线电压 747Vrms）。电机弱磁起始点在 640V 反电动势对应转速 → 几乎不用弱磁即可达最高转速！转矩-转速曲线极宽，恒功率区有效性极高。

权衡：(a) 如果应用不需要宽弱磁范围（如泵、风机类负载），400V + IGBT 更经济；(b) 如果应用需要宽恒功率区（如伺服主轴、电动汽车），650V + SiC 虽然成本高，但省去弱磁算法的复杂性和弱磁区效率损失

### 题目9：PFC 多相交错——电机驱动大功率应用
**问题：** 为 10kW 电机驱动设计 3 相交错 CCM PFC（三相 120° 交错）。计算：(1) 每相功率；(2) 输入电流纹波频率；(3) 与单相 PFC 相比，输出电容纹波电流的降低比例。

**参考答案：**
1. 每相 10/3 = 3.33kW。每相电感可独立设计，功率器件可选用更小规格
2. 纹波频率 = 3 × fs（三相交错等效频率倍增 3 倍）。若 fs=65kHz，等效纹波频率=195kHz → EMI 滤波器更小
3. 输出电容纹波电流：单相 PFC 的 Irms_cap ≈ 0.6 × Iout。三相交错 Irms_cap ≈ 0.2 × Iout（降低 3 倍！）。这意味着输出电容可以用更少的电解电容，寿命更长

交错 PFC 的关键设计要点：(a) 三相电流必须均分 → 每相电流采样和电流环独立；(b) 180°/120° 交错相位必须精确；(c) 多相交错对数字控制器的 PWM 模块要求高（需至少 3 对互补 PWM + 移相能力），推荐用 TI TMS320F28004x 系列的"Global PWM Load"功能实现相位同步

### 题目10：PFC 在电机驱动中的系统集成——完整输入级设计
**问题：** 设计 5kW 伺服驱动器的完整输入级。220VAC 单相输入 → _ → 650VDC 母线 → 逆变器。请给出：(1) EMI 滤波器规格；(2) 软启动/预充电电路；(3) PFC 拓扑和参数；(4) 母线电容组；(5) 放电/泄放电路。并绘制框图。

**参考答案：**
EMI：两级共模扼流圈 + X/Y 电容。共模扼流圈 Lcm1=10mH, Lcm2=3mH；X2 电容 1μF+0.47μF；Y2 电容 4.7nF×2。目标 CISPR 11 Class A。

软启动：继电器并联 NTC（或 PTC+电阻）。上电时 NTC 限制浪涌电流至 < 30A；电容充电至 > 250V 后继电器闭合短路 NTC，PFC 启动。NTC 选 MF72-10D15（10Ω/25°C, 5A）。

PFC：CCM Boost, Vin_rms_min=200V, Vout=650V, fs=100kHz。L_PFC=1.7mH（铁硅铝），MOSFET C3M0065090J(900V/65mΩ)，SiC 二极管 C4D10120D(1200V/10A)。控制器：UCC28070(两相交错)或数字控制。两相交错：每相 2.5kW

母线电容：C_total ≥ 5000/(2×100×32.5)= 769μF（纹波约束）。选 4×470μF/450V 电解两串两并 = 470μF/900V。并联 10μF×3 薄膜电容(高频去耦)

放电电路：安全规范要求断电后 5 秒内母线降到 60V 以下。R_bleed = 5/(C_total×ln(650/60)) = 5/(470e-6×ln(10.8)) = 5/(470e-6×2.38) = 4470Ω → 2×10kΩ/5W 并联
