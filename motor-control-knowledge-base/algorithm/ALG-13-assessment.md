## 一、概念理解题（★★☆☆☆）

### 题目1：三层保护架构
**问题：** 电机控制器采用三层保护架构：硬件保护层（<1μs）、软件保护层（<100μs）、系统响应层（<10ms）。请说明：
1. 为什么需要三层保护而不是只用软件保护？
2. 各层分别适合处理什么类型的故障？

**参考答案：**
1. 需要三层保护的原因：不同故障的发展速度不同，单一软件保护无法覆盖所有场景。短路电流在微秒级即可损坏功率器件，而软件保护最快需要一个控制周期（50μs@20kHz），远来不及。硬件保护（比较器+PWM刹车）可在1μs内响应，是最后一道防线。软件保护处理发展较慢的故障，系统层处理需要协调多种动作的复杂故障。

2. 各层适用故障类型：
   - **硬件保护层：** 短路过流、桥臂直通——需要纳秒/微秒级响应
   - **软件保护层：** 过载过流、过压/欠压、堵转——毫秒级发展，软件可处理
   - **系统响应层：** 过温降额、故障记录、报警输出、自动恢复——秒级发展，需协调多模块

### 题目2：死区效应的成因与影响
**问题：** 为什么逆变器需要插入死区时间？死区时间对FOC控制有什么负面影响？在什么工况下影响最严重？

**参考答案：**
死区时间是为了防止同一桥臂上下两个开关管同时导通（直通），导致电源短路。插入死区后，上下管都关断期间，电流通过续流二极管流通，实际输出电压偏离给定值。

负面影响：
- 输出电压误差导致电流畸变，尤其在低速低调制比时
- 电流波形出现5次、7次谐波，产生6次转矩脉动
- 电流过零点附近畸变最严重（极性判断困难）

影响最严重的工况：低速、轻载（低调制比），此时死区电压占输出电压比例最大。

### 题目3：弱磁控制的必要性
**问题：** 为什么电机高速运行时需要进行弱磁控制？请从反电动势与母线电压的关系解释。

**参考答案：**
电机反电动势与转速成正比：$E = \omega_e \psi_f$。当转速升高到反电动势接近母线电压时：
$\omega_e \psi_f \approx U_{dc}/\sqrt{3}$
此时SVPWM的电压裕度趋近于零，无法提供q轴电流来产生转矩，电机无法继续加速。

弱磁控制通过注入负d轴电流（$i_d < 0$），削弱气隙磁链：
$\psi_{net} = \psi_f + L_d i_d < \psi_f$
降低等效磁链后，反电动势降低，释放电压裕度给q轴电流，电机可以继续在更高转速下运行。代价是转矩能力下降（恒功率区）。

## 二、计算分析题（★★★☆☆）

### 题目4：过流保护阈值设计
**问题：** 某电机额定电流5A，最大过载电流10A（短时），采样电阻5mΩ，运放增益20倍。请计算：
1. 硬件比较器过流保护阈值对应的DAC输出电压（设安全系数1.5倍最大电流）；
2. 软件过流预警和跳闸阈值（分别取1.2倍和1.5倍额定电流）；
3. 如果DAC为12位、参考电压3.3V，阈值对应的DAC数字值是多少？

**参考答案：**
1. 硬件保护阈值电流：$I_{th} = 1.5 \times 10 = 15\text{A}$
   采样电压：$V_{sense} = I_{th} \times R_{sense} \times G_{opamp} = 15 \times 0.005 \times 20 = 1.5\text{V}$

2. 软件预警阈值：$I_{warn} = 1.2 \times 5 = 6\text{A}$
   软件跳闸阈值：$I_{trip} = 1.5 \times 5 = 7.5\text{A}$

3. DAC数字值：$DAC_{val} = V_{th} / V_{ref} \times 4095 = 1.5 / 3.3 \times 4095 = 1861$

### 题目5：温度降额计算
**问题：** 某电机控制器过温保护参数：$T_{warn} = 80°C$，$T_{trip} = 100°C$，额定电流 $I_{rated} = 10\text{A}$。NTC参数：$R_{25} = 10\text{k}\Omega$，$B = 3950$。
1. 当温度为90°C时，电流限幅值是多少？
2. 计算NTC在90°C时的电阻值；
3. 如果NTC安装在散热器上，功率管结温到NTC的热阻导致15°C温差，实际结温已达多少？保护是否可靠？

**参考答案：**
1. 降额系数：$K = (T_{trip} - T) / (T_{trip} - T_{warn}) = (100 - 90) / (100 - 80) = 0.5$
   电流限幅：$I_{limit} = I_{rated} \times K = 10 \times 0.5 = 5\text{A}$

2. NTC电阻：$R_T = R_{25} \times e^{B(1/T - 1/T_{25})}$

   其中：
   - $R_T$：温度$T$下的NTC热敏电阻阻值 ($\Omega$)
   - $R_{25}$：25°C时的标称电阻值 ($\Omega$)
   - $B$：NTC材料常数 (K)
   - $T$：当前绝对温度 (K)
   - $T_{25}$：25°C对应的绝对温度，即298.15K

   $T = 90 + 273.15 = 363.15\text{K}$，$T_{25} = 298.15\text{K}$
   $R_T = 10000 \times e^{3950 \times (1/363.15 - 1/298.15)} = 10000 \times e^{3950 \times (-0.000599)} = 10000 \times e^{-2.367} = 10000 \times 0.0938 = 938\Omega$

3. 实际结温：$T_j = T_{NTC} + 15 = 90 + 15 = 105°C$
   结温已超过 $T_{trip} = 100°C$，但NTC仅读到90°C，保护未触发。这说明热延迟导致保护不可靠。解决方案：将 $T_{trip}$ 降低15°C（即设为85°C），或使用结温估算模型补偿热延迟。

### 题目6：死区补偿计算
**问题：** 某逆变器参数：母线电压 $U_{dc} = 24\text{V}$，死区时间 $T_{dt} = 1\mu\text{s}$，PWM频率 $f_{PWM} = 20\text{kHz}$。
1. 计算死区导致的电压误差 $U_{dt}$；
2. 在10%调制比（输出电压约2.4V）时，死区电压误差占比多少？
3. 代码中使用 `sign(i)` 进行补偿，在电流过零点附近有什么问题？如何改进？

**参考答案：**
1. $U_{dt} = U_{dc} \times T_{dt} / T_{PWM} = 24 \times 1\mu\text{s} / 50\mu\text{s} = 0.48\text{V}$

2. 10%调制比时输出电压约 $24 \times 0.1 = 2.4\text{V}$，死区电压误差占比 $0.48 / 2.4 = 20\%$。这是非常严重的畸变，会导致电流波形明显失真。

3. `sign(i)` 在电流过零点附近的问题：
   - 电流过零时极性不确定（受噪声影响），`sign()` 函数在0附近频繁切换，导致补偿方向振荡
   - 实际电流可能因噪声在0附近波动，补偿量在 $+U_{dt}$ 和 $-U_{dt}$ 之间跳变
   改进方法：使用平滑过渡函数替代硬切换：
   $sign_{smooth}(i) = \frac{2}{\pi}\arctan(i/\delta)$
   其中 $\delta$ 为过渡区间宽度（如0.1A），在 $|i| < \delta$ 时线性过渡，避免硬切换。

### 题目7：弱磁电流计算
**问题：** 某IPMSM参数：$p = 4$，$\psi_f = 0.05\text{Wb}$，$L_d = 2\text{mH}$，$L_q = 5\text{mH}$，母线电压 $U_{dc} = 48\text{V}$，额定转速3000rpm。请计算：
1. 基速（不弱磁时的最高转速），设SVPWM线性调制范围 $U_{max} = U_{dc}/\sqrt{3}$；
2. 在2倍基速时，弱磁d轴电流 $i_d^{FW}$ 是多少？（忽略电阻压降，设 $i_q = 5\text{A}$）

**参考答案：**
1. 基速条件：$\omega_e \psi_f = U_{max} = U_{dc}/\sqrt{3} = 48/1.732 = 27.7\text{V}$
   $\omega_e = 27.7 / 0.05 = 554\text{rad/s}$
   机械转速：$n = \omega_e / (2\pi p) \times 60 = 554 / (2\pi \times 4) \times 60 = 1323\text{rpm}$
   （注意：这是 $i_d = 0$ 且 $i_q$ 很小的情况下的理论基速）

2. 2倍基速：$\omega_e = 1108\text{rad/s}$
   弱磁电流：$i_d^{FW} = -\frac{\psi_f}{L_d} + \frac{1}{L_d}\sqrt{\frac{U_{max}^2}{\omega_e^2} - (L_q i_q)^2}$
   $= -\frac{0.05}{0.002} + \frac{1}{0.002}\sqrt{\frac{27.7^2}{1108^2} - (0.005 \times 5)^2}$
   $= -25 + 500\sqrt{0.000624 - 0.000625}$
   根号内为负值，说明 $i_q = 5\text{A}$ 在2倍基速下不可行（电压约束不满足）。需要降低 $i_q$：
   最大 $i_q$：$(L_q i_q)^2 \leq U_{max}^2/\omega_e^2$，$i_q \leq 27.7/(1108 \times 0.005) = 5.0\text{A}$
   刚好在边界上，$i_d^{FW} = -25 + 0 = -25\text{A}$，需要极大的弱磁电流，实际不可行。这说明2倍基速时转矩能力已极度受限。

## 三、工程实践题（★★★★☆）

### 题目8：过流故障状态机的工程分析
**问题：** 代码中过流保护状态机为 `FAULT_NONE → FAULT_WARN → FAULT_TRIP → FAULT_LOCK`。请分析：
1. 从WARN到TRIP需要连续100个周期超过跳闸阈值，这种延时设计的目的是什么？
2. `FAULT_LOCK` 状态没有自动恢复机制，为什么？如果需要自动恢复应如何设计？
3. 硬件比较器过流保护（PWM刹车）触发后，为什么需要软件复位才能恢复？

**参考答案：**
1. 延时设计的目的：防止瞬态干扰（如开关噪声、EMI脉冲）误触发跳闸。100个周期@20kHz = 5ms，短时过流（如启动冲击）不会触发跳闸，只有持续过流才跳闸。这是抗干扰与保护速度的权衡。

2. LOCK状态无自动恢复的原因：过流跳闸通常意味着严重故障（如短路、过载），自动恢复可能导致反复跳闸-恢复循环，加剧设备损坏。如需自动恢复，应设计：
   - 恢复延时：跳闸后等待足够长时间（如30秒）让设备冷却
   - 恢复次数限制：如3次自动恢复后锁定，需人工复位
   - 恢复前自检：检查故障是否已消除（如电流已归零）
   - 降额恢复：恢复后限制电流为额定值的50%

3. PWM刹车后需软件复位的原因：这是安全设计原则——故障发生后必须由软件确认故障已消除才允许恢复运行。自动恢复可能在故障仍然存在时重新启动，导致二次损坏。软件复位流程：读取故障寄存器 → 确认故障类型 → 检查故障是否消除 → 清除故障标志 → 重新使能PWM。

### 题目9：MTPA与弱磁的协同控制
**问题：** 对于IPMSM，全速域控制轨迹为：MTPA → 弱磁 → MTPV。请分析：
1. MTPA和弱磁的控制目标是否矛盾？如何协调？
2. 在MTPA到弱磁的过渡区域，如何保证平滑切换？
3. 如果MTPA轨迹的 $i_d$ 值与弱磁要求的 $i_d$ 值不同，应取哪个？为什么？

**参考答案：**
1. MTPA目标是最小化电流幅值（效率优先），弱磁目标是满足电压约束（运行范围优先）。两者确实矛盾：MTPA可能给出 $i_d = -2\text{A}$，而弱磁要求 $i_d = -10\text{A}$。协调原则：**电压约束优先**——当电压未饱和时使用MTPA，电压饱和时切换到弱磁。即 $i_d^{ref} = \min(i_d^{MTPA}, i_d^{FW})$（取更负的值）。

2. 平滑切换方法：
   - **电压反馈弱磁：** 不直接计算弱磁电流，而是通过电压幅值反馈自动调节 $i_d$。当 $|U_s| > U_{max} - \Delta U$ 时，逐渐增大负 $i_d$；电压裕度恢复后逐渐减小。这种反馈机制天然平滑。
   - **查表法：** 离线计算全速域 $i_d(i_q, \omega)$ 查找表，MTPA和弱磁轨迹在表中自然衔接。
   - 避免硬切换：不要用if-else在MTPA和弱磁之间跳变，而应通过连续的反馈机制过渡。

3. 应取弱磁要求的 $i_d$ 值（更负的值）。原因：电压约束是硬约束——如果电压饱和，电流环无法正常工作，控制性能急剧恶化。MTPA是软约束——不满足只是效率降低，不会导致失稳。因此安全优先：先保证电压不饱和（弱磁），再在电压裕度内优化效率（MTPA）。

### 题目10：参数自整定的工程实现
**问题：** 代码中PI参数自整定函数：
```c
id_pi.kp = para->Ls * bandwidth;
id_pi.ki = para->Rs * bandwidth;
```
请分析：
1. 这个公式对应什么设计方法？推导过程是什么？
2. 如果电机参数（Rs、Ls）测量不准确，对控制性能有什么影响？
3. 提出一种在线辨识Rs的方法，并说明其实现步骤。

**参考答案：**
1. 对应零极点对消法（带宽法）。推导：
   电流环被控对象：$G(s) = 1/(R_s + L_s s)$
   PI控制器：$G_{PI}(s) = K_p + K_i/s$
   零极点对消条件：$K_i/K_p = R_s/L_s$
   期望闭环带宽 $\omega_{bw}$：$K_p = L_s \cdot \omega_{bw}$，$K_i = R_s \cdot \omega_{bw}$
   代码中bandwidth即 $\omega_{bw}$。

2. 参数不准确的影响：
   - Ls偏大→Kp偏大→电流环振荡，超调增大
   - Ls偏小→Kp偏小→电流响应变慢，跟踪误差增大
   - Rs偏大→Ki偏大→积分过快，可能振荡
   - Rs偏小→Ki偏小→稳态误差消除慢
   - 零极点对消失效→闭环出现额外极点，动态性能退化

3. 在线Rs辨识方法——直流注入法：
   步骤：
   a. 在d轴注入小幅度直流电压 $\Delta U_d$（如1V）
   b. 等待电流稳定，测量d轴电流变化 $\Delta I_d$
   c. 计算电阻：$R_s = \Delta U_d / \Delta I_d$
   d. 低通滤波更新Rs估计值
   e. 根据新Rs重新计算Ki
   注意事项：注入电压不能太大（不影响正常控制），需在稳态时进行（避免电感压降干扰），更新速率要慢（避免参数跳变）。
