## 一、概念理解题（★★☆☆☆）

### 题目1：编码器类型与选型
**问题：** 增量式编码器和绝对值编码器在有感FOC应用中有何本质区别？在以下场景中应选择哪种编码器：
1. 工业伺服定位系统（要求掉电不丢位置）
2. 风机水泵调速（成本敏感，无需精确定位）

**参考答案：**
- **本质区别：** 增量式编码器输出A/B/Z三相脉冲信号，只能测量位置增量，掉电后位置信息丢失，上电需通过Z信号或转子定位重新校准；绝对值编码器直接输出绝对位置数字量，掉电后位置保持，无需校准。
- 场景1：应选择绝对值编码器（如SSI/BiSS-C接口），掉电不丢位置，满足定位需求。
- 场景2：可选择增量式编码器或霍尔传感器，成本低，风机水泵只需调速不需精确定位，上电通过IF强拖启动即可。

### 题目2：PWM与ADC同步原理
**问题：** 在FOC电流采样中，为什么必须在PWM开通期间（下桥臂导通时）进行ADC采样？如果采样时刻选择不当会出现什么问题？

**参考答案：**
电流采样电阻通常安装在下桥臂与地之间，只有下桥臂MOSFET导通时相电流才流过采样电阻，此时才能采样到真实的相电流。如果采样时刻不当：
- **在开关动作瞬间采样：** 开关噪声（dv/dt、di/dt）通过PCB寄生参数耦合到采样电路，导致电流读数出现大幅尖峰。
- **在上桥臂导通期间采样：** 采样电阻被旁路，无法获取电流信息。
- **采样点偏离PWM中心：** 电流纹波最大，采样值不能代表平均电流。
因此，ADC注入通道通常由TIM1 TRGO在PWM周期中心位置触发，确保采样点在下桥臂导通期间且远离开关边沿。

### 题目3：IF强拖启动的原理与局限
**问题：** IF（Current-Frequency）强拖启动的基本原理是什么？为什么说它"可能失步"？失步的根本原因是什么？

**参考答案：**
IF强拖启动原理：角度自增（开环给定电角度斜坡），电流闭环控制。即人为给定一个逐渐增加的电角度，同时通过电流环控制 $i_d = 0$、$i_q = I_{start}$，使电机跟随给定角度旋转。

失步原因：IF启动是开环角度给定，电机实际转子位置可能与给定角度不同步。当负载转矩超过启动电流产生的电磁转矩，或角度斜坡加速度过大导致电机跟不上时，转子"滑落"，实际角度滞后于给定角度，即失步。根本原因是缺乏转子位置反馈，无法保证给定角度与实际角度一致。

## 二、计算分析题（★★★☆☆）

### 题目4：电流采样比例系数计算
**问题：** 某FOC系统参数如下：采样电阻 $R_{sense} = 5\text{m}\Omega$，运放增益 $G_{opamp} = 20$，ADC参考电压 $V_{ref} = 3.3\text{V}$，ADC分辨率12位。请计算：
1. 电流比例系数 $iratio$（A/LSB）；
2. 满量程电流（ADC达到最大值时对应的电流）；
3. 电流分辨率（1 LSB对应的电流值）。

**参考答案：**
1. ADC量化步长：$V_{LSB} = V_{ref} / 2^{12} = 3.3 / 4096 = 0.805\text{mV}$
   电流比例系数：$iratio = V_{LSB} / (R_{sense} \times G_{opamp}) = 0.805 \times 10^{-3} / (0.005 \times 20) = 0.00805\text{A/LSB}$

2. 满量程电流：$I_{max} = iratio \times 4095 = 0.00805 \times 4095 \approx 32.96\text{A}$
   （注意：ADC最大值为4095，不是4096）

3. 电流分辨率即 $iratio = 8.05\text{mA}$，即1 LSB对应约8mA电流变化。

### 题目5：速度计算方法对比
**问题：** 某增量式编码器PPR = 2500，控制周期 $T_s = 50\mu\text{s}$（20kHz），电机在3000rpm运行。请分别用M法和T法计算速度，并分析哪种方法更适合该工况。

**参考答案：**
电机转速3000rpm = 50rps。
- **M法（脉冲计数法）：**
  每转脉冲数2500，每秒脉冲数 = 50 × 2500 = 125000
  一个控制周期内脉冲数 $M = 125000 \times 50 \times 10^{-6} = 6.25$
  分辨率：1脉冲对应速度 = $1 / (T_s \times PPR) = 1 / (50 \times 10^{-6} \times 2500) = 8\text{rps} = 480\text{rpm}$
  量化误差：$480 / 3000 = 16\%$，精度较差。

- **T法（周期测量法）：**
  脉冲间隔 = $1 / 125000 = 8\mu\text{s}$
  若时钟频率100MHz，计数 $N = 100 \times 10^6 \times 8 \times 10^{-6} = 800$
  分辨率：1个时钟对应速度变化 = $f_{clock} / (N \times PPR) = 100 \times 10^6 / (800 \times 2500) = 50\text{rps} = 3000\text{rpm}$ 的 $1/800$，精度高。

M法在高速时脉冲数多、分辨率尚可，但3000rpm下量化误差仍达16%。T法在高速时精度高。因此该工况更适合T法或M/T法。实际工程中常用M/T法，结合两者优势。

### 题目6：串级PI与并行PI的区别
**问题：** 代码中实现了串级PI（serial_pid_ctrl）和并行PI（parallel_pid_ctrl）两种控制器，核心区别在于积分对象不同。请分析：
1. 串级PI对P项积分，并行PI对误差积分，两者在抗积分饱和方面有什么差异？
2. 在电机电流环中，哪种更合适？为什么？

**参考答案：**
1. **串级PI：** 积分项 $I_{term} += K_i \times P_{term} = K_i \times K_p \times error$。当输出饱和时，P项立即减小（因为误差减小），积分增量也随之减小，天然具有抗积分饱和特性。即使误差方向不变，P项受输出限幅间接约束，积分增长速度受限。
   **并行PI：** 积分项 $I_{term} += K_i \times error$。只要误差存在，积分项就持续增长，不受输出饱和约束，容易产生积分饱和（windup），需要额外的抗饱和机制（如条件积分、遇限削弱）。

2. 电机电流环中串级PI更合适。原因：电流环频繁工作在限幅区（如启动、加减速），串级PI天然抗饱和，避免积分windup导致的超调和振荡。此外，串级PI的等效增益 $K_i \times K_p$ 随Kp调整自动变化，参数整定更简单。

### 题目7：SVPWM扇区判断与作用时间
**问题：** 已知逆Park变换输出 $v_\alpha = -12\text{V}$，$v_\beta = 20.78\text{V}$，母线电压 $U_{dc} = 24\text{V}$。请计算：
1. SVPWM扇区判断变量 $va, vb, vc$ 及对应扇区；
2. 参考电压矢量的幅值和角度，判断是否在线性调制范围内。

**参考答案：**
1. 扇区判断变量：
   $va = v_\beta = 20.78$
   $vb = (\sqrt{3} \times v_\alpha - v_\beta) / 2 = (1.732 \times (-12) - 20.78) / 2 = (-20.78 - 20.78) / 2 = -20.78$
   $vc = (-\sqrt{3} \times v_\alpha - v_\beta) / 2 = (20.78 - 20.78) / 2 = 0$

   $a = (va > 0) = 1$，$b = (vb > 0) = 0$，$c = (vc > 0) = 0$
   $sextant = (c \ll 2) + (b \ll 1) + a = 0 + 0 + 1 = 1$，对应扇区II。

2. 参考电压矢量：
   幅值：$V_{ref} = \sqrt{v_\alpha^2 + v_\beta^2} = \sqrt{144 + 431.8} = \sqrt{575.8} \approx 24.0\text{V}$
   角度：$\theta = \arctan(v_\beta / v_\alpha) = \arctan(20.78 / (-12)) = 180° - 60° = 120°$（第二象限）
   线性调制范围：$U_{dc}/\sqrt{3} = 24/1.732 \approx 13.86\text{V}$
   $V_{ref} = 24\text{V} > 13.86\text{V}$，已超出线性调制范围，进入过调制区。

## 三、工程实践题（★★★★☆）

### 题目8：电流校准的工程实现与问题分析
**问题：** 代码中电流偏置校准函数 `get_curr_off()` 在电机未通电时采样1000次取平均。请分析：
1. 为什么校准必须在电机未通电时进行？
2. 该校准方法有什么局限性？在什么情况下偏置会发生变化需要重新校准？
3. 代码中 `sum_a` 等变量未初始化，存在什么隐患？如何修正？

**参考答案：**
1. 电机未通电时采样电阻无电流流过，ADC读数仅反映运放偏置电压，此时取平均即为偏置值。如果通电后校准，ADC读数包含实际电流，偏置值将错误。

2. 局限性：运放偏置随温度漂移，上电校准后温度变化可能导致偏置偏移。需要重新校准的情况：
   - 环境温度大幅变化（如从冷态到满载运行后）
   - 长时间运行后运放自热
   - 供电电压波动影响运放偏置
   工程改进：定期在零电流窗口（如PWM全关断瞬间）重新校准，或使用动态偏置跟踪算法。

3. `sum_a, sum_b, sum_c` 声明为局部变量但未初始化为0，在C语言中局部变量初值不确定，可能导致校准结果错误。修正：`float sum_a = 0.0f, sum_b = 0.0f, sum_c = 0.0f;`

### 题目9：启动状态机设计与参数整定
**问题：** 代码中的启动状态机流程为：`mt_start → mt_precharge → mt_align → mt_run`。请分析：
1. 预充电阶段（10个控制周期）的作用是什么？时间是否合理？
2. 转子定位阶段（200个控制周期，IF模式）的定位电流和定位时间如何确定？
3. 从IF模式切换到 `spd_curr` 模式时，为什么需要清除PI积分项（`pid_clear`）？不清除会怎样？

**参考答案：**
1. 预充电阶段的作用：等待母线电容充电至稳定电压，避免上电瞬间浪涌电流。10个控制周期（20kHz下约0.5ms）可能偏短，实际预充电时间取决于母线电容大小和限流电阻，通常需要几十毫秒到几百毫秒。建议增加预充电完成判断条件（如母线电压达到额定值的90%）。

2. 定位电流：通常取 $(0.5 \sim 1.0) \times I_{rated}$，太小无法可靠定位，太大可能引起机械冲击。定位时间：需大于机械系统过渡过程时间 $t > 5J/B$（J为惯量，B为阻尼），200个周期（10ms@20kHz）对小惯量电机足够，对大惯量负载可能不足。

3. 切换时清除PI积分项的原因：IF模式下PI积分项积累的是开环角度下的控制量，切换到闭环后角度来源突变（从自增角度变为编码器角度），如果不清除积分项，其残留值会导致电压输出跳变，引起电流冲击。不清除会导致切换瞬间电流尖峰，严重时触发过流保护。

### 题目10：PI参数自动计算的工程分析
**问题：** 代码中 `foc_update_current_ctrl_gain()` 函数实现了PI参数自动计算：
```c
id_pi.kp = mt.para.Ls * bandwidth * i_base;
id_pi.ki = mt.para.Rs / mt.para.Ls * 0.00005f;
```
请分析：
1. Kp公式中乘以 `i_base` 的作用是什么？如果不乘以 `i_base` 会怎样？
2. Ki公式中的 `0.00005f` 代表什么？这个值是否合理？
3. 如果电机运行后温度升高导致Rs增大30%，对控制性能有什么影响？如何应对？

**参考答案：**
1. `i_base` 是电流基准值（标幺化系数），作用是将Kp从物理单位（V/A）转换为标幺值，使PI输出与电压基准匹配。如果不乘以 `i_base`，Kp的量纲不匹配，PI输出值域错误，可能导致输出过小（电流跟踪不上）或过大（振荡发散）。

2. `0.00005f` 是采样周期 $T_s$。Ki的推导：$K_i = R_s / \tau_c = R_s \cdot \omega_c$，在离散实现中 $K_i = K_{i,continuous} \times T_s$。$0.00005\text{s} = 50\mu\text{s}$，对应20kHz控制频率，数值合理。但代码中Ki公式与理论公式不一致——理论应为 $K_i = R_s \cdot \omega_c \cdot T_s$，而代码为 $K_i = (R_s / L_s) \times T_s$，缺少带宽项，这可能是串级PI的特殊实现。

3. Rs增大30%的影响：
   - Ki偏小（Ki应与Rs成正比），积分速度变慢，稳态误差消除变慢
   - 零极点对消条件 $K_i/K_p = R_s/L_s$ 被破坏，对消失效，电流环出现振荡倾向
   - 低速时电阻压降占比大，Rs误差导致磁链观测偏差
   应对措施：在线Rs辨识（如直流注入法），根据温度传感器补偿Rs，或使用MRAS自适应调整Ki。
