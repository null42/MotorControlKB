## 一、概念理解题（★★☆☆☆）

### 题目1：Clarke变换的物理意义与简化条件
**问题：** Clarke变换将三相静止坐标系(ABC)转换为两相静止坐标系(αβ)，请说明：
1. 为什么可以将三相系统简化为两相系统？其数学前提是什么？
2. 在实际代码实现中，`i_alpha = i_a; i_beta = (i_b - i_c) * ONE_BY_SQRT3;` 用了什么简化条件？如果该条件不满足会怎样？

**参考答案：**
1. 三相系统可以简化为两相系统，数学前提是三相电流之和为零（$i_a + i_b + i_c = 0$），即三相平衡条件。这意味着三相变量中只有两个是独立的，第三个可由前两个线性表示。几何上，三相绕组在空间互差120°，其合成磁动势可以用两个正交分量(αβ)完全描述。
2. 代码利用了三相平衡条件 $i_a + i_b + i_c = 0$，推导出 $i_c = -i_a - i_b$，代入标准Clarke变换矩阵后化简得到。如果三相不平衡（如绕组匝间短路、单相断路），则 $i_a + i_b + i_c \neq 0$，简化公式将产生误差，导致αβ轴电流出现零序分量泄漏，Park变换后dq轴电流含有二倍频纹波。

### 题目2：Park变换中d轴和q轴的物理含义
**问题：** 在Park变换中，d轴和q轴分别代表什么物理含义？为什么说FOC实现了"类似直流电机"的控制？请结合直流电机的励磁电流和电枢电流进行类比说明。

**参考答案：**
- **d轴（直轴）：** 方向与转子磁链方向一致，d轴电流 $i_d$ 控制磁场强度（励磁）。对于表贴式PMSM，通常令 $i_d = 0$；对于IPMSM，$i_d$ 可为负值实现弱磁。
- **q轴（交轴）：** 方向与转子磁链垂直，q轴电流 $i_q$ 控制转矩大小，与转矩成正比。

类比直流电机：$i_d$ 对应直流电机的励磁电流 $I_f$，$i_q$ 对应电枢电流 $I_a$。在直流电机中，励磁和电枢天然解耦，可以独立控制；FOC通过坐标变换使交流电机也获得了这种解耦特性——d轴管磁场、q轴管转矩，各司其职。这正是FOC的核心价值：将交流电机的控制复杂度降低到直流电机水平。

### 题目3：等幅值变换与功率等效变换的区别
**问题：** Clarke变换存在等幅值变换和功率等效变换两种形式，其系数分别为 $\frac{2}{3}$ 和 $\sqrt{\frac{2}{3}}$。请说明：
1. 两种变换的物理区别是什么？
2. 在电机控制代码中通常使用哪种？为什么？

**参考答案：**
1. **等幅值变换**（系数 $\frac{2}{3}$）：变换后αβ变量的幅值与ABC变量幅值相等，即 $I_{\alpha\beta,peak} = I_{abc,peak}$。但功率不守恒，$P_{\alpha\beta} = \frac{3}{2}P_{abc}$。
   **功率等效变换**（系数 $\sqrt{\frac{2}{3}}$）：变换前后功率守恒，$P_{\alpha\beta} = P_{abc}$。但幅值关系为 $I_{\alpha\beta,peak} = \sqrt{\frac{3}{2}}I_{abc,peak}$。

2. 电机控制代码中通常使用等幅值变换。原因：等幅值变换后电流、电压的数值与实际物理量幅值一致，便于调试和参数整定（如PI限幅值可直接对应实际电流值）。功率不守恒的影响可通过系数 $\frac{3}{2}$ 在转矩计算中补偿。

### 题目4：SVPWM的基本概念
**问题：** SVPWM（空间矢量脉宽调制）与SPWM相比有什么本质区别？三相逆变器有多少种开关状态？其中有效矢量和零矢量各有多少个？

**参考答案：**
- **本质区别：** SPWM是相电压独立调制，三相分别生成正弦参考波与三角载波比较；SVPWM将三相电压视为一个空间矢量整体，通过相邻两个有效矢量和零矢量的时间分配来合成任意参考矢量，电压利用率更高（线性调制范围达 $U_{dc}/\sqrt{3}$，比SPWM提高约15%）。
- 三相逆变器有 $2^3 = 8$ 种开关状态。
- 其中有效矢量6个（V1~V6），零矢量2个（V0、V7）。

## 二、计算分析题（★★★☆☆）

### 题目5：Clarke变换与Park变换的数值计算
**问题：** 某三相PMSM当前三相电流为 $i_a = 10\text{A}$，$i_b = -5\text{A}$，$i_c = -5\text{A}$，转子电角度 $\theta_e = 30°$。请计算：
1. 经过Clarke变换后的 $i_\alpha$ 和 $i_\beta$；
2. 经过Park变换后的 $i_d$ 和 $i_q$；
3. 如果 $i_d$ 的PI控制器输出 $v_d = 0$，$i_q$ 的PI控制器输出 $v_q = 24\text{V}$，计算逆Park变换后的 $v_\alpha$ 和 $v_\beta$。

**参考答案：**
1. **Clarke变换：**
   $i_\alpha = i_a = 10\text{A}$
   $i_\beta = \frac{1}{\sqrt{3}}(i_b - i_c) = \frac{1}{\sqrt{3}}(-5 - (-5)) = 0\text{A}$
   验证：$i_a + i_b + i_c = 10 - 5 - 5 = 0$，满足三相平衡条件。

2. **Park变换：** $\theta_e = 30°$，$\cos 30° = \frac{\sqrt{3}}{2}$，$\sin 30° = \frac{1}{2}$
   $i_d = i_\alpha \cos\theta + i_\beta \sin\theta = 10 \times \frac{\sqrt{3}}{2} + 0 = 5\sqrt{3} \approx 8.66\text{A}$
   $i_q = -i_\alpha \sin\theta + i_\beta \cos\theta = -10 \times \frac{1}{2} + 0 = -5\text{A}$

3. **逆Park变换：**
   $v_\alpha = v_d \cos\theta - v_q \sin\theta = 0 \times \frac{\sqrt{3}}{2} - 24 \times \frac{1}{2} = -12\text{V}$
   $v_\beta = v_d \sin\theta + v_q \cos\theta = 0 \times \frac{1}{2} + 24 \times \frac{\sqrt{3}}{2} = 12\sqrt{3} \approx 20.78\text{V}$

### 题目6：dq轴耦合分析与前馈解耦
**问题：** 某SPMSM参数为 $R_s = 0.5\Omega$，$L_s = 1\text{mH}$，$\psi_f = 0.01\text{Wb}$，极对数 $p = 4$。当前运行状态为 $\omega_e = 600\text{rad/s}$，$i_d = 0\text{A}$，$i_q = 10\text{A}$。请计算：
1. dq轴电压方程中的耦合项和反电动势项分别是多少？
2. 如果不进行前馈解耦，d轴电压中由q轴电流引起的耦合电压占d轴总电压的比例约为多少？（忽略电阻压降）

**参考答案：**
1. **耦合项：**
   d轴耦合项：$-\omega_e L_s i_q = -600 \times 0.001 \times 10 = -6\text{V}$
   q轴耦合项：$\omega_e L_d i_d = 600 \times 0.001 \times 0 = 0\text{V}$

   **反电动势项：** $\omega_e \psi_f = 600 \times 0.01 = 6\text{V}$

2. 在 $i_d = 0$ 的情况下，d轴电压方程简化为：
   $u_d = R_s i_d + L_s \frac{di_d}{dt} - \omega_e L_s i_q = 0 + 0 + (-6) = -6\text{V}$

   d轴电压完全由q轴耦合项决定，占比100%。这说明在高速运行时，dq轴耦合非常严重，如果不进行前馈解耦，d轴电流将无法维持在0，导致磁场定向失败。前馈解耦项 $u_d^{dec} = -\omega_e \hat{L}_q i_q = 6\text{V}$ 可完全补偿此耦合。

### 题目7：电流环PI参数设计
**问题：** 某SPMSM参数为 $R_s = 0.178\Omega$，$L_s = 95\mu\text{H}$。要求电流环带宽为 $500\text{Hz}$（$\omega_c = 3141.6\text{rad/s}$）。
1. 用零极点对消法计算电流环PI参数 $K_p$ 和 $K_i$；
2. 验证PI零点是否与电机极点对消；
3. 说明速度环带宽应如何设计。

**参考答案：**
1. 零极点对消法：$\tau_c = 1/\omega_c = 1/3141.6 \approx 0.318\text{ms}$
   $K_p = L_s / \tau_c = 95 \times 10^{-6} / 0.318 \times 10^{-3} = 0.2987$
   $K_i = R_s / \tau_c = 0.178 / 0.318 \times 10^{-3} = 559.7$

2. PI零点：$K_i / K_p = 559.7 / 0.2987 = 1873.8\text{rad/s}$
   电机极点：$R_s / L_s = 0.178 / 95 \times 10^{-6} = 1873.7\text{rad/s}$
   两者基本相等，验证了对消条件成立。

3. 速度环带宽应为电流环带宽的1/5~1/10，即 $\omega_{c,spd} = 50 \sim 100\text{Hz}$。带宽递减原则确保内环（电流环）对外环（速度环）的指令跟踪足够快，避免内外环动态耦合。

## 三、工程实践题（★★★★☆）

### 题目8：转矩方程与电机选型
**问题：** 某IPMSM参数为 $p = 4$，$\psi_f = 0.05\text{Wb}$，$L_d = 2\text{mH}$，$L_q = 5\text{mH}$。在两种控制策略下：
1. $i_d = 0$ 控制：$i_q = 20\text{A}$ 时的电磁转矩是多少？
2. MTPA控制：相同电流幅值 $I_s = 20\text{A}$ 下，MTPA轨迹给出的 $i_d$ 和 $i_q$ 各是多少？电磁转矩是多少？
3. MTPA相比 $i_d = 0$ 控制转矩提升了多少百分比？

**参考答案：**
1. $i_d = 0$ 控制：
   $T_e = \frac{3}{2} p \psi_f i_q = \frac{3}{2} \times 4 \times 0.05 \times 20 = 6.0\text{N·m}$

2. MTPA控制：$I_s = 20\text{A}$，约束 $i_d^2 + i_q^2 = 400$
   MTPA轨迹：$i_d^{MTPA} = \frac{\psi_f}{2(L_q - L_d)} - \sqrt{\frac{\psi_f^2}{4(L_q - L_d)^2} + i_q^2}$
   代入：$L_q - L_d = 3\text{mH}$
   $\frac{\psi_f}{2(L_q - L_d)} = \frac{0.05}{2 \times 0.003} = 8.333$
   这是一个隐式方程，需迭代求解。利用MTPA条件 $i_d = \frac{(L_d - L_q)}{\psi_f} i_d i_q$ 的关系，近似解为：
   $i_d \approx -8.7\text{A}$，$i_q \approx 18.0\text{A}$
   $T_e = \frac{3}{2} \times 4 \times [0.05 \times 18.0 + (0.002 - 0.005) \times (-8.7) \times 18.0]$
   $= 6 \times [0.9 + (-0.003) \times (-156.6)] = 6 \times [0.9 + 0.47] = 8.22\text{N·m}$

3. 转矩提升：$(8.22 - 6.0) / 6.0 \times 100\% \approx 37\%$
   IPMSM利用磁阻转矩可显著提升单位电流输出转矩，这是MTPA控制的核心价值。

### 题目9：坐标变换角度误差的极对数放大效应
**问题：** 某电机极对数 $p = 7$，使用14位磁编码器（16384线/圈）。
1. 编码器的机械角度分辨率是多少？
2. 对应的电角度分辨率是多少？
3. 如果磁编码器安装偏心导致0.1°机械角度误差，对应的电角度误差是多少？这对FOC控制有什么影响？
4. 如果换成极对数 $p = 2$ 的电机，同样的安装误差影响如何？

**参考答案：**
1. 机械角度分辨率：$360° / 16384 \approx 0.022°$
2. 电角度分辨率：$0.022° \times 7 \approx 0.154°$
3. 电角度误差：$0.1° \times 7 = 0.7°$。在Park变换中，角度误差 $\Delta\theta$ 导致d轴和q轴电流交叉耦合：$i_d' = i_d \cos\Delta\theta + i_q \sin\Delta\theta$，$i_q' = -i_d \sin\Delta\theta + i_q \cos\Delta\theta$。当 $\Delta\theta = 0.7°$ 时，$\sin(0.7°) \approx 0.0122$，即q轴电流的1.22%泄漏到d轴，导致磁场定向偏差，产生转矩脉动。
4. $p = 2$ 时，电角度误差仅 $0.2°$，影响显著减小。高极对数电机对编码器精度和安装精度要求更高，这是硬件选型时必须考虑的约束。

### 题目10：磁链参数温度漂移对转矩精度的影响
**问题：** 某PMSM在25°C时永磁体磁链 $\psi_f = 0.05\text{Wb}$，NdFeB永磁体温度系数为 $-0.1\%/°C$。电机运行后温升至125°C。
1. 计算温升后的磁链值；
2. 如果转矩环仍按25°C的磁链参数计算，在 $i_q = 20\text{A}$、$p = 4$ 的条件下，转矩估算偏差是多少？
3. 提出至少两种工程解决方案。

**参考答案：**
1. 温升100°C，磁链下降：$\psi_f(125°C) = 0.05 \times (1 - 0.001 \times 100) = 0.05 \times 0.9 = 0.045\text{Wb}$
2. 按25°C参数估算的转矩：$T_{est} = \frac{3}{2} \times 4 \times 0.05 \times 20 = 6.0\text{N·m}$
   实际转矩：$T_{act} = \frac{3}{2} \times 4 \times 0.045 \times 20 = 5.4\text{N·m}$
   偏差：$6.0 - 5.4 = 0.6\text{N·m}$，相对偏差10%。
3. 工程解决方案：
   - **温度补偿查找表：** 在电机内埋入温度传感器，建立磁链-温度查找表，实时补偿 $\psi_f$。
   - **在线磁链辨识：** 通过空载反电动势或MRAS方法在线辨识 $\psi_f$，动态更新参数。
   - **转矩闭环控制：** 用转矩传感器或基于电流/速度的转矩估算构成转矩闭环，自动补偿参数偏差。
