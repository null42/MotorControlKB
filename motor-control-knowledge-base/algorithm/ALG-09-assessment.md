## 一、概念理解题（★★☆☆☆）

### 题目1：凸极效应的物理本质
**问题：** 高频注入法为什么只适用于IPMSM而不适用于SPMSM？请从磁路结构角度解释凸极效应的物理本质。

**参考答案：**
IPMSM的永磁体嵌入转子内部，导致d轴和q轴磁路截然不同：
- **d轴磁路**穿过永磁体区域，NdFeB永磁体磁导率接近空气（$\mu_r \approx 1$），磁阻大，电感小（$L_d$ 小）
- **q轴磁路**不穿过永磁体，通过铁芯，铁芯磁导率高，磁阻小，电感大（$L_q$ 大）
结果：$L_d < L_q$，凸极率 $\xi = (L_q - L_d)/(L_d + L_q) > 0$。

SPMSM的永磁体粘贴在转子表面，d轴和q轴磁路都经过气隙和永磁体，磁阻近似相等，$L_d \approx L_q$，凸极率 $\xi \approx 0$。

高频注入依赖凸极效应（电感随角度变化）来提取位置信息。SPMSM无凸极效应，注入高频电压后电流响应不随角度变化，无法提取位置信息，因此不适用。

### 题目2：脉振注入与方波注入的对比
**问题：** 脉振高频注入和方波高频注入在注入信号形式和解调方法上有什么本质区别？方波注入为什么带宽更宽？

**参考答案：**
| 特性 | 脉振注入 | 方波注入 |
|------|---------|---------|
| 注入信号 | d轴正弦电压 $U_h \cos(\omega_h t)$ | d轴方波电压 $\pm U_h$ |
| 解调方法 | 乘法解调：$i_{qh} \times \sin(\omega_h t)$ + LPF | 差分解调：前后半周期电流差分 |
| 带宽 | 受限于LPF（截止频率远小于注入频率） | 无需LPF，差分直接提取，带宽宽 |
| 采样要求 | 常规采样即可 | 需要高采样率（方波边沿处采样） |

方波注入带宽更宽的原因：脉振注入的解调信号需经过低通滤波器提取直流分量，LPF的截止频率必须远小于注入频率，导致角度跟踪带宽受限（通常<50Hz）。方波注入利用前后半周期的电流差分直接提取角度误差，无需窄带滤波，角度跟踪带宽可达数百Hz，适合快速动态响应。

### 题目3：N/S极模糊问题
**问题：** 高频注入存在180°角度模糊问题，即无法区分转子的N极和S极方向。请解释产生模糊的数学原因，并说明至少两种N/S极判断方法。

**参考答案：**
数学原因：解调信号中角度误差以 $\cos(2\theta_{err})$ 形式出现，而 $\cos(2\theta_{err}) = \cos(2(\theta_{err} + 180°))$，即 $\theta_{err}$ 和 $\theta_{err} + 180°$ 产生完全相同的电流响应，无法区分。

N/S极判断方法：
1. **脉冲注入法：** 依次注入6个有效电压矢量，比较各矢量产生的峰值电流。N极方向对应的电感最小，峰值电流最大。可靠性高但增加启动时间。
2. **Id偏置法：** 在d轴注入直流偏置电流，比较正负半周的d轴电流响应。磁饱和效应导致N极方向电流更大。实现简单但需要稳定条件。
3. **磁饱和法：** 利用大电流时磁饱和的非线性特性，N极方向磁饱和更严重。无需额外注入但对电机参数敏感。

## 二、计算分析题（★★★☆☆）

### 题目4：凸极率与信噪比
**问题：** 某IPMSM参数：$L_d = 2\text{mH}$，$L_q = 5\text{mH}$，$\psi_f = 0.05\text{Wb}$。
1. 计算凸极率 $\xi$；
2. 注入电压 $U_h = 10\text{V}$，注入频率 $f_h = 1000\text{Hz}$，计算q轴高频电流响应中角度误差信号的幅值（相对于基波分量）；
3. 如果凸极率降低到 $\xi = 0.1$，信噪比会如何变化？

**参考答案：**
1. $\xi = (L_q - L_d)/(L_d + L_q) = (5 - 2)/(5 + 2) = 3/7 \approx 0.429$

2. $L_{avg} = (L_d + L_q)/2 = 3.5\text{mH}$，$\omega_h = 2\pi \times 1000 = 6283\text{rad/s}$
   q轴高频电流基波幅值：$I_{qh,base} = U_h / (\omega_h L_{avg}) = 10 / (6283 \times 0.0035) = 0.454\text{A}$
   角度误差信号幅值：$\xi \times I_{qh,base} = 0.429 \times 0.454 = 0.195\text{A}$
   相对幅值：$\xi = 0.429$，即误差信号约为基波的42.9%。

3. 凸极率降至0.1时，误差信号相对幅值仅10%，信噪比大幅下降。在电流采样噪声不变的情况下，角度估算精度显著恶化。凸极率 $\xi > 0.1$ 是高频注入可行性的基本门槛。

### 题目5：PLL参数设计
**问题：** 高频注入PLL的闭环传递函数为 $G_{PLL}(s) = (K_p s + K_i)/(s^2 + K_p s + K_i)$，自然频率 $\omega_n = \sqrt{K_i}$，阻尼比 $\zeta = K_p / (2\sqrt{K_i})$。
1. 要求PLL带宽50Hz，阻尼比0.707，计算 $K_p$ 和 $K_i$；
2. 如果注入频率 $f_h = 1000\text{Hz}$，PLL带宽是否满足设计准则？为什么？

**参考答案：**
1. $\omega_n = 2\pi \times 50 = 314.16\text{rad/s}$
   $K_i = \omega_n^2 = 314.16^2 = 98696$
   $K_p = 2\zeta\omega_n = 2 \times 0.707 \times 314.16 = 444.3$

2. 设计准则：PLL带宽 $\omega_{PLL}$ 应远小于注入频率 $\omega_h$。
   $\omega_{PLL} = 314.16\text{rad/s}$，$\omega_h = 6283\text{rad/s}$
   $\omega_{PLL} / \omega_h = 0.05$，即PLL带宽仅为注入频率的5%，满足远小于的要求。这确保PLL不会跟踪注入频率分量，只跟踪角度误差的缓慢变化。

### 题目6：注入频率选择
**问题：** 某电机运行频率范围0~200Hz，PWM开关频率10kHz。
1. 注入频率 $f_h$ 应满足什么约束条件？
2. 如果选择 $f_h = 500\text{Hz}$，一个注入周期内有多少个PWM脉冲？是否足够？
3. 如果选择 $f_h = 2000\text{Hz}$，有什么问题？

**参考答案：**
1. 约束条件：$f_{base} \ll f_h \ll f_{sw}/10$
   即 $200\text{Hz} \ll f_h \ll 1000\text{Hz}$
   综合考虑，$f_h$ 应在500~800Hz范围。

2. $f_h = 500\text{Hz}$：一个注入周期 = 2ms，PWM周期 = 0.1ms，一个注入周期内有20个PWM脉冲。足够实现SVPWM调制和电流采样，但脉振注入的解调LPF截止频率需低于500Hz，角度跟踪带宽受限。

3. $f_h = 2000\text{Hz}$：不满足 $f_h \ll f_{sw}/10 = 1000\text{Hz}$ 的约束。一个注入周期仅5个PWM脉冲，注入波形严重失真，SVPWM无法精确合成注入电压，解调精度大幅下降。

## 三、工程实践题（★★★★☆）

### 题目7：高频注入代码实现分析
**问题：** 脉振高频注入代码中，信号处理链为：
```
q轴电流 → BPF(带通滤波) → 乘以sin(ωh·t) → LPF(低通滤波) → 误差信号 → PI → 角度积分
```
请分析：
1. 为什么需要BPF？直接用LPF提取高频分量行不行？
2. 乘以 $\sin(\omega_h t)$ 的作用是什么？这是哪种解调方法？
3. 如果BPF的中心频率与实际注入频率不匹配（如电机参数变化导致），会有什么影响？

**参考答案：**
1. BPF的作用是提取注入频率附近的高频电流分量，同时抑制基波电流和开关频率噪声。直接用LPF不行——LPF无法区分基波（低频）和高频分量，基波电流幅值远大于高频响应，会淹没位置信息。BPF的中心频率对准注入频率，只允许高频分量通过。

2. 乘以 $\sin(\omega_h t)$ 是乘法解调（同步解调）。原理：高频电流 $i_{qh} = A[1 + \xi\cos(2\theta_{err})]\sin(\omega_h t)$，乘以 $\sin(\omega_h t)$ 后：

   其中：
   - $A$：高频电流响应幅值 (A)
   - $\xi$：电机凸极率，$\xi = (L_q - L_d)/(L_d + L_q)$
   - $\theta_{err}$：估计角度与真实角度的误差 (rad)
   - $\omega_h$：高频注入角频率 (rad/s)

   $i_{qh} \times \sin(\omega_h t) = \frac{A}{2}[1 + \xi\cos(2\theta_{err})] - \frac{A}{2}[1 + \xi\cos(2\theta_{err})]\cos(2\omega_h t)$
   LPF滤除 $2\omega_h$ 分量后，得到包含角度误差的直流信号。

3. BPF中心频率偏移的影响：
   - 高频电流信号被BPF衰减，信噪比下降
   - BPF引入额外相位延迟，导致解调后的误差信号相位偏移
   - 严重时角度估算出现稳态偏差或振荡
   工程对策：BPF参数应根据注入频率精确设计，或使用自适应BPF跟踪注入频率变化。

### 题目8：全速域切换的工程实现
**问题：** 代码中全速域切换函数 `ANGLE_TRANSIT()` 使用线性加权：
```c
if (spd >= spdHi) angle = angleSMO;
else if (spd >= spdLo) {
    Ksmo = (spd - spdLo) * scale;
    angle = angleHFI*(1-Ksmo) + angleSMO*Ksmo;
}
else angle = angleHFI;
```
请分析：
1. 线性加权在角度域上是否合理？如果HFI角度为10°、SMO角度为350°，加权结果是否正确？
2. 如何解决上述角度跳变问题？
3. 切换过程中如果SMO尚未收敛，加权结果会怎样？如何保护？

**参考答案：**
1. 线性加权在角度域上存在严重问题。如果HFI角度10°、SMO角度350°，$K_{smo} = 0.5$ 时加权结果为 $(10 \times 0.5 + 350 \times 0.5) = 180°$，而正确结果应接近0°（10°和350°仅相差20°）。这是因为角度是周期量，直接线性加权忽略了2π周期性。

2. 解决方法：在加权前将两个角度对齐到同一周期。具体做法：计算角度差 $\Delta\theta = \theta_{SMO} - \theta_{HFI}$，将 $\Delta\theta$ 归一化到 $[-\pi, \pi)$，然后 $\theta = \theta_{HFI} + (1-K_{smo}) \times 0 + K_{smo} \times \Delta\theta$。这样10°和350°的差为-20°（而非340°），加权结果为10° + 0.5×(-20°) = 0°，正确。

3. 如果SMO未收敛，其角度估算值可能偏差很大（甚至随机），加权后角度严重偏离真实值，导致电流失控。保护措施：
   - 切换前验证SMO收敛性（观测电流误差小于阈值）
   - 在SMO未收敛时强制 $K_{smo} = 0$，不参与加权
   - 监测加权后角度变化率，异常时回退到HFI

### 题目9：高频注入的噪声与实用性
**问题：** 高频注入法在实际应用中面临可听噪声问题。请分析：
1. 注入频率在听觉范围（20Hz~20kHz）内时，为什么会产生可听噪声？
2. 如何在不降低位置估算精度的前提下降低可听噪声？提出至少三种方案。
3. 某些应用（如电动汽车转向助力电机）对噪声极其敏感，高频注入是否适合？替代方案是什么？

**参考答案：**
1. 注入的高频电压在电机绕组中产生高频电流，高频电流与永磁体磁场相互作用产生高频电磁力，导致定子铁芯和绕组高频振动，振动通过空气传播即为可听噪声。注入频率在500~2000Hz时，恰好位于人耳最敏感的频段（1~4kHz），噪声明显。

2. 降低噪声的方案：
   - **提高注入频率至超声波范围（>20kHz）：** 但受PWM频率限制（$f_h < f_{sw}/10$），需要更高的开关频率
   - **方波注入替代脉振注入：** 方波注入的频谱分散在奇次谐波上，单一频率分量幅值较小，噪声听起来更"沙沙"而非"嗡嗡"，主观感受更好
   - **注入幅值自适应：** 低速时大注入（位置精度优先），速度稍高后减小注入幅值（噪声优先），在精度和噪声间平衡
   - **随机注入频率：** 将注入频率在一定范围内随机跳变，将窄带噪声展宽为宽带噪声，降低峰值声压级

3. 电动汽车转向助力电机对噪声极其敏感，高频注入产生的可听噪声不可接受。替代方案：
   - 双编码器冗余方案（成本高但无噪声）
   - 基于INFORM的方法（利用PWM开关动作本身，无需额外注入）
   - 低速反电动势增强技术（特殊调制策略增强低速可观测性）
