## 一、参数计算题（★★★☆☆）

### 题目1：四电阻差分放大器CMRR计算

**问题：** 设计一个差分放大器用于电流采样：$R_1=R_3=1k\Omega$，$R_2=R_4=20k\Omega$（增益 $G=20$）。使用1%精度的0805贴片电阻，电阻温度系数为 $\pm 100ppm/°C$。

请计算：
1. 最差情况下的CMRR（假设最差匹配组合）
2. 若共模电压为10V（地弹导致的瞬态共模），输出共模误差电压
3. 若改用0.1%电阻匹配网络，CMRR可提升到多少？
4. 温度从25°C变化到85°C时，若四个电阻温度系数差异达 $\pm 50ppm/°C$，额外CMRR劣化多少？

**参考答案：**
1. $\varepsilon_{worst}=2\times 1\%=2\%$（最差匹配：R1max/R2min, R3min/R4max）
   $CMRR=(1+20)/(4\times 0.02)=21/0.08=262.5\approx 48.4dB$
2. $V_{out\_cm}=V_{cm}\times 10^{-CMRR/20}\times G=10\times 10^{-48.4/20}\times 20=10\times 0.0038\times 20=0.76V$
   在3.3V ADC量程中占23%——不可接受！
3. $\varepsilon=0.1\%$，$CMRR=(1+20)/(4\times 0.001)=21/0.004=5250\approx 74.4dB$
   $V_{out\_cm}=10\times 10^{-74.4/20}\times 20=10\times 1.9\times 10^{-4}\times 20=38mV$——基本可接受
4. $\Delta T=60°C$，电阻变化 $\Delta R=R\times 50ppm\times 60=R\times 0.003=0.3\%$
   额外失配 $\varepsilon_T\approx 0.3\%$，$CMRR_T=(1+20)/(4\times 0.003)\approx 1750\approx 64.9dB$
   比常温CMRR（74.4dB）下降了9.5dB！

### 题目2：运放增益带宽与噪声计算

**问题：** 电流采样电路：shunt电阻 $10m\Omega$，运放增益 $G=100$，运放GBW=10MHz。PWM频率 $f_{sw}=16kHz$。

请计算：
1. 闭环带宽 $f_{BW}$ 和可以准确放大的最高信号频率 $f_{max}$（增益误差<1%）
2. 对16kHz PWM噪声的衰减（一阶滚降-20dB/dec）
3. 若需要信号带宽5kHz（含电流谐波），应选择多大GBW的运放？

**参考答案：**
1. $f_{BW}=GBW/G=10MHz/100=100kHz$
   增益误差<1%：$f_{max}=f_{BW}\times\sqrt{(1/0.99)^2-1}\approx f_{BW}\times 0.14\approx 14kHz$
2. 16kHz处处于运放开环增益拐点以下（$f_{BW}=100kHz$），衰减仅 $-20\times\log_{10}(100kHz/16kHz)\approx -16dB$→不够！
3. $GBW>5kHz\times 100\times 10=5MHz$（10倍裕量）。但注意GBW越大对PWM噪声衰减越差→需额外加LPF

## 二、概念辨析题（★★★☆☆）

### 题目3：虚短虚断的工程边界

**问题：** 判断以下说法的正误并给出理由：

1. 虚短在运放输出饱和时依然成立
2. 虚断对JFET输入运放始终成立，因为栅极绝缘
3. 同相放大器中虚短意味着 $V_+=V_-$，但这不是因为两输入端短接
4. 对于开环使用的运放（作比较器），虚短不成立
5. CMOS运放的输入偏置电流 $I_{bias}$ 远小于BJT运放，因此虚断条件更"真"

**参考答案：**
1. ❌ 错误。饱和时运放输出不能再通过负反馈调整 $V_-$，$V_+\neq V_-$，虚短打破
2. ❌ 错误。JFET栅极仍有pn结反向漏电流（$\sim pA$级），且有过压保护二极管的漏电流，极端高温下pA级也会增大为nA级
3. ✅ 正确。虚短是运放通过负反馈主动调节的结果：$V_+-V_-=V_{out}/A_{OL}\rightarrow 0$，不是物理短接
4. ✅ 正确。开环无负反馈，输出饱和到电源轨，$V_+$和$V_-$可以有任意差值
5. ✅ 正确。CMOS运放 $I_{bias}\approx 1pA$，BJT运放 $I_{bias}\approx 100nA$，差5个数量级

### 题目4：电机驱动中运放选型优先级排序

**问题：** 将以下运放参数按电机驱动电流采样应用的重要性排序（从高到低），并说明理由：
- A. 增益带宽积（GBW）
- B. 输入失调电压（$V_{os}$）及温漂
- C. 共模抑制比（CMRR）
- D. 压摆率（SR）
- E. 输入偏置电流（$I_{bias}$）

**参考答案：**
**排序：B > C > A > D > E**

理由：
1. **$V_{os}$（最高优先级）**：shunt电压仅数十mV，$V_{os}=2mV$在G=50下输出误差100mV，直接导致电流测量偏移—这是系统性误差，无法通过滤波消除
2. **CMRR**：PWM开关产生极大共模干扰（dv/dt达kV/μs），CMRR不足会导致共模噪声转化为差模误差，在开关瞬间产生巨大尖峰
3. **GBW**：决定带宽，影响信号保真度和抗混叠；但不能盲目选大，需与LPF配合
4. **SR**：需要足够快以跟上PWM边沿噪声，否则运放进入饱和—恢复延迟导致采样值错误
5. **$I_{bias}$**：影响输入阻抗设计，但在电流采样中shunt电阻很小（mΩ级），源阻抗低，$I_{bias}$误差可忽略

## 三、设计应用题（★★★★☆）

### 题目5：三相电流采样完整信号链设计

**问题：** 设计一个7.5kW伺服驱动器的相电流采样信号链。给定条件：
- 三相低压侧shunt：$5m\Omega$，额定 $I_{rms}=15A$，峰值 $I_{peak}=30A$
- ADC：12bit，$V_{ref}=3.3V$，采样率 $f_s=20kHz$，同步采样
- PWM频率 $f_{sw}=16kHz$

请设计：
1. 确定运放增益，使ADC量程最大化利用（考虑过载裕量20%）
2. 选择运放型号及关键参数
3. 设计抗混叠LPF（截止频率和阶数）
4. 计算整体信号链的电流分辨率

**参考答案：**
1. $V_{sense\_peak}=30A\times 5m\Omega=150mV$
   过载裕量20%：$V_{sense\_max}=150\times 1.2=180mV$
   增益 $G=3.3V/180mV=18.33\rightarrow$选 $G=18$（$V_{out\_max}=180\times 18=3.24V<3.3V$ ✅）

2. 选**OPA2333**（双通道零漂移）：
   - $V_{os}=10\mu V$ → 输出offset $=10\mu V\times 18=0.18mV$（0.06%满量程）
   - GBW=350kHz → $f_{BW}=350k/18=19.4kHz$（足够信号带宽）
   - CMRR=130dB → 共模抑制极佳
   - 轨到轨输出（RRIO），单电源3.3V可工作

3. LPF设计：
   - 信号带宽：DC~2kHz（电流基波+主要谐波）
   - 抗混叠要求：$f_c<f_s/2=10kHz$，且对16kHz PWM衰减足够
   - 选 $f_c=5kHz$ 二阶Butterworth LPF
   - 16kHz衰减 $\approx -20dB$（每阶10dB×log(16/5)=10dB，二阶=20dB）
   - 配合同步采样实现足够衰减
   - 二阶LPF在2kHz处相位 $\approx -28°$，在FOC角度补偿中可修正

4. ADC分辨率 $=3.3V/4096=0.806mV/LSB$
   折算电流分辨率 $=0.806mV/18/5m\Omega=8.95mA/LSB$
   考虑 $V_{os}$ 和噪声：实际分辨率 $\approx 15mA$（约 $0.1\%$ 额定电流）✅

### 题目6：硬件过流保护比较器设计

**问题：** 设计硬件过流保护电路。已知：
- shunt $5m\Omega$，过流阈值 $I_{trip}=45A$（1.5×额定峰值）
- 比较器输出直接接到MCU的PWM急停引脚（TZ/BRK），要求总延迟 $<500ns$

请设计：
1. 过流阈值对应的shunt电压和比较器参考电压
2. 若运放用作比较器（TLV3501），计算总保护延迟
3. 设计迟滞（hysteresis）防止噪声反复触发

**参考答案：**
1. $V_{sense\_trip}=45A\times 5m\Omega=225mV$
   比较器同相输入 $\gets$ 放大后的电流信号（G=18→$225mV\times 18=4.05V>3.3V$！）
   → 需要独立的比较器路径，不经过G=18运放
   → 用专用比较器，$V_{ref}=225mV$（直接比较shunt电压）

2. TLV3501延迟分析：
   - 比较器传播延迟 $t_{pd}=4.5ns$
   - 输出上升时间 $t_r\approx 2ns$
   - MCU TZ输入到PWM关断 $t_{TZ}\approx 25ns$
   - 总延迟 $\approx 4.5+2+25\approx 32ns$ ← 极快！✅✅
   （对比DSP软件过流：ADC+中断+FOC任务+PWM更新 $\approx 5-10\mu s$）

3. 迟滞设计：
   噪声幅度 $\approx 5mV$（在shunt端，考虑PWM噪声）
   $V_{TH_H}=225mV+10mV=235mV$
   $V_{TH_L}=225mV-10mV=215mV$
   迟滞 $=20mV$
   反馈电阻网络：$R_1=10k\Omega$（输出到同相端），$R_2=100k\Omega$（同相端到Vref）
   迟滞 $\approx V_{OH}\times R_2/(R_1+R_2)=3.3V\times 100k/110k\approx 3V$→太大！
   改用：$R_1=1M\Omega, R_2=10k\Omega$→迟滞 $\approx 3.3V\times 10k/1010k\approx 33mV$ ✅

## 四、故障诊断题（★★★★☆）

### 题目7：电流采样异常噪声诊断

**问题：** 一台3kW伺服驱动器，客户反馈电流波形"毛刺很大"。已知：
- 运放：OPA2333，G=20，输出端有二阶LPF（fc=2kHz）
- ADC：12bit，fs=20kHz，PWM同步触发采样
- 现象：ADC读数在PWM开关时刻附近有±60LSB的尖峰（正常噪声±3LSB）

请分析：
1. 可能的原因（至少3个）
2. 给出逐项验证方案
3. 给出解决措施

**参考答案：**
1. 可能原因：
   (a) **LPF fc太高**：2kHz对16kHz PWM仅衰减约36dB（二阶），尖峰虽衰减但仍可见
   (b) **采样时刻不对**：同步采样触发点偏移，采到了PWM边沿而非中心点
   (c) **地弹耦合**：大电流PWM开关导致模拟地电位波动，通过运放参考地窜入信号
   (d) **PCB布局问题**：差分走线未紧密耦合，或靠近PWM开关节点

2. 验证方案：
   - 示波器同时测运放输出和PWM波形→确认尖峰是否与PWM边沿对齐
   - 修改ADC触发点偏移（前后各5μs）→观察尖峰变化
   - 测运放供电轨和参考地→看是否有PWM频率纹波
   - 断开电机负载（无大电流）→观察噪声水平，区分是传导还是辐射

3. 解决措施：
   - 将LPF fc降至1.5kHz（增加对16kHz衰减约2.5dB）
   - 校准ADC触发点为PWM计数器中心值（SVPWM的T0/2）
   - 模拟地与功率地单点连接（星形接地）
   - 运放电源增加LC滤波（10μH+100μF+0.1μF）

### 题目8：运放输出饱和导致FOC失控

**问题：** 高速伺服电机在6000rpm时FOC突然失控。分析发现运放输出在高速时"卡"在3.2V不变。已知：
- 运放：MCP6022（GBW=10MHz, SR=7V/μs, RRIO）
- G=50，shunt=10mΩ，额定I=10A
- 6000rpm时电流基频 $f_e=P\times n/60=4\times 6000/60=400Hz$
- 问题只在高速时发生，低速正常

请分析根本原因并提出解决方案。

**参考答案：**
**根因分析：**
1. 额定电流10A时shunt电压=100mV，运放输出=100mV×50=5V→超出3.3V ADC范围→运放已饱和！
   → 但低速时电流可能没达到额定值，或增益设计有误
2. 重新审视：$V_{out\_max}=3.3V$→允许最大shunt电压=3.3/50=66mV→最大电流=6.6A
   → 10A额定下只有66%量程→运放在大电流时必然饱和！
3. 高速时需要弱磁电流（Id负），相电流可能大于低速时的Iq→更容易触发饱和

**解决方案：**
- 减小增益至 $G=3.3V/(10A\times 10m\Omega\times 1.2)=3.3/0.12=27.5$→选G=25
- 或更换shunt为 $5m\Omega$：$V_{sense}=50mV$→G=3.3/0.06=55→选G=50合适
- 验证：$V_{os}=2mV$（MCP6022）→输出offset=2mV×50=100mV→12%满量程→不够精密！
  → 最终方案：$5m\Omega$ shunt + OPA2333(Vos=10μV)，G=50

## 五、综合案例题（★★★★★）

### 题目9：从噪声分析到系统级优化

**问题：** 一个FOC电流环的噪声预算分析。已知参数：
- shunt $10m\Omega$，满量程电流 $I_{fs}=20A$（$V_{sense\_fs}=200mV$）
- ADC 12bit, 3.3V, $SNR_{ADC}=68dB$
- 运放 G=16（$V_{out\_fs}=3.2V$），$e_n=55nV/\sqrt{Hz}$（OPA333），$f_{BW}=100kHz$
- LPF：二阶，fc=8kHz

请完成：
1. 计算运放输出的总RMS噪声（积分到LPF截止频率）
2. 计算ADC的量化噪声和热噪声
3. 计算总噪声预算并折算为电流噪声
4. 判断该噪声水平对电流环的影响（电流环设计带宽500Hz）

**参考答案：**
1. 运放输出噪声：
   等效噪声带宽（二阶LPF）$f_{ENB}=1.11\times f_c=1.11\times 8k=8.88kHz$
   运放RMS噪声 $=e_n\times\sqrt{f_{ENB}}\times G=55nV/\sqrt{Hz}\times\sqrt{8880}\times 16$
   $=55nV\times 94.2\times 16=82.9\mu V_{RMS}$

   电阻热噪声（反馈电阻 $R_f=16k\Omega, R_g=1k\Omega$，带宽8.88kHz，25°C）：
   $e_{nR}=\sqrt{4kTR\times f_{ENB}}=\sqrt{4\times 1.38e-23\times 298\times 16k\times 8880}=1.53\mu V_{RMS}$
   电阻噪声放大G=16→$24.5\mu V_{RMS}$
   
   运放总输出噪声 $\approx\sqrt{82.9^2+24.5^2}=86.5\mu V_{RMS}$

2. ADC量化噪声：$LSB=3.3/4096=0.806mV$
   $e_{q\_RMS}=LSB/\sqrt{12}=0.806/\sqrt{12}=0.233mV_{RMS}=233\mu V_{RMS}$
   ADC热噪声：$SNR=68dB$→$e_{ADC\_RMS}=3.3V\times 10^{-68/20}=3.3\times 0.000398=1.31mV_{RMS}$

3. 总噪声：$e_{total\_RMS}=\sqrt{86.5^2+233^2+1310^2}\approx 1332\mu V_{RMS}$
   ADC热噪声主导！
   折算电流噪声：$I_{noise\_RMS}=1332\mu V/16/10m\Omega=8.33A_{RMS}$ ← 太大！

   ★ 但这是宽带噪声！经过数字滤波后：
   电流环带宽500Hz→数字LPF降低带宽→噪声 $\propto\sqrt{BW}$：
   $I_{noise\_500Hz}=8.33\times\sqrt{500/8880}=8.33\times 0.237=1.97A_{RMS}$
   在满量程20A中占比约10%→仍然偏高！

4. 对电流环影响：1.97A的RMS噪声→电流环PI输出高频抖动→转矩脉动
   优化方向：(a) 降低LPF fc至2kHz→噪声降为 $1.97\times\sqrt{2/8}=0.99A$(5%)；(b) 增加ADC过采样(OSR=16)→有效分辨率提升→噪声降为 $1.97/4=0.49A$(2.5%)；(c) 增大shunt→提高信噪比

### 题目10：共模瞬态抑制的全链路设计

**问题：** 一台11kW伺服驱动器的电流采样面临极端共模瞬态挑战：
- $dv/dt=10kV/\mu s$（SiC MOSFET逆变器）
- 低压侧shunt，但地弹导致共模瞬态 $\pm 20V$，持续约50ns
- 要求运放输出在共模瞬态后200ns内恢复到正确值（误差<1%）

请设计完整的共模抑制方案，包括：
1. 前端共模滤波
2. 运放/仪放选型
3. PCB布局要点
4. 验证时序是否满足200ns恢复时间

**参考答案：**
1. **前端共模滤波：**
   - 共模扼流圈：$L_{cm}=100\mu H$（共模电感），$C_{cm}=100pF$（Y电容到地）
   - 截止频率 $f_{c\_cm}=1/(2\pi\sqrt{L_{cm}C_{cm}})=1/(2\pi\sqrt{100\mu H\times 100pF})\approx 1.6MHz$
   - 对50ns瞬态（等效频率 $\approx 1/50ns=20MHz$）衰减 $\approx 20MHz/1.6MHz\approx 12.5\times$（一阶约22dB）
   - 差模RC滤波：$R_{diff}=100\Omega, C_{diff}=1nF$，$f_{c\_diff}=1.6MHz$

2. **运放选型：**
   - 必须选**仪表放大器**而非四电阻差分
   - INA821：CMRR=120dB@G=10，共模范围 $\pm 15V$（±5V供电）
   - 或专用电流检测：INA240，CMRR=120dB，Vcm=-4V~+80V，带宽400kHz（G=20）
   - 增益G=16（同前例），$f_{BW}=400k/16=25kHz$

3. **PCB布局：**
   - 差分对走线：等长、紧密耦合（间距<走线宽度），远离PWM开关节点（>5mm）
   - 地平面：模拟地（AGND）与功率地（PGND）单点连接（星形接地），连接点靠近shunt
   - 运放退耦：100nF+10μF，紧贴电源引脚
   - 屏蔽：必要时差分走线两侧加GND铜皮

4. **恢复时间验证：**
   - 共模滤波后剩余共模 $\approx 20V/12.5=1.6V$（估算）
   - INA240 CMRR=120dB→$10^6$衰减→$1.6V/10^6=1.6\mu V$→可忽略✅
   - 关键路径：前级RC滤波器建立时间 $\approx 5\times R_{diff}\times C_{diff}=5\times 100\times 1nF=500ns$
   - 超过200ns要求！→改为 $R_{diff}=39\Omega, C_{diff}=1nF$→建立时间 $\approx 195ns$✅
   - 运放overload recovery：INA240<100ns→满足200ns
   - 总恢复时间 $\approx 195+100=295ns$，略超但接近200ns→接受✅
