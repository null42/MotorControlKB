# EE-04: MOSFET 原理与特性

**副标题：从转移特性到开关损耗——三相逆变器的核心功率器件**

---

## 1. 📌 核心摘要 ★★★☆☆ 🔰📚

**一句话讲清楚**：MOSFET（Metal-Oxide-Semiconductor Field-Effect Transistor）是电机驱动的核心功率开关——通过栅极电压(Vgs)控制漏极电流(Id)，导通电阻Rds(on)决定导通损耗，栅极电荷Qg决定开关速度，体二极管决定续流能力。三相逆变器的六个MOSFET直接决定了整个电驱系统的效率、发热和可靠性。

**认知挂钩**：很多人以为MOSFET就是个"电压控制的开关"，**选型时只看耐压和电流就下单**——这是导致工程灾难的常见错误！Rds(on)随温度上升50%~100%，Qg决定栅极驱动功耗，体二极管trr可能长达500ns，SOA在线性模式下急剧缩小。

**与电机控制的关联**：
- 🔗 **Rds(on)正温度系数**：并联MOSFET时自动均流 → 利于扩容
- 🔗 **栅极电荷Qg**：决定开关速度 → `tsw = Qg / Ig_drive`
- 🔗 **体二极管trr**：决定死区时间下限和反向恢复损耗
- 🔗 **SOA曲线**：电机堵转/启动时MOSFET可能进入线性模式 → 烧毁风险

## 3. 💡 直观理解 ★★☆☆☆ 🔰💡

### 类比1：MOSFET就像"电子水龙头"

```
普通水龙头               MOSFET
───────────             ──────
旋转手柄    ↔    栅极电压 Vgs（控制端）
水流        ↔    漏极电流 Id（被控量）
阀门开度    ↔    沟道导电能力

关键区别：MOSFET是电压控制！
  不需要持续电流维持导通（BJT需要Ib）
  栅极是绝缘的(SiO2层)→ 输入阻抗极高 >10MΩ
  
但：栅极有电容Ciss → 开关时需要充放电！
  → 需要瞬时栅极驱动大电流!
```

### 类比2：Rds(on)就像"水管的摩擦阻力"

```
水管越粗 → 水阻越小
沟道越宽 → Rds(on)越小

温度升高时：晶格振动加剧 → 电子移动受阻 → Rds(on)增大
25°C时5mΩ → 100°C时8~10mΩ!
→ 发热更多 → 温度更高 → Rds(on)更大 → 恶性循环!
→ 但因为并联管也发热，Rds(on)同时增大→自动均流（好处！）
```

### 关键参数速查

| 参数 | 符号 | 物理意义 | 电机驱动选型影响 |
|------|------|---------|----------------|
| 漏源耐压 | Vdss | 最大漏源电压 | 母线电压×1.5~2倍 |
| 导通电阻 | Rds(on) | 导通时DS间电阻 | 决定导通损耗 |
| 栅极阈值 | Vgs(th) | 开始导通的栅压 | 3.3V MCU能否直接驱动 |
| 栅极电荷 | Qg | 开关所需总电荷 | 决定开关速度和驱动功耗 |
| 米勒电荷 | Qgd | 米勒平台电荷 | 影响dV/dt和开关损耗 |
| 输入电容 | Ciss | Cgs+Cgd | 影响驱动峰值电流需求 |
| 体二极管trr | trr | 反向恢复时间 | 决定死区大小 |

## 4. 🔬 技术原理 ★★★☆☆ 📚

### 4.1 MOSFET结构

N沟道增强型垂直MOSFET（VMOS/VDMOS，电机驱动主流）：

```
    S(源极)    G(栅极)    D(漏极)
      │         │         │
      ├─────┬───┤         │
      │  N+ │P  │  N+     │  N-漂移区(决定耐压)
      │     │   │         │
      └──┬──┴───┴────┬────┘
         │   P-body  │
         └────┬──────┘
              │
            N+衬底 → D(漏极)

工作原理：
  1. Vgs=0：P-body无导电沟道 → 截止
  2. Vgs>Vgs(th)：栅极正压吸引P-body中电子 → 形成N型反型层(沟道) → 导通
  3. Vgs越大 → 沟道越深 → Rds(on)越小
```

### 4.2 转移特性 Id vs Vgs

饱和区（恒流区）：
$$
I_d = \frac{1}{2} \mu_n C_{ox} \frac{W}{L} (V_{gs} - V_{th})^2
$$

线性区（欧姆区——电机驱动正常工作区）：
$$
R_{ds(on)} = \frac{1}{\mu_n C_{ox} \frac{W}{L} (V_{gs} - V_{th})}
$$

**关键洞察**：Vgs越大→Rds(on)越小！12V驱动比10V驱动Rds(on)可能低20%！

### 4.3 Rds(on)温度特性——最致命的忽略项

硅MOSFET的Rds(on)具有**正温度系数**：
$$
R_{ds(on)}(T) = R_{ds(on)}(25^\circ C) \times \left(\frac{T}{300}\right)^{2.3 \sim 2.5}
$$

```
典型数据：
  IRFB4110: Rds_on=3.7mΩ@25°C → 5.8mΩ@100°C(+57%)
  IPP075N15N3: Rds_on=7.5mΩ@25°C → 14mΩ@125°C(+87%)
  
工程教训：
  如果按25°C的Rds_on计算损耗 → 严重低估实际发热!
  应按实际结温迭代计算: Tj = Ta + P×RθJA
```

**正温度系数的好处**：并联时自动均流——电流大的管发热多→Rds(on)增大→电流自动减少！

### 4.4 栅极电荷与开关过程

MOSFET开启过程（感性负载，电机驱动典型）：

**阶段1**：Vgs从0→Vth（Cgs充电），Id=0，Vds=Vdc，持续时间t1

**阶段2**：Vgs在米勒平台Vplateau（Cgd充电主导），Id上升到负载电流，持续时间t2

**阶段3**：Vgs从Vplateau→Vdrive，Vds从Vdc下降到导通压降，持续时间t3

总栅极电荷：
$$
Q_{g\_total} = Q_{gs} + Q_{gd} + Q_{g(剩余)}
$$

栅极驱动功耗：
$$
P_{drive} = Q_g \times V_{gs} \times f_{PWM}
$$

### 4.5 开关损耗

开通损耗：
$$
E_{on} \approx \frac{1}{2} V_{dc} \times I_d \times t_{on} + Q_{rr} \times V_{dc}
$$

关断损耗：
$$
E_{off} \approx \frac{1}{2} V_{dc} \times I_d \times t_{off}
$$

总开关损耗：
$$
P_{sw} = (E_{on} + E_{off}) \times f_{PWM}
$$

```
示例(Vdc=48V, Id=20A, ton=50ns, toff=30ns, f=20kHz)：
  Eon ≈ 0.5×48×20×50ns = 24μJ
  Eoff ≈ 0.5×48×20×30ns = 14.4μJ
  Psw = 38.4μJ × 20000 = 0.77W/管 → 6管 ≈ 4.6W
```

### 4.6 体二极管

每个MOSFET内部寄生PN结二极管（S→D方向），电机驱动中：优点=续流、缺点=反向恢复trr差(100~500ns)

```
体二极管反向恢复损耗：
  Prr = f_PWM × Vdc × Qrr
  
低压48V: Prr = 20k×48×500nC = 0.48W ← 可接受
高压400V: Prr = 20k×400×500nC = 4W ← 必须外加SiC二极管!
```

### 4.7 SOA（安全工作区）

限制边界：(1)Rds(on)线(最低电压/最大电流)；(2)脉冲电流限制(封装引线熔断)；(3)功耗限制(热限制)；(4)击穿电压限制Vdss

电机驱动风险：启动/堵转时可能进入线性模式→Vds大+Id大→瞬时功耗巨大→超出SOA→烧毁

## 5. 🔗 交叉视角 ★★★☆☆ 💡

### 5.1 Rds(on) → 逆变器效率

三相逆变器导通损耗：
$$
P_{cond} = 3 \times I_{rms}^2 \times R_{ds(on)}(T_j)
$$

更低Rds(on)=更高效，但意味着更大芯片→更大Ciss→更大开关损耗。存在最优平衡点。

### 5.2 Qg → PWM频率上限

栅极驱动功耗限定了PWM频率上限。Si MOSFET(Qg=100~200nC)适合<50kHz；GaN(Qg<10nC)可到MHz级。

### 5.3 体二极管trr → 死区时间

最小死区必须>trr_max+安全余量。trr差的MOSFET(>200ns)→死区≥400ns→PWM分辨率降低→电流畸变

## 6. 🎯 工程案例 ★★★★☆ 🎯

### 案例1：按25°C Rds_on计算→过温烧毁

选用IRF3205(Rds_on=8mΩ, Vdss=55V)用于48V/20A驱动。计算P=20²×8mΩ=3.2W→可接受。实际：Rds_on@100°C≈16mΩ→P=6.4W！且55V耐压对48V母线裕量仅1.15×→瞬态尖峰击穿风险。

解决：选IRFB4110(Vdss=100V, Rds_on=3.7mΩ@25°C→5.8mΩ@100°C)→P=2.32W ✅

### 案例2：栅极驱动不足→开关损耗暴增

MCU GPIO(8mA)→100Ω→IRFP460(Qg=210nC)。ton≈210nC/8mA=26μs！PWM周期50μs，开通占一半→大部分时间在线性区→严重发热。

解决：加栅极驱动IC→Ig=2A→ton=105ns ✅

### 案例3：并联MOSFET布局不对称→不均流

3×IRFB4110并联，一个管温度高20°C。PCB走线不对称→共源极电感不同→动态均流失效。

解决：对称布局+各管源极独立Kelvin连接+各管独立栅极电阻(10Ω)

### 案例4：Si vs GaN —— 低压驱动的未来

48V/1kW BLDC: Si IRFB4110(Rds_on=3.7mΩ,Qg=150nC) vs GaN EPC2024(Rds_on=1.5mΩ,Qg=4.8nC)
GaN优势：无体二极管trr、死区<20ns；劣势：Vgs_max=6V脆弱、成本高
结论：20kHz用Si足够；>100kHz或极致效率→GaN

### 案例5：三相逆变器选型速查

| 功率 | 母线 | 推荐MOSFET | Rds_on | Qg |
|------|------|-----------|--------|-----|
| 500W | 48V | IRFB4110 | 3.7mΩ | 150nC |
| 2kW | 72V | IRFB7430 | 1.3mΩ | 210nC |

## 7. 📝 实践练习

### 练习1：Rds_on温度折算

Rds_on@25°C=5mΩ，Tj=110°C，k=(T/300)^2.4。求实际Rds_on和Id=15A时的导通损耗。

*参考答案：Rds_on(110)=5×((273+110)/300)^2.4=5×1.277^2.4=5×1.79=8.95mΩ；P=15²×8.95m=2.01W*

### 练习2：栅极驱动时间

Qg=100nC, Qgd=30nC, Vdrive=12V, Rg=10Ω, Vplateau=5V, Vth=3V。估算ton。

*参考答案：Ig_avg≈12/10=1.2A(峰值)；ton≈100nC/1.2A=83ns*

### 练习3：开关损耗

Vdc=310V, Id=10A, ton=40ns, toff=30ns, f=16kHz, Qrr=800nC。求总开关损耗。

*参考答案：Eon=0.5×310×10×40n=62μJ；Prr=16k×310×800n=3.97W；Eoff=0.5×310×10×30n=46.5μJ；Psw=(62+46.5)μJ×16k+3.97=5.71W→反恢主导！换SiC*
