## 一、参数计算题（★★☆☆☆）

### 题目1：RC滤波电路计算
**问题：** 某电流采样电路中，ADC采样频率为40kHz，需要在采样前滤除高频噪声。要求低通滤波截止频率为采样频率的1/5，已知R = 100Ω。请计算：
1. 所需电容C的数值
2. 该滤波器的RC时间常数τ
3. 如果电容使用了MLCC（X7R材质），温度从25°C升至85°C时，电容值可能变化±15%，截止频率会漂移到什么范围？

**参考答案：**
1. fc = 40kHz/5 = 8kHz；C = 1/(2π × 100 × 8000) = 1/(5.027×10⁶) ≈ 199nF → 选220nF标称值
2. τ = R×C = 100 × 220e-9 = 22μs
3. C_min = 220 × 0.85 = 187nF → fc_max = 1/(2π×100×187e-9) = 8.51kHz；C_max = 220 × 1.15 = 253nF → fc_min = 1/(2π×100×253e-9) = 6.29kHz。设计时应考虑最差情况fc_max < 奈奎斯特频率(20kHz)，确保始终满足抗混叠要求 ✅

### 题目2：母线电容纹波计算
**问题：** 一台PMSM驱动的直流母线电压为48V，PWM频率为16kHz，额定负载电流为20A。请计算：
1. 使用Cdc = 1000μF电解电容时的母线电压纹波ΔVripple
2. 如果另外再并联一个100μF陶瓷电容（ESL = 5nH），该陶瓷电容能分担多少纹波电流？
3. 电解电容的ESR = 0.8Ω，计算仅电解电容时的ESR发热功率

**参考答案：**
1. ΔVripple = I/(2×f_PWM×C) = 20/(2×16000×1000e-6) = 0.625V，纹波率 0.625/48 = 1.3%
2. 在开关频率16kHz下：Z_陶瓷 = 1/(2π×16000×100e-6) = 0.0995Ω（容抗）；ESL阻抗 = 2π×16000×5e-9 = 0.0005Ω；总阻抗 ≈ 0.1Ω；Z_电解_ESR主导 ≈ 0.8Ω。电流分流比 ≈ Z_电解/Z_陶瓷 ≈ 0.8/0.1 ≈ 8:1 → 陶瓷电容分担约 20/9 ≈ 2.2A的纹波电流
3. P_ESR = I_rms² × ESR。纹波电流近似三角波，Irms ≈ I_load/√3 = 20/1.732 = 11.55A。P = 11.55² × 0.8 = 106.7W！⚠️ 这远超电解电容的纹波电流额定值，说明单靠电解电容结构不合理——需要并联多个电解电容或大量MLCC降低总ESR

### 题目3：LR时间常数与电流环带宽
**问题：** 电机参数：Rs = 0.3Ω，Ld = Lq = 1.5mH。请计算：
1. 电机的电气时间常数τ_elec
2. 基于τ_elec的自然电流环带宽上限ωc_max
3. 若希望电流环带宽达到2000 rad/s（对应约318Hz），Kp应设为多少？该设计是否可行？为什么？

**参考答案：**
1. τ_elec = L/R = 1.5e-3/0.3 = 5ms
2. ωc_max = 1/τ = 200 rad/s（约32Hz）
3. Kp = L × ωc = 0.0015 × 2000 = 3.0。设计带宽2000 rad/s远超自然响应极限200 rad/s（10倍！），但这是可行的——因为PWM逆变器提供的等效电压远大于电阻压降（例如48V母线 vs 10A×0.3Ω=3V的IR压降），通过前馈补偿+PI调节，实际电压输出可达数十伏，驱动di/dt远超自然响应。但需验证：相位裕度是否>45°，输出是否饱和，数字控制延迟是否可接受

## 二、概念辨析题（★★☆☆☆）

### 题目4：RLC元件特性判断
**问题：** 以下关于RLC元件的描述，请逐条判断正误并说明理由：
1. 理想电感在直流稳态下相当于短路
2. 理想电容在直流稳态下相当于开路
3. 电解电容的ESR远小于同容量MLCC的ESR
4. 电感串联时总电感增大，电容串联时总电容也增大
5. 电机绕组的电感是恒定值，与电流大小无关

**参考答案：**
1. ✅ 正确。Z_L = jωL，直流时ω=0 → Z_L=0 → 短路
2. ✅ 正确。Z_C = 1/(jωC)，直流时ω=0 → Z_C=∞ → 开路
3. ❌ 错误，恰恰相反。同容量下MLCC的ESR通常为2~10mΩ，电解电容的ESR为0.5~2Ω，MLCC的ESR低2~3个数量级
4. ❌ 错误。电感串联：L_total = L1+L2（增大，正确）；电容串联：1/C_total = 1/C1+1/C2（总电容减小，错误）
5. ❌ 错误。电机绕组电感随电流增大（铁芯饱和）而减小，随温度变化而微小变化，不是恒定值

## 三、设计应用题（★★★☆☆）

### 题目5：电机电流采样滤波网络设计
**问题：** 设计一个三电阻法电流采样的滤波电路。已知条件：PWM频率 = 20kHz，每个PWM周期在中心点采样一次（即采样频率 = 20kHz）。最大相电流 = 30A，采样电阻Rshunt = 5mΩ。请设计：
1. RC低通滤波的截止频率（考虑抗混叠和相位延迟的折中）
2. R和C的具体取值
3. 评估该滤波引入的相位延迟对电流环的影响

**参考答案：**
1. 奈奎斯特频率 = 10kHz。截止频率取奈奎斯特的1/5~1/3：fc = 2~3kHz。太高 → 抗混叠不足；太低 → 相位延迟大。折中选择fc = 3kHz
2. 选择R = 100Ω（避免过大增加源阻抗影响ADC驱动），C = 1/(2π×100×3000) = 530nF → 标称值470nF，实际fc = 1/(2π×100×470e-9) = 3.39kHz
3. 相位延迟：φ = -arctan(f/fc)，在1kHz（电流环基频附近）：φ = -arctan(1000/3390) = -16.4°。电流环通常设计在500~1500Hz，16°的相位延迟意味着相位裕度减少16°，需在PI设计中预留余量。若相位裕度不足，可适当提高fc或使用数字滤波器补偿相位

### 题目6：母线电容组合设计
**问题：** 设计一个48V/500W电机驱动的母线电容组。已知：PWM频率 = 20kHz，满载电流 = 15A，加速峰值电流 = 40A（持续2ms），允许母线电压跌落 < 2.4V（5%）。可用元件：470μF/63V电解电容（ESR = 0.3Ω，纹波额定2A），10μF/50V MLCC（ESR = 5mΩ），100μF/63V电解（ESR = 0.6Ω，纹波额定0.8A）。请设计最优组合。

**参考答案：**
首先确定最小电容值：
- 纹波约束：C ≥ 15/(2×20000×2.4) = 156μF
- 瞬态约束：C ≥ 40×0.002/2.4 = 33333μF ← 瞬态要求远超纹波要求！

需重新评估：2ms瞬态太苛刻，如果电流环响应可优化到500μs：C = 40×0.0005/2.4 = 8333μF。

方案：18×470μF并联 = 8460μF。ΔVdip = 40×0.0005/8460e-6 = 2.36V ✅ <2.4V

ESR_total = 0.3/18 = 0.017Ω。纹波电流每电容分担约15/18 = 0.83A < 2A额定 ✅

加MLCC：3×10μF，ESR_total = 5/3 = 1.7mΩ，分担高频纹波。

最终方案：18×470μF电解 + 3×10μF MLCC，总电容8490μF。发热：P_ESR_total = (15/√3)² × 0.017 = 1.28W ✅

## 四、故障诊断题（★★★★☆）

### 题目7：电流采样异常——寄生振荡
**问题：** 一台FOC驱动器，ADC读到的相电流波形在PWM开关瞬间出现高频振荡（约50MHz），幅值约0.5V，叠加在正常电流波形上。已知采样电阻Rshunt = 10mΩ（2512封装），采样运放增益 = 20倍。请分析：
1. 异常振荡的可能来源（至少3种）
2. 如何通过实验区分不同来源
3. 给出针对性解决方案

**参考答案：**
1. 可能来源：(a) 采样电阻的寄生电感（典型2512封装≈2nH）在MOSFET开关di/dt（100A/μs）下产生Vspike=2nH×100A/μs=200mV → 经20倍放大=4V；(b) PCB走线电感形成LC谐振——走线L+采样电阻寄生C+运放输入C，谐振频率f=1/(2π√(LC))；(c) 运放输入端共模噪声耦合，MOSFET开关产生的高dv/dt通过PCB寄生电容耦合到运放输入端
2. 区分方法：(a) 用差分探头直接测量Rshunt两端，若有50MHz振荡则是物理电压振荡 → L+C谐振；(b) 若无振荡但ADC读数有，则是运放/ADC环节问题 → 共模耦合；(c) 对比PWM周期中点采样vs边沿采样，若中点无问题则确认是开关瞬态耦合
3. 解决：(a) 改用4端子Kelvin封装采样电阻，消除引线电感；(b) 在运放输入端加RC滤波fc≈5MHz；(c) PCB布线，采样信号远离开关节点的dv/dt源；(d) 软件在PWM周期中点采样避开开关沿 ✅✅✅

### 题目8：母线电容过热分析
**问题：** 一台72V/3kW的电机驱动器，使用6个470μF/100V电解电容并联（总2820μF），ESR_single = 0.5Ω。运行30分钟后，热成像显示电容温度高达95°C（环境40°C）。请分析：
1. 计算每个电解电容分担的纹波电流有效值（总纹波电流 ≈ 额定电流40A的1/√3）
2. 计算单个电容的ESR发热功率
3. 为什么发热如此严重？给出改进方案

**参考答案：**
1. Iripple_rms ≈ Iphase/√3 = 40/1.732 = 23.1A。并联6个，总ESR = 0.5/6 = 0.0833Ω。各电容电流按导纳(1/ESR)分配，相等时每个分担23.1/6 = 3.85A
2. P_single = 3.85² × 0.5 = 7.41W！6个总发热 = 44.5W！⚠️ 在封闭机壳内，热阻≈15°C/W → 温升 = 44.5×15/6 ≈ 111°C → 远超实测95°C（说明有部分散热）
3. 原因：ESR过大导致严重发热 + 电流分配可能不均（PCB铜厚不够，并联支路阻抗不一致）。改进：(a) 使用低ESR电解（如0.1Ω品级→每个发热降至1.48W）；(b) 增加并联数量（12×470μF→电流减半→发热降至1/4=1.85W/个）；(c) 并联≥20μF MLCC分担高频纹波；(d) 改善散热布局

## 五、综合案例题（★★★★☆）

### 题目9：RLC综合分析——电机相电流采样完整链路
**问题：** 从电机相线电流到MCU ADC数字量，分析整个信号链中的RLC影响，给定：
- 采样电阻Rshunt = 5mΩ（2512, 寄生电感2nH）
- 差分运放增益 = 50，输入RC滤波R=47Ω, C=100pF
- ADC输入RC滤波R=10Ω, C=1nF
- MOSFET开关时间 = 30ns, di/dt = 50A/30ns = 1667A/μs

请计算每个环节的时间响应和频率响应，以及最终信号带宽和总延迟。

**参考答案：**
寄生电感影响：Vspike = 2nH × 1667A/μs = 3.33V → 叠加在真实50mV(10A×5mΩ)上 → 信噪比极差！
运放前RC滤波：fc = 1/(2π×47×100e-12) = 33.9MHz, τ = 47×100pF = 4.7ns → 对50MHz振荡有部分衰减
ADC前RC滤波：fc = 1/(2π×10×1e-9) = 15.9MHz, τ = 10ns
总带宽由最慢环节决定 ≈ 15.9MHz
总延迟 ≈ 4.7+10 = 14.7ns，结合运放建立时间(≈50ns)，总延迟≈65ns

关键结论：这些RC滤波器几乎不影响正常信号（电流环带宽<5kHz），但能衰减开关噪声。然而Vspike=3.33V × 50(增益) = 166.5V远超运放输出范围 → 运放必然饱和 → 必须在采样电阻端解决（改用Kelvin或避开采样时刻）

### 题目10：系统设计——48V/2kW电机驱动母线电容组完整设计
**问题：** 完整设计母线电容组，满足：(a) 稳态纹波<1%；(b) 50A/1ms加速瞬态电压跌落<5%；(c) 制动回馈能量100J吸收（Vdc_max<52V）；(d) 电容总发热<10W。提供元件选型、数量、布局建议。

**参考答案：**
给定：Vdc=48V, I=40A(连续)/50A(峰值), f_PWM=20kHz

(a) 稳态纹波：ΔV<0.48V → C ≥ 40/(2×20000×0.48) = 2083μF
(b) 瞬态跌落：ΔV<2.4V → C ≥ 50×0.001/2.4 = 20833μF → 瞬态约束主导！
(c) 制动能量：½C×(52²-48²) ≥ 100J → C×(2704-2304) ≥ 200 → C ≥ 200/400 = 500μF ← 远小于瞬态要求

方案：45×470μF/63V电解电容（总21150μF）。选低ESR型号（ESR_single≤0.15Ω），并联后ESR_total=0.15/45=3.3mΩ。纹波电流分配：总Irms≈23A，每电容≈0.51A。总发热：P=23²×0.0033=1.75W ✅ <10W

布局建议：(a) 大面积铜皮并联，保证均流；(b) 靠近MOSFET桥臂，缩短高频环路；(c) 电容矩阵排列，正负极交替以抵消磁场；(d) 预留温度传感器监控电容温度

> ⚠️ 工程注：21150μF/48V非常庞大！实际工程中会结合：(a) 提高开关频率；(b) 软件限制电流上升率缓解瞬态需求；(c) 采用薄膜电容+电解混搭提高高频性能
