## 一、参数计算题（★★★☆☆）

### 题目1：自举电容容量计算

**问题：** 设计一个24V/300W BLDC驱动器的自举电路。已知：
- MOSFET：IRFZ44N（$V_{DSS}=55V, R_{DS(on)}=17.5m\Omega$），$Q_{g\_total}=63nC$ @$V_{GS}=10V$
- 栅极驱动IC：IR2101，高侧静态电流 $I_{qbs}=50\mu A$（最大），自举二极管 $V_f=1V$
- PWM频率 $f_{sw}=20kHz$，最大占空比 $D_{max}=95\%$
- 最长不刷新时间 = 最低电机转速对应的换相周期（按10rpm计算，P=4）

请计算：
1. 最低转速下自举电容不刷新时间
2. 最小自举电容容量（允许 $\Delta V_{boot}=0.5V$）
3. 若电机堵转（转速=0，固定导通扇区），自举电容能维持多久？

**参考答案：**
1. 10rpm→$f_e=10\times 4/60=0.667Hz$→换相周期$T_{comm}=1/(0.667\times 6)\approx 250ms$
   不刷新时间≈250ms（在这期间下管不会导通刷新自举电容）

2. $Q_{total}=Q_g + I_{qbs}\times T_{on\_max}=63nC+50\mu A\times 250ms=63nC+12500nC=12563nC$
   $C_{boot\_min}=12563nC/0.5V\approx 25.1\mu F$→选$33\mu F/35V$或$47\mu F/35V$

   **但这不现实！** 250ms内还要开关 250ms×20kHz=5000次→总Qg消耗=$63nC\times 5000=315\mu C$！
   加上IC静态$12.5\mu C$→总计$327.5\mu C$→$C=327.5\mu C/0.5V=655\mu F$→太大！
   
   **结论：极低速自举驱动不可行！需要电荷泵辅助或隔离电源。**

3. 堵转时 $f_{sw}=20kHz, D=95\%$→下管导通时间仅$1/20k\times 0.05=2.5\mu s$→刷新窗口极小！
   实际上驱动IC在2.5μs内能否完成充电需要验证（IR2101需要最小$t_{on\_low}\approx 500ns$→$2.5\mu s>500ns$ 可行）
   但$33\mu F$电容充电电流$I=C\times\Delta V/\Delta t=33\mu F\times 1V/2.5\mu s=13.2A$→远超Vcc能提供的电流！
   →**大自举电容在短刷新窗口下充电不充分**→电容实际电压会持续下降

### 题目2：H桥MOSFET导通损耗计算

**问题：** 一个24V/10A直流有刷电机通过H桥（4个IRFZ44N）驱动，PWM频率20kHz。

请计算：
1. 正转时（Q1+Q4导通），H桥的总导通损耗（$T_j=25°C$和$T_j=100°C$）
2. 若改用能耗制动（Q1+Q3导通），制动电流=15A，制动损耗
3. 比较PWM调速和直流调速的损耗差异

**参考答案：**
1. 正转路径：Vdc→Q1→电机→Q4→GND（2个MOSFET串联）
   $R_{DS(on)}25°C=17.5m\Omega, R_{DS(on)}100°C\approx 17.5\times 1.7=29.75m\Omega$
   $I=10A$
   $25°C: P_{cond}=I^2\times(R_{Q1}+R_{Q4})=100\times(0.0175+0.0175)=3.5W$
   $100°C: P_{cond}=100\times(0.02975+0.02975)=5.95W$
   加上开关损耗：$E_{sw}\approx 0.05mJ$→$P_{sw}=0.05mJ\times 20kHz=1W$/管×2=2W
   $P_{total}(100°C)\approx 5.95+2=7.95W$
   
   2个TO-220封装（$R_{th(ja)}=62°C/W$自然散热）→$\Delta T=7.95\times 62/2=246°C$→必须加散热器！

2. 能耗制动：Q1导通→电机一端接Vdc→Q3导通→电机另一端接Vdc（短路）
   电流路径：电机→Q1→Q3→电机（电流在H桥内部循环，不经过电源）
   $P_{brake}=I^2\times(R_{Q1}+R_{Q3}+R_{motor})=15^2\times(0.03+0.03+0.5)=225\times 0.56=126W$
   注意：这里电机绕组电阻（0.5Ω）消耗了大部分制动能量！
   两个MOSFET各消耗约6.75W→需要散热器

3. PWM调速：额外的开关损耗导致效率低于纯直流调速
   但PWM调速可实现无级变速→工程上以2~3%的效率损失换取调速性能是值得的

## 二、概念辨析题（★★★☆☆）

### 题目3：电流采样位置分析

**问题：** 一个BLDC驱动器有三种电流采样方案可选。请判断以下说法的正误并给出理由：

1. 母线单电阻采样可以用于FOC控制的电流环反馈
2. 低端三电阻采样需要在下管导通期间读取ADC
3. 相线串联电阻采样的运放需要极高CMRR
4. 低端采样的共模电压始终为0，因为接地点参考相同
5. 母线采样适合BLDC方波驱动的过流保护

**参考答案：**
1. ❌ 错误。母线电流是斩波后的脉冲波形，不是连续相电流。FOC需要三相瞬时相电流用于Clarke/Park变换，母线单电阻无法直接提供。需要复杂的相电流重构算法（单电阻FOC），且存在盲区。
2. ✅ 正确。低端shunt在下管导通时才串联进电流回路，下管关断时（续流阶段）shunt两端电压为0→必须在PWM的下管导通窗口内同步采样。
3. ✅ 正确。相线shunt的共模电压在0~Vdc之间随PWM跳变，需要运放具有高CMRR（>80dB）来抑制共模噪声和dv/dt耦合。
4. ❌ 错误。即使shunt一端接地，在PWM开关时大电流在PCB地线上产生压降（地弹），实际共模并非始终0V。且不同shunt的"地"之间也有电位差。
5. ✅ 正确。母线采样简单、成本低，配合比较器可快速判断过流（总电流>阈值→立即关断）。BLDC方波驱动不需要精确的瞬时相电流控制，母线峰值电流或平均电流足够用于保护。

### 题目4：自举vs隔离驱动选型

**问题：** 以下场景应选自举驱动还是隔离DC-DC驱动？各给出理由：
1. 48V/500W BLDC电调（无人机），$D_{max}=100\%$需要
2. 540V/7.5kW伺服驱动器逆变级
3. 24V/100W直流电机H桥
4. 400V/3kW光伏逆变器

**参考答案：**
1. **自举不可行，需要隔离电源**。无人机需要100%占空比（满油门），自举电容无法刷新→必须用隔离DC-DC为上管提供独立栅极电源。或改用PMOS上管（但PMOS性能差）。
2. **隔离DC-DC驱动**。540V高压需要电气隔离（安规要求），自举仅在低压侧可行。通常用光耦隔离+隔离DC-DC供电的栅极驱动IC。
3. **自举驱动**。24V低压、成本敏感、不需要隔离。自举方案简单可靠，100nF~1μF自举电容即可。
4. **隔离DC-DC**。400V中压光伏，需要隔离（安全+法规），且光伏逆变器通常需要100%占空比能力。

## 三、设计应用题（★★★★☆）

### 题目5：BLDC六步换相的完整软件状态机设计

**问题：** 设计一个基于霍尔传感器的BLDC六步换相状态机。已知：
- 三个霍尔传感器（HA, HB, HC），120°间隔安装
- 每60°电角度产生一个霍尔状态变化（共6个合法状态）
- 合法霍尔序列：101→100→110→010→011→001

请设计：
1. 霍尔状态→MOSFET开关真值表（6行×6管）
2. 非法霍尔状态（000或111）的处理策略
3. 换相方向切换（正反转）的逻辑

**参考答案：**
1. **真值表**（1=上管导通/PWM调制，0=下管导通，Z=高阻/关断）：

```
霍尔 │扇区│ AH │ AL │ BH │ BL │ CH │ CL │ 电流 │ 说明
─────┼───┼────┼────┼────┼────┼────┼────┼─────┼────────
 101 │ 1 │PWM │ Z  │ Z  │ Z  │ Z  │ ON │ A→C │ (Q1 PWM, Q6 ON)
 100 │ 2 │PWM │ Z  │ Z  │ ON │ Z  │ Z  │ A→B │ (Q1 PWM, Q4 ON)
 110 │ 3 │ Z  │ Z  │PWM │ Z  │ Z  │ ON │ C→B │ (Q3 PWM, Q6 ON)
 010 │ 4 │ Z  │ ON │PWM │ Z  │ Z  │ Z  │ C→A │ (Q3 PWM, Q2 ON)
 011 │ 5 │ Z  │ ON │ Z  │ Z  │PWM │ Z  │ B→A │ (Q5 PWM, Q2 ON)
 001 │ 6 │ Z  │ Z  │ Z  │ ON │PWM │ Z  │ B→C │ (Q5 PWM, Q4 ON)
```

2. **非法状态处理（000/111）：**
   - 000：三个霍尔全部低电平→可能霍尔供电故障、或传感器损坏
   - 111：三个霍尔全部高电平→同上
   - 策略：(a) 保持上一个合法状态的输出→等待霍尔恢复；(b) 若持续>10ms非法→触发故障→停机；(c) 记录最后一次合法霍尔状态用于诊断

3. **正反转切换逻辑：**
   正转A→C→B→A→C→B（转子逆时针，霍尔序列递增）
   反转只需在换相表中"上下管互换"即可：
   ```
   正转: 上管PWM调制+下管常通
   反转: 下管PWM调制+上管常通（电流反向）
   ```
   或等价地，将霍尔序列反向映射：
   正转: 101→AH+C-；反转: 101→CH+A-（电流C→A）

### 题目6：死区时间对BLDC转矩脉动的影响

**问题：** 一台4极BLDC电机在500rpm运行时出现明显的"咔哒"声（每转4次）。已知PWM频率16kHz，死区0.8μs。请分析：
1. 咔哒声的频率是多少？与什么物理现象对应？
2. 死区导致的有效电压损失（$V_{dc}=48V$）
3. 给出3种抑制换相转矩脉动的方案

**参考答案：**
1. 500rpm→$f_{mech}=500/60=8.33Hz$→每转4次咔哒=$4\times 8.33=33.3Hz$
   4极电机每转电周期数=P=4→每转换相次数=$6\times 4=24$次
   咔哒频率$4\times 8.33=33.3Hz$，对应换相频率=$24\times 8.33=200Hz$→咔哒频率仅为换相频率的1/6→说明是**某个特定扇区换相**时问题严重（可能是某个MOSFET开关特性与其他不同）

2. 死区电压损失：$\Delta V=V_{dc}\times t_{dead}\times f_{pwm}=48\times 0.8\mu s\times 16kHz=0.614V$
   在BLDC中，死区仅在换相瞬间才有明显影响（正常PWM上下管不是互补的→死区不参与每周期损耗），换相时的死区导致的是**电流重建延迟**：$\Delta t_{rebuild}=L\times\Delta I/V_{dc}$→若L=100μH, $\Delta I=5A, V_{dc}=48V$→$\Delta t=100\mu\times 5/48\approx 10.4\mu s$→在这10μs内转矩跌落→机械振动

3. **抑制方案：**
   (a) 换相时使用**重叠换相**：在关断Q1之前先开通Q3→0.5~1μs后关断Q1→电流无缝转移，代价是短时环流
   (b) **换相后占空比补偿**：换相后立即将占空比提升$20\sim30\%$→持续$50\sim100\mu s$→补偿电流重建期间的转矩损失
   (c) **升级到FOC**：从六步方波升级到SVPWM→从根本上消除换相转矩脉动

## 四、故障诊断题（★★★★☆）

### 题目7：BLDC缺相故障诊断

**问题：** 一台BLDC电机启动后抖动严重，无法正常旋转。测量发现：
- 三相电流中A相电流始终≈0（其他两相正常交替）
- 霍尔传感器信号正常（6种状态依次出现）
- MOSFET栅极驱动波形正常（示波器看到PWM和开通信号正常）

请分析A相无电流的可能原因（MOSFET→PCB→shunt→电机），并给出逐项验证方案。

**参考答案：**
**可能原因（从硅片到电机绕组）：**

1. **A相MOSFET损坏（开路）：**
   - Q1或Q2内部bonding wire熔断→栅极正常但漏-源开路
   - 验证：断电后用万用表二极管档测D-S→正常应有一个体二极管（正向0.4~0.7V）

2. **PCB走线或焊点断裂：**
   - 大电流导致焊点热疲劳→A相功率路径开路
   - 验证：目检+万用表通断档（断电测），或用热成像看异常热点

3. **A相shunt电阻烧断开路（低端采样方案）：**
   - 过流导致shunt电阻爆裂→A相下管到GND路径断开
   - 验证：测量shunt电阻阻值（应为mΩ级，若无穷大→已开路）

4. **电机A相绕组断路：**
   - 电机内部断线或端子松脱
   - 验证：断开驱动器，用LCR表测电机三相电感→A相电感异常大（开路）或无穷

5. **栅极驱动虽有波形但驱动能力不足（驱动IC部分损坏）：**
   - 栅极波形的上升沿变缓→MOSFET在线性区时间长→过热保护→最终彻底开路
   - 验证：对比A相和其他相MOSFET的栅极波形上升时间→若明显偏慢→驱动IC问题

**逐项验证方案：**
1. 断电→万用表二极管档测所有MOSFET D-S→对比三相
2. 目检+通断档测A相功率路径（D-S-Motor端子）
3. 测A相下管S到GND之间shunt电阻→应为mΩ级
4. 断开电机线→LCR测三相绕组→对比电感值和电阻值
5. 上电（低压限流）→示波器同时测三相MOSFET的Vds波形→看是否有一相异常

### 题目8：自举电容反复失效分析

**问题：** 一台电动滑板车BLDC控制器，每隔约2周自举电容$C_{boot}$（$10\mu F/50V$ MLCC）失效（短路）。更换后正常2周又失效。已知：
- $V_{dc}=36V$（10S锂电池），$V_{cc}=12V$（LDO供电）
- 自举二极管：1N4148（$V_{RRM}=75V, I_F=200mA$）
- 运行工况：频繁启停，最高转速对应电频率约500Hz

请分析电容反复失效的原因，给出永久解决方案。

**参考答案：**
**根因分析：**
1. **自举电压应力：** 上管导通时 $V_{boot}=V_{dc}+V_{cc}=36+12=48V$。$10\mu F/50V$ MLCC→耐压裕量仅4%！MLCC在DC bias下容量下降且耐压无余量→瞬态过冲即可击穿。

2. **自举二极管反向恢复**：1N4148是**小信号二极管**（$t_{rr}=4ns$），开关速度快但反向恢复时产生尖峰。上管开通的dv/dt通过二极管的结电容耦合→$V_{boot}$瞬间过冲→超50V耐压→MLCC介质击穿。

3. **MLCC的老化+DC bias：** $10\mu F/50V$ X7R MLCC在48V DC bias下容量可能仅剩$30\sim 40\%$（约$3\sim 4\mu F$），实际自举储能下降→栅极驱动不足→上管进入线性区→耗散功率↑→局部温升↑→MLCC加速老化→短路。

4. **频繁启停→大电流冲击→PCB+电容机械应力→MLCC微裂纹→吸湿→短路。**

**永久解决方案：**
1. 自举电容改用 $10\mu F/100V$ X7R MLCC（耐压裕量从4%升到108%）✅
2. 自举二极管改用 **US1M**（$V_{RRM}=1000V, I_F=1A, t_{rr}=75ns$，超快恢复）→反向尖峰更小
3. 电容两端并联 $100nF/100V$ MLCC→高频去耦
4. 实际容量选 $22\mu F/100V$→充足裕量
5. 增加 $15V$ TVS管（SMBJ15A）在 $V_{boot}$ 对S_H之间→钳位过压

## 五、综合案例题（★★★★★）

### 题目9：从H桥到三相FOC逆变器的完整演进

**问题：** 你现在有一个H桥直流电机驱动板（4个IRF540N MOSFET），需要改造为BLDC/FOC三相驱动板。硬件不变（增加2个MOSFET即可凑齐6管），请完成：

1. 硬件改造方案（增加2个同型号MOSFET，完成三相桥）
2. 三相blade驱动6步换相状态机（霍尔传感器接口）
3. 从6步换相升级到FOC需要的硬件变更
4. 若想进一步升级到FOC，电流采样方案选择与设计

**参考答案：**
1. **硬件改造：**
   - 现有4×IRF540N→增加2×IRF540N→6管三相半桥
   - H桥排列：(Q1+Q2) = A相桥臂，(Q3+Q4) = B相桥臂，(Q5+Q6) = C相桥臂
   - 三个半桥输出分别接电机A/B/C相
   - 续流二极管：IRF540N内置体二极管即可（$t_{rr}$较大但$f_{sw}<20kHz$可接受）
   - 若原板没有自举驱动→需要增加三相栅极驱动IC（如FD6288→3个自举通道+3个低端驱动）

2. **六步换相状态机（同题目5）：**
   - 霍尔输入→6状态查表→控制6个PWM通道
   - 转速计算：$n=60\times f_{hall}/(6\times P)$→$f_{hall}$=霍尔变化频率
   - 速度环：目标速度→PI→PWM占空比

3. **升级FOC的硬件变更：**
   - 必须的：(a) 增加低端三电阻采样或相线电流采样；(b) 运放信号调理电路（G=10~50）
   - 强烈建议的：(c) MCU升级→带高级定时器（支持互补PWM+死区插入+ADC触发）→如STM32G4/F3系列
   - 可选的：(d) 增加编码器/霍尔接口→高分辨率位置反馈

4. **电流采样方案：**
   - 成本敏感→**低端三电阻**+3通道差分运放（INA180系列）：
     - shunt: $5m\Omega$，G=50→$I_{max}=30A\to V_{out}=30\times 0.005\times 50=7.5V$→太大！
     - 改进：shunt=$2m\Omega$，G=20→$V_{out}=30\times 0.002\times 20=1.2V$→在3.3V ADC范围内✅
     - 同步采样：PWM中心触发ADC→采样三相下管导通时的电流
   - 性能优先→**相线串联采样**+INA240（集成式，Vcm=-4~+80V，CMRR=120dB）

### 题目10：BLDC六步换相→SVPWM FOC的转矩脉动对比

**问题：** 一台IPM-BLDC电机参数：P=4, Ld=0.5mH, Lq=1.0mH, $\psi_f=0.05Wb$, $V_{dc}=48V$。在$I_q=10A, n=3000rpm$工况下，分析：

1. BLDC六步换相的转矩脉动来源和幅值（估算）
2. FOC SVPWM在相同工况下的转矩脉动
3. 从效率角度分析两种方案的差异

**参考答案：**
1. **BLDC转矩脉动来源：**
   (a) **换相脉动**（主要）：每60°电角度换相一次→电流从0重建→$T_e$跌落
      重建时间$\approx L\times I/V_{dc}=1mH\times 10A/48V\approx 208\mu s$
      3000rpm→$f_e=3000\times 4/60=200Hz$→电周期=5ms→60°扇区=833μs
      重建时间占扇区$208/833=25\%$→转矩跌落时间占比大
      脉动幅值≈$25\%\times T_{rated}$
   
   (b) **方波驱动谐波**：6步相电流为**方波**→含5次（16.7%）、7次（12.5%）谐波
      谐波转矩脉动：5次谐波→6倍频脉动（$T_e$以300Hz波动）
      总转矩脉动**THD(T)**≈30~50%

2. **FOC SVPWM转矩脉动：**
   - 电流为正弦波→谐波含量极低（取决于电流环带宽和PI参数）
   - 死区导致少量6倍频脉动（可通过死区补偿抑制）
   - 齿槽转矩（电机本体）→FOC也无法完全消除
   - 总转矩脉动**THD(T)**≈2~8%（主要来自PWM纹波+死区+齿槽）

3. **效率对比：**
   - **BLDC方波：** 方波电流的5次+7次谐波→在电机绕组中产生**额外铜损**
     $P_{cu\_harm}=3\times I_{harm}^2\times R_s$，谐波电流≈基波的18%→额外铜损≈$3\times(10\times 0.18)^2\times R_s=3\times 3.24\times R_s=9.72R_s$
     基波铜损=$3\times 10^2\times R_s=300R_s$→额外约3.2%
     加上铁损（高次谐波磁通）→总效率损失≈**+5~8%**
     
   - **FOC正弦：** 正弦电流→几乎无电流谐波→铜损接近理论最小值
     开关损耗：SVPWM对称调制→开关损耗≈BLDC的1.5×（每相每周期2次 vs 1.33次）
     总效率：FOC通常**高3~5%**（铜铁损优化大于开关损耗增加）
     
   - **综合评价：** FOC转矩脉动降低**80~90%**，效率提升**3~5%**→这就是为什么现代伺服几乎全部采用FOC
