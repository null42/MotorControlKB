## 一、参数计算题（★★☆☆☆）

### 题目1：整流滤波电容计算
**问题：** 一台单相220VAC/50Hz输入的小功率变频器，桥式整流后滤波，负载等效直流电流3A，允许纹波电压ΔV = 10V。请计算：(1) 整流后纹波频率；(2) 所需最小滤波电容；(3) 若改用三相380VAC输入，同样负载条件下所需电容

**参考答案：**
(1) 单相全波整流纹波频率 = 2×f_line = 100Hz
(2) C = I/(2×f×ΔV) = 3/(2×100×10) = 1500μF → 选2200μF/400V（20%容差余量）
(3) 三相整流纹波频率 = 6×f_line = 300Hz；C = 3/(2×300×10) = 500μF → 三相整流对电容需求显著降低（仅1/3！）

---

### 题目2：自举二极管选型计算
**问题：** 半桥驱动电路，VCC=15V，自举电容Cboot=2.2μF，PWM频率=20kHz，低边最小导通时间=5μs。自举二极管动态电阻=1.5Ω，Vf=0.4V。请计算：(1) 充电时间常数τ；(2) 5μs内Cboot能充到多少电压；(3) 该电压是否足够（推荐Vgs=10V）？

**参考答案：**
(1) τ = R×C = 1.5×2.2e-6 = 3.3μs
(2) t=5μs ≈ 1.52τ → Vboot = (VCC-Vf)×(1-e^(-1.52)) = 14.6×0.78 = 11.4V → 满足10V推荐值 ✅
(3) 若PWM升至50kHz → 最小导通=2μs ≈ 0.6τ → Vboot=14.6×0.45=6.6V ❌ → 需减小Cboot或加电荷泵

---

### 题目3：反向恢复损耗计算
**问题：** Si FRD续流二极管，Qrr=300nC，Vdc=300V，f_PWM=16kHz。计算：(1) 单管反向恢复损耗；(2) 三相逆变器6管总损耗；(3) 改用SiC肖特基节省的功率

**参考答案：**
(1) P_rr_single = f_PWM×Vdc×Qrr = 16000×300×300e-9 = 1.44W
(2) P_rr_total = 6×1.44 = 8.64W
(3) SiC Qrr≈0 → 节省8.64W。在3kW系统中效率提升 ≈ 8.64/3000 = 0.29%。虽百分比小，但省去了反向恢复EMI和额外散热需求

## 二、概念辨析题（★★☆☆☆）

### 题目4：二极管特性判断
**问题：** 判断正误并说明理由：(1) 肖特基Vf低，适合所有整流；(2) SiC肖特基trr=0因其为多数载流子器件；(3) 二极管Vf随温度升高而升高；(4) 1N4007可用作20kHz PWM续流

**参考答案：**
(1) ❌ 肖特基耐压通常<200V，不适合高压整流（380VAC→537VDC）。高压应选Si FRD或SiC
(2) ✅ 正确。SiC肖特基为多数载流子（电子）导电，无少数载流子存储效应 → trr≈0
(3) ❌ 硅管dVf/dT≈-2mV/°C，温升Vf反降（负温度系数）
(4) ❌ 1N4007 trr≈2~5μs，在20kHz下的周期仅50μs，trr占比高达10%，完全无法正常续流

## 三、设计应用题（★★★☆☆）

### 题目5：三相整流桥设计
**问题：** 设计5.5kW变频器输入整流级。输入：三相380VAC±10%/50Hz，效率93%。选整流桥型号、计算滤波电容、设计浪涌限制

**参考答案：**
DC功率=5500/0.93=5914W；Vdc_max=1.35×380×1.1=564V；Idc=5914/564=10.5A
整流桥：I_rated≥10.5×2.5=26A，V_rated≥564×1.5=846V → 选GBJ3510(35A/1000V)或MDS50A模块
滤波电容：f_ripple=300Hz，ΔV=28V(5%)，C=10.5/(2×300×28)=625μF → 4×680μF两串两并（等效680μF/800V）
浪涌：100Ω/50W预充电电阻+继电器延迟1s短路，NTC备用

### 题目6：逆变器续流方案对比
**问题：** 对比48V/500W和400V/30kW两种电机驱动的续流二极管方案

**参考答案：**
48V/500W：MOSFET体二极管trr≈50~150ns，低压下反向恢复损耗小。外部可选肖特基SS34(40V/3A)并联提升效率。推荐直接用体二极管即可，外加肖特基边际收益有限

400V/30kW：体二极管trr大(100~500ns)+高压高di/dt→反向恢复严重。必须外加SiC肖特基(如C3D10060A,600V/10A,trr≈0)。否则EMI超标+额外损耗+桥臂直通风险

---

## 四、故障诊断题（★★★★☆）

### 题目7：自举电压跌落诊断
**问题：** 48V电机驱动板，PWM=20kHz，占空比>90%时高边Vgs从12V跌至5V，MOSFET发热。自举二极管BAT54S，Cboot=1μF。分析原因给出3+解决方案

**参考答案：**
计算：低边开通=5μs，τ=2μs，5μs=2.5τ→理论上充到92%≈11V。但实际5V说明：
(a) BAT54S高温反向漏电流大。肖特基85°C漏电可达100μA，在95%周期(47.5μs)内严重放电；
(b) MLCC在DC偏压下有效容量退化——1μF/50V在15V偏压下可能仅剩0.3μF；
(c) 栅极驱动器IC静态功耗高温增大

解决：(1)换ES1J超快恢复（漏电<5μA）；(2) Cboot用C0G/NP0材质或增大到2.2μF以上；(3)最高占空比限制95%；(4)隔离DC-DC替代自举

---

### 题目8：续流二极管过热分析
**问题：** 逆变器MUR1560(15A/600V)续流，Iphase_rms=8A，f_PWM=16kHz，二极管壳温150°C。分析发热，给出方案

**参考答案：**
导通损耗：P_cond=Vf×I_avg≈1.5V×(8×0.5)=6W（占空比50%）
反向恢复：Qrr≈150nC→Prr=16000×300×150e-9=0.72W
总P≈7W；RθJC=2°C/W→结温升14°C。壳温150°C→结温164°C，接近175°C极限！

导通损耗占85%为主导。解决：(1)换SiC肖特基降低总损耗；(2)改善散热降低壳温；(3)降低PWM频率减小开关损耗

## 五、综合案例题（★★★★☆）

### 题目9：体二极管反向恢复与死区
**问题：** MOSFET IRFP460(500V/20A，体二极管trr=460ns典型)。半桥驱动感性负载，Vdc=310V，两管Rds_on=0.27Ω。计算最小安全死区并分析死区过短的后果

**参考答案：**
后果：死区<trr→对管开通时体二极管仍在反向恢复(等效导通)→直通！I_shoot=310/(0.27+0.27)=574A→瞬间烧毁

最小死区：t_dead>trr_max+t_delay_mismatch+margin=460+50+100=610ns，取700~800ns
权衡：死区长→有效PWM分辨率低→电流畸变大。优化：(a)并SiC肖特基→体二极管几乎不导通→死区可缩至200ns；(b)用GaN HEMT(trr=0)→死区<50ns

### 题目10：5kW伺服驱动器全链路二极管设计
**问题：** 完整设计5kW伺服中所有二极管应用：输入整流、续流、自举、保护。给出选型和理由

**参考答案：**
输入三相整流：GBJ3510(35A/1000V)，Idc≈10A需2.5×裕量+1000V耐压。加NTC+预充电继电器
母线均压：电解串联各并100kΩ/2W电阻+反并1N5408防反接
续流：SiC C3D10060A(600V/10A)×6，Qrr≈0消反向恢复→提效降EMI
自举：ES1J(600V/1A超快恢复,trr=35ns)，漏电<5μA优于肖特基高温表现
栅极保护：BZX84C15(15V/300mW)×6，GS并联防过压
制动单元：MUR1560(15A/600V超快恢复)，制动电阻续流用

> 关键原则：高频PWM→肖特基或超快恢复；高压高di/dt→SiC肖特基；自举→超快恢复（非肖特基，避高温漏电）；整流→普通二极管即可
