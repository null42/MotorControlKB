串口助手 — 使用说明

连接方式

  • 优先使用 Web Serial API（Chrome/Edge 89+），点击"连接"选择串口
  • 若浏览器不支持，自动回退到 WebSocket 代理模式，需运行串口代理服务
  • 勾选 Demo模式 可在无串口时使用模拟FOC电机数据

协议配置

  • AA55固定帧: 帧头[AA 55] + 通道数[1B] + 数据类型[1B] + 数据[N*4B] + 帧尾[55 AA]
  • Modbus RTU: 标准Modbus从站响应帧解析
  • SLIP/COBS: 透明传输协议
  • 自定义: 可配置帧头、帧尾、偏移、CRC校验

五种模式

  • 纯文本: 收发原始数据，支持HEX/ASCII双显、周期发送、快捷命令
  • 示波器: 多通道时域波形，支持触发、时基调节、十字光标
  • XY路径: 两通道XY轨迹图，四象限显示
  • 3D姿态: Three.js六色立方体显示姿态角，参考体、误差面板
  • 数据监视: 实时数值表格 + HEX数据流并排显示

导出

  • CSV: 带序号和时间戳的通道数据导出
  • PNG: Canvas/Three.js截图
  • 录制/回放: 录制串口数据到内存，离线回放调试
  • 配置: 通道/协议/视图配置保存到localStorage
