参考链接: https://github.com/Sguan-ZhouQing/SguanFOC_Library

平台: STM32F405RGT6 + 标准库

---

## 一、分层思想

```
┌─────────────────────────────────────────────────────────┐
│  APP  (RenMinWanSui_FOC/)   面向算法                     │
│    FOC 算法 / 状态机 / 滤波 / 观测器 / 协议                │
└─────────────────────────────────────────────────────────┘
                       ↑↓  (语义接口)
┌─────────────────────────────────────────────────────────┐
│  BSP  (BSP/)                面向应用的薄封装              │
│    bsp_pwm / bsp_current / bsp_vbus / bsp_debug          │
└─────────────────────────────────────────────────────────┘
                       ↑↓  (寄存器调用)
┌─────────────────────────────────────────────────────────┐
│  HARDWARE  (HARDWARE/)      面向芯片                     │
│    adc.c / timer.c / usart2.c / LED.c                    │
└─────────────────────────────────────────────────────────┘
                       ↑↓
              STM32 标准库 (LIB/) + 内核 (SYSTEM/)
```

**HARDWARE vs BSP 的边界**:
- HARDWARE: 写"怎么操作这颗芯片"  -- 寄存器、标准库、中断
- BSP:      写"算法想要什么功能"  -- `bsp_pwm_set_duty(da, db, dc)`、`bsp_current_get(&ia, &ib, &ic)`
- 换芯片时, HARDWARE 全部重写, BSP 基本不动, APP 完全不动

## 三、`MotorRT_t` 全局实时状态

**核心数据结构**: `RenMinWanSui_FOC/RtState/motor_realtime.h` 中的 `MotorRT_t`, 全局唯一实例 `g_motor`.

**设计原则**:
1. **只放跨模块共享的实时闭环量** (三相电流、角度、转速、电压、占空比、参考值、状态)
2. **算法内部状态** (PID 积分、SMO 滑模面、滤波器上一拍) 留在各自 `.c` 的 `static` 变量里
3. **写入纪律**: 每个字段约定唯一写入者, 见 `motor_realtime.h` 注释

## 四、四步走开发路线

| 版本 | 内容 | 验收 |
|---|---|---|
| **v1.0** | 基础目录 + 外设 + SVPWM 开环 (强拖) | 三相波形对称, VOFA+ 上传三相电流, ADC 与 PWM 中断对齐 |
| **v2.0** | 有感闭环: ABZ + Clark/Park + 三环 PID + 状态机 + 保护 | 速度环阶跃响应, 带载性能, 保护触发恢复 |
| **v3.0** | 无感闭环: SMO + PLL + IF/VF 启动 + 切换平滑 | 低速启动, 中高速切换, 抗扰 |
| **v3.1** | 引入 FreeRTOS (电流环仍在中断) | 任务划分清晰 |
| **v4.0** | 扩展: ADRC / MTPA / 弱磁 / 陷波 / CAN / 参数辨识 | 按需 |

每个版本独立目录 (`FOC_F405_v2.0` ...), 每个版本根目录放 `CHANGELOG.md`.

## 五、关键约定

- **单位**: 角度 rad、角速度 rad/s、电流 A、电压 V (TODO: 全部填到 Config/)
- **任务时间片** (v1.0 先骨架):
  - 电流环: PWM 中断, 10~20 kHz
  - 速度环: 1~2 kHz
  - 位置环: 500 Hz
  - 通信/状态机: 主循环 或 1ms
- **调试通道**: VOFA+ JustFloat 二进制 (4 字节 float * N 通道 + 帧尾 0x00 0x00 0x80 0x7F)

## 六、命名规范

- 目录:    PascalCase, 无连字符 (`RtState`, `FocMath`)
- 文件:    snake_case  (`bsp_pwm.c`, `motor_realtime.h`)
- 函数:    snake_case  (`bsp_current_get`, `pid_calc`)
- 类型:    PascalCase + `_t` 后缀  (`MotorRT_t`, `pid_t`)
- 全局变量: `g_` 前缀  (`g_motor`)
- 宏:      全大写  (`FOC_PWM_FREQ_HZ`)
- 静态变量: `s_` 前缀  (`s_zero_phB`)
